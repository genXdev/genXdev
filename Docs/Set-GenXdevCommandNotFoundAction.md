# Set-GenXdevCommandNotFoundAction

## SYNOPSIS
Sets up custom command not found handling

## SYNTAX

```
Set-GenXdevCommandNotFoundAction [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Configures PowerShell to handle unknown commands by either navigating to
directories.

## EXAMPLES

### EXAMPLE 1
```
Set-GenXdevCommandNotFoundAction
```
