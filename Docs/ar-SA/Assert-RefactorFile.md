# Assert-RefactorFile

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> ينفذ عملية إعادة هيكلة على ملف مصدر باستخدام بيئة التطوير المتكاملة (IDE) المحددة ونموذج موجه الذكاء الاصطناعي.

## Description

تقوم هذه الوظيفة بأتمتة عملية إعادة الهيكلة من خلال إعداد موجه ذكاء اصطناعي استنادًا إلى تعريف إعادة الهيكلة، واكتشاف أو تحديد بيئة التطوير المتكاملة المناسبة (VS Code أو Visual Studio)، وفتح الملف المستهدف مع الموجه المُعد. تتعامل الوظيفة مع معالجة قالب الموجه، واكتشاف بيئة التطوير، وأتمتة سير عمل إعادة الهيكلة.

## Syntax

```powershell
Assert-RefactorFile -Path <String> [[-RefactorSettings] <GenXdev.Helpers.RefactorSettings>] [-AllowLongRunningTests] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String | ✅ | — | 0 | — | المسار إلى الملف المصدر لتحسينه |
| `-RefactorSettings` | GenXdev.Helpers.RefactorSettings | — | — | 1 | `[GenXdev.Helpers.RefactorSettings]::new()` | تعريف إعادة الهيكلة يحتوي على الإعدادات وقالب المطالبة |
| `-AllowLongRunningTests` | SwitchParameter | — | — | Named | — | The `-AllowLongRunningTests` parameter. |

## Examples

### $refactorDef = Get-RefactorDefinition -Type "Documentation" Assert-RefactorFile -RefactorDefinition $refactorDef -Path "MyScript.ps1"

```powershell
$refactorDef = Get-RefactorDefinition -Type "Documentation"
Assert-RefactorFile -RefactorDefinition $refactorDef -Path "MyScript.ps1"
```

### Assert-RefactorFile -RefactorDefinition $refactorDef -Path ".\Functions\Test.ps1"

```powershell
Assert-RefactorFile -RefactorDefinition $refactorDef -Path ".\Functions\Test.ps1"
```
