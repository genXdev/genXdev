# Convert-PhysicsUnit

> **Module:** GenXdev.Helpers.Physics | **Type:** Function | **Aliases:** —

## Synopsis

> تحويل قيمة من وحدة فيزيائية إلى أخرى ضمن نفس الفئة.

## Description

تقوم هذه الدالة بتحويل الكميات الفيزيائية بين الوحدات المتوافقة في فئات مثل الطول والوقت والسرعة وغيرها. تستنتج الفئة من الوحدات وتطرح خطأ في حالة التحويلات غير المتوافقة.

## Syntax

```powershell
Convert-PhysicsUnit -Value <Double> -FromUnit <String> -ToUnit <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Value` | Double | ✅ | — | 0 | — | القيمة الرقمية للتحويل |
| `-FromUnit` | String | ✅ | — | 1 | — | وحدة قيمة الإدخال |
| `-ToUnit` | String | ✅ | — | 2 | — | وحدة الإخراج المطلوبة |

## Examples

### Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"

```powershell
Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"
```

يحول 100 متر إلى قدم.

### Convert-PhysicsUnit 10 "seconds" "minutes"

```powershell
Convert-PhysicsUnit 10 "seconds" "minutes"
```

يقوم بتحويل 10 ثوانٍ إلى دقائق باستخدام المعلمات الموضعية.
