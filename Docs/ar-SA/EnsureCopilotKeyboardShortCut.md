# EnsureCopilotKeyboardShortCut

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> يقوم بتكوين اختصارات لوحة المفاتيح لـ GitHub Copilot Chat في Visual Studio Code.

## Description

تضمن هذه الوظيفة أن ميزة إرفاق الملفات في GitHub Copilot Chat تحتوي على اختصار لوحة مفاتيح مناسب (Ctrl+Shift+Alt+F12) تم تكوينه في Visual Studio Code.

سيؤدي ذلك إلى إزالة أي اختصارات إرفاق Copilot موجودة واستبدالها بالأمر الصحيح الحالي (github.copilot.chat.attachFile).

كما يضيف اختصار Alt+` (backtick) لتبديل اللوحة الموسعة.

## Syntax

```powershell
EnsureCopilotKeyboardShortCut [<CommonParameters>]
```

## Examples

### EnsureCopilotKeyboardShortCut

```powershell
EnsureCopilotKeyboardShortCut
```
