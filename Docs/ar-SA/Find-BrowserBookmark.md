# Find-BrowserBookmark

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `bookmarks`

## Synopsis

> البحث عن الإشارات المرجعية من متصفح ويب واحد أو أكثر.

## Description

يبحث في الإشارات المرجعية من متصفحات Microsoft Edge أو Google Chrome أو Mozilla Firefox.
يعيد الإشارات المرجعية التي تطابق واحدًا أو أكثر من استعلامات البحث في اسمها أو عنوان URL أو مسار المجلد. إذا لم يتم تقديم أي استعلامات، فإنه يعيد جميع الإشارات المرجعية من المتصفحات المحددة.

## Syntax

```powershell
Find-BrowserBookmark [[-Queries] <String[]>] [-Chrome] [-Count <Int32>] [-Edge] [-Firefox] [-PassThru] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Queries` | String[] | — | ✅ (ByValue, ByPropertyName) | 0 | — | عبارات البحث للعثور على الإشارات المرجعية المتطابقة 🌐 *Supports wildcards* |
| `-Edge` | SwitchParameter | — | — | Named | — | البحث في إشارات Microsoft Edge المرجعية |
| `-Chrome` | SwitchParameter | — | — | Named | — | البحث في إشارات جوجل كروم المرجعية |
| `-Firefox` | SwitchParameter | — | — | Named | — | ابحث في إشارات فايرفوكس المرجعية |
| `-Count` | Int32 | — | — | Named | `99999999` | الحد الأقصى لعدد النتائج المراد إرجاعها |
| `-PassThru` | SwitchParameter | — | — | Named | — | قم بإرجاع كائنات الإشارات المرجعية بدلاً من عناوين URL فقط |

## Examples

### Find-BrowserBookmark -Query "github" -Edge -Chrome -Count 10 Searches Edge and Chrome bookmarks for "github", returns first 10 URLs

```powershell
Find-BrowserBookmark -Query "github" -Edge -Chrome -Count 10
Searches Edge and Chrome bookmarks for "github", returns first 10 URLs
```

### bookmarks powershell -e -ff -PassThru Searches Edge and Firefox bookmarks for "powershell", returns full objects

```powershell
bookmarks powershell -e -ff -PassThru
Searches Edge and Firefox bookmarks for "powershell", returns full objects
```
