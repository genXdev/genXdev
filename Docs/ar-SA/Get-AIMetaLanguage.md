# Get-AIMetaLanguage

> **Module:** GenXdev.AI.Queries | **Type:** Cmdlet | **Aliases:** `getimgmetalang`

## Synopsis

> يحصل على اللغة الافتراضية المهيأة لعمليات بيانات الصور الوصفية.

## Description

* يقوم هذا الأمر باسترداد اللغة الافتراضية المستخدمة من قبل وحدة GenXdev.AI
  لعمليات بيانات الصور الوصفية.
* يتحقق أولاً من المتغيرات العامة (ما لم يتم تحديد SkipSession)، ثم
  يتراجع إلى التفضيلات الثابتة، وأخيراً يستخدم الإعدادات الافتراضية للنظام.

## Syntax

```powershell
Get-AIMetaLanguage [[-Language] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Language` | String | — | — | 0 | — | اللغة الافتراضية لعمليات بيانات الصورة الوصفية |
| `-PreferencesDatabasePath` | String | — | — | Named | — | مسار قاعدة البيانات لملفات بيانات التفضيلات |
| `-SessionOnly` | SwitchParameter | — | — | Named | `False` | استخدم الإعدادات البديلة المخزنة في الجلسة لتفضيلات الذكاء الاصطناعي مثل اللغة ومجموعات الصور وما إلى ذلك |
| `-ClearSession` | SwitchParameter | — | — | Named | `False` | مسح إعدادات الجلسة (المتغير العام) قبل الاسترجاع |
| `-SkipSession` | SwitchParameter | — | — | Named | `False` | لا تستخدم الإعدادات البديلة المخزنة في الجلسة لتفضيلات الذكاء الاصطناعي مثل اللغة ومجموعات الصور، إلخ |

## Examples

### Example 1

```powershell
Get-AIMetaLanguage
```

الحصول على اللغة المهيأة حالياً من المتغيرات العامة أو التفضيلات.

### Example 2

```powershell
Get-AIMetaLanguage -SkipSession
```

احصل على اللغة المُهيأة من التفضيلات الدائمة، متجاهلاً الجلسة.

### Example 3

```powershell
Get-AIMetaLanguage -ClearSession
```

امسح إعداد الجلسة واحصل على اللغة من التفضيلات الدائمة.
