# Get-ApparentSizeAtArmLength

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> يحسب الحجم الظاهري لجسم على مسافة الذراع.

## Description

يحسب الحجم الظاهري باستخدام تقريب الزاوية الصغيرة.

## Syntax

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters <Double> -SizeInMeters <Double> [[-ArmLengthInMeters] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | المسافة إلى الكائن بالأمتار |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | الحجم الفعلي للكائن بالأمتار |
| `-ArmLengthInMeters` | Double | — | — | 2 | `0` | طول الذراع بالأمتار (القيمة الافتراضية: 0.7) |
| `-As` | String | — | — | 3 | — | الوحدة المستخدمة لحجم الإخراج |

## Examples

### Example 1

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters 10 -SizeInMeters 1 -As "centimeters"
```

يحسب الحجم الظاهري لجسم طوله 1 متر على مسافة 10 أمتار.

### Example 2

```powershell
Get-ApparentSizeAtArmLength 10 1
```

تحسب الحجم الظاهري باستخدام المعلمات الموضعية.
