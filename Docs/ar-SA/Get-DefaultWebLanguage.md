# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> يحصل على مفتاح اللغة الافتراضية للويب بناءً على إعدادات اللغة الحالية للنظام.

## Description

* يقوم باسترداد إعدادات اللغة والثقافة الحالية للنظام وتعيينها إلى مفتاح قاموس لغة الويب المقابل المستخدم من قبل خدمات الترجمة.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

إرجاع "الإنجليزية" لنظام إنجليزي، و"الهولندية" لنظام هولندي، إلخ.
