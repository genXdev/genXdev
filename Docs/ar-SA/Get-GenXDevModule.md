# Get-GenXDevModule

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> استرداد جميع وحدات GenXDev من مسار محدد.

## Description

تبحث هذه الوظيفة في هيكل الدليل عن وحدات GenXdev، باستثناء أي منها يحتوي على '.local' في الاسم. لكل وحدة صالحة تم العثور عليها، تقوم بإرجاع مجلد الإصدار الأحدث (1.x) الذي يحتوي على ملف بيان وحدة صالح (.psd1).

## Syntax

```powershell
Get-GenXDevModule [[-Path] <String>] [-IncludeLocal] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String | — | — | 0 | — | المسار الجذر للبحث عن وحدات GenXdev |
| `-IncludeLocal` | SwitchParameter | — | — | Named | — | The `-IncludeLocal` parameter. |

## Examples

### Get-GenXDevModule -Path "C:\PowerShell\Modules"

```powershell
Get-GenXDevModule -Path "C:\PowerShell\Modules"
```

### Get-GenXDevModule "C:\PowerShell\Modules"

```powershell
Get-GenXDevModule "C:\PowerShell\Modules"
```
