# Get-GenXDevNewModulesInOrderOfDependency

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> استرداد وحدات GenXDev بترتيب التبعية.

## Description

ترجع هذه الوظيفة قائمة بوحدات GenXDev مرتبة حسب ترتيب التبعية الصحيح لضمان تحميل الوحدات بشكل صحيح. تقوم أولاً باسترداد جميع معلومات الوحدات ثم ترتيبها بناءً على تبعياتها، بدءًا من الوحدات الأساسية وانتهاءً بالوحدات التابعة. يضمن ذلك تحميل الوحدات بالتسلسل الصحيح.

## Syntax

```powershell
Get-GenXDevNewModulesInOrderOfDependency [[-ModuleName] <String[]>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String[] | — | ✅ (ByValue, ByPropertyName) | 0 | `@('GenXdev*')` | اسم وحدة أو أكثر للتصفية حسبها |

## Examples

### Get-GenXDevNewModulesInOrderOfDependency -ModuleName "GenXdev.Helpers"

```powershell
Get-GenXDevNewModulesInOrderOfDependency -ModuleName "GenXdev.Helpers"
```

### "GenXdev.Console" | Get-GenXDevNewModulesInOrderOfDependency

```powershell
"GenXdev.Console" | Get-GenXDevNewModulesInOrderOfDependency
```
