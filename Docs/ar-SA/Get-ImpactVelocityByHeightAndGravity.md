# Get-ImpactVelocityByHeightAndGravity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> يحتسب سرعة الارتطام من الارتفاع.

## Description

استخدم v = sqrt(2 g h) مع إهمال مقاومة الهواء.

## Syntax

```powershell
Get-ImpactVelocityByHeightAndGravity -HeightInMeters <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-HeightInMeters` | Double | ✅ | — | 0 | `0` | الارتفاع بالأمتار |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 1 | `0` | الجاذبية بوحدة م/ث². الافتراضي 9.81. |
| `-As` | String | — | — | 2 | — | وحدة إخراج للسرعة. الافتراضي 'م/ث'. |

## Examples

### Example 1

```powershell
Get-ImpactVelocityByHeightAndGravity -HeightInMeters 100 -As "km/h"
```

يحسب سرعة الاصطدام لسقوط 100 متر ويخرجها بوحدة كم/ساعة.

### Example 2

```powershell
Get-ImpactVelocityByHeightAndGravity 50
```

تحسب سرعة الاصطدام لسقوط بمقدار 50 مترًا باستخدام الوحدات الافتراضية.
