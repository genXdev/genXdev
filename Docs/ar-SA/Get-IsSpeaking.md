# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> يُرجع قيمة صحيحة إذا كان محرك تحويل النص إلى كلام يتحدث.

## Description

يتحقق من حالة كل من محولي الكلام الافتراضي والمخصص لتحديد ما إذا كان أي منهما يتحدث حاليًا. يُرجع true إذا كان الكلام قيد التقدم، و false بخلاف ذلك.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

يُرجع قيمة صحيحة إذا كان محرك تحويل النص إلى كلام يتحدث.

### Example 2

```powershell
iss
```

إرجاع صحيح إذا كان محرك تحويل النص إلى كلام يتحدث (باستخدام الاسم المستعار).
