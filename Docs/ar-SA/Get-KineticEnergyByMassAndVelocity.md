# Get-KineticEnergyByMassAndVelocity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> تحسب الطاقة الحركية.

## Description

يستخدم KE = 1/2 م ف².

## Syntax

```powershell
Get-KineticEnergyByMassAndVelocity -MassInKilograms <Double> -VelocityInMetersPerSecond <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | الكتلة بالكيلوغرام |
| `-VelocityInMetersPerSecond` | Double | ✅ | — | 1 | `0` | السرعة بالمتر/ثانية |
| `-As` | String | — | — | 2 | — | وحدة إخراج للطاقة |

## Examples

### Example 1

```powershell
Get-KineticEnergyByMassAndVelocity -MassInKilograms 10 -VelocityInMetersPerSecond 5 -As "calories"
```

تحسب الطاقة الحركية لجسم كتلته 10 كجم يتحرك بسرعة 5 م/ث وتعطي الناتج بالسعرات الحرارية.

### Example 2

```powershell
Get-KineticEnergyByMassAndVelocity 5 10
```

تحسب الطاقة الحركية لجسم كتلته 5 كجم يتحرك بسرعة 10 م/ث باستخدام المعاملات الموضعية.
