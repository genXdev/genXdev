# Get-LightTravelTimeByDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> حساب الوقت الذي يستغرقه الضوء لقطع مسافة.

## Description

يُستخدم t = d / c، حيث تعتمد c على الوسط.

## Syntax

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters <Double> [[-As] <String>] [<CommonParameters>]

Get-LightTravelTimeByDistance [[-SpeedOfLightInMetersPerSecond] <Double>] [<CommonParameters>]

Get-LightTravelTimeByDistance -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | المسافة بالأمتار |
| `-SpeedOfLightInMetersPerSecond` | Double | — | — | 1 | `0` | سرعة الضوء بالمتر/ثانية (الافتراضي: 299792458) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | الوسط الذي ينتقل من خلاله الضوء *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | وحدة إخراج للوقت |

## Examples

### Example 1

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters 149597870700 -Medium "water" -As "minutes"
```

تحسب الوقت الذي يستغرقه الضوء لقطع المسافة إلى الشمس عبر الماء، بالدقائق.

### Example 2

```powershell
Get-LightTravelTimeByDistance 300000000 -SpeedOfLightInMetersPerSecond 225000000
```

يحسب الوقت الذي يستغرقه الضوء لقطع 300000000 متر بسرعة 225000000 م/ث.
