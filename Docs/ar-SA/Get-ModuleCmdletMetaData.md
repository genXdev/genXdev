# Get-ModuleCmdletMetaData

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> يقوم باسترداد البيانات الوصفية لجميع أدوات cmdlet في وحدة PowerShell.

## Description

يسترد البيانات الوصفية الكاملة للأوامر (الملخص، الوصف، الوسائط، الأمثلة، المخرجات، والأسماء المستعارة) لكل أمر في الوحدة النمطية المحددة، مع إضافة خصائص SubModuleName و CmdletType إلى كل نتيجة.

يستخدم تعيين الوحدة الفرعية مسارين مستقلين:

- أوامر البرنامج النصي (.ps1): تطابق ملف المصدر مع تعيينات دليل المصدر النقطي (.psm1) (Functions\<SubModuleName>\FileName.ps1).
- الأوامر المُجمَّعة (.dll): مساحة الاسم من ImplementationType.Namespace (مثل GenXdev.FileSystem).

## Syntax

```powershell
Get-ModuleCmdletMetaData -ModuleName <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-Language <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [-SkipTranslation] [-TranslationInstructions <String>] [<CommonParameters>]
```

## Examples

### Get-ModuleCmdletMetaData -ModuleName 'GenXdev'

```powershell
Get-ModuleCmdletMetaData -ModuleName 'GenXdev'
```

يُرجع بيانات التعريف لجميع أوامر cmdlet في وحدة GenXdev.

### Get-ModuleCmdletMetaData -ModuleName 'GenXdev' -Language 'nl-NL'

```powershell
Get-ModuleCmdletMetaData -ModuleName 'GenXdev' -Language 'nl-NL'
```

إرجاع البيانات الوصفية المترجمة إلى العربية (السعودية) لجميع أوامر GenXdev PowerShell.
