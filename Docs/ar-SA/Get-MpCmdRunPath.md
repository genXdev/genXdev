# Get-MpCmdRunPath

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> يعيد المسار إلى الملف القابل للتنفيذ MpCmdRun.exe الخاص بـ Windows Defender.

## Description

* يقوم بتحديد موقع وإرجاع المسار الكامل لأداة سطر الأوامر الخاصة بـ Windows Defender (MpCmdRun.exe).
* يتحقق الأمر cmdlet من موقع التثبيت القياسي في Program Files ويوفر معالجة مناسبة للأخطاء في حالة عدم العثور على الملف.

## Syntax

```powershell
Get-MpCmdRunPath [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$defenderPath = Get-MpCmdRunPath
```

استرداد المسار الكامل لأداة سطر الأوامر في Windows Defender.
