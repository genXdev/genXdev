# Get-PotentialEnergyByMassHeightAndGravity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> تحسب طاقة الجاذبية الكامنة.

## Description

يستخدم PE = m g h.

## Syntax

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms <Double> -HeightInMeters <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | الكتلة بالكيلوغرام |
| `-HeightInMeters` | Double | ✅ | — | 1 | `0` | الارتفاع بالأمتار |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | الجاذبية بوحدة م/ث² (الافتراضي: 9.81) |
| `-As` | String | — | — | 3 | — | وحدة إخراج للطاقة |

## Examples

### Example 1

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms 10 -HeightInMeters 5 -As "calories"
```

احسب الطاقة الكامنة الثقالية لكتلة مقدارها 10 كجم على ارتفاع 5 أمتار، وأخرج النتيجة بوحدة السعرة الحرارية.

### Example 2

```powershell
Get-PotentialEnergyByMassHeightAndGravity 5 10
```

يوضح استخدام المعاملات الموضعية لحساب الطاقة الكامنة.
