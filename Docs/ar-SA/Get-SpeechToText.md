# Get-SpeechToText

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> يحول ملفات الصوت إلى نص باستخدام نموذج التعرف على الكلام Whisper من OpenAI.

## Description

يعالج ملفات الصوت ويحول الكلام إلى نص باستخدام مكتبة Whisper.NET، التي تطبق نظام التعرف التلقائي على الكلام (ASR) من OpenAI Whisper. يدعم عدة لغات، وإمكانيات الترجمة، وإعدادات جودة نسخ مختلفة.

## Examples

### Example 1

```powershell
Get-SpeechToText -Input "C:\audio\recording.wav"
```

ينسخ ملفًا صوتيًا إلى نص باستخدام الإعدادات الافتراضية.

### Example 2

```powershell
Get-ChildItem "C:\audio\*.wav" | Get-SpeechToText
```

ينسخ جميع ملفات WAV في مجلد.

### Example 3

```powershell
Get-SpeechToText -Input "audio.mp3" -LanguageIn "es" -WithTranslate
```

يقوم بنسخ الصوت الإسباني وترجمته إلى الإنجليزية.

### Example 4

```powershell
Get-SpeechToText -Input "recording.wav" -Passthru -WithTokenTimestamps
```

يُرجع كائنات SegmentData مع معلومات التوقيت الدقيقة.
