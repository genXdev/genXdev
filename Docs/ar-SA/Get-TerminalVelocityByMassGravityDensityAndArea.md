# Get-TerminalVelocityByMassGravityDensityAndArea

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> يحسب السرعة الحدية.

## Description

تستخدم v = sqrt(2 m g / (ρ A C)).

## Syntax

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms <Double> [[-GravityInMetersPerSecondSquared] <Double>] -DensityInKilogramsPerCubicMeter <Double> -AreaInSquareMeters <Double> [[-DragCoefficient] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | الكتلة بالكيلوغرام |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 1 | `0` | الجاذبية بوحدة م/ث² (الافتراضي: 9.81) |
| `-DensityInKilogramsPerCubicMeter` | Double | ✅ | — | 2 | `0` | كثافة السائل بالكيلوغرام لكل متر مكعب |
| `-AreaInSquareMeters` | Double | ✅ | — | 3 | `0` | المساحة المقطعية بالمتر المربع |
| `-DragCoefficient` | Double | — | — | 4 | `0` | معامل السحب (الافتراضي: 0.5) |
| `-As` | String | — | — | 5 | — | وحدة قياس السرعة |

## Examples

### Example 1

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms 80 -DensityInKilogramsPerCubicMeter 1.225 -AreaInSquareMeters 0.7 -DragCoefficient 1.0 -As "mph"
```

تحسب السرعة الحدية لكتلة مقدارها 80 كجم في الهواء بمعامل سحب 1.0، وتخرج النتيجة بوحدة ميل في الساعة.

### Example 2

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea 70 1.225 0.8 0.8
```

يستخدم معلمات موضعية لحساب السرعة النهائية.
