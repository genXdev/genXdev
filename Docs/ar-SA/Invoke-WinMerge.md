# Invoke-WinMerge

> **Module:** GenXdev.AI | **Type:** Function | **Aliases:** —

## Synopsis

> يطلق WinMerge لمقارنة ملفين جنبًا إلى جنب.

## Description

يقوم بتشغيل تطبيق WinMerge لمقارنة الملفات المصدر والهدف في عرض الفرق جنبًا إلى جنب. تتحقق الدالة من وجود كلا ملفي الإدخال وتضمن تثبيت WinMerge بشكل صحيح قبل التشغيل. توفر وظيفة انتظار اختيارية لإيقاف التنفيذ حتى يتم إغلاق WinMerge.

## Syntax

```powershell
Invoke-WinMerge -SourcecodeFilePath <String> -TargetcodeFilePath <String> [-Wait] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-SourcecodeFilePath` | String | ✅ | — | 0 | — | المسار إلى الملف المصدر للمقارنة |
| `-TargetcodeFilePath` | String | ✅ | — | 1 | — | المسار إلى الملف الهدف للمقارنة معه |
| `-Wait` | SwitchParameter | — | — | Named | — | انتظر حتى يتم إغلاق WinMerge قبل المتابعة |

## Examples

### Invoke-WinMerge -SourcecodeFilePath "C:\source\file1.txt" `                 -TargetcodeFilePath "C:\target\file2.txt" `                 -Wait

```powershell
Invoke-WinMerge -SourcecodeFilePath "C:\source\file1.txt" `
                -TargetcodeFilePath "C:\target\file2.txt" `
                -Wait
```

### merge "C:\source\file1.txt" "C:\target\file2.txt"

```powershell
merge "C:\source\file1.txt" "C:\target\file2.txt"
```
