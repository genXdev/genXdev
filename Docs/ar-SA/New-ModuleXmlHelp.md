# New-ModuleXmlHelp

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> يقوم بإنشاء ملفات تعليمات MAML XML لأي وحدة نمطية في PowerShell.

## Description

ينشئ ملفات تعليمات MAML XML لأي وحدة PowerShell من خلال استخراج البيانات الوصفية من جميع أوامر cmdlet في الوحدة باستخدام Get-CmdletMetaData وإنتاج كل من ModuleName.dll-Help.xml (لأوامر cmdlet بلغة C#) وModuleName-help.xml (لأوامر cmdlet النصية).

## Syntax

```powershell
New-ModuleXmlHelp -ModuleName <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-Force] [-Language <String>] [-LinkPrefix <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [-SkipTranslation] [-TranslationInstructions <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String | ✅ | — | 0 | — | اسم وحدة PowerShell النمطية التي سيتم إنشاء تعليمات المساعدة لها |
| `-Language` | String | — | — | Named | `'en-US'` | ar-SA |
| `-Force` | SwitchParameter | — | — | Named | — | استبدال ملفات تعليمات XML الحالية دون طلب تأكيد |
| `-SkipTranslation` | SwitchParameter | — | — | Named | — | @{response=Skip LLM translation; keep help in source language} |
| `-Model` | String | — | — | Named | — | معرف النموذج أو النمط المستخدم للترجمات بالذكاء الاصطناعي |
| `-ApiEndpoint` | String | — | — | Named | — | رابط نقطة نهاية API للترجمات بالذكاء الاصطناعي |
| `-ApiKey` | String | — | — | Named | — | مفتاح API للترجمات المعتمدة بالذكاء الاصطناعي |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | يشير إلى أن LLM لا يدعم مخططات JSON |
| `-LinkPrefix` | String | — | — | Named | — | بادئة URL للرابط الإلكتروني في كل رابط من روابط الأوامر ذات الصلة (مثل: https://github.com/genXdev/genXdev/blob/main/Docs/en-US/) |
| `-TranslationInstructions` | String | — | — | Named | — | تعليمات مخصصة لترجمة LLM (تم تمريرها إلى Get-TextTranslation) |

## Examples

### New-ModuleXmlHelp -ModuleName 'Pester'

```powershell
New-ModuleXmlHelp -ModuleName 'Pester'
```

إنشاء ملف XML تعليمات باللغة الإنجليزية لوحدة Pester في en-US\.

### New-ModuleXmlHelp -ModuleName 'GenXdev' -Language 'nl-NL' -Force  -Model 'deepseek-v4-pro' -ApiKey 'your-api-key' -ApiEndpoint 'https://api.deepseek.com/chat/completions'

```powershell
New-ModuleXmlHelp -ModuleName 'GenXdev' -Language 'nl-NL' -Force  -Model 'deepseek-v4-pro' -ApiKey 'your-api-key' -ApiEndpoint 'https://api.deepseek.com/chat/completions'
```

يقوم بإنشاء ملفات تعليمات XML بالهولندية لـ GenXdev، مع استبدال الملفات الموجودة.

### New-ModuleXmlHelp -ModuleName 'PSReadLine' -WhatIf

```powershell
New-ModuleXmlHelp -ModuleName 'PSReadLine' -WhatIf
```

يعرض الملفات التي سيتم إنشاؤها دون كتابتها فعليًا.

### New-ModuleXmlHelp -ModuleName 'GenXdev' -LinkPrefix `     'https://github.com/genXdev/genXdev/blob/main/Docs/en-US/' -Force

```powershell
New-ModuleXmlHelp -ModuleName 'GenXdev' -LinkPrefix `
    'https://github.com/genXdev/genXdev/blob/main/Docs/en-US/' -Force
```

يقوم بإنشاء ملف XML مساعدة باللغة الإنجليزية مع عناوين URL للإصدار عبر الإنترنت لكل أمر cmdlet تشير إلى مستندات GitHub Markdown. يفتح الأمر Get-Help <cmdlet> -Online المتصفح.
