# Now

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> إرجاع تاريخ ووقت النظام الحاليين ككائن DateTime.

## Description

يوفر طريقة بسيطة للحصول على تاريخ ووقت النظام الحالي بدون أي معلمات. يُرجع كائن DateTime من .NET القياسي يمكن استخدامه لإجراء العمليات الحسابية والتنسيق والمقارنات الخاصة بالتاريخ والوقت.

## Syntax

```powershell
Now [<CommonParameters>]
```

## Examples

### Now Returns the current system date and time as a DateTime object

```powershell
Now
Returns the current system date and time as a DateTime object
```

### $timestamp = Now Stores the current date and time in a variable for later use

```powershell
$timestamp = Now
Stores the current date and time in a variable for later use
```

## Outputs

- `DateTime`
