# Open-ViralSimulation

> **Module:** GenXdev.Queries.Websites | **Type:** Function | **Aliases:** `viral`

## Synopsis

> يفتح لعبة محاكاة فيروسية في متصفح ويب مع خيارات تكوين واسعة.

## Description

يفتح لعبة المحاكاة الفيروسية على https://viral.genxdev.net/ في متصفح ويب محدد مع إعدادات نافذة قابلة للتكوين، وخيارات تحديد الموضع والسلوك. يدعم عدة متصفحات، التصفح الخاص، وضع التطبيق، وميزات إدارة النوافذ الشاملة.

توفر الوظيفة تجربة محاكاة فيروسية تفاعلية مع ميزات مثل:
- دعم متصفحات متعددة (Edge، Chrome، Firefox)
- عناصر تحكم في تحديد موضع النافذة وحجمها
- خيارات وضع ملء الشاشة ووضع التطبيق
- إمكانيات التصفح الخاص
- دعم أتمتة لوحة المفاتيح
- دعم شاشات متعددة

### Open-ViralSimulation -Chrome -Private -NoFullScreen

```powershell
Open-ViralSimulation -Chrome -Private -NoFullScreen
```
