# Remove-JSONComments

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> إزالة التعليقات من محتوى JSON.

## Description

* يعالج محتوى JSON ويزيل التعليقات ذات السطر الواحد والمتعددة الأسطر مع الحفاظ على بنية JSON.
* مفيد لتنظيف ملفات JSON التي تحتوي على تعليقات توثيق قبل التحليل.
* يدعم كلاً من التعليقات ذات السطر الواحد (//) والتعليقات متعددة الأسطر (/* */).

## Syntax

```powershell
Remove-JSONComments -Json <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Json` | String | ✅ | — | 0 | — | لا تخلط بين المحتوى الذي سيتم ترجمته والذي يتكون من نصوص تعليمات PowerShell cmdlet، وبين التعليمات!
لا تقم بإدراج صور أو أي شيء آخر. فقط قم بترجمة المحتوى بأكبر قدر ممكن من البساطة إلى: العربية (المملكة العربية السعودية). |

## Examples

### Example 1

```powershell
$jsonContent = @'
{
    // This is a comment
    "name": "test", /* inline comment */
    "value": 123
}
'@ -split "`n"
Remove-JSONComments -Json $jsonContent
```

يزيل التعليقات من محتوى JSON المخزن في متغير.

### Example 2

```powershell
$jsonContent | Remove-JSONComments
```

يعالج محتوى JSON من خط الأنابيب.
