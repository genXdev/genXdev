# Remove-Refactor

> **Module:** GenXdev.Coding | **Type:** Function | **Aliases:** —

## Synopsis

> يقوم بإزالة مجموعات إعادة الهيكلة من نظام تفضيلات GenXdev.

## Description

يدير مجموعات إعادة الهيكلة في نظام تفضيلات GenXdev عن طريق إزالة الإدخالات المحددة. يعمل مع مجموعات إعادة الهيكلة المخصصة والافتراضية. المجموعات الافتراضية محمية ما لم يُسمح بإزالتها صراحةً. يتم تعريف جميع مجموعات إعادة الهيكلة بالبادئة "refactor_set_" في نظام التفضيلات.

## Syntax

```powershell
Remove-Refactor -Name <String[]> [-ClearSession] [-DefaultValue <String>] [-PreferencesDatabasePath <String>] [-RemoveDefault] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Name` | String[] | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | اسم إعادة الهيكلة، يقبل أحرف البدل 🌐 *Supports wildcards* |
| `-RemoveDefault` | SwitchParameter | — | — | Named | — | التبديل أيضًا لإزالة مجموعة إعادة الهيكلة القياسية |
| `-PreferencesDatabasePath` | String | — | — | Named | — | مسار قاعدة البيانات لملفات بيانات التفضيلات |
| `-DefaultValue` | String | — | ✅ (ByPropertyName) | Named | — | القيمة الافتراضية إذا لم يتم العثور على التفضيل |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | استخدم الإعدادات البديلة المخزنة في الجلسة لتفضيلات البيانات مثل اللغة ومسارات قواعد البيانات، إلخ |
| `-ClearSession` | SwitchParameter | — | — | Named | — | مسح إعدادات الجلسة (المتغير العام) قبل الاسترجاع |
| `-SkipSession` | SwitchParameter | — | — | Named | — | لا تستخدم الإعدادات البديلة المخزنة في الجلسة لتفضيلات البيانات مثل اللغة، مسارات قاعدة البيانات، إلخ |

## Examples

### Remove-Refactor -Name "CustomRefactor" -RemoveDefault Removes a specific refactor set named "CustomRefactor" and allows removal of default sets if matched

```powershell
Remove-Refactor -Name "CustomRefactor" -RemoveDefault
Removes a specific refactor set named "CustomRefactor" and allows removal
of default sets if matched
```
