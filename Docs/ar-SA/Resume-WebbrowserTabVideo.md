# Resume-WebbrowserTabVideo

> **Module:** GenXdev.Webbrowser.Playwright | **Type:** Function | **Aliases:** `wbvideoplay`

## Synopsis

> يستأنف تشغيل الفيديو في علامة تبويب متصفح يوتيوب.

## Description

يبحث عن علامة تبويب YouTube النشطة ويستأنف تشغيل الفيديو عن طريق تنفيذ طريقة play() على أي عناصر فيديو موجودة في الصفحة. إذا لم يتم العثور على علامة تبويب YouTube، تقوم الدالة بإلقاء خطأ. هذه الدالة مفيدة بشكل خاص لأتمتة التحكم في تشغيل الفيديو في جلسات المتصفح.

## Syntax

```powershell
Resume-WebbrowserTabVideo [<CommonParameters>]
```

## Examples

### Resume-WebbrowserTabVideo

```powershell
Resume-WebbrowserTabVideo
```

### wbvideoplay

```powershell
wbvideoplay
```
