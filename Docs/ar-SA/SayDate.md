# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> ينطق التاريخ الحالي باستخدام التوليف الصوتي للنص.

## Description

يقوم بتحويل التاريخ الحالي إلى صيغة اللغة الطبيعية ويعلنه باستخدام محرك تحويل النص إلى كلام في النظام. يتم نطق التاريخ بالصيغة: "اليوم هو [يوم الأسبوع]، [الشهر] [اليوم] [السنة]"

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
