# Search-GenXdevCmdlet

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `searchcmdlet`

## Synopsis

> يبحث عن أمر GenXdev cmdlet، ويفتحه اختيارياً في بيئة تطوير متكاملة (IDE) للتحرير.

## Description

تبحث هذه الدالة عن أوامر GenXdev باستخدام نمط وتفتح اختيارياً ملف المصدر للأمر الذي تم العثور عليه في Visual Studio Code أو Visual Studio للتحرير. وهي تسترجع معلومات الأمر وتوفر اختصارات لوحة المفاتيح للانتقال بسرعة إلى تعريف الدالة.

## Syntax

```powershell
Search-GenXdevCmdlet [[-CmdletName] <String>] [[-ModuleName] <String[]>] [-Code] [-EditCmdlet] [-FromScripts] [-NoLocal] [-OnlyPublished] [-VisualStudio] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CmdletName` | String | — | — | 0 | — | نمط البحث لتصفية أداة الأوامر (cmdlets) 🌐 *Supports wildcards* |
| `-ModuleName` | String[] | — | ✅ (ByValue, ByPropertyName) | 1 | — | أسماء وحدات GenXdev للبحث 🌐 *Supports wildcards* |
| `-NoLocal` | SwitchParameter | — | — | Named | — | تخطي البحث في مسارات الوحدات المحلية |
| `-OnlyPublished` | SwitchParameter | — | — | Named | — | اقتصر البحث على مسارات الوحدات المنشورة فقط |
| `-FromScripts` | SwitchParameter | — | — | Named | — | البحث في ملفات البرامج النصية بدلاً من ملفات الوحدات النمطية |
| `-Code` | SwitchParameter | — | — | Named | — | افتح الأمر الموجود في Visual Studio Code |
| `-VisualStudio` | SwitchParameter | — | — | Named | — | افتح الأمر cmdlet الموجود في Visual Studio |
| `-EditCmdlet` | SwitchParameter | — | — | Named | — | كما يفتح الملف في المحرر بعد العثور عليه |

## Examples

### Search-GenXdevCmdlet -CmdletName "Get-*" -Code Opens first matching cmdlet starting with "Get-" in Visual Studio Code.

```powershell
Search-GenXdevCmdlet -CmdletName "Get-*" -Code
Opens first matching cmdlet starting with "Get-" in Visual Studio Code.
```

### searchcmdlet "Set-GitConfig" -EditCmdlet Searches for Set-GitConfig cmdlet and opens it for editing.

```powershell
searchcmdlet "Set-GitConfig" -EditCmdlet
Searches for Set-GitConfig cmdlet and opens it for editing.
```
