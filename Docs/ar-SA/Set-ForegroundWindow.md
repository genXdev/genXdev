# Set-ForegroundWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> يجلب النافذة المحددة إلى المقدمة ويجعلها النافذة النشطة.

## Description

* يجعل النافذة نافذة أمامية باستخدام طرق Win32 API متعددة لتحقيق أقصى قدر من الموثوقية.
* يحاول أولاً استخدام واجهة برمجة التطبيقات SwitchToThisWindow، ثم يعود إلى SetForegroundWindow إذا لزم الأمر.

## Syntax

```powershell
Set-ForegroundWindow -WindowHandle <IntPtr> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-WindowHandle` | IntPtr | ✅ | — | 0 | `0` | مقبض النافذة لتعيينها كنافذة أمامية |

## Examples

### Example 1

```powershell
$hwnd = (Get-Process notepad).MainWindowHandle
Set-ForegroundWindow -WindowHandle $hwnd
```

يجلب نافذة المفكرة إلى المقدمة باستخدام مقبض النافذة.

### Example 2

```powershell
$hwnd = (Get-Process notepad).MainWindowHandle
Set-ForegroundWindow $hwnd
```

يستخدم معلمة موضعية لبناء جملة موجزة.
