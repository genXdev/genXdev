# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....`

## Synopsis

> ينتقل لأعلى ثلاثة مستويات من الدليل في التسلسل الهرمي لنظام الملفات.

## Description

* يغير دليل العمل الحالي بالانتقال لأعلى ثلاثة أدلة أصلية في التسلسل الهرمي لنظام الملفات.
* يعرض محتويات الدليل الناتج.

## Syntax

```powershell
Set-LocationParent3 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

ينتقل إلى مستوى ثلاثة أدلة للأعلى من الموقع الحالي.

### Example 2

```powershell
....
```

يستخدم الاسم المستعار للتنقل لأعلى ثلاثة مستويات من الدليل.
