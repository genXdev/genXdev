# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> ينتقل لأعلى أربعة مستويات دليل في التسلسل الهرمي لنظام الملفات.

## Description

* ينقل لأعلى أربعة مستويات من الدليل من الموقع الحالي في نظام الملفات.
* بعد التنقل، يعرض محتويات الدليل الناتج.

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

ينتقل لأعلى أربعة مستويات دليل من الموقع الحالي.

### Example 2

```powershell
.....
```

يستخدم الاسم المستعار للتنقل لأعلى أربعة مستويات دليل.
