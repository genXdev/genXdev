# Set-MonitorPowerOn

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `wakemonitor`, `monitoroff`

## Synopsis

> يقوم بتشغيل طاقة الشاشة.

## Description

يستخدم واجهة برمجة تطبيقات ويندوز من خلال GenXdev.Helpers.WindowObj لإيقاظ الشاشة من حالة السكون/إيقاف التشغيل. هذا مفيد لبرامج الأتمتة التي تحتاج إلى ضمان تشغيل الشاشة.

## Syntax

```powershell
Set-MonitorPowerOn [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOn
```

يقوم بتشغيل طاقة الشاشة.

### Example 2

```powershell
wake-monitor
```

يقوم بتشغيل طاقة الشاشة باستخدام اسم مستعار.
