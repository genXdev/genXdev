# Set-WebbrowserTabLocation

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `lt`, `Nav`

## Synopsis

> ينقل علامة تبويب متصفح الويب الحالية إلى عنوان URL محدد.

## Description

يعيّن موقع (URL) علامة تبويب المتصفح المحددة حاليًا. يدعم كلاً من متصفحي Edge و Chrome من خلال مفاتيح اختيارية. يتضمن التنقل التحقق من صحة URL ويضمن تحميل الصفحة بشكل صحيح من خلال العمليات غير المتزامنة.

## Syntax

```powershell
Set-WebbrowserTabLocation -Url <String> [-ByReference <PSObject>] [-Chromium] [-Firefox] [-NoAutoSelectTab] [-Page <Object>] [-Webkit] [<CommonParameters>]

Set-WebbrowserTabLocation [-Edge] [<CommonParameters>]

Set-WebbrowserTabLocation [-Chrome] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |

| `-NoAutoSelectTab` | SwitchParameter | — | — | Named | — | منع تحديد علامة التبويب تلقائيًا |
| `-Edge` | SwitchParameter | — | — | Named | — | التنقل باستخدام متصفح Microsoft Edge *(Parameter set: )* |
| `-Chrome` | SwitchParameter | — | — | Named | — | تصفح باستخدام متصفح جوجل كروم *(Parameter set: )* |
| `-Chromium` | SwitchParameter | — | — | Named | — | التصفح باستخدام Microsoft Edge أو Google Chrome، اعتمادًا على المتصفح الافتراضي |
| `-Firefox` | SwitchParameter | — | — | Named | — | التنقل باستخدام متصفح فايرفوكس |

| `-ByReference` | PSObject | — | — | Named | — | كائن مرجع جلسة المتصفح |

## Examples

### Set-WebbrowserTabLocation -Url "https://github.com/microsoft" -Edge

```powershell
Set-WebbrowserTabLocation -Url "https://github.com/microsoft" -Edge
```
