# Start-VlcMediaPlayerNextInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcnext`

## Synopsis

> يتقدم مشغل وسائط VLC إلى العنصر التالي في قائمة التشغيل الحالية.

## Description

ترسل هذه الدالة الضغط على المفتاح 'n' إلى مشغل الوسائط VLC للانتقال إلى المقطع التالي أو العنصر التالي في قائمة التشغيل الحالية. تركز على نافذة VLC، وترسل الأمر التالي، وتستعيد التركيز إلى وحدة التحكم في PowerShell. تتضمن الدالة دعم ShouldProcess لظهور طلبات التأكيد عند الحاجة.

## Syntax

```powershell
Start-VlcMediaPlayerNextInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerNextInPlaylist

```powershell
Start-VlcMediaPlayerNextInPlaylist
```

### vlcnext

```powershell
vlcnext
```
