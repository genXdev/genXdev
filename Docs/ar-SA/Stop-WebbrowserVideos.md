# Stop-WebbrowserVideos

> **Module:** GenXdev.Webbrowser.Playwright | **Type:** Function | **Aliases:** `wbsst`, `ssst`, `wbvideostop`

## Synopsis

> إيقاف تشغيل الفيديو مؤقتًا في جميع جلسات المتصفح النشطة.

## Description

يقوم بالتكرار عبر جميع جلسات المتصفح النشطة وإيقاف أي فيديوهات قيد التشغيل مؤقتًا عن طريق تنفيذ أوامر JavaScript. تحافظ الوظيفة على حالة الجلسة الأصلية وتتعامل مع الأخطاء بشكل أنيق.

## Syntax

```powershell
Stop-WebbrowserVideos [-Chrome] [-Chromium] [-Edge] [-Firefox] [-Webkit] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Edge` | SwitchParameter | — | — | Named | — | يفتح في مايكروسوفت إيدج |
| `-Chrome` | SwitchParameter | — | — | Named | — | يفتح في جوجل كروم |
| `-Chromium` | SwitchParameter | — | — | Named | — | يُفتح في Microsoft Edge أو Google Chrome، اعتمادًا على المتصفح الافتراضي |
| `-Firefox` | SwitchParameter | — | — | Named | — | يفتح في فايرفوكس |
| `-Webkit` | SwitchParameter | — | — | Named | — | يستخدم متصفح WebKit المُدار عبر Playwright |
