# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> يبدّل حالة الإيقاف المؤقت/التشغيل لمشغل الوسائط VLC.

## Description

ترسل هذه الدالة مفتاح المسافة إلى مشغل الوسائط VLC للتبديل بين حالتي الإيقاف المؤقت والتشغيل. وتستعيد التركيز تلقائيًا إلى النافذة الأصلية بعد إرسال الأمر الرئيسي.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

يقوم بتبديل حالة الإيقاف المؤقت/التشغيل لمشغل الوسائط VLC.

### vlcpause

```powershell
vlcpause
```

يستخدم الاسم المستعار لتبديل حالة الإيقاف المؤقت/التشغيل.

### vlcplay

```powershell
vlcplay
```

يستخدم الاسم المستعار البديل لتبديل حالة الإيقاف المؤقت/التشغيل.
