# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> يقوم بتبديل وضع التكرار في مشغل الوسائط VLC.

## Description

ترسل هذه الدالة أمر المفتاح 'r' إلى مشغل الوسائط VLC لتبديل أنماط التكرار المختلفة (لا تكرار، تكرار الحالي، تكرار الكل). تقوم الدالة بفتح VLC إذا لم يكن قيد التشغيل بالفعل، وتعيد التركيز إلى النافذة السابقة بعد إرسال الأمر.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
