# Test-RefactorLLMSelection

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> يقوم بتقييم ملفات المصدر لتحديد أهلية إعادة الهيكلة باستخدام تحليل LLM.

## Description

يستخدم تحليل نموذج تعلم اللغة (LLM) لتحديد ما إذا كان يجب تحديد ملف شفرة المصدر لإعادة هيكلته بناءً على معايير محددة. تقوم الوظيفة بمعالجة محتوى الملف من خلال استعلام LLM وتُرجع استجابة منطقية.

## Syntax

```powershell
Test-RefactorLLMSelection -RefactorDefinition <GenXdev.Helpers.RefactorDefinition> -Path <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-RefactorDefinition` | GenXdev.Helpers.RefactorDefinition | ✅ | — | 0 | — | تعريف إعادة الهيكلة الذي يحتوي على إعدادات LLM |
| `-Path` | String | ✅ | — | 1 | — | المسار إلى الملف المصدر المطلوب تقييمه |

## Examples

### Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"

```powershell
Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"
```
