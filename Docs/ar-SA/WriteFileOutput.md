# WriteFileOutput

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> يعرض معلومات الملف مع أسماء عرض مرتبطة بصلات لتحسين تجربة المستخدم.

## Description

تقوم هذه الوظيفة بمعالجة كائنات إدخال الملفات وإخراجها بتنسيق سهل الاستخدام مع أسماء ملفات مرتبطة تشعبيًا عند عرضها في وحدة التحكم. وهي تعالج أنواع إدخال مختلفة تشمل السلاسل النصية وكائنات FileInfo وأنواع كائنات أخرى. تكتشف الوظيفة تلقائيًا ما إذا كان الإخراج يتم إعادة توجيهه أو التقاطه في خط أنابيب وتعدل سلوكها وفقًا لذلك.

## Syntax

```powershell
WriteFileOutput -CallerInvocation <Object> -Input <Object> [-FullPaths] [-Prefix <String>] [-RelativeBasePath <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CallerInvocation` | Object | ✅ | ✅ (ByValue) | 0 | — | معلومات الاستدعاء من الدالة الاستدعاءية |
| `-Input` | Object | ✅ | ✅ (ByValue) | 1 | — | كائن الإدخال المراد معالجته، والذي يمكن أن يكون مسار ملف أو كائن |
| `-Prefix` | String | — | — | Named | — | بادئة نص اختيارية تُضاف بداية المخرجات لتوفير سياق إضافي |
| `-RelativeBasePath` | String | — | — | Named | — | المسار الأساسي لإنشاء مسارات ملفات نسبية في المخرجات |
| `-FullPaths` | SwitchParameter | — | — | Named | — | يجبر المخرجات على استخدام المسارات المطلقة الكاملة بدلاً من المسارات النسبية |

## Examples

### WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"

```powershell
WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"
```

### Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_

```powershell
Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_
```
