# resetdefaultmonitor

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> استعادة التكوين الافتراضي للشاشة الثانوية.

## Description

يقوم هذا السكريبت باستعادة تكوين الشاشة الثانوية الافتراضية للنظام، مع ضبط الشاشة الثانوية على القيمة الافتراضية الأصلية. هذا مفيد للمستخدمين الذين يرغبون في العودة إلى إعداد الشاشات المتعددة السابق بعد استخدام تكوينات جنبًا إلى جنب. انظر أيضًا: وظيفة 'sidebyside' للتبديل إلى وضع جنبًا إلى جنب للنوافذ الجديدة.

## Syntax

```powershell
resetdefaultmonitor [<CommonParameters>]
```

## Examples

### secondscreen

```powershell
secondscreen
```

يستعيد التكوين الافتراضي للشاشة الثانوية للنظام.
