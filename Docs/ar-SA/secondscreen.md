# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> تعيين التكوين الافتراضي للشاشة الثانية.

## Description

يضبط السلوك الافتراضي لفتح نوافذ GenXdev ليكون على الشاشة الثانوية.
هذا مفيد للمستخدمين الذين لديهم شاشة واحدة أو الذين يفضلون تخطيطات النوافذ جنباً إلى جنب.
انظر أيضًا إلى cmdlet 'sidebyside' و'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

يضبط الإعدادات الافتراضية لفتحات نافذة GenXdev لتكون على الشاشة الثانوية
