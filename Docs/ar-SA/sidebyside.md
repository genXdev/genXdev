# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> تعيين التكوين الافتراضي للتشغيل جنبًا إلى جنب.

## Description

يضبط السلوك الافتراضي لفتح نوافذ GenXdev ليكون جنبًا إلى جنب مع PowerShell.
هذا مفيد للمستخدمين الذين لديهم شاشة واحدة أو الذين يفضلون تخطيط النوافذ جنبًا إلى جنب.
انظر أيضًا الأمر cmdlet 'secondscreen' و 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

يضبط الإعدادات الافتراضية لفتحات نافذة GenXdev لتكون جنبًا إلى جنب مع PowerShell
