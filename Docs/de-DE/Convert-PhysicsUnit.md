# Convert-PhysicsUnit

> **Module:** GenXdev.Helpers.Physics | **Type:** Function | **Aliases:** —

## Synopsis

> Konvertiert einen Wert von einer physikalischen Einheit in eine andere innerhalb derselben Kategorie.

## Description

Diese Funktion konvertiert physikalische Größen zwischen kompatiblen Einheiten in Kategorien wie Länge, Zeit, Geschwindigkeit usw. Sie leitet die Kategorie aus den Einheiten ab und wirft einen Fehler für inkompatible Konvertierungen.

## Syntax

```powershell
Convert-PhysicsUnit -Value <Double> -FromUnit <String> -ToUnit <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Value` | Double | ✅ | — | 0 | — | Der numerische Wert, der konvertiert werden soll |
| `-FromUnit` | String | ✅ | — | 1 | — | Die Einheit des Eingabewerts |
| `-ToUnit` | String | ✅ | — | 2 | — | Die gewünschte Ausgabeeinheit |

## Examples

### Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"

```powershell
Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"
```

Konvertiert 100 Meter in Fuß.

### Convert-PhysicsUnit 10 "seconds" "minutes"

```powershell
Convert-PhysicsUnit 10 "seconds" "minutes"
```

Konvertiert 10 Sekunden in Minuten mithilfe von Positionsparametern.
