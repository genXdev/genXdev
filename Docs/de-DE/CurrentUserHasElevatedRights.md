# CurrentUserHasElevatedRights

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Überprüft, ob der aktuelle Benutzer über erhöhte Rechte verfügt.

## Description

Ermittelt, ob der aktuelle Windows-Benutzer über Administrator- oder Backup-Operator-Berechtigungen verfügt, indem die Sicherheitsprinzipalrollen überprüft werden. Gibt true zurück, wenn der Benutzer erhöhte Rechte hat, andernfalls false. Implementiert eine robuste Fehlerbehandlung für sicherheits- und zugriffsbezogene Probleme.

## Syntax

```powershell
CurrentUserHasElevatedRights [<CommonParameters>]
```

## Examples

### $hasRights = CurrentUserHasElevatedRights ##############################################################################

```powershell
$hasRights = CurrentUserHasElevatedRights
##############################################################################
```

## Outputs

- `Boolean
Returns true if user has admin or backup operator rights, false otherwise.`
- `Boolean`
