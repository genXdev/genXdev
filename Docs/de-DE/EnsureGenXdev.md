# EnsureGenXdev

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Stellt sicher, dass alle GenXdev-Module ordnungsgemäß geladen werden, indem alle Ensure*-Cmdlets aufgerufen werden.

## Description

Diese Funktion ruft alle GenXdev-Cmdlets ab, die mit "Ensure" oder
"Optimize-Ensure" (für C#-Cmdlets) beginnen, und
führt jedes aus, um sicherzustellen, dass alle erforderlichen GenXdev-Module und
Abhängigkeiten ordnungsgemäß geladen und verfügbar sind. Fehler während der
Ausführung werden abgefangen und als Informationsmeldungen angezeigt.

Optional kann es auch alle NuGet-Pakete herunterladen und laden, die in der
packages.json-Manifestdatei definiert sind.

## Syntax

```powershell
EnsureGenXdev [-ConsentToThirdPartySoftwareInstallation] [-DownloadAllNugetPackages] [-Force] [-ForceConsent] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Force` | SwitchParameter | — | — | Named | — | Erzwingt die Ausführung von Sicherungsvorgängen, selbst wenn diese scheinbar bereits abgeschlossen sind |
| `-DownloadAllNugetPackages` | SwitchParameter | — | — | Named | — | Lädt alle NuGet-Pakete herunter und lädt sie, die in der Manifestdatei packages.json definiert sind |
| `-ForceConsent` | SwitchParameter | — | — | Named | — | Erzwingen Sie eine Zustimmungsaufforderung, selbst wenn die Einstellung für die Installation von Software Dritter festgelegt ist. |
| `-ConsentToThirdPartySoftwareInstallation` | SwitchParameter | — | — | Named | — | Installation von Drittanbietersoftware automatisch zustimmen und persistentes Flag für alle Pakete setzen. |

## Examples

### EnsureGenXdev

```powershell
EnsureGenXdev
```

Dieser Befehl führt alle verfügbaren Ensure*-Cmdlets aus, um die GenXdev-Umgebung zu initialisieren.

### EnsureGenXdev -DownloadAllNugetPackages

```powershell
EnsureGenXdev -DownloadAllNugetPackages
```

Dieser Befehl führt alle Ensure*-Cmdlets aus und lädt außerdem alle in der Manifestdatei packages.json definierten NuGet-Pakete herunter und lädt sie.

### EnsureGenXdev -DownloadAllNugetPackages -ConsentToThirdPartySoftwareInstallation

```powershell
EnsureGenXdev -DownloadAllNugetPackages -ConsentToThirdPartySoftwareInstallation
```

Dieser Befehl führt alle Ensure*-Cmdlets aus und lädt NuGet-Pakete herunter.
