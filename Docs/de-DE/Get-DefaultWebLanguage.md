# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Ruft den Standard-Websprachschlüssel basierend auf den aktuellen Spracheinstellungen des Systems ab.

## Description

* Ruft die aktuellen Systemsprach- und Kultureinstellungen ab und ordnet sie dem entsprechenden Schlüssel des Websprachenwörterbuchs zu, der von Übersetzungsdiensten verwendet wird.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Gibt "Englisch" für ein englischsprachiges System zurück, "Niederländisch" für ein niederländischsprachiges System usw.
