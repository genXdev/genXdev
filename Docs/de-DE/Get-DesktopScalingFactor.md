# Get-DesktopScalingFactor

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Ruft den Windows-Anzeigeskalierungsfaktor (DPI-Einstellung) für einen angegebenen Monitor ab.

## Description

* Ruft den aktuellen Windows-DPI-Skalierungsfaktor ab, der für einen Monitor im System konfiguriert ist.
* Der Skalierungsfaktor wird als Prozentsatz ausgedrückt, wobei 100 für die Standardskalierung (96 DPI) steht. Übliche Werte sind 100, 125, 150 und 200.
* Wenn kein Monitor angegeben wird, wird der Skalierungsfaktor für den primären Bildschirm zurückgegeben.

## Syntax

```powershell
Get-DesktopScalingFactor [[-Monitor] <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | — | — | 0 | `0` | Der zu überprüfende Monitorindex (0 = primärer Monitor) |

## Examples

### Example 1

```powershell
Get-DesktopScalingFactor -Monitor 0
```

Gibt den Skalierungsfaktor-Prozentsatz für den primären Monitor zurück.

### Example 2

```powershell
Get-DesktopScalingFactor 1
```

Gibt den Skalierungsfaktor-Prozentsatz für den zweiten Monitor zurück, wobei ein Positionsparameter verwendet wird.
