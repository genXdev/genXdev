# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss

## Synopsis

> Gibt true zurück, wenn die Text-to-Speech-Engine spricht.

## Description

Überprüft den Zustand sowohl des standardmäßigen als auch des angepassten Sprachsynthesizers, um festzustellen, ob einer derzeit spricht. Gibt „true“ zurück, wenn Sprache aktiv ist, andernfalls „false“.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Gibt true zurück, wenn die Text-to-Speech-Engine spricht.

### Example 2

```powershell
iss
```

Gibt true zurück, wenn die Text-to-Speech-Engine spricht (Alias verwendet).
