# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> Gibt true zurück, wenn die Sprachausgabe-Engine spricht.

## Description

Überprüft den Status sowohl des Standard- als auch des benutzerdefinierten Sprachsynthesizers, um festzustellen, ob einer davon gerade spricht. Gibt true zurück, wenn eine Sprachausgabe läuft, andernfalls false.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Gibt true zurück, wenn die Sprachausgabe-Engine spricht.

### Example 2

```powershell
iss
```

Gibt "true" zurück, wenn die Text-zu-Sprache-Engine spricht (unter Verwendung eines Alias).
