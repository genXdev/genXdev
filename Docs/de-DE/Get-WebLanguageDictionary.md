# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Gibt ein umgekehrtes Wörterbuch für alle von der Google-Suche unterstützten Sprachen zurück.

## Description

* Gibt ein Wörterbuch zurück, das Sprachnamen ihren entsprechenden
  Sprachcodes für alle von Google Search unterstützten Sprachen zuordnet.
* Nützlich für Sprachcode-Nachschlagevorgänge in Webabfragen und Übersetzungsaufgaben.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

Ruft das Wörterbuch ab, das Sprachnamen ihren Codes zuordnet.
