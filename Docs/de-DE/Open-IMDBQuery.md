# Open-IMDBQuery

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** `imdb`

## Synopsis

> Öffnet eine IMDB-Suchanfrage in einem Webbrowser.

## Description

Öffnet eine „Internet Movie Database“-Abfrage in einem Webbrowser mit umfangreichen Konfigurationsoptionen. Diese Funktion bietet eine praktische Hülle um die IMDB-Suchfunktionalität, die es Benutzern ermöglicht, direkt aus PowerShell nach Filmen, TV-Sendungen, Schauspielern und anderen Unterhaltungsinhalten zu suchen.

Die Funktion akzeptiert Suchbegriffe und erstellt ordnungsgemäße IMDB-Such-URLs, öffnet diese dann im angegebenen Webbrowser mit anpassbarer Fensterpositionierung, Browserauswahl und Anzeigeoptionen. Sie unterstützt alle gängigen Browser einschließlich Edge, Chrome und Firefox mit Optionen für privates Surfen, Vollbildmodus und Fensterverwaltung.

Zu den Hauptfunktionen gehören Multi-Monitor-Unterstützung, Sprachlokalisierung, Tastaturautomatisierung und umfassende Browserkonfigurationsoptionen. Die Funktion kann mehrere Suchanfragen gleichzeitig verarbeiten und bietet Optionen sowohl für die interaktive Nutzung als auch für automatisierte Arbeitsabläufe.

## Syntax

```powershell
Open-IMDBQuery -Queries <String[]> [[-Language] <String>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-Monitor <Int32>] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-IMDBQuery -Queries "The Matrix" -Monitor 0

```powershell
Open-IMDBQuery -Queries "The Matrix" -Monitor 0
```

Öffnet eine IMDB-Suche nach "The Matrix" auf dem Standardmonitor.

### imdb "The Matrix" -m 0

```powershell
imdb "The Matrix" -m 0
```

Öffnet eine IMDB-Suche nach "The Matrix" unter Verwendung der Alias- und Kurzformparameter.

### "Inception", "Interstellar" | Open-IMDBQuery -Language "French" -Chrome

```powershell
"Inception", "Interstellar" | Open-IMDBQuery -Language "French" -Chrome
```

Durchsucht IMDB nach mehreren Filmen mit französischen Suchergebnissen in Chrome.

### Open-IMDBQuery -Queries "Tom Hanks" -FullScreen -RestoreFocus

```powershell
Open-IMDBQuery -Queries "Tom Hanks" -FullScreen -RestoreFocus
```

Sucht im Vollbildmodus auf IMDB nach Tom Hanks und gibt dann den Fokus zurück an PowerShell.
