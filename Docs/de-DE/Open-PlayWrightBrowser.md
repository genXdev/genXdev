# Open-PlayWrightBrowser

> **Module:** GenXdev.Webbrowser.Playwright | **Type:** Function | **Aliases:** `spb`

## Synopsis

> Startet einen von Playwright verwalteten Browser mit einem persistenten Benutzerprofil.

## Description

<details>
<summary><b>Expand description</b></summary>

Startet eine Browserinstanz, die von Playwright gesteuert wird. Der Parameter BrowserType bestimmt, welche Browser-Engine und welcher Startmodus verwendet wird:

- ChromeNormal / EdgeNormal: Startet das auf Ihrem Betriebssystem installierte Chrome oder Edge über Playwrights Channel-API, wodurch die Infoleiste „Wird von Automatisierungssoftware gesteuert“ und Probleme mit Anti-Bot-Erkennung vermieden werden.
- ChromiumNormal: Erkennt automatisch Chrome oder Edge (je nachdem, welches installiert und als Systemstandard festgelegt ist) und verwendet die Channel-API. Fällt auf das gebündelte Playwright-Chromium zurück, wenn keines installiert ist.
- ChromiumPlaywright: Verwendet immer das gebündelte Playwright-Chromium-Binary (niemals den im Betriebssystem installierten Browser).
- FirefoxPlaywright: Verwendet das gebündelte Playwright-Firefox-Binary. Die Channel-API ist für Firefox in den Playwright-.NET-Bindungen nicht verfügbar.
- WebKitPlaywright: Verwendet das gebündelte Playwright-WebKit-Binary.

Anti-Erkennungsmaßnahmen für Chromium-basierte Browser:
- Deaktiviert Blink-Automatisierungsflags (entfernt die Infoleiste)
- Setzt Viewport auf null und startet im Vollbildmodus für native Fenstergröße
- Verwendet das Farbschema 'no-preference', um einen weißen Hintergrund zu vermeiden
- Setzt Standard-Browserargumente (no-first-run, no-default-browser-check)

Persistente Profile pro BrowserType werden unter GenXdev AppData gespeichert. Der gestartete Browser und sein Kontext/Seite werden in $Global:GenXdevPlaywright gespeichert, damit sie von anderen Cmdlets verwendet werden können.

</details>

## Examples

### Open-PlayWrightBrowser Launches your OS-installed Chrome or Edge (auto-detected) with a persistent profile and anti-detection measures.

```powershell
Open-PlayWrightBrowser
Launches your OS-installed Chrome or Edge (auto-detected) with a
persistent profile and anti-detection measures.
```

### Open-PlayWrightBrowser -BrowserType EdgeNormal Launches Microsoft Edge via Playwright's Channel API.

```powershell
Open-PlayWrightBrowser -BrowserType EdgeNormal
Launches Microsoft Edge via Playwright's Channel API.
```

### Open-PlayWrightBrowser -BrowserType ChromiumPlaywright Uses the bundled Playwright Chromium binary.

```powershell
Open-PlayWrightBrowser -BrowserType ChromiumPlaywright
Uses the bundled Playwright Chromium binary.
```

### Open-PlayWrightBrowser -BrowserType FirefoxPlaywright -Headless Launches the bundled Firefox in headless mode.

```powershell
Open-PlayWrightBrowser -BrowserType FirefoxPlaywright -Headless
Launches the bundled Firefox in headless mode.
```

### Open-PlayWrightBrowser -Width 1280 -Height 720 -Force Restarts the browser with a 1280x720 viewport.

```powershell
Open-PlayWrightBrowser -Width 1280 -Height 720 -Force
Restarts the browser with a 1280x720 viewport.
```
