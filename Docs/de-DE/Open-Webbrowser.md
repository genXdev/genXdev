# Open-Webbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `wb`

## Synopsis

> Öffnet URLs in einem oder mehreren Browser-Fenstern mit optionaler Positionierung und Gestaltung.

## Description

Diese Funktion bietet einen erweiterten Wrapper für das Starten von Browsern mit umfangreichen Optionen für Fensterpositionierung, Browserauswahl und Verhaltensanpassung. Sie unterstützt mehrere Browser, darunter Edge, Chrome und Firefox, mit Funktionen wie privatem Surfen, Anwendungsmodus und präziser Fensterverwaltung.

Wichtige Funktionen:
- Intelligente Browsererkennung und -auswahl
- Fensterpositionierung (links, rechts, oben, unten, zentriert, Vollbild)
- Multi-Monitor-Unterstützung mit automatischer oder manueller Monitorauswahl
- Unterstützung des privaten/Inkognito-Modus
- Anwendungsmodus für ablenkungsfreies Surfen
- Optionen zum Blockieren von Erweiterungen und Popups
- Fokusverwaltung und Fensterbearbeitung
- Stapelweises Öffnen von URLs in mehreren Browsern
- Tastenanschlag-Automatisierung für Browserfenster

Die Funktion kann automatisch Systemfähigkeiten erkennen und das Verhalten entsprechend anpassen. Für Browser, die nicht auf dem System installiert sind, werden Vorgänge stillschweigend ohne Fehler übersprungen.

## Examples

### wb -PlayWright https://github.com

```powershell
wb -PlayWright https://github.com
```

Öffnet GitHub in einem Playwright-gesteuerten Chromium-Browser.

### Open-Webbrowser -Url "https://github.com"

```powershell
Open-Webbrowser -Url "https://github.com"
```

Öffnet GitHub im Standardbrowser.

### Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left

```powershell
Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left
```

Öffnet Stack Overflow in der linken Hälfte von Monitor 1.

### wb "https://google.com" -m 0 -fs

```powershell
wb "https://google.com" -m 0 -fs
```

Öffnet Google im Vollbildmodus auf dem primären Monitor unter Verwendung von Aliasnamen.

### Open-Webbrowser -Chrome -Private -NewWindow

```powershell
Open-Webbrowser -Chrome -Private -NewWindow
```

Öffnet ein neues Chrome-Fenster im Inkognito-Modus.

### "https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All

```powershell
"https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All
```

Öffnet mehrere URLs in allen installierten Browsern über die Pipeline.

### Open-Webbrowser -Monitor 0 -Right

```powershell
Open-Webbrowser -Monitor 0 -Right
```

Positioniert ein bereits geöffnetes Browserfenster auf der rechten Seite des primären Monitors neu.

### Open-Webbrowser -ApplicationMode -Url "https://app.example.com"

```powershell
Open-Webbrowser -ApplicationMode -Url "https://app.example.com"
```

Öffnet eine Webanwendung im App-Modus ohne Browsersteuerelemente.
