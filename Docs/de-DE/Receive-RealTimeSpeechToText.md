# Receive-RealTimeSpeechToText

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Konvertiert Echtzeit-Audioeingabe mithilfe des Whisper-KI-Modells in Text.

## Description

Dieses Cmdlet erfasst Audio vom Mikrofon oder Desktop und transkribiert es mithilfe des Whisper-KI-Modells in Echtzeit in Text. Es unterstützt verschiedene Audioquellen, Stilleerkennung und mehrere Konfigurationsoptionen für die Spracherkennung.

## Examples

### Example 1

```powershell
Receive-RealTimeSpeechToText
```

Grundlegende Verwendung mit Mikrofon.

### Example 2

```powershell
Receive-RealTimeSpeechToText -UseDesktopAudioCapture
```

Verwendung der Desktop-Audioaufnahme.

### Example 3

```powershell
Receive-RealTimeSpeechToText -AudioDevice "Microphone*"
```

Verwendet ein bestimmtes Audiogerät.

### Example 4

```powershell
Receive-RealTimeSpeechToText -MaxDurationOfSilence "00:00:05"
```

Mit Stille-Erkennung.
