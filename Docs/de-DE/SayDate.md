# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Gibt das aktuelle Datum mithilfe der Sprachsynthese aus.

## Description

Konvertiert das aktuelle Datum in ein natürliches Sprachformat und gibt es über die Text-zu-Sprache-Engine des Systems aus. Das Datum wird im folgenden Format gesprochen: "Es ist [Wochentag], [Monat] [Tag] [Jahr]"

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
