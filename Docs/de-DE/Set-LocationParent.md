# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..

## Synopsis

> Wechselt in das übergeordnete Verzeichnis und listet dessen Inhalt auf.

## Description

* Wechselt eine Ebene im Dateisystem nach oben.
* Zeigt den Inhalt des neuen aktuellen Verzeichnisses an.
* Bietet einen praktischen '..'-Alias für die schnelle Verzeichnisnavigation.

## Syntax

```powershell
Set-LocationParent [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

Wechselt in das übergeordnete Verzeichnis und listet dessen Inhalt auf.

### Example 2

```powershell
..
```

Verwendet den Alias, um zum übergeordneten Verzeichnis zu wechseln.
