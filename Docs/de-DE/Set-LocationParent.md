# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> Wechselt zum übergeordneten Verzeichnis und listet dessen Inhalt auf.

## Description

* Wechselt eine Verzeichnisebene in der Dateisystemhierarchie nach oben.
* Zeigt den Inhalt des neuen aktuellen Verzeichnisses an.
* Bietet einen praktischen Alias '..' für schnelle Verzeichnisnavigation.

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

Wechselt zum übergeordneten Verzeichnis und listet dessen Inhalt auf.

### Example 2

```powershell
..
```

Verwendet den Alias, um in das übergeordnete Verzeichnis zu wechseln.
