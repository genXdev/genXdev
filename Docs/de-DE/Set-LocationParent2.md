# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...

## Synopsis

> Navigiert zwei Verzeichnisebenen in der Dateisystemhierarchie nach oben.

## Description

* Wechselt das aktuelle Arbeitsverzeichnis in das Übergeordnete Verzeichnis (zwei Ebenen höher).
* Zeigt den Inhalt des resultierenden Verzeichnisses an.

## Syntax

```powershell
Set-LocationParent2 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Wechselt in das übergeordnete Verzeichnis und zeigt dessen Inhalt an.
