# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> Navigiert zwei Verzeichnisebenen in der Dateisystemhierarchie nach oben.

## Description

* Ändert das aktuelle Arbeitsverzeichnis in das übergeordnete Verzeichnis des übergeordneten Verzeichnisses
  (zwei Ebenen höher).
* Zeigt den Inhalt des resultierenden Verzeichnisses an.

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Wechselt in das Verzeichnis der übergeordneten Ebene und zeigt dessen Inhalt an.

### Example 2

```powershell
...
```

Verwendet den Alias '...' für denselben Vorgang.
