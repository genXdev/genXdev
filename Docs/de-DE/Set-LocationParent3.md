# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....`

## Synopsis

> Navigiert drei Verzeichnisebenen in der Dateisystemhierarchie nach oben.

## Description

Ändert das aktuelle Arbeitsverzeichnis, indem drei übergeordnete Verzeichnisse in der Dateisystemhierarchie nach oben gewechselt werden.
* Zeigt den Inhalt des resultierenden Verzeichnisses an.

## Syntax

```powershell
Set-LocationParent3 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

Navigiert drei Verzeichnisebenen vom aktuellen Speicherort nach oben.

### Example 2

```powershell
....
```

Navigiert mit dem Alias drei Verzeichnisebenen nach oben.
