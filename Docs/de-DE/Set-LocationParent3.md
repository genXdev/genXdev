# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....

## Synopsis

> Navigiert drei Verzeichnisebenen in der Dateisystemhierarchie nach oben.

## Description

* Ändert das aktuelle Arbeitsverzeichnis, indem es in der Dateisystemhierarchie drei übergeordnete Verzeichnisse nach oben wechselt.
* Zeigt den Inhalt des resultierenden Verzeichnisses an.

## Syntax

```powershell
Set-LocationParent3 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

Navigiert drei Verzeichnisebenen nach oben vom aktuellen Speicherort aus.
