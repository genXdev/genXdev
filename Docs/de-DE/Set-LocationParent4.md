# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....

## Synopsis

> Navigiert vier Verzeichnisebenen in der Dateisystemhierarchie nach oben.

## Description

* Navigiert vier Verzeichnisebenen vom aktuellen Speicherort im Dateisystem nach oben.
* Nach der Navigation wird der Inhalt des resultierenden Verzeichnisses angezeigt.

## Syntax

```powershell
Set-LocationParent4 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Navigiert vier Verzeichnisebenen vom aktuellen Speicherort nach oben.
