# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> Navigiert vier Verzeichnisebenen in der Dateisystemhierarchie nach oben.

## Description

* Wechselt im Dateisystem von der aktuellen Position vier Verzeichnisebenen nach oben.
* Zeigt nach der Navigation den Inhalt des resultierenden Verzeichnisses an.

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Navigiert vom aktuellen Speicherort aus vier Verzeichnisebenen nach oben.

### Example 2

```powershell
.....
```

Verwendet den Alias, um vier Verzeichnisebenen nach oben zu navigieren.
