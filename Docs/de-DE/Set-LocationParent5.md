# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......

## Synopsis

> Navigiert fünf Verzeichnisebenen in der Dateisystemhierarchie nach oben.

## Description

* Wechselt das aktuelle Arbeitsverzeichnis um fünf Elternverzeichnisebenen nach oben.
* Zeigt den Inhalt des resultierenden Verzeichnisses an.
* Bietet einen praktischen Kurzalias '......' für die schnelle Navigation.

## Syntax

```powershell
Set-LocationParent5 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Navigiert fünf Verzeichnisebenen vom aktuellen Standort nach oben.

```powershell
......
```
