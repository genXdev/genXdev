# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> Navigiert fünf Verzeichnisebenen in der Dateisystemhierarchie nach oben.

## Description

* Wechselt das aktuelle Arbeitsverzeichnis um fünf übergeordnete Verzeichnisebenen nach oben.
* Zeigt den Inhalt des resultierenden Verzeichnisses an.
* Bietet den praktischen Kurzbefehl '......' für die schnelle Navigation.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Navigiert fünf Verzeichnisebenen vom aktuellen Standort nach oben.

### Example 2

```powershell
......
```

Navigiert mithilfe des Alias fünf Verzeichnisebenen nach oben.
