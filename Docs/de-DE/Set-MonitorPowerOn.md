# Set-MonitorPowerOn

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `wakemonitor`, `monitoroff`

## Synopsis

> Schaltet die Monitorstromversorgung ein.

## Description

Verwendet die Windows-API über GenXdev.Helpers.WindowObj, um den Monitor aus dem Ruhezustand/Ausgeschalteten Zustand aufzuwecken. Dies ist nützlich für Automatisierungsskripte, die sicherstellen müssen, dass der Monitor eingeschaltet ist.

## Syntax

```powershell
Set-MonitorPowerOn [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOn
```

Schaltet die Monitorstromversorgung ein.

### Example 2

```powershell
wake-monitor
```

Schaltet die Monitorstromversorgung mit einem Alias ein.
