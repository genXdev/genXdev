# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> Schaltet den Pause-/Wiedergabestatus des VLC Media Players um.

## Description

Diese Funktion sendet ein Leerzeichen an den VLC Media Player, um zwischen Pause und Wiedergabe umzuschalten. Nach dem Senden des Tastenbefehls wird der Fokus automatisch wieder auf das ursprüngliche Fenster zurückgesetzt.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

Schaltet den Pausen-/Wiedergabestatus des VLC Media Players um.

### vlcpause

```powershell
vlcpause
```

Verwendet den Alias, um den Pause-/Wiedergabestatus umzuschalten.

### vlcplay

```powershell
vlcplay
```

Verwendet den alternativen Alias, um den Pause-/Wiedergabe-Zustand umzuschalten.
