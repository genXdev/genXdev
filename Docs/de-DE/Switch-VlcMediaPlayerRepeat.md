# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> Schaltet den Wiederholungsmodus im VLC Media Player um.

## Description

Diese Funktion sendet den Befehl der Taste 'r' an den VLC Media Player, um zwischen verschiedenen Wiederholungsmodi (keine Wiederholung, aktuelle Wiederholung, alle wiederholen) umzuschalten. Die Funktion öffnet VLC, falls es nicht bereits läuft, und stellt den Fokus auf das vorherige Fenster zurück, nachdem der Befehl gesendet wurde.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
