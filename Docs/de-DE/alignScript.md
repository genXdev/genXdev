# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Gibt eine Zeichenfolge (mit geänderter Einrückung) einer angegebenen Skriptblockzeichenfolge zurück

## Description

Ändert die Einrückung einer Skriptblock-Zeichenfolge unter Berücksichtigung der ursprünglichen Codeblock-Einrückungen

## Syntax

```powershell
alignScript [<CommonParameters>]
```
