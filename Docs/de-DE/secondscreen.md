# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Legt die Standardkonfiguration für den zweiten Monitor fest.

## Description

Legt das Standardverhalten für GenXdev-Fensteröffnungen auf dem sekundären Monitor fest.
Dies ist nützlich für Benutzer mit einem einzelnen Monitor oder solche, die nebeneinander angeordnete Fensterlayouts bevorzugen.
Siehe auch Cmdlet 'sidebyside' und 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Legt Standardwerte für GenXdev-Fensteröffnungen fest, die auf dem sekundären Monitor angezeigt werden sollen
