# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Legt die standardmäßige nebeneinander Anordnung fest.

## Description

Legt das Standardverhalten für GenXdev-Fensteröffnungen fest, dass diese nebeneinander mit PowerShell angezeigt werden.
Dies ist nützlich für Benutzer mit einem einzelnen Monitor oder solche, die nebeneinanderliegende Fensterlayouts bevorzugen.
Siehe auch Cmdlet 'secondscreen' und 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

Legt Standards für GenXdev-Fensteröffnungen fest, sodass sie nebeneinander mit PowerShell angeordnet werden
