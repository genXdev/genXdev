# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> Navigates up four directory levels in the filesystem hierarchy.

## Description

* Moves up four directory levels from the current location in the
  filesystem.
* After navigation, displays the contents of the resulting directory.

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Navigates up four directory levels from the current location.

### Example 2

```powershell
.....
```

Uses the alias to navigate up four directory levels.
