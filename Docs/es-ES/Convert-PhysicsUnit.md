# Convert-PhysicsUnit

> **Module:** GenXdev.Helpers.Physics | **Type:** Function | **Aliases:** —

## Synopsis

> Convierte un valor de una unidad física a otra dentro de la misma categoría.

## Description

Esta función convierte cantidades físicas entre unidades compatibles en categorías como longitud, tiempo, velocidad, etc. Infiere la categoría a partir de las unidades y genera un error para conversiones incompatibles.

## Syntax

```powershell
Convert-PhysicsUnit -Value <Double> -FromUnit <String> -ToUnit <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Value` | Double | ✅ | — | 0 | — | El valor numérico a convertir |
| `-FromUnit` | String | ✅ | — | 1 | — | La unidad del valor de entrada |
| `-ToUnit` | String | ✅ | — | 2 | — | La unidad de salida deseada |

## Examples

### Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"

```powershell
Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"
```

Convierte 100 metros a pies.

### Convert-PhysicsUnit 10 "seconds" "minutes"

```powershell
Convert-PhysicsUnit 10 "seconds" "minutes"
```

Convierte 10 segundos a minutos usando parámetros posicionales.
