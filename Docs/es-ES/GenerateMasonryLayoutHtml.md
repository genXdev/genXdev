# GenerateMasonryLayoutHtml

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Genera una galería HTML de diseño de mampostería responsive a partir de datos de imágenes.

## Description

Crea una galería HTML interactiva con diseño de cuadrícula de mampostería responsiva para mostrar imágenes. Las características incluyen:
- Diseño de cuadrícula responsiva que se adapta al tamaño de la pantalla
- Información emergente de imágenes que muestra descripciones y palabras clave
- Funcionalidad de copiar ruta de imagen al hacer clic
- Estilo moderno y limpio con efectos al pasar el ratón

## Syntax

```powershell
GenerateMasonryLayoutHtml -Images <Object> [[-FilePath] <String>] [-AutoAnimateRectangles] [-AutoScrollPixelsPerSecond <Int32>] [-CanDelete] [-CanEdit] [-Description <String>] [-EmbedImages] [-ImageUrlPrefix <String>] [-MaxPrintImages <Int32>] [-PageSize <Int32>] [-RootMargin <String>] [-ShowOnlyPictures] [-SingleColumnMode] [-Threshold <Double>] [-Title <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Images` | Object | ✅ | — | 0 | — | Matriz de objetos de imagen con ruta, palabras clave y descripción |
| `-FilePath` | String | — | — | 1 | `$null` | Ruta de salida para el archivo HTML generado |
| `-Title` | String | — | — | Named | `'Photo Gallery'` | Título para la galería |
| `-Description` | String | — | — | Named | `'Hover over images to see face recognition, object detection, and scene classification data'` | Descripción para la galería |
| `-CanEdit` | SwitchParameter | — | — | Named | `$false` | Si la edición está habilitada |
| `-CanDelete` | SwitchParameter | — | — | Named | `$false` | Si la eliminación está habilitada |
| `-EmbedImages` | SwitchParameter | — | — | Named | `$false` | Incrustar imágenes como URL de datos base64 en lugar de URL de archivo para una mejor portabilidad |
| `-ShowOnlyPictures` | SwitchParameter | — | — | Named | — | Muestra solo imágenes en un rectángulo redondeado, sin texto debajo. |
| `-AutoScrollPixelsPerSecond` | Int32 | — | — | Named | `$null` | Desplazamiento automático de la página por esta cantidad de píxeles por segundo (null para desactivar) |
| `-AutoAnimateRectangles` | SwitchParameter | — | — | Named | — | Animar rectángulos (objetos/caras) en el rango visible, alternando cada 300 ms |
| `-SingleColumnMode` | SwitchParameter | — | — | Named | `$false` | Forzar diseño de una sola columna (centrado, 1/3 del ancho de pantalla) |
| `-ImageUrlPrefix` | String | — | — | Named | `''` | Prefijo para anteponer a cada ruta de imagen (por ejemplo, para URL remotas) |
| `-PageSize` | Int32 | — | — | Named | `20` | Número de imágenes a cargar por página (para carga dinámica) |
| `-MaxPrintImages` | Int32 | — | — | Named | `50` | Número máximo de imágenes para cargar en modo de impresión |
| `-RootMargin` | String | — | — | Named | `'1200px'` | rootMargin de IntersectionObserver para el disparador de desplazamiento infinito (por ejemplo, "1200px") |
| `-Threshold` | Double | — | — | Named | `0.1` | Umbral de IntersectionObserver para el activador de desplazamiento infinito |

## Examples

### Create gallery from image array and save to file $images = @(     @{         path = "C:\photos\sunset.jpg"         keywords = @("nature", "sunset", "landscape")         description = @{             short_description = "Mountain sunset"             long_description = "Beautiful sunset over mountain range"         }     } ) GenerateMasonryLayoutHtml -Images $images -FilePath "C:\output\gallery.html"

```powershell

            long_description = "Beautiful sunset over mountain range"
        }
    }
)
GenerateMasonryLayoutHtml -Images $images -FilePath "C:\output\gallery.html"
```
