# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Obtiene la clave de idioma web predeterminada según la configuración de idioma actual del sistema.

## Description

* Recupera la configuración de idioma y cultura actual del sistema y las asigna a la clave de diccionario de idioma web correspondiente utilizada por los servicios de traducción.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Devuelve "Inglés" para un sistema en inglés, "Neerlandés" para un sistema en neerlandés, etc.
