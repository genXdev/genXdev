# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss

## Synopsis

> Devuelve verdadero si el motor de texto a voz está hablando.

## Description

Comprueba el estado de los sintetizadores de voz predeterminado y personalizado para determinar si alguno está hablando actualmente. Devuelve verdadero si se está reproduciendo voz, falso en caso contrario.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Devuelve verdadero si el motor de texto a voz está hablando.

### Example 2

```powershell
iss
```

Devuelve verdadero si el motor de texto a voz está hablando (usando alias).
