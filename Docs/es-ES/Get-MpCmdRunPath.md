# Get-MpCmdRunPath

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Obtiene la ruta al ejecutable MpCmdRun.exe de Windows Defender.

## Description

* Localiza y devuelve la ruta completa de la utilidad de línea de comandos de Windows Defender (MpCmdRun.exe).
* El cmdlet verifica la ubicación de instalación estándar en Archivos de Programa y proporciona un manejo de errores adecuado si no se encuentra el archivo.

## Syntax

```powershell
Get-MpCmdRunPath [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$defenderPath = Get-MpCmdRunPath
```

Recupera la ruta completa de la herramienta de línea de comandos de Windows Defender.
