# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Devuelve un objeto auxiliar de ventana para la ventana principal del terminal de PowerShell.

## Description

* Recupera un objeto auxiliar WindowObj que representa la ventana principal del proceso host de PowerShell actual.
* Esto permite la manipulación e interacción con la propia ventana de terminal.

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

Recupera el objeto WindowObj para la ventana actual del terminal de PowerShell.
