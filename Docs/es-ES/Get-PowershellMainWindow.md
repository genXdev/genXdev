# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Devuelve un objeto auxiliar de ventana para la ventana principal del terminal de PowerShell.

## Description

* Recupera un objeto WindowObj que representa la ventana principal del proceso host de PowerShell actual.
* Esto permite manipular e interactuar con la propia ventana del terminal.

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

Recupera el objeto WindowObj para la ventana del terminal de PowerShell actual.
