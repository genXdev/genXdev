# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Devuelve un diccionario inverso para todos los idiomas compatibles con Google Search.

## Description

* Devuelve un diccionario que mapea nombres de idiomas a sus códigos correspondientes para todos los idiomas compatibles con Google Search.
* Útil para búsquedas de códigos de idioma en consultas web y tareas de traducción.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

Recupera la asignación de nombres de idioma a sus códigos.
