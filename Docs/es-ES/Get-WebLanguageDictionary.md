# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Devuelve un diccionario invertido para todos los idiomas compatibles con la Búsqueda de Google.

## Description

* Devuelve un diccionario que asigna nombres de idiomas a sus códigos de idioma correspondientes para todos los idiomas compatibles con Google Search.
* Útil para búsquedas de códigos de idioma en consultas web y tareas de traducción.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

Recupera el diccionario que asigna nombres de idiomas a sus códigos.
