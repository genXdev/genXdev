# Invoke-AIPowershellCommand

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `hint`

## Synopsis

> Genera y ejecuta comandos de PowerShell utilizando asistencia de IA.

## Description

Utiliza modelos de IA para generar comandos de PowerShell basados en consultas en lenguaje natural. La función puede enviar comandos directamente a la ventana de PowerShell o copiarlos al portapapeles. Aprovecha modelos de IA para interpretar lenguaje natural y convertirlo en comandos ejecutables de PowerShell con soporte completo de parámetros para varios backends de IA.

## Syntax

```powershell
Invoke-AIPowershellCommand -PowershellCmdline <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-LLMQueryType <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-PowershellCmdline` | String | ✅ | — | 0 | — | La consulta en lenguaje natural para generar un comando para |
| `-LLMQueryType` | String | — | — | Named | `'Coding'` | El tipo de consulta de LLM |
| `-Model` | String | — | — | Named | — | El identificador o patrón de modelo a utilizar para operaciones de IA |
| `-ApiEndpoint` | String | — | — | Named | — | La URL del punto de conexión de la API para operaciones de IA |
| `-ApiKey` | String | — | — | Named | — | La clave API para operaciones de IA autenticadas |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | Indica que el LLM no tiene soporte para esquemas JSON |

## Examples

### Invoke-AIPowershellCommand -Query "list all running processes"

```powershell
Invoke-AIPowershellCommand -Query "list all running processes"
```

Genera un comando de PowerShell para listar los procesos en ejecución

### hint "list files modified today"

```powershell
hint "list files modified today"
```

Utiliza el alias para generar un comando para buscar archivos modificados hoy.

### Invoke-AIPowershellCommand -Query "stop service" -Clipboard

```powershell
Invoke-AIPowershellCommand -Query "stop service" -Clipboard
```

Genera un comando para detener un servicio y lo copia al portapapeles.
