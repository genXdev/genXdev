# Open-BuiltWithSiteInfo

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Abre consultas del sitio web BuiltWith en un navegador web.

## Description

Abre una consulta de BuiltWith en un navegador web, con opciones configurables de monitorización y comportamiento del navegador. La función analiza las pilas tecnológicas de sitios web consultando el servicio de perfilado tecnológico de BuiltWith.com, que proporciona información detallada sobre tecnologías web, marcos de trabajo y servicios utilizados por los sitios web.

BuiltWith.com es un servicio integral de consulta tecnológica que identifica tecnologías web, marcos de trabajo, sistemas de gestión de contenido, herramientas de análisis, proveedores de alojamiento y otros componentes de software utilizados por cualquier sitio web. Esta función proporciona una interfaz de PowerShell para analizar rápidamente las pilas tecnológicas sin tener que navegar manualmente al sitio web de BuiltWith.

## Syntax

```powershell
Open-BuiltWithSiteInfo -Queries <String[]> [[-Language] <String>] [[-Monitor] <Int32>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-BuiltWithSiteInfo -Queries "microsoft.com" -Monitor 0

```powershell
Open-BuiltWithSiteInfo -Queries "microsoft.com" -Monitor 0
```

Abre un análisis de BuiltWith de Microsoft.com en el monitor predeterminado.

### "microsoft.com" | Open-BuiltWithSiteInfo -m -1

```powershell
"microsoft.com" | Open-BuiltWithSiteInfo -m -1
```

Analiza Microsoft.com mediante la entrada de la canalización con la posición del monitor descartada.
