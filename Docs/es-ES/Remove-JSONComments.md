# Remove-JSONComments

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Elimina comentarios del contenido JSON.

## Description

* Procesa contenido JSON y elimina comentarios de una sola línea y de múltiples líneas mientras preserva la estructura JSON.
* Útil para limpiar archivos JSON que contienen comentarios de documentación antes de analizarlos.
* Admite tanto comentarios de una sola línea (//) como comentarios de múltiples líneas (/* */).

## Syntax

```powershell
Remove-JSONComments -Json <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Json` | String | ✅ | — | 0 | — | .PARÁMETRO -Directory
Especifica que el cmdlet actúa sobre el directorio si el destino es un directorio.

Si no se especifica este parámetro y el destino es un directorio, el cmdlet muestra un mensaje de confirmación para preguntar si se debe actuar sobre el directorio de destino o sobre los elementos secundarios. Emplee el parámetro Force para suprimir el mensaje de confirmación. |

## Examples

### Example 1

```powershell
$jsonContent = @'
{
    // This is a comment
    "name": "test", /* inline comment */
    "value": 123
}
'@ -split "`n"
Remove-JSONComments -Json $jsonContent
```

Elimina los comentarios del contenido JSON almacenado en una variable.

### Example 2

```powershell
$jsonContent | Remove-JSONComments
```

Procesa contenido JSON de la canalización.
