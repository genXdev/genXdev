# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Habla la fecha actual mediante síntesis de texto a voz.

## Description

Convierte la fecha actual en un formato de lenguaje natural y la anuncia usando el motor de texto a voz del sistema. La fecha se expresa en el formato: "Hoy es [día de la semana], [mes] [día] [año]"

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
