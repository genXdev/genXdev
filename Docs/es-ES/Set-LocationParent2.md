# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...

## Synopsis

> Navega dos niveles de directorio hacia arriba en la jerarquía del sistema de archivos.

## Description

* Cambia el directorio de trabajo actual al directorio abuelo (dos niveles arriba).
* Muestra el contenido del directorio resultante.

## Syntax

```powershell
Set-LocationParent2 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Cambia al directorio padre y muestra su contenido.

```powershell
...
```

Utiliza el alias '...' para la misma operación.
