# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....

## Synopsis

> Navega tres niveles de directorio hacia arriba en la jerarquía del sistema de archivos.

## Description

* Cambia el directorio de trabajo actual subiendo tres niveles en la jerarquía del sistema de archivos.
* Muestra el contenido del directorio resultante.

## Syntax

```powershell
Set-LocationParent3 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

Navega tres niveles de directorio hacia arriba desde la ubicación actual.

### Example 2

```powershell
....
```

Utiliza el alias para subir tres niveles de directorio.
