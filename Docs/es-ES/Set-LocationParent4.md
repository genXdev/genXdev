# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> Navega hacia arriba cuatro niveles de directorio en la jerarquía del sistema de archivos.

## Description

* Sube cuatro niveles de directorio desde la ubicación actual en el sistema de archivos.
* Tras la navegación, muestra el contenido del directorio resultante.

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Navega hacia arriba cuatro niveles de directorio desde la ubicación actual.

### Example 2

```powershell
.....
```

Utiliza el alias para navegar cuatro niveles de directorio hacia arriba.
