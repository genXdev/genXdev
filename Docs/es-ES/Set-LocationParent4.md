# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....

## Synopsis

> Navega cuatro niveles de directorio hacia arriba en la jerarquía del sistema de archivos.

## Description

* Sube cuatro niveles de directorio desde la ubicación actual en el sistema de archivos.
* Después de la navegación, muestra el contenido del directorio resultante.

## Syntax

```powershell
Set-LocationParent4 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Navega hacia arriba cuatro niveles de directorio desde la ubicación actual.

### Example 2

```powershell
.....
```

Uso del alias para subir cuatro niveles de directorio.
