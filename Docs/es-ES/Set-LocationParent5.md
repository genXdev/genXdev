# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> Navega hacia arriba cinco niveles de directorio en la jerarquía del sistema de archivos.

## Description

* Cambia el directorio de trabajo actual subiendo cinco niveles de directorio padre.
* Muestra el contenido del directorio resultante.
* Proporciona un alias abreviado conveniente '......' para una navegación rápida.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Navega cinco niveles de directorio hacia arriba desde la ubicación actual.

### Example 2

```powershell
......
```

Usa el alias para navegar cinco niveles de directorio hacia arriba.
