# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......

## Synopsis

> Navega cinco niveles de directorio hacia arriba en la jerarquía del sistema de archivos.

## Description

* Cambia el directorio de trabajo actual subiendo cinco niveles de directorio padre.
* Muestra el contenido del directorio resultante.
* Proporciona un alias abreviado '......' para navegación rápida.

## Syntax

```powershell
Set-LocationParent5 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Navega cinco niveles de directorio hacia arriba desde la ubicación actual.

### Example 2

```powershell
......
```

Utiliza el alias para navegar cinco niveles de directorio hacia arriba.
