# Set-MonitorPowerOff

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `poweroff`

## Synopsis

> Apaga la alimentación de todos los monitores conectados.

## Description

Utiliza llamadas a la API de Windows para enviar una señal de apagado a todos los monitores conectados. Esto equivale a presionar el botón de encendido físico de sus monitores. Los monitores entrarán en modo de ahorro de energía, pero pueden reactivarse moviendo el ratón o pulsando una tecla.

## Syntax

```powershell
Set-MonitorPowerOff [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOff
```

Apaga todos los monitores conectados.

### Example 2

```powershell
poweroff
```

Apaga todos los monitores conectados usando el alias.
