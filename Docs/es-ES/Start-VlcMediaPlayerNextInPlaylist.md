# Start-VlcMediaPlayerNextInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcnext`

## Synopsis

> Avanza el reproductor multimedia VLC al siguiente elemento de la lista de reproducción actual.

## Description

Esta función envía la pulsación de tecla 'n' a VLC Media Player para saltar a la siguiente pista o elemento multimedia en la lista de reproducción actualmente cargada. Enfoca la ventana de VLC, envía el comando de siguiente, y restaura el foco a la consola de PowerShell. La función incluye soporte ShouldProcess para indicaciones de confirmación cuando sea necesario.

## Syntax

```powershell
Start-VlcMediaPlayerNextInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerNextInPlaylist

```powershell
Start-VlcMediaPlayerNextInPlaylist
```

### vlcnext

```powershell
vlcnext
```
