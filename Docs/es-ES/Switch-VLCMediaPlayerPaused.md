# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> Alterna el estado de pausa/reproducción del Reproductor multimedia VLC.

## Description

Esta función envía una tecla de espacio a VLC Media Player para alternar entre los estados de pausa y reproducción. Restaura automáticamente el foco a la ventana original después de enviar el comando de tecla.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

Alterna el estado de pausa/reproducción de VLC Media Player.

### vlcpause

```powershell
vlcpause
```

Usa el alias para alternar el estado de pausa/reproducción.

### vlcplay

```powershell
vlcplay
```

Utiliza el alias alternativo para alternar el estado de pausa/reproducción.
