# Test-RefactorLLMSelection

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Evalúa los archivos de origen para determinar su elegibilidad para la refactorización mediante análisis de LLM.

## Description

Utiliza el análisis del Modelo de Lenguaje de Aprendizaje (LLM) para determinar si un archivo de código fuente debe seleccionarse para su refactorización según los criterios especificados. La función procesa el contenido del archivo mediante una consulta LLM y devuelve una respuesta booleana.

## Syntax

```powershell
Test-RefactorLLMSelection -RefactorDefinition <GenXdev.Helpers.RefactorDefinition> -Path <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-RefactorDefinition` | GenXdev.Helpers.RefactorDefinition | ✅ | — | 0 | — | La definición de refactorización que contiene configuraciones de LLM |
| `-Path` | String | ✅ | — | 1 | — | La ruta al archivo de origen que se va a evaluar |

## Examples

### Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"

```powershell
Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"
```

### $def | Test-RefactorLLMSelection -Path source.ps1

```powershell
$def | Test-RefactorLLMSelection -Path source.ps1
```
