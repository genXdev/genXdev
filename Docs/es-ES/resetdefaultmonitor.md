# resetdefaultmonitor

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Restaura la configuración predeterminada del monitor secundario.

## Description

Este script restaura la configuración predeterminada del monitor secundario para el sistema,
estableciendo el monitor secundario a su valor predeterminado original.
Esto es útil para los usuarios que desean volver a su configuración anterior de varios monitores después de usar configuraciones lado a lado.
Véase también: la función 'sidebyside' para cambiar al modo lado a lado para nuevas ventanas.

## Syntax

```powershell
resetdefaultmonitor [<CommonParameters>]
```

## Examples

### secondscreen

```powershell
secondscreen
```

Restaura la configuración predeterminada del monitor secundario para el sistema.
