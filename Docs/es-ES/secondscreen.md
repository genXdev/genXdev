# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Establece la configuración predeterminada del segundo monitor.

## Description

Establece el comportamiento predeterminado para las aperturas de ventanas de GenXdev en el monitor secundario.
Esto es útil para usuarios con un solo monitor o aquellos que prefieren diseños de ventanas lado a lado.
Consulte también el cmdlet 'sidebyside' y 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Establece los valores predeterminados para que las aperturas de ventanas de GenXdev estén en el monitor secundario
