# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Establece la configuración predeterminada de lado a lado.

## Description

Establece el comportamiento predeterminado para que las aperturas de ventanas de GenXdev se muestren en paralelo con PowerShell.
Esto es útil para usuarios con un solo monitor o aquellos que prefieren diseños de ventanas en paralelo.
Consulte también el cmdlet 'secondscreen' y 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

Establece los valores predeterminados para que las ventanas de GenXdev se abran en paralelo con PowerShell
