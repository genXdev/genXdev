# Convert-DotNetTypeToLLMType

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Convertit les noms de types .NET en noms de types LLM (modèle de langage).

## Description

Prend un nom de type .NET en entrée et retourne le nom de type simplifié correspondant utilisé dans les modèles de langage. Il gère les types .NET courants et fournit des mappages de types appropriés.

## Syntax

```powershell
Convert-DotNetTypeToLLMType -DotNetType <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DotNetType` | String | ✅ | — | 0 | — | Le nom du type .NET à convertir en un nom de type LLM |

## Examples

### Example 1

```powershell
Convert-DotNetTypeToLLMType -DotNetType "System.String"
```

Convertit un type System.String en son équivalent LLM.
