# Enable-Screensaver

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Démarre l'économiseur d'écran Windows configuré.

## Description

Active l'économiseur d'écran du système Windows en exécutant l'exécutable d'économiseur d'écran par défaut (scrnsave.scr) avec le commutateur /s pour le démarrer immédiatement.

## Syntax

```powershell
Enable-Screensaver [<CommonParameters>]
```

## Examples

### Enable-Screensaver

```powershell
Enable-Screensaver
```
