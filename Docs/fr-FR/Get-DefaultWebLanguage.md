# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Obtient la clé de langue web par défaut en fonction des paramètres de langue actuels du système.

## Description

* Récupère les paramètres de langue et de culture actuels du système et les mappe
  à la clé de dictionnaire de langue web correspondante utilisée par les services
  de traduction.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Retourne « Anglais » pour un système anglais, « Néerlandais » pour un système néerlandais, etc.
