# Get-DesktopScalingFactor

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Récupère le facteur d'échelle d'affichage Windows (paramètre DPI) pour un moniteur spécifié.

## Description

* Obtient le facteur de mise à l'échelle d'affichage Windows actuel configuré pour un moniteur dans le système.
* Le facteur de mise à l'échelle est exprimé en pourcentage, où 100 représente une mise à l'échelle standard (96 DPI). Les valeurs courantes sont 100, 125, 150 et 200.
* Si aucun moniteur n'est spécifié, retourne le facteur de mise à l'échelle pour l'affichage principal.

## Syntax

```powershell
Get-DesktopScalingFactor [[-Monitor] <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | — | — | 0 | `0` | L'indice du moniteur à vérifier (0 = moniteur principal) |

## Examples

### Example 1

```powershell
Get-DesktopScalingFactor -Monitor 0
```

Retourne le pourcentage du facteur de mise à l'échelle pour le moniteur principal.

### Example 2

```powershell
Get-DesktopScalingFactor 1
```

Retourne le pourcentage du facteur d'échelle pour le deuxième moniteur à l'aide d'un paramètre positionnel.
