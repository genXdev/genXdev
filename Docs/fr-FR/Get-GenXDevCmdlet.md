# Get-GenXDevCmdlet

> **Module:** GenXdev | **Type:** Function | **Aliases:** `gcmds`

## Synopsis

> Récupère et liste toutes les applets de commande GenXdev ainsi que leurs détails.

## Description

Recherche dans les modules GenXdev installés et les fichiers de script pour trouver les cmdlets, leurs alias et descriptions. Peut filtrer par modèle de nom et par nom de module. Prend en charge le filtrage par définitions de cmdlets et offre des options de recherche flexibles à la fois dans les chemins de modules locaux et publiés.

## Syntax

```powershell
Get-GenXDevCmdlet [[-CmdletName] <String[]>] [[-DefinitionMatches] <String>] [[-ModuleName] <String[]>] [-ExactMatch] [-FromScripts] [-NoLocal] [-OnlyPublished] [-OnlyReturnModuleNames] [<CommonParameters>]

Get-GenXDevCmdlet [-IncludeScripts] [<CommonParameters>]
```
