# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> Retourne true si le moteur de synthèse vocale est en train de parler.

## Description

Vérifie l'état des synthétiseurs vocaux par défaut et personnalisés pour déterminer si l'un d'eux parle actuellement. Renvoie true si la parole est en cours, false sinon.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Retourne true si le moteur de synthèse vocale est en train de parler.

### Example 2

```powershell
iss
```

Renvoie true si le moteur de synthèse vocale est en train de parler (à l'aide d'un alias).
