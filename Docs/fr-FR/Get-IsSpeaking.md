# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss

## Synopsis

> Renvoie vrai si le moteur de synthèse vocale parle.

## Description

Vérifie l'état des synthétiseurs vocaux par défaut et personnalisé pour déterminer si l'un d'eux est en train de parler. Renvoie vrai si la parole est en cours, faux sinon.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Renvoie vrai si le moteur de synthèse vocale parle.

### Example 2

```powershell
iss
```

Renvoie vrai si le moteur de synthèse vocale parle (en utilisant un alias).
