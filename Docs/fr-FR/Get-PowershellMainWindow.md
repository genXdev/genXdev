# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Retourne un objet d'aide à la fenêtre pour la fenêtre principale du terminal PowerShell.

## Description

* Récupère un objet helper WindowObj qui représente la fenêtre principale du
  processus hôte PowerShell actuel.
* Cela permet la manipulation et l'interaction avec la fenêtre du terminal elle-même.

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

Récupère l'objet WindowObj pour la fenêtre actuelle du terminal PowerShell.
