# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Retourne un dictionnaire inversé pour toutes les langues prises en charge par la recherche Google.

## Description

* Retourne un dictionnaire faisant correspondre les noms de langue à leurs codes de langue correspondants pour toutes les langues prises en charge par la Recherche Google.
* Utile pour les recherches de codes de langue dans les requêtes web et les tâches de traduction.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

Récupère le dictionnaire associant les noms de langues à leurs codes.
