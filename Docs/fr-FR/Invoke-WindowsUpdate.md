# Invoke-WindowsUpdate

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** `updatewindows`

## Synopsis

> Vérifie si Windows est à jour et installe optionnellement les mises à jour disponibles.

## Description

Cette fonction recherche les mises à jour Windows, les mises à jour des packages winget et les mises à jour des modules PowerShell. Elle peut afficher les mises à jour disponibles ou les installer automatiquement. La fonction nécessite des privilèges administratifs pour installer les mises à jour Windows et peut éventuellement redémarrer le système si les mises à jour nécessitent un redémarrage.

## Syntax

```powershell
Invoke-WindowsUpdate [[-Criteria] <String>] [-AutoReboot] [-GroupByCategory] [-IncludeDrivers] [-Install] [-NoBanner] [-NoRebootCheck] [-ProcessOptionalUpdates] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Install` | SwitchParameter | — | — | Named | — | Installer automatiquement les mises à jour Windows disponibles |
| `-AutoReboot` | SwitchParameter | — | — | Named | — | Redémarrer automatiquement si des mises à jour nécessitent un redémarrage |
| `-Criteria` | String | — | — | 0 | `"IsInstalled=0 and IsHidden=0 and AutoSelectOnWebSites=1"` | Critères de recherche personnalisée pour Windows Update |
| `-IncludeDrivers` | SwitchParameter | — | — | Named | — | Inclure les pilotes dans la recherche de mises à jour |
| `-ProcessOptionalUpdates` | SwitchParameter | — | — | Named | — | @{response=Inclure les mises à jour facultatives et inviter à installer chacune d'elles} |
| `-GroupByCategory` | SwitchParameter | — | — | Named | — | Regrouper et colorer la sortie par catégorie de mise à jour |
| `-NoBanner` | SwitchParameter | — | — | Named | — | Désactiver la sortie de bannière/statut |
| `-NoRebootCheck` | SwitchParameter | — | — | Named | — | Ignorer la vérification/notification de l'obligation de redémarrage |

## Examples

### Invoke-WindowsUpdate

```powershell
Invoke-WindowsUpdate
```

Vérifie les mises à jour disponibles pour Windows et winget sans les installer.

### Invoke-WindowsUpdate -Install

```powershell
Invoke-WindowsUpdate -Install
```

Installe automatiquement toutes les mises à jour Windows et winget disponibles.

### updatewindows -Install -AutoReboot

```powershell
updatewindows -Install -AutoReboot
```

Installe toutes les mises à jour et redémarre automatiquement si nécessaire en utilisant l'alias.

### Invoke-WindowsUpdate -GroupByCategory

```powershell
Invoke-WindowsUpdate -GroupByCategory
```

Affiche les mises à jour disponibles regroupées par catégorie pour une meilleure organisation.

### Invoke-WindowsUpdate -Install -ProcessOptionalUpdates

```powershell
Invoke-WindowsUpdate -Install -ProcessOptionalUpdates
```

Installe automatiquement toutes les mises à jour importantes et demande pour chaque mise à jour facultative individuellement, permettant une installation sélective des mises à jour facultatives.

### Invoke-WindowsUpdate -IncludeDrivers -Criteria "IsInstalled=0"

```powershell
Invoke-WindowsUpdate -IncludeDrivers -Criteria "IsInstalled=0"
```

Recherche les mises à jour, y compris les pilotes, avec des critères de recherche personnalisés.
