# Now

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Retourne la date et l'heure actuelles du système sous forme d'un objet DateTime.

## Description

Fournit un moyen simple d'obtenir la date et l'heure système actuelles sans aucun paramètre. Retourne un objet DateTime .NET standard pouvant être utilisé pour les calculs de date/heure, le formatage et les comparaisons.

## Syntax

```powershell
Now [<CommonParameters>]
```

## Examples

### Now Returns the current system date and time as a DateTime object

```powershell
Now
Returns the current system date and time as a DateTime object
```

### $timestamp = Now Stores the current date and time in a variable for later use

```powershell
$timestamp = Now
Stores the current date and time in a variable for later use
```

## Outputs

- `DateTime`
