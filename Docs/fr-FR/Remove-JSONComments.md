# Remove-JSONComments

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Supprime les commentaires du contenu JSON.

## Description

* Traite le contenu JSON et supprime les commentaires sur une seule ligne et
  multilignes tout en préservant la structure JSON.
* Utile pour nettoyer les fichiers JSON contenant des commentaires de documentation
  avant l'analyse.
* Prend en charge les commentaires sur une seule ligne (//) et les commentaires
  multilignes (/* */).

## Syntax

```powershell
Remove-JSONComments -Json <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Json` | String | ✅ | — | 0 | — | L'objet de la requête est l'obtention d'une copie de la base de données.
Le serveur gère les connexions simultanées. |

## Examples

### Example 1

```powershell
$jsonContent = @'
{
    // This is a comment
    "name": "test", /* inline comment */
    "value": 123
}
'@ -split "`n"
Remove-JSONComments -Json $jsonContent
```

Supprime les commentaires d'un contenu JSON stocké dans une variable.

### Example 2

```powershell
$jsonContent | Remove-JSONComments
```

Traite le contenu JSON à partir du pipeline.
