# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Prononce la date actuelle en utilisant la synthèse vocale.

## Description

Convertit la date actuelle dans un format de langage naturel et l'annonce à l'aide du moteur de synthèse vocale du système. La date est prononcée au format : « Nous sommes le [jour de la semaine] [mois] [jour] [année] »

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
