# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...

## Synopsis

> Navigue de deux niveaux vers le haut dans la hiérarchie du système de fichiers.

## Description

* Change le répertoire de travail courant au répertoire grand-parent (deux niveaux au-dessus).
* Affiche le contenu du répertoire résultant.

## Syntax

```powershell
Set-LocationParent2 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Se rend dans le répertoire parent et affiche son contenu.
