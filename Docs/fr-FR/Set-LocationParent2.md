# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> Remonte de deux niveaux de répertoire dans la hiérarchie du système de fichiers.

## Description

* Remplace le répertoire de travail actuel par le répertoire parent du parent
  (deux niveaux vers le haut).
* Affiche le contenu du répertoire résultant.

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Se déplace dans le répertoire parent du répertoire parent et affiche son contenu.

```powershell
...
```

Utilise l'alias '...' pour la même opération.
