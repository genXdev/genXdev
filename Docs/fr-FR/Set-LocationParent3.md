# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....

## Synopsis

> Remonte de trois niveaux de répertoire dans la hiérarchie du système de fichiers.

## Description

* Change le répertoire de travail courant en remontant de trois niveaux
  dans la hiérarchie du système de fichiers.
* Affiche le contenu du répertoire résultant.

## Syntax

```powershell
Set-LocationParent3 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

Remonte trois niveaux de répertoire à partir de l'emplacement actuel.
