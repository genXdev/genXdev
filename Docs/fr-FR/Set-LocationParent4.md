# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....

## Synopsis

> Remonte de quatre niveaux de répertoire dans la hiérarchie du système de fichiers.

## Description

* Remonte de quatre niveaux de répertoires à partir de l'emplacement actuel dans le système de fichiers.
* Après la navigation, affiche le contenu du répertoire résultant.

## Syntax

```powershell
Set-LocationParent4 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Remonte quatre niveaux de répertoire à partir de l'emplacement actuel.

.....
