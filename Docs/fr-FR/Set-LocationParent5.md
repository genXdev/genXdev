# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......

## Synopsis

> Remonte de cinq niveaux dans la hiérarchie du système de fichiers.

## Description

* Change le répertoire de travail courant en remontant de cinq niveaux
  de répertoire parent.
* Affiche le contenu du répertoire résultant.
* Fournit un alias court pratique '......' pour une navigation rapide.

## Syntax

```powershell
Set-LocationParent5 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Remonte de cinq niveaux de répertoire à partir de l'emplacement actuel.

### Example 2

```powershell
......
```

Utilise l'alias pour remonter de cinq niveaux de répertoires.
