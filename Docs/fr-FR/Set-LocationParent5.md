# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> Remonte de cinq niveaux de répertoire dans la hiérarchie du système de fichiers.

## Description

* Modifie le répertoire de travail actuel en remontant de cinq niveaux dans la hiérarchie des répertoires parents.
* Affiche le contenu du répertoire résultant.
* Fournit un alias abrégé pratique '......' pour une navigation rapide.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Remonte de cinq niveaux de répertoire à partir de l'emplacement actuel.

### Example 2

```powershell
......
```

Utilise l'alias pour naviguer cinq niveaux de répertoire vers le haut.
