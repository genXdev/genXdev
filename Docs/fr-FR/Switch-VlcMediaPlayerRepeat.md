# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> Active le mode de répétition dans VLC Media Player.

## Description

Cette fonction envoie la commande de la touche 'r' au lecteur multimédia VLC pour alterner entre les différents modes de répétition (pas de répétition, répéter en cours, répéter tout). La fonction ouvre VLC s'il n'est pas déjà en cours d'exécution et rétablit le focus sur la fenêtre précédente après avoir envoyé la commande.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
