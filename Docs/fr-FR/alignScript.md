# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Retourne une chaîne (avec une indentation modifiée) d'une chaîne de scriptblock fournie.

## Description

Modifie l'indentation d'une chaîne de script tout en respectant les indentations originales du bloc de code

## Syntax

```powershell
alignScript [<CommonParameters>]
```
