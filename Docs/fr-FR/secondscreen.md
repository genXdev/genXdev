# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Définit la configuration par défaut du deuxième moniteur.

## Description

Définit le comportement par défaut pour que les ouvertures de fenêtres GenXdev se fassent sur le moniteur secondaire.
Ceci est utile pour les utilisateurs ayant un seul moniteur ou ceux qui préfèrent les dispositions de fenêtres côte à côte.
Voir également l'applet de commande 'sidebyside' et 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Définit les valeurs par défaut pour les ouvertures de fenêtres GenXdev sur le moniteur secondaire.
