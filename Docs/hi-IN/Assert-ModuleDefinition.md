# Assert-ModuleDefinition

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> AI सहायता का उपयोग करके PowerShell स्रोत कोड फ़ाइलों को रीफ़ैक्टर करने में सहायता करता है।

## Description

यह फ़ंक्शन AI का उपयोग करके PowerShell कोड को रीफैक्टर करने की प्रक्रिया को स्वचालित करता है।
यह प्रॉम्प्ट टेम्पलेट्स का प्रबंधन करता है, सक्रिय IDE (VS Code या Visual Studio) का पता लगाता है,
और कीबोर्ड ऑटोमेशन के माध्यम से रीफैक्टरिंग वर्कफ़्लो को व्यवस्थित करता है।
फ़ंक्शन मॉड्यूल मैनिफ़ेस्ट (.psd1) और मॉड्यूल स्क्रिप्ट (.psm1)
दोनों फ़ाइलों को संभाल सकता है।

## Syntax

```powershell
Assert-ModuleDefinition -ModuleName <String> [-EditPrompt] [-Prompt <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | मॉड्यूल का नाम |
| `-Prompt` | String | — | — | Named | — | एआई प्रॉम्प्ट |
| `-EditPrompt` | SwitchParameter | — | — | Named | — | केवल AI प्रॉम्प्ट को संपादित करने पर स्विच करें |
