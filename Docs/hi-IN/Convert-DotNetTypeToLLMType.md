# Convert-DotNetTypeToLLMType

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> .NET प्रकार नामों को LLM (भाषा मॉडल) प्रकार नामों में परिवर्तित करता है।

## Description

यह .NET प्रकार का नाम इनपुट के रूप में लेता है और भाषा मॉडल में उपयोग किए जाने वाले संबंधित सरलीकृत प्रकार का नाम लौटाता है। यह सामान्य .NET प्रकारों को संभालता है और उपयुक्त प्रकार मैपिंग प्रदान करता है।

## Syntax

```powershell
Convert-DotNetTypeToLLMType -DotNetType <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DotNetType` | String | ✅ | — | 0 | — | LLM प्रकार के नाम में रूपांतरित करने के लिए .NET प्रकार का नाम |

## Examples

### Example 1

```powershell
Convert-DotNetTypeToLLMType -DotNetType "System.String"
```

System.String प्रकार को उसके LLM समकक्ष में रूपांतरित करता है।
