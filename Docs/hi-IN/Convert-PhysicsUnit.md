# Convert-PhysicsUnit

> **Module:** GenXdev.Helpers.Physics | **Type:** Function | **Aliases:** —

## Synopsis

> एक मान को उसी श्रेणी के भीतर एक भौतिक इकाई से दूसरी में परिवर्तित करता है।

## Description

यह फ़ंक्शन लंबाई, समय, गति आदि जैसी श्रेणियों में संगत इकाइयों के बीच भौतिक मात्राओं को परिवर्तित करता है। यह इकाइयों से श्रेणी का अनुमान लगाता है और असंगत रूपांतरणों के लिए त्रुटि देता है।

## Syntax

```powershell
Convert-PhysicsUnit -Value <Double> -FromUnit <String> -ToUnit <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Value` | Double | ✅ | — | 0 | — | कनवर्ट करने के लिए संख्यात्मक मान |
| `-FromUnit` | String | ✅ | — | 1 | — | इनपुट मान की इकाई |
| `-ToUnit` | String | ✅ | — | 2 | — | वांछित आउटपुट यूनिट |

## Examples

### Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"

```powershell
Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"
```

100 मीटर को फीट में बदलता है।

### Convert-PhysicsUnit 10 "seconds" "minutes"

```powershell
Convert-PhysicsUnit 10 "seconds" "minutes"
```

स्थितीय प्राचलों का उपयोग करके 10 सेकंड को मिनटों में परिवर्तित करता है।
