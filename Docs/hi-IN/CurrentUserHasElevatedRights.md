# CurrentUserHasElevatedRights

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> जाँचता है कि वर्तमान उपयोगकर्ता के पास उन्नत अधिकार हैं या नहीं।

## Description

यह निर्धारित करता है कि वर्तमान Windows उपयोगकर्ता के पास उनके सुरक्षा प्रिंसिपल भूमिकाओं की जाँच करके प्रशासनिक या बैकअप ऑपरेटर विशेषाधिकार हैं या नहीं। यदि उपयोगकर्ता के पास उन्नत अधिकार हैं तो true लौटाता है, अन्यथा false। सुरक्षा और पहुँच-संबंधी मुद्दों के लिए मजबूत त्रुटि प्रबंधन लागू करता है।

## Syntax

```powershell
CurrentUserHasElevatedRights [<CommonParameters>]
```

## Examples

### $hasRights = CurrentUserHasElevatedRights ##############################################################################

```powershell
$hasRights = CurrentUserHasElevatedRights
##############################################################################
```

## Outputs

- `Boolean
Returns true if user has admin or backup operator rights, false otherwise.`
- `Boolean`
