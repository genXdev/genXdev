# Enable-Screensaver

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> कॉन्फ़िगर किए गए विंडोज स्क्रीनसेवर को प्रारंभ करता है।

## Description

डिफ़ॉल्ट स्क्रीनसेवर निष्पादन योग्य (scrnsave.scr) को /s स्विच के साथ तुरंत प्रारंभ करके विंडोज सिस्टम स्क्रीनसेवर को सक्रिय करता है।

## Syntax

```powershell
Enable-Screensaver [<CommonParameters>]
```

## Examples

### Enable-Screensaver

```powershell
Enable-Screensaver
```
