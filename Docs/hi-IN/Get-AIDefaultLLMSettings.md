# Get-AIDefaultLLMSettings

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> जेनएक्सडेव.एआई में एआई संचालनों के लिए सभी उपलब्ध डिफ़ॉल्ट एलएलएम सेटिंग्स कॉन्फ़िगरेशन प्राप्त करता है।

## Description

एआई संचालनों के लिए कॉन्फ़िगर किए गए बड़े भाषा मॉडल (LLM) सेटिंग्स का पूरा सेट प्राप्त करता है। परिणामों को क्वेरी प्रकार, मॉडल पहचानकर्ता, API एंडपॉइंट, या API कुंजी द्वारा फ़िल्टर किया जा सकता है। सत्र-आधारित और स्थायी प्राथमिकता-आधारित पुनर्प्राप्ति दोनों का समर्थन करता है, जिसमें सत्र स्थिति साफ़ करने या संग्रहीत प्राथमिकताओं से सीधे पढ़ने के लिए इसे पूरी तरह से दरकिनार करने के विकल्प हैं।

## Syntax

```powershell
Get-AIDefaultLLMSettings [[-LLMQueryType] <String>] [-ApiEndpoint <String>] [-ApiKey <String>] [-ClearSession] [-Model <String>] [-NoSupportForJsonSchema <String>] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```
