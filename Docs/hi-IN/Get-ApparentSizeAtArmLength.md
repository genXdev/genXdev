# Get-ApparentSizeAtArmLength

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> हाथ की दूरी पर किसी वस्तु के स्पष्ट आकार की गणना करता है।

## Description

छोटे कोण सन्निकटन का उपयोग करके स्पष्ट आकार की गणना करता है।

## Syntax

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters <Double> -SizeInMeters <Double> [[-ArmLengthInMeters] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | मीटर में वस्तु की दूरी |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | मीटर में वस्तु का वास्तविक आकार |
| `-ArmLengthInMeters` | Double | — | — | 2 | `0` | मीटर में बांह की लंबाई (डिफ़ॉल्ट: 0.7) |
| `-As` | String | — | — | 3 | — | आउटपुट आकार के लिए इकाई |

## Examples

### Example 1

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters 10 -SizeInMeters 1 -As "centimeters"
```

10 मीटर की दूरी पर 1 मीटर की वस्तु के स्पष्ट आकार की गणना करता है।

### Example 2

```powershell
Get-ApparentSizeAtArmLength 10 1
```

स्थितीय पैरामीटरों का उपयोग करके दृश्य आकार की गणना करता है।
