# Get-ChildProcesses

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> वर्तमान PowerShell प्रक्रिया की सभी वंशज प्रक्रियाओं को पुनर्प्राप्त करता है।

## Description

* सभी चल रही प्रक्रियाओं की जाँच करता है और उन प्रक्रियाओं की पहचान करता है जिनके पैरेंट प्रक्रिया श्रृंखला में वर्तमान PowerShell प्रक्रिया पूर्वज के रूप में है।
* इसमें प्रत्यक्ष चाइल्ड प्रक्रियाएँ और उनके वंशज (पोते-पोतियाँ आदि) दोनों शामिल हैं।

## Syntax

```powershell
Get-ChildProcesses [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ChildProcesses
```

वर्तमान PowerShell सत्र के सभी चाइल्ड प्रोसेस प्राप्त करें।

### Example 2

```powershell
Get-ChildProcesses -Verbose
```

विस्तृत प्रक्रिया पहचान जानकारी दिखाने वाले वर्बोज़ आउटपुट के साथ चाइल्ड प्रक्रियाएँ प्राप्त करें।
