# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> सिस्टम की वर्तमान भाषा सेटिंग्स के आधार पर डिफ़ॉल्ट वेब भाषा कुंजी प्राप्त करता है।

## Description

* वर्तमान सिस्टम भाषा और संस्कृति सेटिंग्स प्राप्त करता है और उन्हें अनुवाद सेवाओं द्वारा उपयोग की जाने वाली संबंधित वेब भाषा शब्दकोश कुंजी से मैप करता है।

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

एक अंग्रेजी सिस्टम के लिए "English", एक डच सिस्टम के लिए "Dutch", आदि लौटाता है।
