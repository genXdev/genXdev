# Get-GenXDevModule

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> निर्दिष्ट पथ से सभी GenXDev मॉड्यूल प्राप्त करता है।

## Description

यह फ़ंक्शन GenXdev मॉड्यूल के लिए निर्देशिका संरचना में खोज करता है,
नाम में '.local' वाले किसी भी मॉड्यूल को छोड़कर। प्रत्येक वैध मॉड्यूल के लिए, यह
सबसे हाल का संस्करण फ़ोल्डर (1.x) लौटाता है जिसमें एक मान्य मॉड्यूल मेनिफ़ेस्ट
(.psd1) फ़ाइल होती है।

## Syntax

```powershell
Get-GenXDevModule [[-Path] <String>] [-IncludeLocal] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String | — | — | 0 | — | GenXdev मॉड्यूल की खोज के लिए मूल पथ |
| `-IncludeLocal` | SwitchParameter | — | — | Named | — | The `-IncludeLocal` parameter. |

## Examples

### Get-GenXDevModule -Path "C:\PowerShell\Modules"

```powershell
Get-GenXDevModule -Path "C:\PowerShell\Modules"
```

### Get-GenXDevModule "C:\PowerShell\Modules"

```powershell
Get-GenXDevModule "C:\PowerShell\Modules"
```
