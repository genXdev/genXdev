# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> यदि टेक्स्ट-टू-स्पीच इंजन बोल रहा है तो सत्य लौटाता है।

## Description

डिफ़ॉल्ट और अनुकूलित दोनों वाक् संश्लेषकों की स्थिति की जाँच करता है कि कोई वर्तमान में बोल रहा है या नहीं। यदि वाक् चल रहा है तो सत्य लौटाता है, अन्यथा असत्य।

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

यदि टेक्स्ट-टू-स्पीच इंजन बोल रहा है तो सत्य लौटाता है।

### Example 2

```powershell
iss
```

यदि टेक्स्ट-टू-स्पीच इंजन बोल रहा है तो सत्य लौटाता है (उपनाम का उपयोग करके)।
