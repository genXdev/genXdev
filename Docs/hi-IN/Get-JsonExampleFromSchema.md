# Get-JsonExampleFromSchema

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> एक JSON स्कीमा परिभाषा से एक मानव-पठनीय उदाहरण JSON स्ट्रिंग उत्पन्न करता है।

## Description

किसी भी JSON स्कीमा (OpenAI json_schema प्रारूप या कच्ची स्कीमा) को एक संक्षिप्त उदाहरण JSON स्ट्रिंग में बदलता है जो LLM सिस्टम निर्देशों में डालने के लिए उपयुक्त है। नेस्टेड ऑब्जेक्ट्स, ऐरे, स्केलर प्रकार, एनम, यूनियन, oneOf/anyOf/allOf रचनाएं, और date-time और email जैसे प्रारूप संकेतों को संभालता है।

यह फ़ंक्शन स्कीमा को पुनरावृत्त रूप से पार करके एक PowerShell ऑब्जेक्ट ट्री बनाता है, फिर ConvertTo-Json के माध्यम से इसे एक संक्षिप्त JSON स्ट्रिंग में क्रमबद्ध करता है।

## Syntax

```powershell
Get-JsonExampleFromSchema [[-Schema] <Object>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Schema` | Object | — | — | 0 | — | एक JSON स्कीमा परिभाषा (JSON स्ट्रिंग या PSCustomObject)। OpenAI json_schema रैपर प्रारूप और रॉ JSON स्कीमा ऑब्जेक्ट का समर्थन करता है। |

## Examples

### $schema = '{ "type": "object", "properties": { "name": { "type": "string" } } }' Get-JsonExampleFromSchema -Schema $schema

```powershell
$schema = '{ "type": "object", "properties": { "name": { "type": "string" } } }'
Get-JsonExampleFromSchema -Schema $schema
```

उत्पन्न करता है: {"name":"..."}

### $openAiSchema = '{ "type": "json_schema", "json_schema": { "name": "response", "strict": true, "schema": { "type": "object", "properties": { "items": { "type": "array", "items": { "type": "object", "properties": { "id": { "type": "integer" }, "label": { "type": "string" } } } } } } } }' Get-JsonExampleFromSchema -Schema $openAiSchema

```powershell
$openAiSchema = '{ "type": "json_schema", "json_schema": { "name": "response",
"strict": true, "schema": { "type": "object", "properties": {
"items": { "type": "array", "items": { "type": "object", "properties": {
"id": { "type": "integer" }, "label": { "type": "string" } } } } } } } }'
Get-JsonExampleFromSchema -Schema $openAiSchema
```

उत्पन्न करता है: {"items":[{"id":0,"label":"..."}]}
