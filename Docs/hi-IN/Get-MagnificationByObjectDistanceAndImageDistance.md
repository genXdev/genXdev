# Get-MagnificationByObjectDistanceAndImageDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> एक लेंस के लिए आवर्धन की गणना करता है।

## Description

m = - (छवि दूरी / वस्तु दूरी) का उपयोग करता है।

## Syntax

```powershell
Get-MagnificationByObjectDistanceAndImageDistance -ObjectDistanceInMeters <Double> -ImageDistanceInMeters <Double> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ObjectDistanceInMeters` | Double | ✅ | — | 0 | `0` | मीटर में वस्तु की दूरी |
| `-ImageDistanceInMeters` | Double | ✅ | — | 1 | `0` | मीटर में छवि की दूरी |

## Examples

### Example 1

```powershell
Get-MagnificationByObjectDistanceAndImageDistance -ObjectDistanceInMeters 0.5 -ImageDistanceInMeters 1
```

0.5m की वस्तु दूरी और 1m की प्रतिबिंब दूरी के साथ आवर्धन की गणना करता है।

### Example 2

```powershell
Get-MagnificationByObjectDistanceAndImageDistance 0.3 0.6
```

स्थितीय प्राचलों का उपयोग करके आवर्धन की गणना करता है।
