# Get-MpCmdRunPath

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> विंडोज डिफेंडर MpCmdRun.exe निष्पादन योग्य का पथ प्राप्त करता है।

## Description

* विंडोज डिफेंडर कमांड-लाइन उपयोगिता (MpCmdRun.exe) का पूरा पथ ढूंढ़ता है और लौटाता है।
* cmdlet प्रोग्राम फ़ाइलों में मानक स्थापना स्थान की जाँच करता है और यदि फ़ाइल नहीं मिलती है तो उपयुक्त त्रुटि प्रबंधन प्रदान करता है।

## Syntax

```powershell
Get-MpCmdRunPath [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$defenderPath = Get-MpCmdRunPath
```

विंडोज डिफ़ेंडर कमांड-लाइन टूल का पूरा पथ प्राप्त करता है।
