# Get-PotentialEnergyByMassHeightAndGravity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> गुरुत्वाकर्षण स्थितिज ऊर्जा की गणना करता है।

## Description

PE = m g h का उपयोग करता है।

## Syntax

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms <Double> -HeightInMeters <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | किलोग्राम में द्रव्यमान |
| `-HeightInMeters` | Double | ✅ | — | 1 | `0` | मीटर में ऊंचाई |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | गुरुत्वाकर्षण m/s² में (डिफ़ॉल्ट: 9.81) |
| `-As` | String | — | — | 3 | — | ऊर्जा के लिए आउटपुट इकाई |

## Examples

### Example 1

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms 10 -HeightInMeters 5 -As "calories"
```

10 किग्रा द्रव्यमान और 5 मीटर ऊंचाई के लिए गुरुत्वीय स्थितिज ऊर्जा की गणना करता है, परिणाम कैलोरी में आउटपुट करता है।

### Example 2

```powershell
Get-PotentialEnergyByMassHeightAndGravity 5 10
```

संभावित ऊर्जा की गणना करने के लिए स्थितीय मापदंडों का उपयोग करने का प्रदर्शन।
