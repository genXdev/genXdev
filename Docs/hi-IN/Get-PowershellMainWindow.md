# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> पावरशेल टर्मिनल की मुख्य विंडो के लिए एक विंडो सहायक वस्तु लौटाता है।

## Description

* वर्तमान PowerShell होस्ट प्रक्रिया की मुख्य विंडो का प्रतिनिधित्व करने वाला WindowObj सहायक ऑब्जेक्ट प्राप्त करता है।
* यह टर्मिनल विंडो में ही हेरफेर और अंतःक्रिया की अनुमति देता है।

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

वर्तमान पावरशेल टर्मिनल विंडो के लिए WindowObj प्राप्त करता है।
