# Get-RefractionAngleByIncidentAngleAndIndices

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> स्नेल के नियम का उपयोग करके अपवर्तन कोण की गणना करता है।

## Description

θ2 = arcsin( (n1 / n2) sin θ1 ) का उपयोग करता है।

## Syntax

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees <Double> -IndexOfRefraction1 <Double> -IndexOfRefraction2 <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-IncidentAngleInDegrees` | Double | ✅ | — | 0 | `0` | डिग्री में आपतन कोण |
| `-IndexOfRefraction1` | Double | ✅ | — | 1 | `0` | प्रथम माध्यम का अपवर्तनांक |
| `-IndexOfRefraction2` | Double | ✅ | — | 2 | `0` | दूसरे माध्यम का अपवर्तनांक |
| `-As` | String | — | — | 3 | — | कोण के लिए आउटपुट इकाई |

## Examples

### Example 1

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees 30 -IndexOfRefraction1 1 -IndexOfRefraction2 1.33 -As "radians"
```

हवा (n=1) से पानी (n=1.33) में 30 डिग्री आपतन कोण पर प्रकाश जाने पर अपवर्तन कोण की गणना करता है।

### Example 2

```powershell
Get-RefractionAngleByIncidentAngleAndIndices 45 1 1.5
```

जब प्रकाश वायु (n=1) से कांच (n=1.5) में 45 डिग्री के कोण पर प्रवेश करता है, तो अपवर्तन कोण की गणना करें।
