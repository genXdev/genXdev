# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Google खोज द्वारा समर्थित सभी भाषाओं के लिए उल्टा शब्दकोश लौटाता है।

## Description

* Google खोज द्वारा समर्थित सभी भाषाओं के लिए भाषा नामों को उनके संबंधित भाषा कोड में मैप करने वाला एक शब्दकोश लौटाता है।
* वेब क्वेरी और अनुवाद कार्यों में भाषा कोड देखने के लिए उपयोगी।

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

भाषा नामों को उनके कोडों में मैप करने वाले शब्दकोश को पुनर्प्राप्त करता है।
