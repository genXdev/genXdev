# Import-GenXdevModules

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** `reloadgenxdev`

## Synopsis

> सभी GenXdev पावरशेल मॉड्यूल को वैश्विक दायरे में आयात करता है।

## Description

जेनएक्सडेव मॉड्यूल के लिए पैरेंट निर्देशिका को स्कैन करता है और प्रत्येक को वैश्विक दायरे में आयात करता है। कार्यशील निर्देशिका को संरक्षित करने के लिए स्थान स्टैक प्रबंधन का उपयोग करता है और सफल और असफल आयातों के लिए दृश्य प्रतिक्रिया प्रदान करता है। आयात प्रक्रिया के दौरान फ़ंक्शन गणना में परिवर्तनों को ट्रैक करता है।

## Syntax

```powershell
Import-GenXdevModules [-DebugFailedModuleDefinitions] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DebugFailedModuleDefinitions` | SwitchParameter | — | — | Named | — | असफल मॉड्यूल परिभाषाओं के लिए डीबग आउटपुट सक्षम करें |

## Examples

### Import-GenXdevModules -DebugFailedModuleDefinitions Imports modules with debug output for failures

```powershell
Import-GenXdevModules -DebugFailedModuleDefinitions
Imports modules with debug output for failures
```

### reloadgenxdev Imports all modules using the alias

```powershell
reloadgenxdev
Imports all modules using the alias
```
