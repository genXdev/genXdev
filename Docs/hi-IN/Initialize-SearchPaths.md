# Initialize-SearchPaths

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> पैकेज प्रबंधन के लिए सिस्टम खोज पथ प्रारंभ और कॉन्फ़िगर करता है।

## Description

* डिफ़ॉल्ट सिस्टम स्थानों, चॉकलेटी पथों, डेवलपमेंट टूल पथों और कस्टम पैकेज पथों को मिलाकर खोज पथों की एक व्यापक सूची तैयार करता है।
* इन समेकित पथों के साथ सिस्टम के PATH पर्यावरण चर को अपडेट करता है।

## Syntax

```powershell
Initialize-SearchPaths [[-WorkspaceFolder] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-WorkspaceFolder` | String | — | — | 0 | — | नोड मॉड्यूल और PowerShell पथों के लिए उपयोग किया जाने वाला कार्यस्थान फ़ोल्डर पथ। |

## Examples

### Example 1

```powershell
Initialize-SearchPaths
```

डिफ़ॉल्ट कार्यक्षेत्र फ़ोल्डर का उपयोग करके खोज पथों को आरंभ करता है।

### Example 2

```powershell
Initialize-SearchPaths -WorkspaceFolder "C:\workspace"
```

एक विशिष्ट कार्यक्षेत्र फ़ोल्डर का उपयोग करके खोज पथ आरंभ करता है।
