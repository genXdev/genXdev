# Invoke-AIPowershellCommand

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `hint`

## Synopsis

> AI सहायता का उपयोग करके PowerShell कमांड उत्पन्न और निष्पादित करता है।

## Description

प्राकृतिक भाषा प्रश्नों के आधार पर PowerShell कमांड उत्पन्न करने के लिए AI मॉडल का उपयोग करता है। यह फ़ंक्शन या तो कमांड को सीधे PowerShell विंडो में भेज सकता है या उन्हें क्लिपबोर्ड पर कॉपी कर सकता है। यह प्राकृतिक भाषा की व्याख्या करने और विभिन्न AI बैकएंड के लिए व्यापक पैरामीटर समर्थन के साथ इसे निष्पादन योग्य PowerShell कमांड में बदलने के लिए AI मॉडल का लाभ उठाता है।

## Syntax

```powershell
Invoke-AIPowershellCommand -PowershellCmdline <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-LLMQueryType <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-PowershellCmdline` | String | ✅ | — | 0 | — | प्राकृतिक भाषा क्वेरी एक कमांड उत्पन्न करने के लिए |
| `-LLMQueryType` | String | — | — | Named | `'Coding'` | LLM क्वेरी का प्रकार |
| `-Model` | String | — | — | Named | — | AI संचालन के लिए उपयोग किए जाने वाले मॉडल पहचानकर्ता या पैटर्न |
| `-ApiEndpoint` | String | — | — | Named | — | AI संचालन के लिए API एंडपॉइंट URL |
| `-ApiKey` | String | — | — | Named | — | प्रमाणित AI संचालनों के लिए API कुंजी |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | इंगित करता है कि LLM के पास JSON स्कीमा के लिए कोई समर्थन नहीं है |

## Examples

### Invoke-AIPowershellCommand -Query "list all running processes"

```powershell
Invoke-AIPowershellCommand -Query "list all running processes"
```

चल रही प्रक्रियाओं की सूची प्राप्त करने के लिए PowerShell कमांड उत्पन्न करता है

### hint "list files modified today"

```powershell
hint "list files modified today"
```

आज संशोधित फ़ाइलों को खोजने के लिए कमांड उत्पन्न करने हेतु उपनाम का उपयोग करता है।

### Invoke-AIPowershellCommand -Query "stop service" -Clipboard

```powershell
Invoke-AIPowershellCommand -Query "stop service" -Clipboard
```

एक सेवा को रोकने का आदेश उत्पन्न करता है और इसे क्लिपबोर्ड पर कॉपी करता है।
