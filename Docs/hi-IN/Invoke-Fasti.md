# Invoke-Fasti

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** `fasti`

## Synopsis

> वर्तमान निर्देशिका में संग्रह फ़ाइलों को उनकी अपनी फ़ोल्डरों में निकालता है और उन्हें बाद में हटा देता है।

## Description

वर्तमान निर्देशिका में पाए जाने वाले सामान्य आर्काइव प्रारूपों (zip, 7z, tar, आदि) को स्वचालित रूप से प्रत्येक आर्काइव के नाम पर अलग फोल्डरों में निकालता है। सफल निष्कर्षण के बाद, मूल आर्काइव फ़ाइलें हटा दी जाती हैं। सिस्टम पर 7-Zip स्थापित होना आवश्यक है।

## Syntax

```powershell
Invoke-Fasti [[-Password] <String>] [-ExtractOutputToo] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Password` | String | — | — | 0 | — | एन्क्रिप्टेड आर्काइव(s) के लिए पासवर्ड दर्ज करें |
| `-ExtractOutputToo` | SwitchParameter | — | — | Named | — | निकाले गए फ़ोल्डरों में पाए गए आर्काइव को पुनरावर्ती रूप से निकालें |

## Examples

### PS C:\Downloads> Invoke-Fasti

```powershell
PS C:\Downloads> Invoke-Fasti
```

### PS C:\Downloads> fasti

```powershell
PS C:\Downloads> fasti
```
