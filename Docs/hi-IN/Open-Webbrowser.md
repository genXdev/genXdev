# Open-Webbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `wb`

## Synopsis

> एक या एकाधिक ब्राउज़र विंडो में वैकल्पिक स्थिति और स्टाइलिंग के साथ URL खोलता है।

## Description

यह फ़ंक्शन ब्राउज़र लॉन्च करने के लिए एक उन्नत रैपर प्रदान करता है जिसमें विंडो पोजिशनिंग, ब्राउज़र चयन और व्यवहार अनुकूलन के व्यापक विकल्प शामिल हैं। यह एज, क्रोम और फ़ायरफ़ॉक्स सहित कई ब्राउज़रों का समर्थन करता है, जिसमें प्राइवेट ब्राउज़िंग, एप्लिकेशन मोड और सटीक विंडो प्रबंधन जैसी सुविधाएँ शामिल हैं।

प्रमुख विशेषताएं:
- स्मार्ट ब्राउज़र डिटेक्शन और चयन
- विंडो पोजिशनिंग (बाएं, दाएं, ऊपर, नीचे, केंद्रित, फुलस्क्रीन)
- स्वचालित या मैन्युअल मॉनिटर चयन के साथ मल्टी-मॉनिटर सपोर्ट
- प्राइवेट/इन्कॉग्निटो ब्राउज़िंग मोड सपोर्ट
- विचलन-मुक्त ब्राउज़िंग के लिए एप्लिकेशन मोड
- एक्सटेंशन और पॉपअप ब्लॉकिंग विकल्प
- फोकस प्रबंधन और विंडो हेरफेर
- कई ब्राउज़रों में बैच URL खोलना
- ब्राउज़र विंडो में कीस्ट्रोक ऑटोमेशन

यह फ़ंक्शन स्वचालित रूप से सिस्टम क्षमताओं का पता लगा सकता है और तदनुसार व्यवहार को समायोजित कर सकता है। सिस्टम पर इंस्टॉल नहीं किए गए ब्राउज़रों के लिए, त्रुटियों के बिना संचालन को चुपचाप छोड़ दिया जाता है।

## Syntax

```powershell
Open-Webbrowser [[-Url] <String[]>] [[-Monitor] <Int32>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-ConsentToThirdPartySoftwareInstallation] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-ForceConsent] [-FullScreen] [-Headless] [-Height <Int32>] [-Input <String>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-PreferPlaywrightBrowser] [-Private] [-RestoreFocus] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### wb -PlayWright https://github.com

```powershell
wb -PlayWright https://github.com
```

Playwright-प्रबंधित क्रोमियम ब्राउज़र में GitHub खोलता है।

### Open-Webbrowser -Url "https://github.com"

```powershell
Open-Webbrowser -Url "https://github.com"
```

GitHub को डिफ़ॉल्ट ब्राउज़र में खोलता है।

### Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left

```powershell
Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left
```

मॉनिटर 1 के बाएं आधे भाग में स्टैक ओवरफ़्लो खोलता है।

### wb "https://google.com" -m 0 -fs

```powershell
wb "https://google.com" -m 0 -fs
```

उपनामों का उपयोग करके प्राथमिक मॉनिटर पर Google को फुलस्क्रीन मोड में खोलता है।

### Open-Webbrowser -Chrome -Private -NewWindow

```powershell
Open-Webbrowser -Chrome -Private -NewWindow
```

गुप्त मोड में एक नया Chrome विंडो खोलता है।

### "https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All

```powershell
"https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All
```

पाइपलाइन के माध्यम से सभी स्थापित ब्राउज़रों में एकाधिक URL खोलता है।

### Open-Webbrowser -Monitor 0 -Right

```powershell
Open-Webbrowser -Monitor 0 -Right
```

पहले से खुली ब्राउज़र विंडो को प्राथमिक मॉनिटर के दाईं ओर पुनः स्थापित करता है।

### Open-Webbrowser -ApplicationMode -Url "https://app.example.com"

```powershell
Open-Webbrowser -ApplicationMode -Url "https://app.example.com"
```

बिना ब्राउज़र नियंत्रण वाले ऐप मोड में एक वेब एप्लिकेशन खोलता है।
