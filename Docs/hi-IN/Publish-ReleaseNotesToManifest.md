# Publish-ReleaseNotesToManifest

> **Module:** GenXdev.Coding | **Type:** Function | **Aliases:** `pubrelnotes`

## Synopsis

> README.md से अपूर्ण रिलीज़ नोट्स को मॉड्यूल मेनिफेस्ट में प्रकाशित करता है।

## Description

README.md फ़ाइल के "## Release notes" अनुभाग को पढ़ता है, सभी अधूरे रिलीज़ नोट्स (☐ से शुरू होने वाली पंक्तियाँ) एकत्र करता है, उन्हें GenXdev.psd1 के ReleaseNotes फ़ील्ड में लिखता है, और README में उन्हें पूर्ण (☐ → ☒) के रूप में चिह्नित करता है। दोनों फ़ाइलों को एक संचालन में परमाणु रूप से अद्यतन किया जाता है।

## Syntax

```powershell
Publish-ReleaseNotesToManifest [[-ManifestPath] <String>] [-UseHomeREADME] [-UseOneDriveREADME] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-UseHomeREADME` | SwitchParameter | — | — | Named | — | पावरशेल प्रोफ़ाइल निर्देशिका में README का उपयोग करें |
| `-UseOneDriveREADME` | SwitchParameter | — | — | Named | — | OneDrive निर्देशिका में README का उपयोग करें |
| `-ManifestPath` | String | — | — | 0 | — | .psd1 मॉड्यूल मेनिफेस्ट का पथ |

## Examples

### Publish-ReleaseNotesToManifest

```powershell
Publish-ReleaseNotesToManifest
```

### Publish-ReleaseNotesToManifest -ManifestPath ".\MyModule.psd1"

```powershell
Publish-ReleaseNotesToManifest -ManifestPath ".\MyModule.psd1"
```
