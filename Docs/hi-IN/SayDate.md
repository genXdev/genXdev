# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> टेक्स्ट-टू-स्पीच संश्लेषण का उपयोग करके वर्तमान तिथि बोलता है।

## Description

वर्तमान तिथि को प्राकृतिक भाषा प्रारूप में परिवर्तित करता है और सिस्टम के टेक्स्ट-टू-स्पीच इंजन का उपयोग करके इसकी घोषणा करता है। तिथि इस प्रारूप में बोली जाती है: "यह [सप्ताह का दिन], [महीना] [दिन] [वर्ष] है"

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
