# Set-AIKnownFacesRootpath

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> GenXdev.AI संचालन में उपयोग होने वाली चेहरे की छवि फ़ाइलों के लिए निर्देशिका सेट करता है।

## Description

यह फ़ंक्शन GenXdev.AI मॉड्यूल द्वारा विभिन्न चेहरा पहचान और AI संचालनों के लिए उपयोग किए जाने वाले वैश्विक चेहरा निर्देशिका को कॉन्फ़िगर करता है। सेटिंग्स को स्थायी रूप से प्राथमिकताओं (डिफ़ॉल्ट) में, केवल वर्तमान सत्र में (-SessionOnly का उपयोग करके) संग्रहीत किया जा सकता है, या सत्र से साफ़ किया जा सकता है (-ClearSession का उपयोग करके)।

## Syntax

```powershell
Set-AIKnownFacesRootpath [[-FacesDirectory] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FacesDirectory` | String | — | — | 0 | — | चेहरे की छवि फ़ाइलों के लिए निर्देशिका पथ |
| `-PreferencesDatabasePath` | String | — | — | Named | — | प्राथमिकता डेटा फ़ाइलों के लिए डेटाबेस पथ |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | भाषा, छवि संग्रह आदि जैसे AI वरीयताओं के लिए सत्र में संग्रहीत वैकल्पिक सेटिंग्स का उपयोग करें |
| `-ClearSession` | SwitchParameter | — | — | Named | — | भाषा, छवि संग्रह आदि जैसी AI प्राथमिकताओं के लिए सत्र में संग्रहीत वैकल्पिक सेटिंग्स साफ़ करें |
| `-SkipSession` | SwitchParameter | — | — | Named | — | एआई प्राथमिकताओं जैसे भाषा, इमेज कलेक्शन आदि के लिए सत्र में संग्रहीत वैकल्पिक सेटिंग्स का उपयोग न करें |

## Examples

### Set-AIKnownFacesRootpath -FacesDirectory "C:\Faces"

```powershell
Set-AIKnownFacesRootpath -FacesDirectory "C:\Faces"
```

प्राथमिकताओं में स्थायी रूप से फ़ेस निर्देशिका सेट करता है।

### Set-AIKnownFacesRootpath "C:\FacePictures"

```powershell
Set-AIKnownFacesRootpath "C:\FacePictures"
```

प्राथमिकताओं में स्थायी रूप से फ़ेस निर्देशिका सेट करता है।

### Set-AIKnownFacesRootpath -FacesDirectory "C:\TempFaces" -SessionOnly

```powershell
Set-AIKnownFacesRootpath -FacesDirectory "C:\TempFaces" -SessionOnly
```

केवल वर्तमान सत्र (ग्लोबल वैरिएबल) के लिए फेस डायरेक्टरी सेट करता है।

### Set-AIKnownFacesRootpath -ClearSession

```powershell
Set-AIKnownFacesRootpath -ClearSession
```

सत्र फ़ेस डायरेक्टरी सेटिंग (ग्लोबल वेरिएबल) को स्थायी प्राथमिकताओं पर प्रभाव डाले बिना साफ़ करता है।
