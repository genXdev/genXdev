# Set-AILLMSettings

> **Module:** GenXdev.AI | **Type:** Function | **Aliases:** —

## Synopsis

> GenXdev.AI में AI संचालन के लिए LLM सेटिंग्स सेट करता है।

## Description

यह फ़ंक्शन GenXdev.AI मॉड्यूल द्वारा विभिन्न AI संचालनों के लिए उपयोग की जाने वाली LLM (बड़ी भाषा मॉडल) सेटिंग्स को प्रबंधित करता है। सेटिंग्स को प्राथमिकताओं में स्थायी रूप से (डिफ़ॉल्ट), केवल वर्तमान सत्र में (-SessionOnly का उपयोग करके), या सत्र से साफ़ किया जा सकता है (-ClearSession का उपयोग करके)। फ़ंक्शन यह सुनिश्चित करता है कि सत्र सेटिंग्स साफ़ करने के अलावा कम से कम एक सेटिंग पैरामीटर प्रदान किया गया हो।

## Syntax

```powershell
Set-AILLMSettings -LLMQueryType <String> [[-Model] <String>] [[-ApiEndpoint] <String>] [[-ApiKey] <String>] [-ClearSession] [-NoSupportForImageUpload] [-NoSupportForJsonSchema] [-NoSupportForToolCalls] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Examples

### Set-AILLMSettings -LLMQueryType "Coding" -Model "*Qwen*14B*"

```powershell
Set-AILLMSettings -LLMQueryType "Coding" -Model "*Qwen*14B*"
```

कोडिंग क्वेरी प्रकार के लिए LLM सेटिंग्स को प्राथमिकताओं में स्थायी रूप से सेट करता है।

### Set-AILLMSettings -LLMQueryType "SimpleIntelligence" -Model "maziyarpanahi/llama-3-groq-8b-tool-use" -SessionOnly

```powershell
Set-AILLMSettings -LLMQueryType "SimpleIntelligence" -Model "maziyarpanahi/llama-3-groq-8b-tool-use" -SessionOnly
```

SimpleIntelligence के लिए LLM सेटिंग्स को केवल वर्तमान सत्र के लिए सेट करता है।

### Set-AILLMSettings -LLMQueryType "Pictures" -ClearSession

```powershell
Set-AILLMSettings -LLMQueryType "Pictures" -ClearSession
```

चित्र क्वेरी प्रकार के लिए सत्र LLM सेटिंग्स को स्थायी प्राथमिकताओं को प्रभावित किए बिना साफ़ करता है।

### Set-AILLMSettings "Coding" "*Qwen*14B*"

```powershell
Set-AILLMSettings "Coding" "*Qwen*14B*"
```

स्थितीय प्राचलों का उपयोग करके कोडिंग क्वेरी प्रकार के लिए LLM सेटिंग्स सेट करता है।
