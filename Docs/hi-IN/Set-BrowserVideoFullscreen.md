# Set-BrowserVideoFullscreen

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `fsvideo`

## Synopsis

> वर्तमान ब्राउज़र टैब में मिले पहले वीडियो तत्व को अधिकतम करता है।

## Description

वर्तमान वेबपेज पर पहले वीडियो तत्व को खोजने और अधिकतम करने के लिए जावास्क्रिप्ट कोड निष्पादित करता है। दृश्यता सुनिश्चित करने के लिए वीडियो को अधिकतम z-इंडेक्स के साथ पूरे व्यूपोर्ट को कवर करने के लिए सेट किया गया है। एक साफ पूर्ण स्क्रीन अनुभव के लिए पेज स्क्रॉलबार छिपाए जाते हैं।

## Syntax

```powershell
Set-BrowserVideoFullscreen [-Chrome] [-Chromium] [-Edge] [-Firefox] [-Webkit] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Edge` | SwitchParameter | — | — | Named | — | माइक्रोसॉफ्ट एज ब्राउज़र का उपयोग करें |
| `-Chrome` | SwitchParameter | — | — | Named | — | Google Chrome ब्राउज़र का उपयोग करें |
| `-Chromium` | SwitchParameter | — | — | Named | — | माइक्रोसॉफ्ट एज या गूगल क्रोम का उपयोग करें, यह इस पर निर्भर करता है कि डिफ़ॉल्ट ब्राउज़र कौन सा है |
| `-Firefox` | SwitchParameter | — | — | Named | — | फ़ायरफ़ॉक्स ब्राउज़र का उपयोग करें |
| `-Webkit` | SwitchParameter | — | — | Named | — | Playwright-प्रबंधित WebKit ब्राउज़र का उपयोग करें |

## Examples

### Set-BrowserVideoFullscreen

```powershell
Set-BrowserVideoFullscreen
```
