# Set-ForegroundWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> निर्दिष्ट विंडो को अग्रभूमि में लाता है और इसे सक्रिय विंडो बनाता है।

## Description

* विंडो को अग्रभूमि विंडो बनाने के लिए एकाधिक Win32 API विधियों का उपयोग करता है अधिकतम विश्वसनीयता के लिए।
* पहले SwitchToThisWindow API का उपयोग करके प्रयास करता है, फिर आवश्यकता पड़ने पर SetForegroundWindow पर वापस आ जाता है।

## Syntax

```powershell
Set-ForegroundWindow -WindowHandle <IntPtr> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-WindowHandle` | IntPtr | ✅ | — | 0 | `0` | अग्रभूमि विंडो के रूप में सेट करने के लिए विंडो हैंडल |

## Examples

### Example 1

```powershell
$hwnd = (Get-Process notepad).MainWindowHandle
Set-ForegroundWindow -WindowHandle $hwnd
```

विंडो हैंडल का उपयोग करके नोटपैड विंडो को अग्रभूमि में लाता है।

### Example 2

```powershell
$hwnd = (Get-Process notepad).MainWindowHandle
Set-ForegroundWindow $hwnd
```

संक्षिप्त वाक्यविन्यास के लिए स्थितीय पैरामीटर का उपयोग करता है।
