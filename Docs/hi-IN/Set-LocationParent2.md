# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> फाइल सिस्टम पदानुक्रम में दो निर्देशिका स्तर ऊपर जाता है।

## Description

* वर्तमान कार्य निर्देशिका को दादा-दादी निर्देशिका (दो स्तर ऊपर) में बदलता है।
* परिणामी निर्देशिका की सामग्री प्रदर्शित करता है।

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

मूल निर्देशिका में बदलता है और उसकी सामग्री प्रदर्शित करता है।

```powershell
...
```
