# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> फ़ाइल सिस्टम पदानुक्रम में चार निर्देशिका स्तर ऊपर नेविगेट करता है।

## Description

* फ़ाइल सिस्टम में वर्तमान स्थान से चार निर्देशिका स्तर ऊपर जाता है।
* नेविगेशन के बाद, परिणामी निर्देशिका की सामग्री प्रदर्शित करता है।

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

वर्तमान स्थान से चार निर्देशिका स्तर ऊपर जाता है।

### Example 2

```powershell
.....
```

निर्देशिका के चार स्तर ऊपर जाने के लिए उपनाम का उपयोग करता है।
