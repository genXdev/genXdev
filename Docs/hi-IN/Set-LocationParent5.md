# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> फ़ाइल सिस्टम पदानुक्रम में पांच निर्देशिका स्तर ऊपर नेविगेट करता है।

## Description

* पांच जनक निर्देशिका स्तरों पर ऊपर जाकर वर्तमान कार्यशील निर्देशिका को बदलता है।
* परिणामी निर्देशिका की सामग्री प्रदर्शित करता है।
* त्वरित नेविगेशन के लिए एक सुविधाजनक शॉर्टकड उपनाम '......' प्रदान करता है।

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

वर्तमान स्थान से पाँच निर्देशिका स्तर ऊपर जाता है।

```powershell
......
```
