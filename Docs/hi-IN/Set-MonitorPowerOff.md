# Set-MonitorPowerOff

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `poweroff`

## Synopsis

> सभी जुड़े मॉनिटरों की बिजली बंद कर देता है।

## Description

सभी जुड़े मॉनिटरों को पावर-ऑफ सिग्नल भेजने के लिए Windows API कॉल का उपयोग करता है। यह आपके मॉनिटरों पर भौतिक पावर बटन दबाने के बराबर है। मॉनिटर पावर सेविंग मोड में चले जाएंगे लेकिन माउस हिलाने या कोई कुंजी दबाने पर जाग सकते हैं।

## Syntax

```powershell
Set-MonitorPowerOff [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOff
```

सभी कनेक्टेड मॉनिटर बंद कर देता है।

### Example 2

```powershell
poweroff
```

उपनाम का उपयोग करके सभी कनेक्टेड मॉनिटर बंद कर देता है।
