# Set-MonitorPowerOn

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `wakemonitor`, `monitoroff`

## Synopsis

> मॉनिटर की शक्ति चालू करता है।

## Description

GenXdev.Helpers.WindowObj के माध्यम से विंडोज API का उपयोग करके मॉनिटर को स्लीप/पावर ऑफ स्थिति से जगाता है। यह ऑटोमेशन स्क्रिप्ट के लिए उपयोगी है जिन्हें यह सुनिश्चित करने की आवश्यकता होती है कि मॉनिटर चालू है।

## Syntax

```powershell
Set-MonitorPowerOn [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOn
```

मॉनिटर की शक्ति चालू करता है।

### Example 2

```powershell
wake-monitor
```

एक उपनाम का उपयोग करके मॉनिटर की बिजली चालू करता है।
