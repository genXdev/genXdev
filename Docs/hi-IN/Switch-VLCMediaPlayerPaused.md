# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> VLC मीडिया प्लेयर के पॉज़/प्ले स्थिति को टॉगल करता है।

## Description

यह फ़ंक्शन VLC मीडिया प्लेयर में रुकने और चलने की स्थिति के बीच टॉगल करने के लिए एक स्पेस कुंजी भेजता है। यह कुंजी कमांड भेजने के बाद स्वचालित रूप से मूल विंडो को फ़ोकस पुनर्स्थापित करता है।

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

VLC मीडिया प्लेयर के पॉज़/प्ले स्थिति को टॉगल करता है।

### vlcpause

```powershell
vlcpause
```

रोक/चलाने की स्थिति को टॉगल करने के लिए उपनाम का उपयोग करता है।

### vlcplay

```powershell
vlcplay
```

विराम/चलन स्थिति को टॉगल करने के लिए वैकल्पिक उपनाम का उपयोग करता है।
