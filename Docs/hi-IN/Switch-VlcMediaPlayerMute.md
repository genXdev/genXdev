# Switch-VlcMediaPlayerMute

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcmute`, `vlcunmute`

## Synopsis

> VLC मीडिया प्लेयर की म्यूट स्थिति को टॉगल करता है।

## Description

यह फ़ंक्शन VLC मीडिया प्लेयर को म्यूट और अनम्यूट ऑडियो स्थितियों के बीच टॉगल करने के लिए 'm' कुंजी भेजता है। फ़ंक्शन VLC पर फ़ोकस करता है, म्यूट/अनम्यूट कमांड भेजता है, और फिर पहले से सक्रिय विंडो पर फ़ोकस पुनर्स्थापित करता है।

## Syntax

```powershell
Switch-VlcMediaPlayerMute [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerMute

```powershell
Switch-VlcMediaPlayerMute
```

पूर्ण फ़ंक्शन नाम का उपयोग करके VLC मीडिया प्लेयर के म्यूट स्थिति को टॉगल करें।

### vlcmute

```powershell
vlcmute
```

संक्षिप्त उपनाम का उपयोग करके VLC मीडिया प्लेयर की म्यूट स्थिति को टॉगल करता है।
