# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> VLC मीडिया प्लेयर में पुनरावृत्ति मोड को टॉगल करता है।

## Description

यह फ़ंक्शन VLC मीडिया प्लेयर को विभिन्न रिपीट मोड (कोई रिपीट नहीं, वर्तमान रिपीट, सभी रिपीट) के बीच टॉगल करने के लिए 'r' कुंजी कमांड भेजता है। यह फ़ंक्शन यदि VLC पहले से नहीं चल रहा है तो उसे खोलता है और कमांड भेजने के बाद पिछली विंडो पर फोकस बहाल करता है।

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
