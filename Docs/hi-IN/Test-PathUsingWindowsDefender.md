# Test-PathUsingWindowsDefender

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `virusscan`, `HasNoVirus`

## Synopsis

> Windows Defender का उपयोग करके मैलवेयर के लिए फ़ाइलों या निर्देशिकाओं को स्कैन करता है।

## Description

* विंडोज डिफेंडर के कमांड-लाइन इंटरफेस (MpCmdRun.exe) का उपयोग करके निर्दिष्ट फ़ाइलों या निर्देशिकाओं का लक्षित स्कैन करता है।
* केवल डिटेक्शन मोड में या स्वचालित थ्रेट रेमेडिएशन सक्षम के साथ स्कैन कर सकता है।
* यदि कोई खतरा नहीं पाया जाता है तो true लौटाता है, यदि खतरे पाए जाते हैं या स्कैन विफल होता है तो false लौटाता है।

## Syntax

```powershell
Test-PathUsingWindowsDefender -FilePath <String> [-EnableRemediation] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | स्कैन किए जाने वाले फ़ाइल या निर्देशिका का पथ |
| `-EnableRemediation` | SwitchParameter | — | — | Named | `False` | विंडोज डिफेंडर को खतरों पर कार्रवाई करने का निर्देश देता है |

## Examples

### Example 1

```powershell
Test-PathUsingWindowsDefender -FilePath "C:\Downloads\file.exe" -Verbose
```

निर्दिष्ट फ़ाइल को स्कैन करता है और विस्तृत आउटपुट प्रदर्शित करता है।

### Example 2

```powershell
virusscan "C:\Downloads\file.exe" -EnableRemediation
```

उपनाम का उपयोग करके फ़ाइल को स्कैन करता है और सुधार सक्षम करता है।

### Example 3

```powershell
"C:\Downloads\file.exe" | HasNoVirus
```

एक उपनाम का उपयोग करके फ़ाइल पथ को cmdlet में पाइप करता है।
