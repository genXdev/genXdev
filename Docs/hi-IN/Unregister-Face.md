# Unregister-Face

> **Module:** GenXdev.AI.DeepStack | **Type:** Function | **Aliases:** —

## Synopsis

> डीपस्टैक से अपने पहचानकर्ता द्वारा एक पंजीकृत चेहरे को हटाता है।

## Description

यह फ़ंक्शन अपने अद्वितीय पहचानकर्ता का उपयोग करके DeepStack चेहरा पहचान प्रणाली से एक चेहरा हटाता है। यह पंजीकृत चेहरे के डेटा को सिस्टम से स्थायी रूप से हटाने के लिए API समापन बिंदु के साथ संचार करता है।

## Syntax

```powershell
Unregister-Face -Identifier <String> [[-ContainerName] <String>] [[-VolumeName] <String>] [[-ServicePort] <Int32>] [[-HealthCheckTimeout] <Int32>] [[-HealthCheckInterval] <Int32>] [[-ImageName] <String>] [-Force] [-NoDockerInitialize] [-ShowWindow] [-UseGPU] [<CommonParameters>]
```

| Name | Type | Required | Pipeline | Position | Default | Description |

| `-ContainerName` | String | — | — | 1 | `'deepstack_face_recognition'` | डॉकर कंटेनर का नाम |

| `-ServicePort` | Int32 | — | — | 3 | `5000` | डीपस्टैक सेवा के लिए पोर्ट नंबर |
| `-HealthCheckTimeout` | Int32 | — | — | 4 | `60` | सेवा स्वास्थ्य जांच के लिए प्रतीक्षा करने का अधिकतम समय सेकंड में |
| `-HealthCheckInterval` | Int32 | — | — | 5 | `3` | स्वास्थ्य जांच प्रयासों के बीच सेकंड में अंतराल |

| `-ShowWindow` | SwitchParameter | — | — | Named | — | आरंभीकरण के दौरान Docker Desktop विंडो दिखाएं |
