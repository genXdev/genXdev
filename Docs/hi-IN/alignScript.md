# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> प्रदान किए गए स्क्रिप्टब्लॉक स्ट्रिंग का एक स्ट्रिंग (बदले गए इंडेंटेशन के साथ) लौटाता है

## Description

एक स्क्रिप्टब्लॉक स्ट्रिंग के इंडेंटेशन को बदलता है, मूल कोड-ब्लॉक इंडेंटेशन का सम्मान करते हुए

## Syntax

```powershell
alignScript [<CommonParameters>]
```
