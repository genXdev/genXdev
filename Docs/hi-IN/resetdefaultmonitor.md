# resetdefaultmonitor

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> डिफ़ॉल्ट द्वितीयक मॉनिटर कॉन्फ़िगरेशन को पुनर्स्थापित करता है।

## Description

यह स्क्रिप्ट सिस्टम के लिए डिफ़ॉल्ट द्वितीयक मॉनिटर कॉन्फ़िगरेशन को पुनर्स्थापित करता है,
द्वितीयक मॉनिटर को मूल डिफ़ॉल्ट मान पर सेट करता है।
यह उन उपयोगकर्ताओं के लिए उपयोगी है जो साइड-बाय-साइड कॉन्फ़िगरेशन का उपयोग करने के बाद अपने पिछले मल्टी-मॉनिटर सेटअप पर वापस जाना चाहते हैं।
यह भी देखें: नई विंडो के लिए साइड-बाय-साइड मोड पर स्विच करने के लिए 'sidebyside' फ़ंक्शन।

## Syntax

```powershell
resetdefaultmonitor [<CommonParameters>]
```

## Examples

### secondscreen

```powershell
secondscreen
```

सिस्टम के लिए डिफ़ॉल्ट द्वितीयक मॉनिटर कॉन्फ़िगरेशन को पुनर्स्थापित करता है।
