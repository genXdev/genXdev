# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> डिफ़ॉल्ट दूसरे मॉनिटर कॉन्फ़िगरेशन सेट करता है।

## Description

GenXdev विंडो खोलने के लिए द्वितीयक मॉनिटर पर डिफ़ॉल्ट व्यवहार सेट करता है।
यह एकल मॉनिटर वाले उपयोगकर्ताओं या साथ-साथ विंडो लेआउट पसंद करने वालों के लिए उपयोगी है।
सहायक कमांडलेट 'sidebyside' और 'restoredefaultmonitor' भी देखें।

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```
