# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> डिफ़ॉल्ट साइड-बाय-साइड कॉन्फ़िगरेशन सेट करता है।

## Description

GenXdev विंडो खोलने के लिए डिफ़ॉल्ट व्यवहार को PowerShell के साथ साइड-बाय-साइड सेट करता है।
यह एक मॉनिटर वाले उपयोगकर्ताओं या उन लोगों के लिए उपयोगी है जो साइड-बाय-साइड विंडो लेआउट पसंद करते हैं।
cmdlet 'secondscreen' और 'restoredefaultmonitor' भी देखें

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

GenXdev विंडो खोलने के लिए PowerShell के साथ एक-दूसरे के बगल में डिफ़ॉल्ट सेट करता है
