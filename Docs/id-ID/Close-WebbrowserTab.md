# Close-WebbrowserTab

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `ct`, `CloseTab`

## Synopsis

> Menutup tab peramban web yang sedang dipilih.

## Description

Menutup tab peramban web yang saat ini dipilih menggunakan metode CloseAsync() dari ChromeDriver. Jika tidak ada tab yang dipilih, fungsi akan secara otomatis mencoba memilih tab terakhir yang digunakan sebelum menutupnya.

## Syntax

```powershell
Close-WebbrowserTab [-Chrome] [-Chromium] [-Edge] [-Firefox] [-Webkit] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Edge` | SwitchParameter | — | — | Named | — | Navigasi menggunakan browser Microsoft Edge |
| `-Chrome` | SwitchParameter | — | — | Named | — | Navigasi menggunakan browser Google Chrome |
| `-Chromium` | SwitchParameter | — | — | Named | — | Dibuka di Microsoft Edge atau Google Chrome, tergantung pada peramban bawaan apa yang digunakan |
| `-Firefox` | SwitchParameter | — | — | Named | — | Membuka di Firefox |
| `-Webkit` | SwitchParameter | — | — | Named | — | Menggunakan browser WebKit yang dikelola oleh Playwright |

## Examples

### Close-WebbrowserTab Closes the currently active browser tab

```powershell
Close-WebbrowserTab
Closes the currently active browser tab
```

### ct Uses the alias to close the currently active browser tab

```powershell
ct
Uses the alias to close the currently active browser tab
```
