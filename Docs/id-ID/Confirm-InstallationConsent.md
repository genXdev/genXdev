# Confirm-InstallationConsent

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Mengonfirmasi persetujuan pengguna untuk menginstal perangkat lunak pihak ketiga, menggunakan preferensi untuk pilihan yang persisten.

## Description

<details>
<summary><b>Expand description</b></summary>

Fungsi ini bertindak sebagai mekanisme ShouldProcess khusus untuk instalasi perangkat lunak. Fungsi ini memeriksa preferensi pengguna untuk menentukan apakah instalasi otomatis diizinkan untuk aplikasi yang ditentukan. Jika tidak ada preferensi yang ditetapkan, fungsi ini akan meminta pengguna dengan penjelasan yang jelas tentang apa yang akan diinstal, sumbernya, potensi risiko, dan opsi untuk mengizinkan atau menolak instalasi (dengan opsi untuk pilihan satu kali atau permanen).

Hal ini memastikan persetujuan eksplisit pengguna sebelum melanjutkan instalasi apa pun, membantu mengurangi potensi risiko hukum dengan memerlukan tindakan afirmatif dari pengguna. Prompt dengan jelas menyatakan bahwa pembuat modul (GenXdev) tidak bertanggung jawab atas perangkat lunak pihak ketiga, dan bahwa pengguna menyetujui instalasi dengan risiko mereka sendiri.

Preferensi disimpan dalam file JSON, memungkinkan pengguna untuk menetapkan "selalu izinkan" untuk aplikasi tertentu atau mengaktifkan persetujuan global untuk semua instalasi pihak ketiga GenXdev, sehingga nyaman namun tetap sah secara hukum.

Jika izin ditolak (atau preferensi diatur untuk menolak), fungsi mengembalikan $false dan tidak melanjutkan instalasi. Jika diizinkan, fungsi mengembalikan $true.

</details>
## Syntax

```powershell
Confirm-InstallationConsent -ApplicationName <String> -Source <String> [-ConsentToThirdPartySoftwareInstallation] [-Description <String>] [-ForceConsent] [-Publisher <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ApplicationName` | String | ✅ | — | 0 | — | Nama aplikasi atau perangkat lunak yang sedang diinstal. |
| `-Source` | String | ✅ | — | 1 | — | Sumber instalasi (misalnya, Winget, PowerShell Gallery). |
| `-Description` | String | — | — | Named | `"This software is required for certain features in the GenXdev modules."` | Deskripsi opsional perangkat lunak dan tujuannya. |
| `-Publisher` | String | — | — | Named | `"Third-party vendor"` | Penerbit atau vendor opsional dari perangkat lunak. |
| `-ForceConsent` | SwitchParameter | — | — | Named | — | Paksa prompt meskipun preferensi telah diatur. |
| `-ConsentToThirdPartySoftwareInstallation` | SwitchParameter | — | — | Named | — | Secara otomatis menyetujui instalasi perangkat lunak pihak ketiga dan mengatur bendera persisten. |

## Examples

### if (Confirm-InstallationConsent -ApplicationName "Docker Desktop" `         -Source "Winget") {     # Proceed with installation     Microsoft.WinGet.Client\Install-WinGetPackage `         -Id "Docker.DockerDesktop" }

```powershell
if (Confirm-InstallationConsent -ApplicationName "Docker Desktop" `
        -Source "Winget") {
    # Proceed with installation
    Microsoft.WinGet.Client\Install-WinGetPackage `
        -Id "Docker.DockerDesktop"
}
```

Ini memeriksa persetujuan sebelum menginstal Docker Desktop melalui Winget.

### Confirm-InstallationConsent -ApplicationName "Pester" `     -Source "PowerShell Gallery" -Publisher "Pester Team" `     -Description "Required for unit testing in PowerShell modules."

```powershell
Confirm-InstallationConsent -ApplicationName "Pester" `
    -Source "PowerShell Gallery" -Publisher "Pester Team" `
    -Description "Required for unit testing in PowerShell modules."
```

Peringatan dengan informasi detail sebelum menginstal modul Pester.
