# Convert-DotNetTypeToLLMType

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Mengonversi nama tipe .NET menjadi nama tipe LLM (Language Model).

## Description

Mengambil nama tipe .NET sebagai input dan mengembalikan nama tipe sederhana yang sesuai yang digunakan dalam Model Bahasa. Ini menangani tipe .NET umum dan menyediakan pemetaan tipe yang sesuai.

## Syntax

```powershell
Convert-DotNetTypeToLLMType -DotNetType <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DotNetType` | String | ✅ | — | 0 | — | Nama tipe .NET untuk dikonversi menjadi nama tipe LLM |

## Examples

### Example 1

```powershell
Convert-DotNetTypeToLLMType -DotNetType "System.String"
```

Mengonversi tipe System.String ke padanan LLM-nya.
