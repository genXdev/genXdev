# ConvertFrom-DiplomaticSpeak

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `undiplomatize`

## Synopsis

> Mengubah bahasa diplomatis atau taktis menjadi bahasa yang langsung, jelas, dan terus terang.

## Description

Fungsi ini mengambil bahasa diplomatik dan menerjemahkannya untuk mengungkapkan makna sebenarnya di balik bahasa yang sopan atau benar secara politis. Fungsi ini menggunakan model bahasa AI untuk mengubah ekspresi eufemistik menjadi pernyataan langsung, membuat komunikasi menjadi tidak ambigu dan mudah dipahami. Fungsi ini sangat berguna untuk menganalisis pernyataan politik, komunikasi bisnis, atau teks apa pun di mana makna sebenarnya mungkin tersembunyi oleh bahasa diplomatik.
