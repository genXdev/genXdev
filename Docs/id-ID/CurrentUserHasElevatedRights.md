# CurrentUserHasElevatedRights

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Memeriksa apakah pengguna saat ini memiliki hak yang ditinggikan.

## Description

Menentukan apakah pengguna Windows saat ini memiliki hak administratif atau operator cadangan dengan memeriksa peran prinsipal keamanan mereka. Mengembalikan true jika pengguna memiliki hak yang ditingkatkan, false jika tidak. Menerapkan penanganan kesalahan yang kuat untuk masalah keamanan dan akses.

## Syntax

```powershell
CurrentUserHasElevatedRights [<CommonParameters>]
```

## Examples

### $hasRights = CurrentUserHasElevatedRights ##############################################################################

```powershell
$hasRights = CurrentUserHasElevatedRights
##############################################################################
```

## Outputs

- `Boolean
Returns true if user has admin or backup operator rights, false otherwise.`
- `Boolean`
