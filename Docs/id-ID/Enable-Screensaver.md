# Enable-Screensaver

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Memulai screensaver Windows yang telah dikonfigurasi.

## Description

Mengaktifkan screensaver sistem Windows dengan menjalankan executable screensaver bawaan (scrnsave.scr) dengan sakelar /s untuk memulainya secara langsung.

## Syntax

```powershell
Enable-Screensaver [<CommonParameters>]
```

## Examples

### Enable-Screensaver

```powershell
Enable-Screensaver
```
