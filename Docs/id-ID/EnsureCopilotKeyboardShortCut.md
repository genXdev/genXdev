# EnsureCopilotKeyboardShortCut

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Mengonfigurasi pintasan keyboard GitHub Copilot Chat di Visual Studio Code.

## Description

Fungsi ini memastikan bahwa fitur lampiran file GitHub Copilot Chat memiliki pintasan keyboard yang tepat (Ctrl+Shift+Alt+F12) yang dikonfigurasi di Visual Studio Code.

Ini akan menghapus pintasan lampiran Copilot yang ada dan menggantinya dengan perintah yang benar saat ini (github.copilot.chat.attachFile).

Juga menambahkan pintasan Alt+` (backtick) untuk mengalihkan panel yang dimaksimalkan.

## Syntax

```powershell
EnsureCopilotKeyboardShortCut [<CommonParameters>]
```

## Examples

### EnsureCopilotKeyboardShortCut

```powershell
EnsureCopilotKeyboardShortCut
```
