# EnsureNuGetAssembly

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** ‚Äî

## Synopsis

> Mengunduh dan memuat rakitan .NET dari paket NuGet berdasarkan kunci atau ID paket.

## Description

Fungsi ini memastikan bahwa rakitan paket NuGet yang ditentukan diunduh dan dimuat ke dalam sesi PowerShell saat ini. Ini mendukung deteksi dan pemuatan otomatis dengan cadangan ke manifes JSON untuk penggantian konfigurasi, menangani kasus tepi seperti dependensi rintisan (misalnya, SQLite). Instalasi menggunakan CLI dotnet secara eksklusif untuk manajemen paket yang andal dan memuat rakitan langsung dari cache NuGet global tanpa menyalin. Pemuatan malas memeriksa apakah tipe sudah tersedia sebelum melanjutkan.

## Syntax

```powershell
EnsureNuGetAssembly -PackageKey <String> [-ConsentToThirdPartySoftwareInstallation] [-Description <String>] [-Destination <String>] [-ForceConsent] [-ForceLatest] [-ManifestPath <String>] [-Publisher <String>] [-TypeName <String>] [-Version <String>] [<CommonParameters>]
```
