# Expand-Path

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** `ep`

## Synopsis

> Memperluas referensi file yang diberikan menjadi nama jalur lengkap.

## Description

Memperluas referensi file yang diberikan menjadi jalur lengkap, terkait dengan direktori pengguna saat ini. Dapat secara opsional memastikan bahwa direktori atau file ada.

## Syntax

```powershell
Expand-Path -FilePath <String> [-CreateDirectory] [-CreateFile] [-DeleteExistingFile] [-DirectoryMustExist] [-FileMustExist] [-ForceDrive <Char>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Jalur untuk diperluas |
| `-CreateDirectory` | SwitchParameter | — | — | Named | — | Akan membuat direktori jika belum ada |
| `-CreateFile` | SwitchParameter | — | — | Named | — | Akan membuat file kosong jika file tersebut tidak ada |
| `-DeleteExistingFile` | SwitchParameter | — | — | Named | — | Akan menghapus file jika sudah ada |
| `-ForceDrive` | Char | — | — | Named | `'*'` | Akan memaksa penggunaan drive tertentu |
| `-FileMustExist` | SwitchParameter | — | — | Named | — | Akan melempar kesalahan jika file tidak ada |
| `-DirectoryMustExist` | SwitchParameter | — | — | Named | — | Akan memunculkan kesalahan jika direktori tidak ada |

## Examples

### Expand-Path -FilePath ".\myfile.txt" -CreateFile

```powershell
Expand-Path -FilePath ".\myfile.txt" -CreateFile
```

### ep ~\documents\test.txt -CreateFile

```powershell
ep ~\documents\test.txt -CreateFile
```
