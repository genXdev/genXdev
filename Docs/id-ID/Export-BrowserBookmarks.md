# Export-BrowserBookmarks

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Mengekspor markah browser ke file JSON.

## Description

Cmdlet Export-BrowserBookmarks mengekspor markah dari peramban web yang ditentukan (Microsoft Edge, Google Chrome, atau Mozilla Firefox) ke file JSON. Hanya satu jenis peramban yang dapat ditentukan dalam satu waktu. Markah diekspor dengan pelestarian penuh struktur dan metadatanya.

## Syntax

```powershell
Export-BrowserBookmarks -OutputFile <String> [-Chrome] [-Edge] [<CommonParameters>]

Export-BrowserBookmarks [-Firefox] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-OutputFile` | String | ✅ | — | 0 | — | Path ke file JSON tempat bookmark akan disimpan |
| `-Chrome` | SwitchParameter | — | — | Named | — | Ekspor markah dari Google Chrome |
| `-Edge` | SwitchParameter | — | — | Named | — | Ekspor bookmark dari Microsoft Edge |
| `-Firefox` | SwitchParameter | — | — | Named | — | Ekspor markah dari Mozilla Firefox *(Parameter set: )* |

## Examples

### Export-BrowserBookmarks -OutputFile "C:\MyBookmarks.json" -Edge

```powershell
Export-BrowserBookmarks -OutputFile "C:\MyBookmarks.json" -Edge
```

### Export-BrowserBookmarks "C:\MyBookmarks.json" -Chrome

```powershell
Export-BrowserBookmarks "C:\MyBookmarks.json" -Chrome
```
