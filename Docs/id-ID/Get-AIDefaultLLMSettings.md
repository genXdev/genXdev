# Get-AIDefaultLLMSettings

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Mendapatkan semua konfigurasi pengaturan LLM default yang tersedia untuk operasi AI di GenXdev.AI.

## Description

Mengambil rangkaian lengkap pengaturan default Model Bahasa Besar (LLM) yang dikonfigurasi untuk operasi AI. Hasil dapat difilter berdasarkan jenis kueri, pengidentifikasi model, titik akhir API, atau kunci API. Mendukung pengambilan berbasis sesi dan preferensi persisten, dengan opsi untuk menghapus status sesi atau melewatinya sepenuhnya untuk membaca langsung dari preferensi yang tersimpan.

## Syntax

```powershell
Get-AIDefaultLLMSettings [[-LLMQueryType] <String>] [-ApiEndpoint <String>] [-ApiKey <String>] [-ClearSession] [-Model <String>] [-NoSupportForJsonSchema <String>] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-LLMQueryType` | String | — | — | 0 | `'SimpleIntelligence'` | Jenis kueri LLM untuk mendapatkan pengaturan |
| `-Model` | String | — | — | Named | — | Filter konfigurasi berdasarkan pengenal atau pola model |
| `-ApiEndpoint` | String | — | — | Named | — | Memfilter konfigurasi berdasarkan URL endpoint API |
| `-ApiKey` | String | — | — | Named | — | Filter konfigurasi berdasarkan kunci API |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | Menunjukkan bahwa LLM tidak memiliki dukungan untuk skema JSON |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Gunakan pengaturan alternatif yang disimpan dalam sesi untuk preferensi AI seperti Bahasa, koleksi Gambar, dll |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Hapus pengaturan sesi (Variabel global) sebelum mengambil |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Jalur database untuk file data preferensi |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Lewati pengaturan sesi dan dapatkan dari preferensi atau default saja |

## Examples

### Get-AIDefaultLLMSettings -LLMQueryType "Coding"

```powershell
Get-AIDefaultLLMSettings -LLMQueryType "Coding"
```

Mendapatkan semua konfigurasi default yang tersedia untuk tipe kueri Coding.
