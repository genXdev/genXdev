# Get-ActiveUser

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** `gusers`

## Synopsis

> Mengambil daftar nama pengguna unik dari proses sistem yang sedang aktif.

## Description

Mengkueri semua proses yang berjalan pada sistem, mengekstrak nama pengguna yang terkait untuk setiap proses, dan mengembalikan daftar pengguna yang tidak terduplikasi yang memiliki proses aktif. Ini berguna untuk administrasi sistem dan pemantauan keamanan.

## Syntax

```powershell
Get-ActiveUser [<CommonParameters>]
```

## Examples

### Get-ActiveUser Returns a list of all unique usernames with active processes.

```powershell
Get-ActiveUser
Returns a list of all unique usernames with active processes.
```

### gusers Uses the alias to get the same results. ##############################################################################

```powershell
gusers
Uses the alias to get the same results.
##############################################################################
```
