# Get-AtEyeLengthSizeInMM

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung ukuran semu dalam mm dari suatu objek sejauh panjang lengan.

## Description

Menghitung ukuran suatu objek yang akan tampak jika dilihat pada jarak lengan orang dewasa rata-rata (sekitar 0,7 meter), berdasarkan ukuran sebenarnya dan jaraknya. Perhitungan menggunakan pendekatan sudut kecil untuk ukuran sudut.

## Syntax

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters <Double> -SizeInMeters <Double> [-EyeToArmLengthInMeters <Double>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | Jarak ke objek dalam meter. |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | Ukuran sebenarnya dari objek dalam meter. |
| `-EyeToArmLengthInMeters` | Double | — | — | Named | `0` | Jarak lengan dalam meter. Nilai default adalah 0,7. |

## Examples

### Example 1

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters 10 -SizeInMeters 1
```

Menghitung ukuran tampak pada jarak lengan untuk objek berukuran 1 meter pada jarak 10 meter.
