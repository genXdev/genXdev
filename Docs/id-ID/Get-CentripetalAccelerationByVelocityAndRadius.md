# Get-CentripetalAccelerationByVelocityAndRadius

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung percepatan sentripetal.

## Description

Menggunakan a = v² / r.

## Syntax

```powershell
Get-CentripetalAccelerationByVelocityAndRadius -VelocityInMetersPerSecond <Double> -RadiusInMeters <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-VelocityInMetersPerSecond` | Double | ✅ | — | 0 | `0` | Kecepatan dalam m/s |
| `-RadiusInMeters` | Double | ✅ | — | 1 | `0` | Radius dalam meter |
| `-As` | String | — | — | 2 | — | Satuan output untuk percepatan |

## Examples

### Example 1

```powershell
Get-CentripetalAccelerationByVelocityAndRadius -VelocityInMetersPerSecond 10 -RadiusInMeters 5 -As "g"
```

Menghitung percepatan sentripetal untuk kecepatan 10 m/s dan jari-jari 5 m, hasil dalam satuan g.

### Example 2

```powershell
Get-CentripetalAccelerationByVelocityAndRadius 20 10
```

Menghitung percepatan sentripetal untuk kecepatan 20 m/s dan radius 10 m menggunakan parameter posisional.
