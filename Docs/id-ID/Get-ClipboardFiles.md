# Get-ClipboardFiles

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `getclipfiles`, `gcbf`

## Synopsis

> Mendapatkan file dari clipboard Windows yang diatur untuk operasi file seperti salin/tempel.

## Description

* Fungsi ini mengambil jalur file dari clipboard Windows yang sebelumnya telah diatur untuk operasi file.
* Fungsi ini secara otomatis menangani mode threading STA dan MTA, memastikan kompatibilitas di berbagai konteks eksekusi PowerShell.
* Fungsi ini memvalidasi keberadaan file dan hanya mengembalikan file/direktori yang ada sebagai objek yang mirip dengan output Get-ChildItem atau Get-Item.

## Syntax

```powershell
Get-ClipboardFiles [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ClipboardFiles
```

Dapatkan semua file yang saat ini ada di clipboard sebagai objek sistem file.

### Example 2

```powershell
$clipboardFiles = Get-ClipboardFiles
$clipboardFiles | ForEach-Object { Write-Host $_.FullName }
```

Dapatkan file klipboard dan tampilkan jalur lengkapnya.

### Example 3

```powershell
Get-ClipboardFiles | Where-Object { $_.Extension -eq '.txt' }
```

Dapatkan hanya file teks dari clipboard.
