# Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung frekuensi yang tergeser Doppler.

## Description

Menggunakan f' = f * (v + vo) / (v - vs), kecepatan positif menuju satu sama lain.

## Syntax

```powershell
Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed -OriginalFrequencyInHertz <Double> -SourceSpeedInMetersPerSecond <Double> -ObserverSpeedInMetersPerSecond <Double> [[-As] <String>] [<CommonParameters>]

Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed [[-SpeedOfSoundInMetersPerSecond] <Double>] [<CommonParameters>]

Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-OriginalFrequencyInHertz` | Double | ✅ | — | 0 | `0` | Frekuensi asli dalam Hz |
| `-SourceSpeedInMetersPerSecond` | Double | ✅ | — | 1 | `0` | Kecepatan sumber dalam m/s (positif menuju pengamat) |
| `-ObserverSpeedInMetersPerSecond` | Double | ✅ | — | 2 | `0` | Kecepatan pengamat dalam m/s (positif menuju sumber) |
| `-SpeedOfSoundInMetersPerSecond` | Double | — | — | 3 | `0` | Kecepatan suara dalam m/s (default: 343) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 3 | — | Media *(Parameter set: )* |
| `-As` | String | — | — | 4 | — | Satuan output untuk frekuensi |

## Examples

### Example 1

```powershell
Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed -OriginalFrequencyInHertz 440 -SourceSpeedInMetersPerSecond 10 -ObserverSpeedInMetersPerSecond 5 -Medium "water" -As "kilohertz"
```

Menghitung frekuensi pergeseran Doppler untuk nada 440 Hz dengan sumber bergerak pada 10 m/s menuju pengamat, pengamat bergerak pada 5 m/s menuju sumber, dalam medium air, keluaran dalam kilohertz.

### Example 2

```powershell
Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed 440 10 0 -SpeedOfSoundInMetersPerSecond 1480
```

Menghitung frekuensi pergeseran Doppler untuk nada 440 Hz dengan sumber bergerak pada 10 m/s, pengamat diam, kecepatan bunyi 1480 m/s.
