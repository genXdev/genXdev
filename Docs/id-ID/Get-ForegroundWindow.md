# Get-ForegroundWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Mendapatkan handle dari jendela latar depan yang sedang aktif.

## Description

* Cmdlet ini mengambil handle jendela (HWND) dari jendela yang saat ini berada di latar depan dan memiliki fokus keyboard.
* Cmdlet ini menggunakan fungsi API Windows melalui P/Invoke untuk berinteraksi dengan pustaka User32.dll.

## Syntax

```powershell
Get-ForegroundWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ForegroundWindow
```

Mengambil dan mengembalikan handle IntPtr dari jendela yang sedang aktif.

### Example 2

```powershell
$windowHandle = Get-ForegroundWindow
Write-Host "Active window handle: $windowHandle"
```

Menyimpan handle jendela latar depan dalam variabel dan menampilkannya.
