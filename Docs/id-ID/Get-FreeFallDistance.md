# Get-FreeFallDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung jarak jatuh bebas untuk durasi waktu yang diberikan.

## Description

Menghitung jarak jatuh bebas menggunakan metode numerik yang memperhitungkan hambatan udara dan kecepatan terminal.

## Syntax

```powershell
Get-FreeFallDistance -DurationInSeconds <Double> [[-TerminalVelocityInMetersPerSecond] <Double>] [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | Durasi waktu jatuh dalam detik |
| `-TerminalVelocityInMetersPerSecond` | Double | — | — | 1 | `0` | Kecepatan terminal dalam meter per detik (bawaan: 53) |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Percepatan gravitasi dalam m/s² (default: 9,81) |
| `-As` | String | — | — | 3 | — | Satuan untuk jarak keluaran |

## Examples

### Example 1

```powershell
Get-FreeFallDistance -DurationInSeconds 10 -TerminalVelocityInMetersPerSecond 53 -As "feet"
```

Menghitung jarak jatuh dalam 10 detik dalam satuan kaki.

### Example 2

```powershell
Get-FreeFallDistance 5
```

Menghitung jarak dalam 5 detik dalam meter.
