# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> Mengembalikan true jika mesin text-to-speech sedang berbicara.

## Description

Memeriksa status dari synthesizer ucapan bawaan dan yang disesuaikan untuk menentukan apakah salah satunya sedang berbicara. Mengembalikan true jika ucapan sedang berlangsung, false jika tidak.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Mengembalikan true jika mesin text-to-speech sedang berbicara.

### Example 2

```powershell
iss
```

Mengembalikan true jika mesin text-to-speech sedang berbicara (menggunakan alias).
