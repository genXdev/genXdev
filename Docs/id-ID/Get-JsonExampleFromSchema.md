# Get-JsonExampleFromSchema

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Menghasilkan contoh string JSON yang dapat dibaca manusia dari definisi Skema JSON.

## Description

Mengonversi skema JSON apa pun (format json_schema OpenAI atau skema mentah) menjadi string JSON contoh yang ringkas dan cocok untuk disuntikkan ke dalam instruksi sistem LLM. Menangani objek bersarang, array, tipe skalar, enum, union, komposisi oneOf/anyOf/allOf, dan petunjuk format seperti date-time dan email.

Fungsi ini membangun pohon objek PowerShell dengan menelusuri skema secara rekursif, lalu menyerialisasikannya menjadi string JSON yang ringkas melalui ConvertTo-Json.

## Syntax

```powershell
Get-JsonExampleFromSchema [[-Schema] <Object>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Schema` | Object | — | — | 0 | — | Definisi Skema JSON (string JSON atau PSCustomObject). Mendukung format wrapper json_schema OpenAI dan objek Skema JSON mentah. |

## Examples

### $schema = '{ "type": "object", "properties": { "name": { "type": "string" } } }' Get-JsonExampleFromSchema -Schema $schema

```powershell
$schema = '{ "type": "object", "properties": { "name": { "type": "string" } } }'
Get-JsonExampleFromSchema -Schema $schema
```

@{name=...}

### $openAiSchema = '{ "type": "json_schema", "json_schema": { "name": "response", "strict": true, "schema": { "type": "object", "properties": { "items": { "type": "array", "items": { "type": "object", "properties": { "id": { "type": "integer" }, "label": { "type": "string" } } } } } } } }' Get-JsonExampleFromSchema -Schema $openAiSchema

```powershell
$openAiSchema = '{ "type": "json_schema", "json_schema": { "name": "response",
"strict": true, "schema": { "type": "object", "properties": {
"items": { "type": "array", "items": { "type": "object", "properties": {
"id": { "type": "integer" }, "label": { "type": "string" } } } } } } } }'
Get-JsonExampleFromSchema -Schema $openAiSchema
```
