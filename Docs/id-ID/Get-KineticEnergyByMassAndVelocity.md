# Get-KineticEnergyByMassAndVelocity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung energi kinetik.

## Description

Menggunakan KE = 1/2 m v².

## Syntax

```powershell
Get-KineticEnergyByMassAndVelocity -MassInKilograms <Double> -VelocityInMetersPerSecond <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Massa dalam kg |
| `-VelocityInMetersPerSecond` | Double | ✅ | — | 1 | `0` | Kecepatan dalam m/s |
| `-As` | String | — | — | 2 | — | Satuan keluaran untuk energi |

## Examples

### Example 1

```powershell
Get-KineticEnergyByMassAndVelocity -MassInKilograms 10 -VelocityInMetersPerSecond 5 -As "calories"
```

Menghitung energi kinetik untuk massa 10 kg yang bergerak dengan kecepatan 5 m/s dan menghasilkan output dalam kalori.

### Example 2

```powershell
Get-KineticEnergyByMassAndVelocity 5 10
```

Menghitung energi kinetik untuk massa 5 kg yang bergerak pada kecepatan 10 m/s menggunakan parameter posisi.
