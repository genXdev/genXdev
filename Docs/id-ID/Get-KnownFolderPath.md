# Get-KnownFolderPath

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** `folder`

## Synopsis

> Mendapatkan jalur folder yang dikenal di Windows menggunakan API Shell32 Windows.

## Description

Mengambil jalur folder yang dikenal di Windows tertentu menggunakan fungsi SHGetKnownFolderPath dari API Shell32. Metode ini lebih andal daripada menggunakan variabel lingkungan karena bekerja secara konsisten di berbagai versi Windows dan pengaturan bahasa. Mendukung semua folder standar Windows yang dikenal seperti Documents, Downloads, AppData, dll.

## Syntax

```powershell
Get-KnownFolderPath -KnownFolder <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-KnownFolder` | String | ✅ | — | 0 | — | Nama folder yang dikenal di Windows untuk mendapatkan jalurnya |

## Examples

### Get-KnownFolderPath -KnownFolder 'Documents' Returns: C:\Users\Username\Documents

```powershell
Get-KnownFolderPath -KnownFolder 'Documents'
Returns: C:\Users\Username\Documents
```

### folder Downloads Returns: C:\Users\Username\Downloads using the alias

```powershell
folder Downloads
Returns: C:\Users\Username\Downloads using the alias
```

### cd (folder Desktop) Changes to the Desktop folder using the alias

```powershell
cd (folder Desktop)
Changes to the Desktop folder using the alias
```
