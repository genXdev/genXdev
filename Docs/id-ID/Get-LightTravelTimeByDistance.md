# Get-LightTravelTimeByDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung waktu yang dibutuhkan cahaya untuk menempuh suatu jarak.

## Description

Menggunakan t = d / c, dengan c tergantung medium.

## Syntax

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters <Double> [[-As] <String>] [<CommonParameters>]

Get-LightTravelTimeByDistance [[-SpeedOfLightInMetersPerSecond] <Double>] [<CommonParameters>]

Get-LightTravelTimeByDistance -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | Jarak dalam meter |
| `-SpeedOfLightInMetersPerSecond` | Double | — | — | 1 | `0` | Kecepatan cahaya dalam m/s (default: 299792458) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | Media tempat cahaya merambat *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | Satuan keluaran untuk waktu |

## Examples

### Example 1

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters 149597870700 -Medium "water" -As "minutes"
```

Menghitung waktu yang dibutuhkan cahaya untuk menempuh jarak ke matahari melalui air, dalam menit.

### Example 2

```powershell
Get-LightTravelTimeByDistance 300000000 -SpeedOfLightInMetersPerSecond 225000000
```

Menghitung waktu yang dibutuhkan cahaya untuk menempuh jarak 300000000 meter pada kecepatan 225000000 m/s.
