# Get-MpCmdRunPath

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Mendapatkan jalur ke executable MpCmdRun.exe milik Windows Defender.

## Description

* Menemukan dan mengembalikan jalur lengkap ke utilitas baris perintah Windows Defender (MpCmdRun.exe).
* Cmdlet memeriksa lokasi instalasi standar di Program Files dan memberikan penanganan kesalahan yang sesuai jika file tidak ditemukan.

## Syntax

```powershell
Get-MpCmdRunPath [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$defenderPath = Get-MpCmdRunPath
```

Mengambil jalur lengkap ke alat baris perintah Windows Defender.
