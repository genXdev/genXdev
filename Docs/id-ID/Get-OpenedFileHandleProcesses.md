# Get-OpenedFileHandleProcesses

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Mengambil proses yang memiliki handle file terbuka ke file yang ditentukan.

## Description

* Menggunakan alat handle.exe dari Sysinternals untuk mengidentifikasi proses yang saat ini memiliki pegangan terbuka ke satu atau lebih file yang ditentukan.
* Berguna untuk menentukan proses mana yang mencegah operasi file seperti penghapusan atau modifikasi.
* Memerlukan handle.exe dari suite Sysinternals untuk diinstal dan tersedia di path sistem.

## Syntax

```powershell
Get-OpenedFileHandleProcesses -FilePath <String[]> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String[] | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Jalur ke file yang akan diperiksa untuk menangani pegangan terbuka |

## Examples

### Example 1

```powershell
Get-OpenedFileHandleProcesses -FilePath "C:\temp\example.txt"
```

Mengidentifikasi semua proses yang memiliki handle terbuka ke file yang ditentukan.

### Example 2

```powershell
"file1.txt", "file2.txt" | Get-OpenedFileHandleProcesses
```

Memeriksa beberapa file melalui input pipeline untuk proses yang memiliki handel terbuka.
