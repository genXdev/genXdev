# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Mengembalikan objek pembantu jendela untuk jendela utama terminal PowerShell.

## Description

* Mengambil objek pembantu WindowObj yang mewakili jendela utama dari
  proses host PowerShell saat ini.
* Ini memungkinkan manipulasi dan interaksi dengan jendela terminal itu sendiri.

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

Mengambil WindowObj untuk jendela terminal PowerShell saat ini.
