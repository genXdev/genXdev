# Get-ProjectileRangeByInitialSpeedAndAngle

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung jangkauan proyektil.

## Description

Menggunakan rumus R = (v² sin(2θ)) / g untuk gerak proyektil ideal.

## Syntax

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle -InitialSpeedInMetersPerSecond <Double> -AngleInDegrees <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InitialSpeedInMetersPerSecond` | Double | ✅ | — | 0 | `0` | Kecepatan awal dalam m/s |
| `-AngleInDegrees` | Double | ✅ | — | 1 | `0` | Sudut peluncuran dalam derajat |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Gravitasi dalam m/s² (default: 9,81) |
| `-As` | String | — | — | 3 | — | Unit keluaran untuk rentang |

## Examples

### Example 1

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle -InitialSpeedInMetersPerSecond 20 -AngleInDegrees 45 -As "feet"
```

Menghitung jangkauan menggunakan rumus gerak proyektil.

### Example 2

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle 30 30
```

Menghitung rentang dengan parameter posisional.
