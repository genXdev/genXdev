# Get-RefractionAngleByIncidentAngleAndIndices

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung sudut refraksi menggunakan hukum Snell.

## Description

Menggunakan θ2 = arcsin( (n1 / n2) sin θ1 ).

## Syntax

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees <Double> -IndexOfRefraction1 <Double> -IndexOfRefraction2 <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-IncidentAngleInDegrees` | Double | ✅ | — | 0 | `0` | Sudut datang dalam derajat |
| `-IndexOfRefraction1` | Double | ✅ | — | 1 | `0` | Indeks bias medium pertama |
| `-IndexOfRefraction2` | Double | ✅ | — | 2 | `0` | Indeks bias medium kedua |
| `-As` | String | — | — | 3 | — | Satuan keluaran untuk sudut |

## Examples

### Example 1

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees 30 -IndexOfRefraction1 1 -IndexOfRefraction2 1.33 -As "radians"
```

Menghitung sudut refraksi saat cahaya melewati dari udara (n=1) ke air (n=1.33) pada sudut datang 30 derajat.

### Example 2

```powershell
Get-RefractionAngleByIncidentAngleAndIndices 45 1 1.5
```

Menghitung sudut refraksi ketika cahaya melewati dari udara (n=1) ke kaca (n=1.5) pada sudut datang 45 derajat.
