# Get-ResonantFrequencyByLengthAndSpeed

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung frekuensi resonansi untuk pipa tertutup.

## Description

Menggunakan f = v / (4 L) untuk frekuensi fundamental.

## Syntax

```powershell
Get-ResonantFrequencyByLengthAndSpeed -LengthInMeters <Double> [[-As] <String>] [<CommonParameters>]

Get-ResonantFrequencyByLengthAndSpeed -SpeedInMetersPerSecond <Double> [<CommonParameters>]

Get-ResonantFrequencyByLengthAndSpeed -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-LengthInMeters` | Double | ✅ | — | 0 | `0` | Panjang dalam meter |
| `-SpeedInMetersPerSecond` | Double | ✅ | — | 1 | `0` | Kecepatan gelombang dalam m/s *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | Media *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | Satuan output untuk frekuensi |

## Examples

### Example 1

```powershell
Get-ResonantFrequencyByLengthAndSpeed -LengthInMeters 0.5 -Medium "air" -As "kilohertz"
```

Menghitung frekuensi resonansi untuk pipa sepanjang 0,5 meter di udara, keluaran dalam kilohertz.

### Example 2

```powershell
Get-ResonantFrequencyByLengthAndSpeed 1 -SpeedInMetersPerSecond 343
```

Menghitung frekuensi resonansi untuk pipa sepanjang 1 meter dengan kecepatan gelombang 343 m/s.
