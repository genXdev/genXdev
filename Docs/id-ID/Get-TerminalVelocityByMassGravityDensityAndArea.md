# Get-TerminalVelocityByMassGravityDensityAndArea

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung kecepatan terminal.

## Description

Menggunakan v = sqrt(2 m g / (ρ A C)).

## Syntax

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms <Double> [[-GravityInMetersPerSecondSquared] <Double>] -DensityInKilogramsPerCubicMeter <Double> -AreaInSquareMeters <Double> [[-DragCoefficient] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Massa dalam kg |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 1 | `0` | Gravitasi dalam m/s² (default: 9,81) |
| `-DensityInKilogramsPerCubicMeter` | Double | ✅ | — | 2 | `0` | Kepadatan fluida dalam kg/m³ |
| `-AreaInSquareMeters` | Double | ✅ | — | 3 | `0` | Luas penampang dalam m² |
| `-DragCoefficient` | Double | — | — | 4 | `0` | Koefisien hambatan (standar: 0.5) |
| `-As` | String | — | — | 5 | — | Satuan output untuk kecepatan |

## Examples

### Example 1

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms 80 -DensityInKilogramsPerCubicMeter 1.225 -AreaInSquareMeters 0.7 -DragCoefficient 1.0 -As "mph"
```

Menghitung kecepatan terminal untuk massa 80 kg di udara dengan koefisien hambatan 1,0, menghasilkan output dalam mph.

### Example 2

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea 70 1.225 0.8 0.8
```

Menggunakan parameter posisi untuk menghitung kecepatan terminal.
