# Get-TimeOfFlightByInitialVelocityAndAngle

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Menghitung waktu terbang untuk proyektil.

## Description

Menggunakan T = (2 v sinθ) / g untuk gerak ideal.

## Syntax

```powershell
Get-TimeOfFlightByInitialVelocityAndAngle -InitialVelocityInMetersPerSecond <Double> -AngleInDegrees <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InitialVelocityInMetersPerSecond` | Double | ✅ | — | 0 | `0` | Kecepatan awal dalam m/s |
| `-AngleInDegrees` | Double | ✅ | — | 1 | `0` | Sudut peluncuran dalam derajat |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Gravitasi dalam m/s² (default: 9,81) |
| `-As` | String | — | — | 3 | — | Satuan keluaran untuk waktu |

## Examples

### Example 1

```powershell
Get-TimeOfFlightByInitialVelocityAndAngle -InitialVelocityInMetersPerSecond 20 -AngleInDegrees 45 -As "minutes"
```

Menghitung waktu terbang untuk proyektil yang diluncurkan pada 20 m/s pada sudut 45 derajat, output dalam menit.

### Example 2

```powershell
Get-TimeOfFlightByInitialVelocityAndAngle 30 30
```

Menghitung waktu terbang untuk proyektil yang diluncurkan pada 30 m/s dengan sudut 30 derajat.
