# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Mengembalikan kamus terbalik untuk semua bahasa yang didukung oleh Google Search.

## Description

* Mengembalikan kamus yang memetakan nama bahasa ke kode bahasa yang sesuai untuk semua bahasa yang didukung oleh Google Search.
* Berguna untuk pencarian kode bahasa dalam kueri web dan tugas penerjemahan.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

Mengambil pemetaan kamus dari nama bahasa ke kode mereka.
