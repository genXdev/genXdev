# Initialize-ScheduledTaskScripts

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Membuat tugas terjadwal yang menjalankan skrip PowerShell pada interval yang ditentukan.

## Description

Membuat dan mengonfigurasi tugas terjadwal yang menjalankan skrip PowerShell pada berbagai interval termasuk:
- Saat startup sistem
- Saat logon pengguna
- Setiap jam pada hari tertentu (misalnya, Senin pukul 13:00)
- Setiap hari pada jam tertentu (misalnya, setiap hari pukul 15:00)
Setiap tugas berjalan dengan hak istimewa yang ditinggikan dalam konteks pengguna saat ini.

## Syntax

```powershell
Initialize-ScheduledTaskScripts [[-FilePath] <String>] [[-Prefix] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | — | — | 0 | `''` | Jalur direktori tempat skrip tugas akan dibuat |
| `-Prefix` | String | — | — | 1 | `'PS'` | Awalan untuk nama tugas terjadwal |

## Examples

### Initialize-ScheduledTaskScripts -FilePath "C:\Tasks" -Prefix "MyTasks"

```powershell
Initialize-ScheduledTaskScripts -FilePath "C:\Tasks" -Prefix "MyTasks"
```

### Initialize-ScheduledTaskScripts

```powershell
Initialize-ScheduledTaskScripts
```
