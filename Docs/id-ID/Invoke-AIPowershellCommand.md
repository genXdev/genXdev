# Invoke-AIPowershellCommand

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `hint`

## Synopsis

> Menghasilkan dan menjalankan perintah PowerShell menggunakan bantuan AI.

## Description

Menggunakan model AI untuk menghasilkan perintah PowerShell berdasarkan kueri bahasa alami. Fungsi ini dapat mengirim perintah langsung ke jendela PowerShell atau menyalinnya ke clipboard. Ini memanfaatkan model AI untuk menafsirkan bahasa alami dan mengubahnya menjadi perintah PowerShell yang dapat dieksekusi dengan dukungan parameter yang komprehensif untuk berbagai backend AI.

## Syntax

```powershell
Invoke-AIPowershellCommand -PowershellCmdline <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-LLMQueryType <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-PowershellCmdline` | String | ✅ | — | 0 | — | Permintaan bahasa alami untuk menghasilkan sebuah perintah |
| `-LLMQueryType` | String | — | — | Named | `'Coding'` | Jenis kueri LLM |
| `-Model` | String | — | — | Named | — | Pengidentifikasi atau pola model yang digunakan untuk operasi AI |
| `-ApiEndpoint` | String | — | — | Named | — | URL titik akhir API untuk operasi AI |
| `-ApiKey` | String | — | — | Named | — | Kunci API untuk operasi AI yang telah diautentikasi |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | Menunjukkan bahwa LLM tidak memiliki dukungan untuk skema JSON |

## Examples

### Invoke-AIPowershellCommand -Query "list all running processes"

```powershell
Invoke-AIPowershellCommand -Query "list all running processes"
```

Menghasilkan perintah PowerShell untuk mendaftar proses yang sedang berjalan

### hint "list files modified today"

```powershell
hint "list files modified today"
```

Menggunakan alias untuk menghasilkan perintah untuk mencari file yang dimodifikasi hari ini.

### Invoke-AIPowershellCommand -Query "stop service" -Clipboard

```powershell
Invoke-AIPowershellCommand -Query "stop service" -Clipboard
```

Menghasilkan perintah untuk menghentikan layanan dan menyalinnya ke clipboard.
