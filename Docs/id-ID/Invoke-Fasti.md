# Invoke-Fasti

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** `fasti`

## Synopsis

> Mengekstrak file arsip di direktori saat ini ke folder mereka sendiri dan menghapusnya setelahnya.

## Description

Secara otomatis mengekstrak format arsip umum (zip, 7z, tar, dll.) yang ditemukan di direktori saat ini ke dalam folder terpisah yang dinamai sesuai dengan setiap arsip. Setelah ekstraksi berhasil, file arsip asli akan dihapus. Memerlukan 7-Zip terinstal di sistem.

## Syntax

```powershell
Invoke-Fasti [[-Password] <String>] [-ExtractOutputToo] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Password` | String | — | — | 0 | — | Masukkan kata sandi untuk arsip yang terenkripsi |
| `-ExtractOutputToo` | SwitchParameter | — | — | Named | — | Secara rekursif ekstrak arsip yang ditemukan di dalam folder yang telah diekstrak |

## Examples

### PS C:\Downloads> Invoke-Fasti

```powershell
PS C:\Downloads> Invoke-Fasti
```

### PS C:\Downloads> fasti

```powershell
PS C:\Downloads> fasti
```
