# Invoke-GenXdevPSFormatter

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Memformat file skrip PowerShell menggunakan aturan pemformatan PSScriptAnalyzer.

## Description

Fungsi ini menerapkan aturan pemformatan PowerShell ke file skrip menggunakan cmdlet Invoke-Formatter milik PSScriptAnalyzer. Fungsi ini dapat memproses file individual atau memformat banyak file dalam direktori secara rekursif. Fungsi ini menggunakan pengaturan pemformatan yang dapat disesuaikan dan menyediakan pencatatan detail tentang proses pemformatan.

## Syntax

```powershell
Invoke-GenXdevPSFormatter -Path <String> [-Range <Int32[]>] [-Recurse] [-Settings <Object>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Menentukan jalur ke file skrip yang akan diformat. |
| `-Settings` | Object | — | — | Named | — | Sebuah hashtable pengaturan atau jalur ke file data PowerShell (.psd1) yang berisi pengaturan format. |
| `-Range` | Int32[] | — | — | Named | — | Rentang di mana pemformatan harus dilakukan sebagai array dari empat bilangan bulat: nomor baris awal, nomor kolom awal, nomor baris akhir, nomor kolom akhir. |
| `-Recurse` | SwitchParameter | — | — | Named | — | Memproses file secara rekursif di subdirektori. |

## Examples

### Invoke-GenXdevPSFormatter -Path "C:\Scripts\MyScript.ps1"

```powershell
Invoke-GenXdevPSFormatter -Path "C:\Scripts\MyScript.ps1"
```

### Invoke-GenXdevPSFormatter -Path "C:\Scripts" -Recurse

```powershell
Invoke-GenXdevPSFormatter -Path "C:\Scripts" -Recurse
```

### "MyScript.ps1" | Invoke-GenXdevPSFormatter -Settings @{IncludeRules=@('PSUseCorrectCasing')}

```powershell
"MyScript.ps1" | Invoke-GenXdevPSFormatter -Settings @{IncludeRules=@('PSUseCorrectCasing')}
```
