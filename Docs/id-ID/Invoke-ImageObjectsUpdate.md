# Invoke-ImageObjectsUpdate

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `imageobjectdetection`

## Synopsis

> Memperbarui metadata deteksi objek untuk file gambar dalam direktori yang ditentukan.

## Description

Fungsi ini memproses gambar dalam direktori yang ditentukan untuk mendeteksi objek menggunakan kecerdasan buatan. Fungsi ini membuat file metadata JSON yang berisi objek yang terdeteksi, posisinya, skor kepercayaan, dan label. Fungsi ini mendukung pemrosesan batch dengan ambang kepercayaan yang dapat dikonfigurasi dan secara opsional dapat melewati file metadata yang sudah ada atau mencoba ulang deteksi yang sebelumnya gagal.

## Syntax

```powershell
Invoke-ImageObjectsUpdate [[-ImageDirectories] <String[]>] [-ClearSession] [-Language <String>] [-LLMQueryType <String>] [-Model <String>] [-OnlyNew] [-PreferencesDatabasePath <String>] [-Recurse] [-RetryFailed] [-SessionOnly] [-SkipSession] [-TimeoutSeconds <Int32>] [<CommonParameters>]
```

## Examples

### Invoke-ImageObjectsUpdate -ImageDirectories @("C:\Photos", "D:\Pictures") -Recurse

```powershell
Invoke-ImageObjectsUpdate -ImageDirectories @("C:\Photos", "D:\Pictures") -Recurse
```

Contoh ini memproses semua gambar di C:\Photos dan D:\Pictures serta semua subdirektorinya menggunakan pengaturan default dengan ambang batas kepercayaan 0,5.

### Invoke-ImageObjectsUpdate @("C:\Photos", "C:\Archive") -RetryFailed -OnlyNew

```powershell
Invoke-ImageObjectsUpdate @("C:\Photos", "C:\Archive") -RetryFailed -OnlyNew
```

Contoh ini hanya memproses gambar baru dan mencoba ulang gambar yang sebelumnya gagal di beberapa direktori menggunakan sintaks parameter posisional.

### Invoke-ImageObjectsUpdate -ImageDirectories "C:\Photos" -UseGPU `     -ConfidenceThreshold 0.7

```powershell
Invoke-ImageObjectsUpdate -ImageDirectories "C:\Photos" -UseGPU `
    -ConfidenceThreshold 0.7
```

Contoh ini menggunakan akselerasi GPU dengan ambang kepercayaan yang lebih tinggi yaitu 0,7 untuk deteksi objek yang lebih akurat tetapi lebih sedikit.
