# Now

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Mengembalikan tanggal dan waktu sistem saat ini sebagai objek DateTime.

## Description

Menyediakan cara sederhana untuk mendapatkan tanggal dan waktu sistem saat ini tanpa parameter apa pun. Mengembalikan objek DateTime .NET standar yang dapat digunakan untuk perhitungan datetime, pemformatan, dan perbandingan.

## Syntax

```powershell
Now [<CommonParameters>]
```

## Examples

### Now Returns the current system date and time as a DateTime object

```powershell
Now
Returns the current system date and time as a DateTime object
```

### $timestamp = Now Stores the current date and time in a variable for later use

```powershell
$timestamp = Now
Stores the current date and time in a variable for later use
```

## Outputs

- `DateTime`
