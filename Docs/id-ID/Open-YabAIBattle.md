# Open-YabAIBattle

> **Module:** GenXdev.Queries.Websites | **Type:** Function | **Aliases:** `yabbattle`

## Synopsis

> Membuka permainan YabAI Battle di peramban web.

## Description

Fungsi ini meluncurkan permainan YabAI Battle di peramban web dengan opsi kustomisasi yang luas untuk penempatan jendela, pemilihan peramban, dan mode tampilan. Ini menyediakan pembungkus di sekitar fungsi GenXdev\Open-Webbrowser dengan konstruksi URL khusus permainan dan penanganan parameter.

Fungsi ini mendukung dua mode permainan:
- Mode Battle: Mulai pertarungan AI vs AI baru
- Mode Spectate: Tonton pertarungan AI yang sudah ada

Opsi penempatan peramban meliputi mode kiri, kanan, atas, bawah, tengah, dan layar penuh dengan dukungan multi-monitor. Fungsi ini secara otomatis menangani pengaturan mode aplikasi dan manajemen ekstensi peramban untuk pengalaman bermain yang optimal.

## Syntax

```powershell
Open-YabAIBattle [[-Language] <String>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-Monitor <Int32>] [-NewWindow] [-NoApplicationMode] [-NoBorders] [-NoBrowserExtensions] [-NoFullScreen] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-ShowWindow] [-SideBySide] [-SkipSession] [-SpectateOnly] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-YabAIBattle -Edge -Private -NoFullScreen

```powershell
Open-YabAIBattle -Edge -Private -NoFullScreen
```

Membuka game Battle YabAI di Microsoft Edge menggunakan mode penjelajahan pribadi tanpa layar penuh.

### yabbattle -e -incognito -nfs

```powershell
yabbattle -e -incognito -nfs
```

Membuka game menggunakan alias untuk Edge, mode penyamaran, dan tanpa layar penuh.

### Open-YabAIBattle -SpectateOnly -Chrome

```powershell
Open-YabAIBattle -SpectateOnly -Chrome
```

Membuka permainan dalam mode tonton menggunakan Google Chrome untuk menyaksikan pertarungan AI.

### Open-YabAIBattle -Language "Spanish" -Monitor 1 -Left

```powershell
Open-YabAIBattle -Language "Spanish" -Monitor 1 -Left
```

Membuka game dengan antarmuka bahasa Spanyol yang diposisikan di sisi kiri monitor 1.
