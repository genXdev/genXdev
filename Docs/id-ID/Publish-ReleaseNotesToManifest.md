# Publish-ReleaseNotesToManifest

> **Module:** GenXdev.Coding | **Type:** Function | **Aliases:** `pubrelnotes`

## Synopsis

> Menerbitkan catatan rilis yang belum selesai dari README.md ke manifes modul.

## Description

Membaca bagian "## Catatan rilis" dari file README.md, mengumpulkan semua
catatan rilis yang belum selesai (baris yang dimulai dengan - ☐), menuliskannya ke dalam
bidang ReleaseNotes di GenXdev.psd1, dan menandainya sebagai selesai (☐ → ☒)
di README. Kedua file diperbarui secara atomik dalam satu operasi.

## Syntax

```powershell
Publish-ReleaseNotesToManifest [[-ManifestPath] <String>] [-UseHomeREADME] [-UseOneDriveREADME] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-UseHomeREADME` | SwitchParameter | — | — | Named | — | Gunakan README di direktori profil PowerShell |
| `-UseOneDriveREADME` | SwitchParameter | — | — | Named | — | Gunakan README di direktori OneDrive |
| `-ManifestPath` | String | — | — | 0 | — | Jalur ke manifes modul .psd1 |

## Examples

### Publish-ReleaseNotesToManifest

```powershell
Publish-ReleaseNotesToManifest
```

### Publish-ReleaseNotesToManifest -ManifestPath ".\MyModule.psd1"

```powershell
Publish-ReleaseNotesToManifest -ManifestPath ".\MyModule.psd1"
```
