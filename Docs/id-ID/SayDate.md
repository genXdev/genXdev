# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Mengucapkan tanggal saat ini menggunakan sintesis teks-ke-ucapan.

## Description

Mengonversi tanggal saat ini ke dalam format bahasa alami dan mengumumkannya menggunakan mesin text-to-speech sistem. Tanggal diucapkan dalam format: "Hari ini adalah [hari dalam minggu], [bulan] [tanggal] [tahun]"

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
