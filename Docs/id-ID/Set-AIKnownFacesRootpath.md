# Set-AIKnownFacesRootpath

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> Mengatur direktori untuk file gambar wajah yang digunakan dalam operasi GenXdev.AI.

## Description

Fungsi ini mengkonfigurasi direktori wajah global yang digunakan oleh modul GenXdev.AI untuk berbagai operasi pengenalan wajah dan AI. Pengaturan dapat disimpan secara persisten dalam preferensi (default), hanya di sesi saat ini (menggunakan -SessionOnly), atau dihapus dari sesi (menggunakan -ClearSession).

## Syntax

```powershell
Set-AIKnownFacesRootpath [[-FacesDirectory] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FacesDirectory` | String | — | — | 0 | — | Path direktori untuk file gambar wajah |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Jalur database untuk file data preferensi |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Gunakan pengaturan alternatif yang disimpan dalam sesi untuk preferensi AI seperti Bahasa, koleksi Gambar, dll |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Hapus pengaturan alternatif yang disimpan di sesi untuk preferensi AI seperti Bahasa, Koleksi Gambar, dll |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Jangan gunakan pengaturan alternatif yang disimpan dalam sesi untuk preferensi AI seperti Bahasa, Koleksi gambar, dll. |

## Examples

### Set-AIKnownFacesRootpath -FacesDirectory "C:\Faces"

```powershell
Set-AIKnownFacesRootpath -FacesDirectory "C:\Faces"
```

Menyetel direktori wajah secara persisten dalam preferensi.

### Set-AIKnownFacesRootpath "C:\FacePictures"

```powershell
Set-AIKnownFacesRootpath "C:\FacePictures"
```

Menyetel direktori wajah secara persisten dalam preferensi.

### Set-AIKnownFacesRootpath -FacesDirectory "C:\TempFaces" -SessionOnly

```powershell
Set-AIKnownFacesRootpath -FacesDirectory "C:\TempFaces" -SessionOnly
```

Mengatur direktori wajah hanya untuk sesi saat ini (variabel Global).

### Set-AIKnownFacesRootpath -ClearSession

```powershell
Set-AIKnownFacesRootpath -ClearSession
```

Menghapus pengaturan direktori sesi wajah (variabel Global) tanpa memengaruhi preferensi persisten.
