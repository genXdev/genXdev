# Set-AILLMSettings

> **Module:** GenXdev.AI | **Type:** Function | **Aliases:** —

## Synopsis

> Mengatur pengaturan LLM untuk operasi AI di GenXdev.AI.

## Description

Ini        [int] $TimeoutSeconds,
        ###############################################################################
        [Parameter(
            Mandatory = $false,
            HelpMessage = 'Apakah endpoint tidak mendukung format respons json_schema'

        [switch] $NoSupportForJsonSchema,
        ###############################################################################
        [Parameter(

            HelpMessage = 'Apakah endpoint tidak mendukung fungsionalitas unggah gambar'
        )]
        [switch] $NoSupportForImageUpload,

                'preferensi AI')
        )]
        [switch] $ClearSession, menyimpan pengaturan LLM (Large Language Model) yang digunakan oleh
modul GenXdev.AI untuk berbagai operasi AI. Pengaturan dapat disimpan secara persisten
dalam preferensi (default), hanya di sesi saat ini (menggunakan -SessionOnly), atau
dihapus dari sesi (menggunakan -ClearSession). Fungsi memvalidasi bahwa setidaknya
satu parameter pengaturan disediakan kecuali saat menghapus pengaturan sesi.

## Syntax

```powershell
Set-AILLMSettings -LLMQueryType <String> [[-Model] <String>] [[-ApiEndpoint] <String>] [[-ApiKey] <String>] [-ClearSession] [-NoSupportForImageUpload] [-NoSupportForJsonSchema] [-NoSupportForToolCalls] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Examples

### Set-AILLMSettings -LLMQueryType "Coding" -Model "*Qwen*14B*"

```powershell
Set-AILLMSettings -LLMQueryType "Coding" -Model "*Qwen*14B*"
```

Mengatur pengaturan LLM untuk tipe kueri Coding secara persisten di preferensi.

### Set-AILLMSettings -LLMQueryType "SimpleIntelligence" -Model "maziyarpanahi/llama-3-groq-8b-tool-use" -SessionOnly

```powershell
Set-AILLMSettings -LLMQueryType "SimpleIntelligence" -Model "maziyarpanahi/llama-3-groq-8b-tool-use" -SessionOnly
```

Mengatur pengaturan LLM untuk SimpleIntelligence hanya untuk sesi saat ini.

### Set-AILLMSettings -LLMQueryType "Pictures" -ClearSession

```powershell
Set-AILLMSettings -LLMQueryType "Pictures" -ClearSession
```

Menghapus pengaturan sesi LLM untuk tipe kueri Pictures tanpa mempengaruhi preferensi permanen.

### Set-AILLMSettings "Coding" "*Qwen*14B*"

```powershell
Set-AILLMSettings "Coding" "*Qwen*14B*"
```

Menyetel pengaturan LLM untuk jenis kueri Coding menggunakan parameter posisi.
