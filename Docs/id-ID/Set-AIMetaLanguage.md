# Set-AIMetaLanguage

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> Menetapkan bahasa default dan secara opsional direktori gambar untuk operasi metadata gambar GenXdev.AI.

## Description

Fungsi ini mengonfigurasi bahasa default global untuk operasi metadata gambar dalam modul GenXdev.AI. Secara opsional, fungsi ini juga dapat mengatur direktori gambar global. Kedua pengaturan tersebut disimpan di penyimpanan preferensi modul untuk digunakan di seluruh sesi.

## Syntax

```powershell
Set-AIMetaLanguage [[-Language] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Language` | String | — | — | 0 | — | Bahasa default untuk operasi metadata gambar |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Jalur database untuk file data preferensi |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Gunakan pengaturan alternatif yang disimpan dalam sesi untuk preferensi AI seperti Bahasa, koleksi Gambar, dll |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Hapus pengaturan alternatif yang disimpan di sesi untuk preferensi AI seperti Bahasa, Koleksi Gambar, dll |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Jangan gunakan pengaturan alternatif yang disimpan dalam sesi untuk preferensi AI seperti Bahasa, Koleksi gambar, dll. |

## Examples

### Set-AIMetaLanguage -Language "Spanish" -ImageDirectories @("C:\Images", "D:\Photos")

```powershell
Set-AIMetaLanguage -Language "Spanish" -ImageDirectories @("C:\Images", "D:\Photos")
```

Menyetel direktori bahasa dan gambar secara persisten di preferensi.

### Set-AIMetaLanguage "French"

```powershell
Set-AIMetaLanguage "French"
```

Mengatur bahasa secara persisten dalam preferensi.

### Set-AIMetaLanguage -Language "German" -SessionOnly

```powershell
Set-AIMetaLanguage -Language "German" -SessionOnly
```

Menetapkan bahasa hanya untuk sesi saat ini (variabel Global).

### Set-AIMetaLanguage -ClearSession

```powershell
Set-AIMetaLanguage -ClearSession
```

Menghapus pengaturan bahasa sesi (variabel global) tanpa mempengaruhi preferensi persisten.
