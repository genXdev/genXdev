# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> Mengubah lokasi saat ini ke direktori induk dan menampilkan daftar isinya.

## Description

* Naik satu tingkat direktori dalam hierarki sistem file.
* Menampilkan konten direktori baru saat ini.
* Menyediakan alias '..' yang nyaman untuk navigasi direktori cepat.

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

Mengubah ke direktori induk dan menampilkan daftar isinya.

### Example 2

```powershell
..
```

Menggunakan alias untuk berpindah ke direktori induk.
