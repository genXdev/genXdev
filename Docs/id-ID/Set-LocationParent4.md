# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> Naik empat tingkat direktori dalam hierarki sistem berkas.

## Description

* Memindahkan ke atas empat tingkat direktori dari lokasi saat ini dalam sistem file.
* Setelah navigasi, menampilkan isi dari direktori hasil.

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Menavigasi naik empat tingkat direktori dari lokasi saat ini.

### Example 2

```powershell
.....
```

Menggunakan alias untuk naik empat tingkat direktori.
