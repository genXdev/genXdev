# Set-WebbrowserTabLocation

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `lt`, `Nav`

## Synopsis

> Menavigasi tab browser web saat ini ke URL yang ditentukan.

## Description

Menetapkan lokasi (URL) dari tab peramban web yang saat ini dipilih. Mendukung peramban Edge dan Chrome melalui sakelar opsional. Navigasi mencakup validasi URL dan memastikan pemuatan halaman yang tepat melalui operasi asinkron.

## Syntax

```powershell
Set-WebbrowserTabLocation -Url <String> [-ByReference <PSObject>] [-Chromium] [-Firefox] [-NoAutoSelectTab] [-Page <Object>] [-Webkit] [<CommonParameters>]

Set-WebbrowserTabLocation [-Edge] [<CommonParameters>]

Set-WebbrowserTabLocation [-Chrome] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Url` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | URL yang akan dinavigasi |
| `-NoAutoSelectTab` | SwitchParameter | — | — | Named | — | Mencegah pemilihan tab otomatis |
| `-Edge` | SwitchParameter | — | — | Named | — | Navigasi menggunakan browser Microsoft Edge *(Parameter set: )* |
| `-Chrome` | SwitchParameter | — | — | Named | — | Navigasi menggunakan browser Google Chrome *(Parameter set: )* |
| `-Chromium` | SwitchParameter | — | — | Named | — | Navigasi menggunakan Microsoft Edge atau Google Chrome, tergantung pada peramban default yang digunakan |
| `-Firefox` | SwitchParameter | — | — | Named | — | Navigasi menggunakan peramban Firefox |
| `-Webkit` | SwitchParameter | — | — | Named | — | Navigasi menggunakan peramban WebKit yang dikelola Playwright |
| `-Page` | Object | — | — | Named | — | Referensi objek halaman browser |
| `-ByReference` | PSObject | — | — | Named | — | Objek referensi sesi browser |

## Examples

### Set-WebbrowserTabLocation -Url "https://github.com/microsoft" -Edge

```powershell
Set-WebbrowserTabLocation -Url "https://github.com/microsoft" -Edge
```
