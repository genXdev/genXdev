# Start-VlcMediaPlayerNextInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcnext`

## Synopsis

> Memajukan VLC Media Player ke item berikutnya dalam daftar putar saat ini.

## Description

Fungsi ini mengirimkan tekanan tombol 'n' ke VLC Media Player untuk melompat ke trek berikutnya atau item media berikutnya dalam daftar putar yang sedang dimuat. Fungsi ini memfokuskan jendela VLC, mengirimkan perintah berikutnya, dan mengembalikan fokus ke konsol PowerShell. Fungsi ini menyertakan dukungan ShouldProcess untuk prompt konfirmasi bila diperlukan.

## Syntax

```powershell
Start-VlcMediaPlayerNextInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerNextInPlaylist

```powershell
Start-VlcMediaPlayerNextInPlaylist
```

### vlcnext

```powershell
vlcnext
```
