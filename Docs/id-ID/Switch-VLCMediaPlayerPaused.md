# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> Mengalihkan status jeda/putar dari VLC Media Player.

## Description

Fungsi ini mengirimkan tombol spasi ke VLC Media Player untuk beralih antara keadaan jeda dan pemutaran. Fungsi ini secara otomatis mengembalikan fokus ke jendela asli setelah mengirimkan perintah tombol.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

Mengalihkan status jeda/putar VLC Media Player.

### vlcpause

```powershell
vlcpause
```

Menggunakan alias untuk beralih status jeda/putar.

### vlcplay

```powershell
vlcplay
```

Menggunakan alias alternatif untuk mengganti status jeda/putar.
