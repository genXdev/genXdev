# Switch-VlcMediaPlayerMute

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcmute`, `vlcunmute`

## Synopsis

> Mengubah status bisu VLC Media Player.

## Description

Fungsi ini mengirimkan tombol 'm' ke VLC Media Player untuk beralih antara status audio senyap dan tidak senyap. Fungsi ini memfokuskan VLC, mengirim perintah senyap/tidak senyap, lalu mengembalikan fokus ke jendela yang sebelumnya aktif.

## Syntax

```powershell
Switch-VlcMediaPlayerMute [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerMute

```powershell
Switch-VlcMediaPlayerMute
```

Mengaktifkan/menonaktifkan status bisu VLC Media Player menggunakan nama fungsi lengkap.

### vlcmute

```powershell
vlcmute
```

Mengubah status bisu pemutar media VLC menggunakan alias pendek.
