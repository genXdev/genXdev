# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> Mengaktifkan mode pengulangan di VLC Media Player.

## Description

Fungsi ini mengirimkan perintah tombol 'r' ke VLC Media Player untuk beralih di antara mode ulang yang berbeda (tanpa ulang, ulang saat ini, ulang semua). Fungsi ini membuka VLC jika belum berjalan dan mengembalikan fokus ke jendela sebelumnya setelah mengirimkan perintah.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
