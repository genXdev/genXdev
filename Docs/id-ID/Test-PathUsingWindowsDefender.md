# Test-PathUsingWindowsDefender

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `virusscan`, `HasNoVirus`

## Synopsis

> Memindai file atau direktori untuk malware menggunakan Windows Defender.

## Description

* Melakukan pemindaian tertarget pada file atau direktori tertentu menggunakan antarmuka baris perintah Windows Defender (MpCmdRun.exe).
* Dapat memindai dalam mode hanya deteksi atau dengan perbaikan ancaman otomatis yang diaktifkan.
* Mengembalikan true jika tidak ada ancaman yang terdeteksi, false jika ancaman ditemukan atau pemindaian gagal.

## Syntax

```powershell
Test-PathUsingWindowsDefender -FilePath <String> [-EnableRemediation] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Jalur ke file atau direktori yang akan dipindai |
| `-EnableRemediation` | SwitchParameter | — | — | Named | `False` | Menginstruksikan Windows Defender untuk mengambil tindakan terhadap ancaman |

## Examples

### Example 1

```powershell
Test-PathUsingWindowsDefender -FilePath "C:\Downloads\file.exe" -Verbose
```

Memindai file yang ditentukan dan menampilkan output verbose.

### Example 2

```powershell
virusscan "C:\Downloads\file.exe" -EnableRemediation
```

Memindai file menggunakan alias dan mengaktifkan perbaikan.

### Example 3

```powershell
"C:\Downloads\file.exe" | HasNoVirus
```

Menyalurkan jalur file ke cmdlet menggunakan alias.
