# Test-RefactorLLMSelection

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Mengevaluasi file sumber untuk kelayakan refaktorisasi menggunakan analisis LLM.

## Description

Menggunakan analisis Model Pembelajaran Bahasa (LLM) untuk menentukan apakah sebuah file kode sumber harus dipilih untuk direfaktor berdasarkan kriteria yang ditentukan. Fungsi ini memproses konten file melalui kueri LLM dan mengembalikan respons boolean.

## Syntax

```powershell
Test-RefactorLLMSelection -RefactorDefinition <GenXdev.Helpers.RefactorDefinition> -Path <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-RefactorDefinition` | GenXdev.Helpers.RefactorDefinition | ✅ | — | 0 | — | Definisi refactor yang berisi pengaturan LLM |
| `-Path` | String | ✅ | — | 1 | — | Jalur ke file sumber untuk dievaluasi |

## Examples

### Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"

```powershell
Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"
```

### $def | Test-RefactorLLMSelection -Path source.ps1

```powershell
$def | Test-RefactorLLMSelection -Path source.ps1
```
