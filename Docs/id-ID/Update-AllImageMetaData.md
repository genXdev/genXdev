# Update-AllImageMetaData

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `updateallimages`

## Synopsis

> Memperbarui kata kunci gambar, wajah, objek, dan pemandangan secara batch di beberapa direktori sistem.

## Description

Fungsi ini secara sistematis memproses gambar di berbagai direktori sistem untuk memperbarui kata kunci, data pengenalan wajah, data deteksi objek, dan data klasifikasi pemandangan menggunakan layanan AI. Ini mencakup folder media penyimpanan, file sistem, unduhan, OneDrive, dan folder gambar pribadi.

Fungsi ini memproses gambar dengan melalui setiap direktori dan memproses file satu per satu. Fungsi DeepStack (wajah, objek, pemandangan) dilakukan terlebih dahulu, diikuti dengan pembuatan kata kunci dan deskripsi.

Ini memungkinkan keluaran data terstruktur untuk operasi pipeline seperti:
Update-AllImageMetaData | Export-ImageIndex

## Syntax

```powershell
Update-AllImageMetaData -ImageDirectories <String[]> [-ApiEndpoint <String>] [-ApiKey <String>] [-AutoUpdateFaces] [-ClearSession] [-ConfidenceThreshold <Double>] [-ContainerName <String>] [-FacesDirectory <String>] [-Force] [-HealthCheckInterval <Int32>] [-HealthCheckTimeout <Int32>] [-ImageName <String>] [-Language <String>] [-Model <String>] [-NoDockerInitialize] [-NoRecurse] [-NoSupportForJsonSchema <String>] [-PassThru] [-PreferencesDatabasePath <String>] [-RedoAll] [-RetryFailed] [-ServicePort <Int32>] [-SessionOnly] [-SkipSession] [-TimeoutSeconds <Int32>] [-UseGPU] [-VolumeName <String>] [<CommonParameters>]
```
