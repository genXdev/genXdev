# WriteFileOutput

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Menampilkan informasi file dengan nama tampilan yang dapat diklik untuk pengalaman pengguna yang lebih baik.

## Description

Fungsi ini memproses objek input file dan menampilkannya dalam format yang ramah pengguna dengan nama file yang tertaut secara hipertaut saat ditampilkan di konsol. Fungsi ini menangani berbagai jenis input termasuk string, objek FileInfo, dan tipe objek lainnya. Fungsi secara otomatis mendeteksi apakah output dialihkan atau ditangkap dalam pipeline dan menyesuaikan perilakunya sesuai.

## Syntax

```powershell
WriteFileOutput -CallerInvocation <Object> -Input <Object> [-FullPaths] [-Prefix <String>] [-RelativeBasePath <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CallerInvocation` | Object | ✅ | ✅ (ByValue) | 0 | — | Informasi pemanggilan dari fungsi yang memanggil |
| `-Input` | Object | ✅ | ✅ (ByValue) | 1 | — | Objek input yang akan diproses, dapat berupa jalur file atau objek |
| `-Prefix` | String | — | — | Named | — | Awalan string opsional yang akan ditambahkan ke tampilan output untuk konteks tambahan |
| `-RelativeBasePath` | String | — | — | Named | — | Jalur dasar untuk menghasilkan jalur file relatif dalam output |
| `-FullPaths` | SwitchParameter | — | — | Named | — | Memaksa output untuk menggunakan jalur absolut lengkap, bukan jalur relatif |

## Examples

### WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"

```powershell
WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"
```

### Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_

```powershell
Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_
```
