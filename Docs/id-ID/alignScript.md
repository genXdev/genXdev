# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Mengembalikan string (dengan perubahan indentasi) dari string blok skrip yang diberikan

## Description

Mengubah indentasi string blok skrip dengan tetap memperhatikan indentasi kode asli

## Syntax

```powershell
alignScript [<CommonParameters>]
```
