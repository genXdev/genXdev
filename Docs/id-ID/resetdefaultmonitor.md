# resetdefaultmonitor

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Mengembalikan konfigurasi monitor sekunder default.

## Description

Skrip ini mengembalikan konfigurasi monitor sekunder default untuk sistem,
menetapkan monitor sekunder ke nilai default aslinya.
Ini berguna bagi pengguna yang ingin kembali ke pengaturan multi-monitor sebelumnya setelah menggunakan konfigurasi berdampingan.
Lihat juga: fungsi 'sidebyside' untuk beralih ke mode berdampingan untuk jendela baru.

## Syntax

```powershell
resetdefaultmonitor [<CommonParameters>]
```

## Examples

### secondscreen

```powershell
secondscreen
```

Memulihkan konfigurasi monitor sekunder default untuk sistem.
