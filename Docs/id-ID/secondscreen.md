# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Mengatur konfigurasi monitor kedua default.

## Description

Mengatur perilaku default pembukaan jendela GenXdev agar berada di monitor sekunder.
Ini berguna bagi pengguna dengan satu monitor atau yang lebih suka tata letak jendela berdampingan.
Lihat juga cmdlet 'sidebyside' dan 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Mengatur default untuk jendela GenXdev agar terbuka di monitor sekunder
