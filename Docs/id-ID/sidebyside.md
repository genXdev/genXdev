# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Menetapkan konfigurasi standar side-by-side.

## Description

Mengatur perilaku default pembukaan jendela GenXdev agar berdampingan dengan PowerShell.
Ini berguna bagi pengguna dengan satu monitor atau yang lebih menyukai tata letak jendela berdampingan.
Lihat juga cmdlet 'secondscreen' dan 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

Menetapkan default untuk jendela GenXdev agar ditampilkan berdampingan dengan PowerShell
