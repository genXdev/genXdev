# Assert-GenXdevCmdletTests

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `improvecmdlettests`

## Synopsis

> Assert e migliora i test unitari di un cmdlet GenXdev specificato.

## Description

Questa funzione aiuta a mantenere e migliorare i test unitari per i cmdlet di GenXdev:
1. Creando file di test se non esistono
2. Aprendo il cmdlet in VS Code
3. Preparando e applicando prompt AI per la generazione/miglioramento dei test
4. Gestendo il flusso di lavoro di esecuzione dei test

## Syntax

```powershell
Assert-GenXdevCmdletTests -CmdletName <String> [[-Prompt] <String>] [[-PromptKey] <String>] [-AssertFailedTest] [-ContinuationHandled] [-EditPrompt] [-FromScripts] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CmdletName` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Il nome del cmdlet per migliorare gli unit-test per |
| `-Prompt` | String | — | — | 1 | `''` | Testo del prompt AI personalizzato da utilizzare |
| `-PromptKey` | String | — | — | 2 | — | La chiave del prompt AI da utilizzare per la selezione del modello |
| `-EditPrompt` | SwitchParameter | — | — | Named | — | Passa alla modifica solo del prompt dell'IA |
| `-AssertFailedTest` | SwitchParameter | — | — | Named | — | Indica il fallimento di un test |
| `-FromScripts` | SwitchParameter | — | — | Named | — | Cerca nei file di script invece dei moduli |
| `-ContinuationHandled` | SwitchParameter | — | — | Named | — | The `-ContinuationHandled` parameter. |
