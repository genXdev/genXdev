# Assert-GenXdevDependencyUsage

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `checkgenxdevdependencies`

## Synopsis

> Convalida l'utilizzo delle dipendenze tra i moduli GenXdev per garantire che venga mantenuta la corretta gerarchia dei moduli.

## Description

Questa funzione analizza i moduli GenXdev per assicurarsi che seguano la corretta gerarchia delle dipendenze. Verifica che i moduli facciano riferimento solo a dipendenze elencate nel loro manifest RequiredModules e previene le dipendenze circolari convalidando che i moduli non facciano riferimento a moduli che vengono successivi nella catena di dipendenze.

## Syntax

```powershell
Assert-GenXdevDependencyUsage [[-ModuleName] <String[]>] [-FromScripts] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String[] | — | ✅ (ByValue, ByPropertyName) | 1 | `@('GenXdev*')` | Filtro da applicare ai nomi dei moduli 🌐 *Supports wildcards* |
| `-FromScripts` | SwitchParameter | — | — | Named | — | Cerca nei file script invece dei file modulo |

## Examples

### Assert-GenXdevDependencyUsage -ModuleName "GenXdev.Coding"

```powershell
Assert-GenXdevDependencyUsage -ModuleName "GenXdev.Coding"
```

### checkgenxdevdependencies "GenXdev*" -FromScripts

```powershell
checkgenxdevdependencies "GenXdev*" -FromScripts
```
