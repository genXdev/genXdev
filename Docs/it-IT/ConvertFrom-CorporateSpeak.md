# ConvertFrom-CorporateSpeak

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `uncorporatize`

## Synopsis

> Converte il linguaggio aziendale formale e professionale in un linguaggio diretto e chiaro utilizzando l'IA.

## Description

Questa funzione elabora il testo in input per trasformare comunicazioni diplomatiche e aziendali in un linguaggio più diretto e chiaro. Può accettare input direttamente tramite parametri, dalla pipeline o dagli appunti di sistema. La funzione sfrutta modelli di intelligenza artificiale per analizzare e riformulare il testo, preservando l'intento originale.

    "schema": {
      "proprietà": {

        }
      },
      "richiesti": [
        "risposta"
      ],
      "tipo": "oggetto"
    }
  },
  "tipo": "schema_json"
}
