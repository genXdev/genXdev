# ConvertTo-HashTable

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Converte un PSCustomObject in un HashTable in modo ricorsivo.

## Description

* Questa funzione converte un PSCustomObject e tutte le sue proprietà PSCustomObject annidate in HashTables.
* Gestisce matrici e altri tipi di raccolta elaborando ogni elemento in modo ricorsivo.

## Syntax

```powershell
ConvertTo-HashTable -InputObject <Object> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputObject` | Object | ✅ | ✅ (ByValue) | 0 | — | Il PSCustomObject da convertire in una HashTable |

## Examples

### Example 1

```powershell
$object = [PSCustomObject]@{
    Name = "John"
    Age = 30
    Details = [PSCustomObject]@{
        City = "New York"
    }
}
$hashTable = ConvertTo-HashTable -InputObject $object
```

Converte un PSCustomObject in una HashTable con proprietà annidate.
