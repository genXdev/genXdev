# Enable-Screensaver

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Avvia lo screensaver di Windows configurato.

## Description

Attiva lo screensaver del sistema Windows eseguendo il file eseguibile predefinito dello screensaver (scrnsave.scr) con l'opzione /s per avviarlo immediatamente.

## Syntax

```powershell
Enable-Screensaver [<CommonParameters>]
```

## Examples

### Enable-Screensaver

```powershell
Enable-Screensaver
```
