# EnsureGenXdev

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Garantisce che tutti i moduli GenXdev siano caricati correttamente richiamando tutti i cmdlet Ensure*.

## Description

Questa funzione recupera tutti i cmdlet GenXdev che iniziano con "Ensure" o
"Optimize-Ensure" (per cmdlet c#) ed
esegue ciascuno di essi per garantire che tutti i moduli GenXdev richiesti e
le dipendenze siano correttamente caricati e disponibili per l'uso. Eventuali errori durante
l'esecuzione vengono intercettati e visualizzati come messaggi informativi.

Opzionalmente, può anche scaricare e caricare tutti i pacchetti NuGet definiti nel
file manifest packages.json.

## Syntax

```powershell
EnsureGenXdev [-ConsentToThirdPartySoftwareInstallation] [-DownloadAllNugetPackages] [-Force] [-ForceConsent] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Force` | SwitchParameter | — | — | Named | — | Forza l'esecuzione delle operazioni di assicurazione anche se sembrano già completate |
| `-DownloadAllNugetPackages` | SwitchParameter | — | — | Named | — | Scarica e carica tutti i pacchetti NuGet definiti nel file manifesto packages.json |
| `-ForceConsent` | SwitchParameter | — | — | Named | — | Forza una richiesta di consenso anche se è impostata una preferenza per l'installazione di software di terze parti. |
| `-ConsentToThirdPartySoftwareInstallation` | SwitchParameter | — | — | Named | — | Consenti automaticamente l'installazione di software di terze parti e imposta il flag persistente per tutti i pacchetti. |

## Examples

### EnsureGenXdev

```powershell
EnsureGenXdev
```

Questo comando esegue tutti i cmdlet Ensure* disponibili per inizializzare l'ambiente GenXdev.

### EnsureGenXdev -DownloadAllNugetPackages

```powershell
EnsureGenXdev -DownloadAllNugetPackages
```

Questo comando esegue tutti i cmdlet Ensure* e scarica e carica tutti i pacchetti NuGet definiti nel file manifest packages.json.

### EnsureGenXdev -DownloadAllNugetPackages -ConsentToThirdPartySoftwareInstallation

```powershell
EnsureGenXdev -DownloadAllNugetPackages -ConsentToThirdPartySoftwareInstallation
```

Questo comando esegue tutti i cmdlet Ensure* e scarica i pacchetti NuGet.
