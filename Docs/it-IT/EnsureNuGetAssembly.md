# EnsureNuGetAssembly

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Scarica e carica assembly .NET da pacchetti NuGet in base a chiave o ID del pacchetto.

## Description

Questa funzione garantisce che gli assembly specificati del pacchetto NuGet vengano scaricati e caricati nella sessione corrente di PowerShell. Supporta il rilevamento e il caricamento automatici con fallback a un manifest JSON per le sostituzioni di configurazione, gestendo casi limite come le dipendenze stub (ad esempio, SQLite). L'installazione utilizza esclusivamente l'interfaccia a riga di comando dotnet per una gestione affidabile dei pacchetti e carica gli assembly direttamente dalla cache NuGet globale senza copiarli. Il caricamento lazy verifica se il tipo è già disponibile prima di procedere.

## Syntax

```powershell
EnsureNuGetAssembly -PackageKey <String> [-ConsentToThirdPartySoftwareInstallation] [-Description <String>] [-Destination <String>] [-ForceConsent] [-ForceLatest] [-ManifestPath <String>] [-Publisher <String>] [-TypeName <String>] [-Version <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-PackageKey` | String | ✅ | — | 0 | — | Chiave del pacchetto da packages.json o identificativo diretto del pacchetto NuGet. |
| `-ManifestPath` | String | — | — | Named | — | Percorso verso packages.json; per impostazione predefinita è la radice del modulo se omesso. |
| `-Version` | String | — | — | Named | — | Versione specifica; se omessa, usa la versione più alta da JSON o l'ultima. |
| `-TypeName` | String | — | — | Named | — | NomeTipo da verificare per il caricamento. |
| `-ForceLatest` | SwitchParameter | — | — | Named | — | Ripiega all'ultima versione se quella esatta non riesce. |
| `-Destination` | String | — | — | Named | — | Destinazione di installazione personalizzata; per impostazione predefinita è la cache persistente o globale locale. |
| `-Description` | String | — | — | Named | — | Descrizione opzionale del software e del suo scopo per il consenso. |
| `-Publisher` | String | — | — | Named | — | Editore o fornitore facoltativo del software per il consenso. |
| `-ForceConsent` | SwitchParameter | — | — | Named | — | Forza una richiesta anche se la preferenza è impostata per il consenso. |
| `-ConsentToThirdPartySoftwareInstallation` | SwitchParameter | — | — | Named | — | Consenti automaticamente l'installazione di software di terze parti e imposta il flag persistente. |

## Examples

### EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core"

```powershell
EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core"
```

### EnsureNuGetAssembly -PackageKey "Microsoft.Playwright" -Version "1.54.0" -TypeName "Microsoft.Playwright.Playwright"

```powershell
EnsureNuGetAssembly -PackageKey "Microsoft.Playwright" -Version "1.54.0" -TypeName "Microsoft.Playwright.Playwright"
```

### EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core" -ConsentToThirdPartySoftwareInstallation

```powershell
EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core" -ConsentToThirdPartySoftwareInstallation
```
