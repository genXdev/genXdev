# Export-BrowserBookmarks

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Esporta i segnalibri del browser in un file JSON.

## Description

Il cmdlet Export-BrowserBookmarks esporta i segnalibri da un browser web specificato (Microsoft Edge, Google Chrome o Mozilla Firefox) in un file JSON. Può essere specificato un solo tipo di browser alla volta. I segnalibri vengono esportati con piena conservazione della loro struttura e dei metadati.

## Syntax

```powershell
Export-BrowserBookmarks -OutputFile <String> [-Chrome] [-Edge] [<CommonParameters>]

Export-BrowserBookmarks [-Firefox] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-OutputFile` | String | ✅ | — | 0 | — | Percorso del file JSON in cui verranno salvati i segnalibri |
| `-Chrome` | SwitchParameter | — | — | Named | — | Esporta i segnalibri da Google Chrome |
| `-Edge` | SwitchParameter | — | — | Named | — | Esporta i segnalibri da Microsoft Edge |
| `-Firefox` | SwitchParameter | — | — | Named | — | Esporta segnalibri da Mozilla Firefox *(Parameter set: )* |

## Examples

### Export-BrowserBookmarks -OutputFile "C:\MyBookmarks.json" -Edge

```powershell
Export-BrowserBookmarks -OutputFile "C:\MyBookmarks.json" -Edge
```

### Export-BrowserBookmarks "C:\MyBookmarks.json" -Chrome

```powershell
Export-BrowserBookmarks "C:\MyBookmarks.json" -Chrome
```
