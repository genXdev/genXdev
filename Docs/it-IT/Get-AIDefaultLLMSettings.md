# Get-AIDefaultLLMSettings

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Ottiene tutte le configurazioni predefinite delle impostazioni LLM disponibili per le operazioni AI in GenXdev.AI.

## Description

Recupera il set completo di impostazioni predefinite del Large Language Model (LLM) configurate per le operazioni di IA. I risultati possono essere filtrati per tipo di query, identificatore del modello, endpoint API o chiave API. Supporta sia il recupero basato su sessione che sulle preferenze persistenti, con opzioni per cancellare lo stato della sessione o bypassarlo completamente per leggere direttamente dalle preferenze memorizzate.

## Syntax

```powershell
Get-AIDefaultLLMSettings [[-LLMQueryType] <String>] [-ApiEndpoint <String>] [-ApiKey <String>] [-ClearSession] [-Model <String>] [-NoSupportForJsonSchema <String>] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-LLMQueryType` | String | — | — | 0 | `'SimpleIntelligence'` | Il tipo di query LLM per cui ottenere le impostazioni |
| `-Model` | String | — | — | Named | — | Filtra le configurazioni per identificatore o pattern del modello |
| `-ApiEndpoint` | String | — | — | Named | — | Filtra le configurazioni per URL dell'endpoint API |
| `-ApiKey` | String | — | — | Named | — | Filtra le configurazioni per chiave API |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | Indica che LLM non supporta gli schemi JSON |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Utilizza impostazioni alternative memorizzate nella sessione per le preferenze dell'IA, come lingua, raccolte di immagini, ecc. |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Cancellare l'impostazione della sessione (variabile globale) prima di recuperare |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Percorso del database per i file dei dati delle preferenze |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Ignora le impostazioni della sessione e utilizza solo quelle delle preferenze o predefinite |

## Examples

### Get-AIDefaultLLMSettings -LLMQueryType "Coding"

```powershell
Get-AIDefaultLLMSettings -LLMQueryType "Coding"
```

Ottiene tutte le configurazioni predefinite disponibili per il tipo di query di codifica.
