# Get-AIKnownFacesRootpath

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> Ottiene la directory configurata per i file immagine dei volti utilizzati nelle operazioni di GenXdev.AI.

## Description

Questa funzione recupera la directory facciale globale utilizzata dal modulo GenXdev.AI per varie operazioni di riconoscimento facciale e AI. Controlla prima le variabili Globali (a meno che non sia specificato SkipSession), poi utilizza le preferenze persistenti e infine utilizza le impostazioni predefinite di sistema.

## Syntax

```powershell
Get-AIKnownFacesRootpath [[-FacesDirectory] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FacesDirectory` | String | — | — | 0 | — | Percorso della directory per i file immagine del volto |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Utilizza impostazioni alternative memorizzate nella sessione per le preferenze dell'IA, come lingua, raccolte di immagini, ecc. |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Cancellare l'impostazione della sessione (variabile globale) prima di recuperare |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Percorso del database per i file dei dati delle preferenze |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Non utilizzare impostazioni alternative memorizzate nella sessione per le preferenze di IA come lingua, raccolte di immagini, ecc. |

## Examples

### Get-AIKnownFacesRootpath

```powershell
Get-AIKnownFacesRootpath
```

Ottiene la directory dei volti attualmente configurata dalle variabili globali o dalle preferenze.

### Get-AIKnownFacesRootpath -SkipSession

```powershell
Get-AIKnownFacesRootpath -SkipSession
```

Recupera solo la directory dei volti configurata dalle preferenze persistenti, ignorando qualsiasi impostazione di sessione.

### Get-AIKnownFacesRootpath -ClearSession

```powershell
Get-AIKnownFacesRootpath -ClearSession
```

Cancella l'impostazione della directory delle facce della sessione e quindi ottiene la directory dalle preferenze persistenti.

### Get-AIKnownFacesRootpath "C:\MyFaces"

```powershell
Get-AIKnownFacesRootpath "C:\MyFaces"
```

Restituisce la directory specificata dopo aver espanso il percorso.
