# Get-AIMetaLanguage

> **Module:** GenXdev.AI.Queries | **Type:** Cmdlet | **Aliases:** `getimgmetalang`

## Synopsis

> Ottiene la lingua predefinita configurata per le operazioni sui metadati delle immagini.

## Description

* Questo cmdlet recupera la lingua predefinita utilizzata dal modulo GenXdev.AI
  per le operazioni sui metadati delle immagini.
* Controlla prima le variabili Globali (a meno che non sia specificato SkipSession), poi
  ricade sulle preferenze persistenti e, infine, utilizza le impostazioni predefinite di sistema.

## Syntax

```powershell
Get-AIMetaLanguage [[-Language] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Language` | String | — | — | 0 | — | La lingua predefinita per le operazioni sui metadati delle immagini |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Percorso del database per i file dei dati delle preferenze |
| `-SessionOnly` | SwitchParameter | — | — | Named | `False` | Utilizza impostazioni alternative memorizzate nella sessione per le preferenze dell'IA, come lingua, raccolte di immagini, ecc. |
| `-ClearSession` | SwitchParameter | — | — | Named | `False` | Cancellare l'impostazione della sessione (variabile globale) prima di recuperare |
| `-SkipSession` | SwitchParameter | — | — | Named | `False` | Non utilizzare impostazioni alternative memorizzate nella sessione per le preferenze di IA come lingua, raccolte di immagini, ecc. |

## Examples

### Example 1

```powershell
Get-AIMetaLanguage
```

Ottieni la lingua attualmente configurata dalle variabili globali o dalle preferenze.

### Example 2

```powershell
Get-AIMetaLanguage -SkipSession
```

Ottieni la lingua configurata dalle preferenze persistenti, ignorando la sessione.

### Example 3

```powershell
Get-AIMetaLanguage -ClearSession
```

Cancella le impostazioni della sessione e recupera la lingua dalle preferenze persistenti.
