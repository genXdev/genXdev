# Get-AtEyeLengthSizeInMM

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Calcola la dimensione apparente in mm di un oggetto alla distanza di un braccio.

## Description

Calcola le dimensioni che un oggetto sembrerebbe avere se visto alla distanza media del braccio di un adulto (circa 0,7 metri), date le sue dimensioni reali e la distanza. Il calcolo utilizza l'approssimazione del piccolo angolo per la dimensione angolare.

## Syntax

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters <Double> -SizeInMeters <Double> [-EyeToArmLengthInMeters <Double>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | La distanza dall'oggetto in metri. |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | La dimensione effettiva dell'oggetto in metri. |
| `-EyeToArmLengthInMeters` | Double | — | — | Named | `0` | La distanza di lunghezza del braccio in metri. Il valore predefinito è 0,7. |

## Examples

### Example 1

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters 10 -SizeInMeters 1
```

Calcola la dimensione apparente a distanza di braccio per un oggetto di 1 metro di dimensione a 10 metri di distanza.
