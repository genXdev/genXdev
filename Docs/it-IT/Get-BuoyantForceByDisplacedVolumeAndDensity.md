# Get-BuoyantForceByDisplacedVolumeAndDensity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Calcola la forza di galleggiamento.

## Description

Usa F = ρ V g.

## Syntax

```powershell
Get-BuoyantForceByDisplacedVolumeAndDensity -DisplacedVolumeInCubicMeters <Double> -FluidDensityInKilogramsPerCubicMeter <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DisplacedVolumeInCubicMeters` | Double | ✅ | — | 0 | `0` | Volume spostato in m³ |
| `-FluidDensityInKilogramsPerCubicMeter` | Double | ✅ | — | 1 | `0` | Densità del fluido in kg/m³ |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Gravità in m/s² (predefinito: 9,81) |
| `-As` | String | — | — | 3 | — | Unità di output per la forza |
