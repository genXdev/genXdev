# Get-CurrentFocusedProcess

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Recupera l'oggetto processo della finestra che ha attualmente lo stato attivo della tastiera.

## Description

* Questo cmdlet utilizza chiamate API di Windows tramite P/Invoke per identificare e restituire l'oggetto Process associato alla finestra attualmente in primo piano.
* Sfrutta le funzioni GetForegroundWindow e GetWindowThreadProcessId della DLL User32.dll per determinare quale finestra ha il focus e ottenere il relativo ID di processo.

## Syntax

```powershell
Get-CurrentFocusedProcess [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$focusedProcess = Get-CurrentFocusedProcess
Write-Host "Active window process: $($focusedProcess.ProcessName)"
```

Recupera l'oggetto processo per la finestra che ha attualmente il focus della tastiera.
