# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Ottiene la chiave della lingua web predefinita in base alle impostazioni della lingua corrente del sistema.

## Description

* Recupera le impostazioni correnti della lingua e della cultura di sistema e le mappa
  alla corrispondente chiave del dizionario linguistico web utilizzata dai servizi
  di traduzione.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Restituisce "English" per un sistema inglese, "Dutch" per un sistema olandese, ecc.
