# Get-DefaultWebbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Restituisce il browser web predefinito configurato per l'utente corrente.

## Description

* Recupera informazioni sul browser web predefinito del sistema interrogando
  il Registro di sistema di Windows.
* Restituisce una tabella hash contenente il nome del browser, la descrizione, il percorso
  dell'icona e il percorso dell'eseguibile.
* La funzione controlla sia le preferenze utente che le registrazioni del browser a livello di sistema
  per determinare il browser predefinito.

## Syntax

```powershell
Get-DefaultWebbrowser [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebbrowser | Format-List
```

Ottieni informazioni dettagliate sul browser predefinito.

### Example 2

```powershell
$browser = Get-DefaultWebbrowser
& $browser.Path https://www.github.com/
```

Avvia il browser predefinito con un URL specifico.
