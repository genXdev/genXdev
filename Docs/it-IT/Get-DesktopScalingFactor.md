# Get-DesktopScalingFactor

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Recupera il fattore di scala di visualizzazione di Windows (impostazione DPI) per un monitor specificato.

## Description

* Ottiene il fattore di scalatura del display di Windows corrente configurato per un monitor nel sistema.
* Il fattore di scalatura è espresso come percentuale dove 100 rappresenta la scalatura standard (96 DPI). I valori comuni sono 100, 125, 150 e 200.
* Se non viene specificato alcun monitor, restituisce il fattore di scalatura per il display primario.

## Syntax

```powershell
Get-DesktopScalingFactor [[-Monitor] <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | — | — | 0 | `0` | L'indice del monitor da controllare (0 = monitor principale) |

## Examples

### Example 1

```powershell
Get-DesktopScalingFactor -Monitor 0
```

Restituisce la percentuale del fattore di ridimensionamento per il monitor principale.

### Example 2

```powershell
Get-DesktopScalingFactor 1
```

Restituisce la percentuale del fattore di scala per il secondo monitor utilizzando il parametro posizionale.
