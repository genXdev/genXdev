# Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Calcola la frequenza spostata per effetto Doppler.

## Description

Usa f' = f * (v + vo) / (v - vs), velocità positive verso l'altro.

## Syntax

```powershell
Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed -OriginalFrequencyInHertz <Double> -SourceSpeedInMetersPerSecond <Double> -ObserverSpeedInMetersPerSecond <Double> [[-As] <String>] [<CommonParameters>]

Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed [[-SpeedOfSoundInMetersPerSecond] <Double>] [<CommonParameters>]

Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-OriginalFrequencyInHertz` | Double | ✅ | — | 0 | `0` | Frequenza originale in Hz |
| `-SourceSpeedInMetersPerSecond` | Double | ✅ | — | 1 | `0` | Velocità della sorgente in m/s (positiva verso l'osservatore) |
| `-ObserverSpeedInMetersPerSecond` | Double | ✅ | — | 2 | `0` | Velocità dell'osservatore in m/s (positiva verso la sorgente) |
| `-SpeedOfSoundInMetersPerSecond` | Double | — | — | 3 | `0` | Velocità del suono in m/s (predefinito: 343) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 3 | — | Il medium *(Parameter set: )* |
| `-As` | String | — | — | 4 | — | Unità di output per la frequenza |

## Examples

### Example 1

```powershell
Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed -OriginalFrequencyInHertz 440 -SourceSpeedInMetersPerSecond 10 -ObserverSpeedInMetersPerSecond 5 -Medium "water" -As "kilohertz"
```

Calcola la frequenza shiftata per effetto Doppler per un tono di 440 Hz con sorgente che si muove a 10 m/s verso l'osservatore, osservatore che si muove a 5 m/s verso la sorgente, in ambiente acquatico, output in kilohertz.

### Example 2

```powershell
Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed 440 10 0 -SpeedOfSoundInMetersPerSecond 1480
```

Calcola la frequenza Doppler spostata per un tono di 440 Hz con sorgente in movimento a 10 m/s, osservatore fermo, velocità del suono 1480 m/s.
