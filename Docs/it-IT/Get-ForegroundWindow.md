# Get-ForegroundWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Ottiene l'handle della finestra in primo piano attualmente attiva.

## Description

* Questo cmdlet recupera l'handle della finestra (HWND) della finestra che è
  attualmente in primo piano e ha il focus della tastiera.
* Utilizza le funzioni API di Windows tramite P/Invoke per interagire con la
  libreria User32.dll.

## Syntax

```powershell
Get-ForegroundWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ForegroundWindow
```

Recupera e restituisce l'handle IntPtr della finestra attualmente attiva.

### Example 2

```powershell
$windowHandle = Get-ForegroundWindow
Write-Host "Active window handle: $windowHandle"
```

Memorizza l'handle della finestra in primo piano in una variabile e lo visualizza.
