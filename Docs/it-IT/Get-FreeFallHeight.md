# Get-FreeFallHeight

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Calcola l'altezza di caduta durante la caduta libera per una data durata temporale.

## Description

Calcola la distanza percorsa in caduta libera utilizzando un metodo numerico che tiene conto della resistenza dell'aria e della velocità terminale.

## Syntax

```powershell
Get-FreeFallHeight -DurationInSeconds <Double> [[-TerminalVelocityInMs] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | La durata della caduta in secondi |
| `-TerminalVelocityInMs` | Double | — | — | 1 | `0` | La velocità terminale in metri al secondo (predefinito: 53 m/s per un essere umano) |
| `-As` | String | — | — | 2 | — | L'unità per l'altezza dell'output |

## Examples

### Example 1

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -TerminalVelocityInMs 53
```

Calcola l'altezza caduta in 10 secondi con la velocità terminale umana predefinita.

### Example 2

```powershell
Get-FreeFallHeight 5
```

Calcola l'altezza caduta in 5 secondi utilizzando il parametro posizionale e la velocità terminale predefinita.

### Example 3

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -As "feet"
```

Calcola l'altezza caduta in 10 secondi e restituisce il risultato in piedi.
