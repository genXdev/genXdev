# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> Restituisce true se il motore di sintesi vocale sta parlando.

## Description

Controlla lo stato sia del sintetizzatore vocale predefinito che di quello personalizzato per determinare se uno dei due sta attualmente parlando. Restituisce true se il parlato è in corso, false in caso contrario.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Restituisce true se il motore di sintesi vocale sta parlando.

### Example 2

```powershell
iss
```

Restituisce true se il motore di sintesi vocale sta parlando (tramite alias).
