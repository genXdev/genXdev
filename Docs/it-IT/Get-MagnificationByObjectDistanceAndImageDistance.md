# Get-MagnificationByObjectDistanceAndImageDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Calcola l'ingrandimento per una lente.

## Description

Utilizza m = - (distanza dell'immagine / distanza dell'oggetto).

## Syntax

```powershell
Get-MagnificationByObjectDistanceAndImageDistance -ObjectDistanceInMeters <Double> -ImageDistanceInMeters <Double> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ObjectDistanceInMeters` | Double | ✅ | — | 0 | `0` | Distanza dell'oggetto in metri |
| `-ImageDistanceInMeters` | Double | ✅ | — | 1 | `0` | Distanza dell'immagine in metri |

## Examples

### Example 1

```powershell
Get-MagnificationByObjectDistanceAndImageDistance -ObjectDistanceInMeters 0.5 -ImageDistanceInMeters 1
```

Calcola l'ingrandimento con distanza dell'oggetto 0.5 m e distanza dell'immagine 1 m.

### Example 2

```powershell
Get-MagnificationByObjectDistanceAndImageDistance 0.3 0.6
```

Calcola l'ingrandimento utilizzando parametri posizionali.
