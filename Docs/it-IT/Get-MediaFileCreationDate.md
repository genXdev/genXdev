# Get-MediaFileCreationDate

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** —

## Synopsis

> Estrae la data di creazione ottimale per file multimediali e altri file.

## Description

Tenta diverse strategie per determinare una data di creazione o acquisizione accurata per il file specificato. Le strategie includono la lettura dei metadati EXIF dell'immagine, l'analisi delle informazioni di data/ora dai nomi dei file e il fallback all'ora dell'ultima scrittura del file quando non sono disponibili altre informazioni affidabili. La funzione restituisce sempre un [DateTime]; quando non è possibile determinare una data, restituisce [DateTime]::MinValue.

## Syntax

```powershell
Get-MediaFileCreationDate -FilePath <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | The `-FilePath` parameter. |

## Examples
