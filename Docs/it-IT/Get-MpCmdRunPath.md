# Get-MpCmdRunPath

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Ottiene il percorso dell'eseguibile MpCmdRun.exe di Windows Defender.

## Description

* Trova e restituisce il percorso completo dell'utilità della riga di comando di Windows Defender (MpCmdRun.exe).
* Il cmdlet controlla il percorso di installazione standard in Program Files e fornisce una gestione degli errori appropriata se il file non viene trovato.

## Syntax

```powershell
Get-MpCmdRunPath [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$defenderPath = Get-MpCmdRunPath
```

Recupera il percorso completo dello strumento da riga di comando di Windows Defender.
