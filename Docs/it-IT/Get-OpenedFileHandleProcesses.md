# Get-OpenedFileHandleProcesses

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Recupera i processi che hanno handle di file aperti su file specificati.

## Description

* Utilizza lo strumento handle.exe di Sysinternals per identificare i processi che
  hanno attualmente handle aperti a uno o più file specificati.
* Utile per determinare quali processi impediscono operazioni sui file
  come l'eliminazione o la modifica.
* Richiede handle.exe dalla suite Sysinternals, che deve essere installato e
  disponibile nel percorso di sistema.

## Syntax

```powershell
Get-OpenedFileHandleProcesses -FilePath <String[]> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String[] | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Percorso dei file da controllare per individuare handle aperti |

## Examples

### Example 1

```powershell
Get-OpenedFileHandleProcesses -FilePath "C:\temp\example.txt"
```

Identifica tutti i processi che hanno handle aperti al file specificato.

### Example 2

```powershell
"file1.txt", "file2.txt" | Get-OpenedFileHandleProcesses
```

Controlla più file tramite input della pipeline per processi con handle aperti.
