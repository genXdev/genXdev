# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Restituisce un oggetto helper per la finestra principale del terminale PowerShell.

## Description

* Recupera un oggetto helper WindowObj che rappresenta la finestra principale del processo host di PowerShell corrente.
* Ciò consente la manipolazione e l'interazione con la finestra del terminale stessa.

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

Recupera l'oggetto WindowObj per la finestra del terminale PowerShell corrente.
