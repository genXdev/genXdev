# Get-ProjectileRangeByInitialSpeedAndAngle

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Calcola la gittata di un proiettile.

## Description

Usa la formula R = (v² sin(2θ)) / g per il moto parabolico ideale.

## Syntax

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle -InitialSpeedInMetersPerSecond <Double> -AngleInDegrees <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InitialSpeedInMetersPerSecond` | Double | ✅ | — | 0 | `0` | Velocità iniziale in m/s |
| `-AngleInDegrees` | Double | ✅ | — | 1 | `0` | Angolo di lancio in gradi |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Gravità in m/s² (predefinito: 9,81) |
| `-As` | String | — | — | 3 | — | Unità di output per intervallo |

## Examples

### Example 1

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle -InitialSpeedInMetersPerSecond 20 -AngleInDegrees 45 -As "feet"
```

Calcola la gittata utilizzando la formula del moto del proiettile.

### Example 2

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle 30 30
```

Calcola l'intervallo con parametri posizionali.
