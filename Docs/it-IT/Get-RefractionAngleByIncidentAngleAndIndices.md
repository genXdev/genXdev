# Get-RefractionAngleByIncidentAngleAndIndices

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Calcola l'angolo di rifrazione utilizzando la legge di Snell.

## Description

Usa θ2 = arcsin( (n1 / n2) sin θ1 ).

## Syntax

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees <Double> -IndexOfRefraction1 <Double> -IndexOfRefraction2 <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-IncidentAngleInDegrees` | Double | ✅ | — | 0 | `0` | Angolo di incidenza in gradi |
| `-IndexOfRefraction1` | Double | ✅ | — | 1 | `0` | Indice di rifrazione del primo mezzo |
| `-IndexOfRefraction2` | Double | ✅ | — | 2 | `0` | Indice di rifrazione del secondo mezzo |
| `-As` | String | — | — | 3 | — | Unità di output per l'angolo |

## Examples

### Example 1

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees 30 -IndexOfRefraction1 1 -IndexOfRefraction2 1.33 -As "radians"
```

Calcola l'angolo di rifrazione quando la luce passa dall'aria (n=1) all'acqua (n=1.33) con un'incidenza di 30 gradi.

### Example 2

```powershell
Get-RefractionAngleByIncidentAngleAndIndices 45 1 1.5
```

Calcola l'angolo di rifrazione quando la luce passa dall'aria (n=1) al vetro (n=1.5) con incidenza a 45 gradi.
