# Get-SoundTravelDistanceByTime

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Calcola la distanza percorsa dal suono in un determinato tempo.

## Description

Utilizza la formula distanza = velocità * tempo, con la velocità predefinita del suono nell'aria.

## Syntax

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds <Double> [[-As] <String>] [<CommonParameters>]

Get-SoundTravelDistanceByTime [[-SpeedOfSoundInMetersPerSecond] <Double>] [<CommonParameters>]

Get-SoundTravelDistanceByTime -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-TimeInSeconds` | Double | ✅ | — | 0 | `0` | Il tempo in secondi |
| `-SpeedOfSoundInMetersPerSecond` | Double | — | — | 1 | `0` | Velocità del suono in m/s (predefinito: 343) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | Il mezzo attraverso cui il suono viaggia *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | L'unità per la distanza di output |

## Examples

### Example 1

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds 5 -Medium "water" -As "kilometers"
```

Calcola la distanza percorsa dal suono nell'acqua in 5 secondi e converte il risultato in chilometri.

### Example 2

```powershell
Get-SoundTravelDistanceByTime 10 -SpeedOfSoundInMetersPerSecond 1480
```

Calcola la distanza utilizzando una velocità del suono specifica.
