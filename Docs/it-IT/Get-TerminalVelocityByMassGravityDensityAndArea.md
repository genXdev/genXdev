# Get-TerminalVelocityByMassGravityDensityAndArea

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Calcola la velocità terminale.

## Description

Utilizza v = sqrt(2 m g / (ρ A C)).

## Syntax

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms <Double> [[-GravityInMetersPerSecondSquared] <Double>] -DensityInKilogramsPerCubicMeter <Double> -AreaInSquareMeters <Double> [[-DragCoefficient] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Massa in kg |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 1 | `0` | Gravità in m/s² (predefinito: 9,81) |
| `-DensityInKilogramsPerCubicMeter` | Double | ✅ | — | 2 | `0` | Densità del fluido in kg/m³ |
| `-AreaInSquareMeters` | Double | ✅ | — | 3 | `0` | Area della sezione trasversale in m² |
| `-DragCoefficient` | Double | — | — | 4 | `0` | Coefficiente di resistenza aerodinamica (valore predefinito: 0.5) |
| `-As` | String | — | — | 5 | — | Output unità per la velocità |

## Examples

### Example 1

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms 80 -DensityInKilogramsPerCubicMeter 1.225 -AreaInSquareMeters 0.7 -DragCoefficient 1.0 -As "mph"
```

Calcola la velocità terminale per una massa di 80 kg in aria con un coefficiente di resistenza di 1,0, restituendo il risultato in mph.

### Example 2

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea 70 1.225 0.8 0.8
```

Utilizza parametri posizionali per calcolare la velocità terminale.
