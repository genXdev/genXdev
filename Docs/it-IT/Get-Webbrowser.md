# Get-Webbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Restituisce una raccolta di browser web moderni installati.

## Description

* Scopre e restituisce dettagli sui browser web moderni installati nel sistema.
* Recupera informazioni tra cui nome, descrizione, percorso dell'icona, percorso eseguibile e stato del browser predefinito interrogando il registro di Windows.
* Restituisce solo i browser che hanno le capacità richieste registrate in Windows.

## Syntax

```powershell
Get-Webbrowser [-Chrome] [-Chromium] [-Edge] [-Firefox] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Edge` | SwitchParameter | — | — | Named | `False` | Seleziona le istanze del browser Microsoft Edge *(Parameter set: )* |
| `-Chrome` | SwitchParameter | — | — | Named | `False` | Seleziona le istanze del browser Google Chrome *(Parameter set: )* |
| `-Chromium` | SwitchParameter | — | — | Named | `False` | Seleziona il browser predefinito basato su Chromium *(Parameter set: )* |
| `-Firefox` | SwitchParameter | — | — | Named | `False` | Seleziona le istanze del browser Firefox *(Parameter set: )* |

## Examples

### Example 1

```powershell
Get-Webbrowser | Select-Object Name, Description | Format-Table
```

Restituisce una raccolta di tutti i browser web moderni installati.

### Example 2

```powershell
Get-Webbrowser | Where-Object { $_.IsDefaultBrowser }
```

Filtri per mostrare solo il browser predefinito del sistema.
