# Import-BrowserBookmarks

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Importa i preferiti da un file o da una raccolta in un browser web.

## Description

Importa i segnalibri in Microsoft Edge o Google Chrome da un file CSV o da una raccolta di oggetti segnalibro. I segnalibri vengono aggiunti alla barra dei segnalibri del browser o a cartelle specificate. Al momento l'importazione da Firefox non è supportata.

## Syntax

```powershell
Import-BrowserBookmarks [[-InputFile] <String>] [<CommonParameters>]

Import-BrowserBookmarks [[-Bookmarks] <Array>] [<CommonParameters>]

Import-BrowserBookmarks [-Chrome] [-Edge] [-Firefox] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputFile` | String | — | — | 0 | — | Percorso del file CSV con i segnalibri da importare *(Parameter set: )* |
| `-Bookmarks` | Array | — | — | 0 | — | Raccolta di segnalibri da importare *(Parameter set: )* |
| `-Chrome` | SwitchParameter | — | — | Named | — | Importa in Google Chrome |
| `-Edge` | SwitchParameter | — | — | Named | — | Importa in Microsoft Edge |
| `-Firefox` | SwitchParameter | — | — | Named | — | Importa in Firefox (non supportato) |

## Examples

### Import-BrowserBookmarks -InputFile "C:\MyBookmarks.csv" -Edge Imports bookmarks from the CSV file into Microsoft Edge.

```powershell
Import-BrowserBookmarks -InputFile "C:\MyBookmarks.csv" -Edge
Imports bookmarks from the CSV file into Microsoft Edge.
```

### $bookmarks = @(     @{         Name = "Microsoft";         URL = "https://microsoft.com";         Folder = "Tech"     } ) Import-BrowserBookmarks -Bookmarks $bookmarks -Chrome Imports a collection of bookmarks into Google Chrome.

```powershell
$bookmarks = @(
    @{
        Name = "Microsoft";
        URL = "https://microsoft.com";
        Folder = "Tech"
    }
)
Import-BrowserBookmarks -Bookmarks $bookmarks -Chrome
Imports a collection of bookmarks into Google Chrome.
```
