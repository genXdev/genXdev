# Initialize-ScheduledTaskScripts

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Crea attività pianificate che eseguono script PowerShell a intervalli specificati.

## Description

Crea e configura attività pianificate che eseguono script PowerShell a vari intervalli, inclusi:
- Avvio del sistema
- Accesso utente
- Ogni ora di giorni specifici (es. lunedì alle 13:00)
- Giornalmente a ore specifiche (es. ogni giorno alle 15:00)
Ogni attività viene eseguita con privilegi elevati nel contesto dell'utente corrente.

## Syntax

```powershell
Initialize-ScheduledTaskScripts [[-FilePath] <String>] [[-Prefix] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | — | — | 0 | `''` | Il percorso della directory in cui verranno creati gli script delle attività |
| `-Prefix` | String | — | — | 1 | `'PS'` | Prefisso per i nomi delle attività pianificate |

## Examples

### Initialize-ScheduledTaskScripts -FilePath "C:\Tasks" -Prefix "MyTasks"

```powershell
Initialize-ScheduledTaskScripts -FilePath "C:\Tasks" -Prefix "MyTasks"
```

### Initialize-ScheduledTaskScripts

```powershell
Initialize-ScheduledTaskScripts
```
