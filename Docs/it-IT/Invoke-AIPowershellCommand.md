# Invoke-AIPowershellCommand

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `hint`

## Synopsis

> Genera ed esegue comandi PowerShell utilizzando l'assistenza AI.

## Description

Utilizza modelli di intelligenza artificiale per generare comandi PowerShell basati su query in linguaggio naturale. La funzione può inviare i comandi direttamente alla finestra di PowerShell o copiarli negli appunti. Sfrutta modelli di intelligenza artificiale per interpretare il linguaggio naturale e convertirlo in comandi PowerShell eseguibili con supporto completo per vari backend di intelligenza artificiale.

## Syntax

```powershell
Invoke-AIPowershellCommand -PowershellCmdline <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-LLMQueryType <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-PowershellCmdline` | String | ✅ | — | 0 | — | La query in linguaggio naturale per generare un comando per |
| `-LLMQueryType` | String | — | — | Named | `'Coding'` | Il tipo di query LLM |
| `-Model` | String | — | — | Named | — | L'identificatore o modello da utilizzare per le operazioni AI |
| `-ApiEndpoint` | String | — | — | Named | — | L'URL dell'endpoint API per le operazioni di IA |
| `-ApiKey` | String | — | — | Named | — | La chiave API per le operazioni AI autenticate |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | Indica che LLM non supporta gli schemi JSON |

## Examples

### Invoke-AIPowershellCommand -Query "list all running processes"

```powershell
Invoke-AIPowershellCommand -Query "list all running processes"
```

Genera un comando PowerShell per elencare i processi in esecuzione

### hint "list files modified today"

```powershell
hint "list files modified today"
```

Utilizza l'alias per generare un comando per trovare i file modificati oggi.

### Invoke-AIPowershellCommand -Query "stop service" -Clipboard

```powershell
Invoke-AIPowershellCommand -Query "stop service" -Clipboard
```

Genera un comando per arrestare un servizio e lo copia negli appunti.
