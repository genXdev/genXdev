# Invoke-ImageObjectsUpdate

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `imageobjectdetection`

## Synopsis

> Aggiorna i metadati di rilevamento oggetti per file immagine in una directory specificata.

## Description

Questa funzione elabora le immagini in una directory specificata per rilevare oggetti utilizzando l'intelligenza artificiale. Crea file JSON di metadati contenenti gli oggetti rilevati, le loro posizioni, i punteggi di confidenza e le etichette. La funzione supporta l'elaborazione batch con soglie di confidenza configurabili e può opzionalmente saltare i file di metadati esistenti o riprovare le rilevazioni fallite in precedenza.

## Syntax

```powershell
Invoke-ImageObjectsUpdate [[-ImageDirectories] <String[]>] [-ClearSession] [-Language <String>] [-LLMQueryType <String>] [-Model <String>] [-OnlyNew] [-PreferencesDatabasePath <String>] [-Recurse] [-RetryFailed] [-SessionOnly] [-SkipSession] [-TimeoutSeconds <Int32>] [<CommonParameters>]
```

## Examples

### Invoke-ImageObjectsUpdate -ImageDirectories @("C:\Photos", "D:\Pictures") -Recurse

```powershell
Invoke-ImageObjectsUpdate -ImageDirectories @("C:\Photos", "D:\Pictures") -Recurse
```

Questo esempio elabora tutte le immagini in C:\Photos e D:\Pictures e tutte le sottodirectory utilizzando le impostazioni predefinite con soglia di confidenza 0.5.

### Invoke-ImageObjectsUpdate @("C:\Photos", "C:\Archive") -RetryFailed -OnlyNew

```powershell
Invoke-ImageObjectsUpdate @("C:\Photos", "C:\Archive") -RetryFailed -OnlyNew
```

Questo esempio elabora solo le nuove immagini e riprova quelle fallite in precedenza in più directory utilizzando la sintassi dei parametri posizionali.

### Invoke-ImageObjectsUpdate -ImageDirectories "C:\Photos" -UseGPU `     -ConfidenceThreshold 0.7

```powershell
Invoke-ImageObjectsUpdate -ImageDirectories "C:\Photos" -UseGPU `
    -ConfidenceThreshold 0.7
```

Questo esempio utilizza l'accelerazione GPU con una soglia di confidenza più alta di 0.7 per rilevamenti di oggetti più accurati ma in numero inferiore.
