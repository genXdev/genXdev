# Merge-TranslationCache

> **Module:** GenXdev.AI | **Type:** Function | **Aliases:** —

## Synopsis

> Gestisce la cache di traduzione persistente con scritture su disco in batch per lingua.

## Description

Mantiene una cache di traduzione in memoria con ambito modulo per Get-TextTranslation. La cache è suddivisa in file JSON per lingua sotto $env:LOCALAPPDATA\GenXdev.PowerShell\. Le chiamate per aggiungere voci vengono raggruppate — le scritture su disco avvengono solo ogni 100 mutazioni per lingua, o quando viene chiamato -PersistNow. -PersistNow cancella anche la cache in memoria per prevenire una crescita illimitata tra i batch di elaborazione.

## Syntax

```powershell
Merge-TranslationCache [-GetCache] [<CommonParameters>]

Merge-TranslationCache -Key <String> -Language <String> -Value <String> [<CommonParameters>]

Merge-TranslationCache -PersistNow [<CommonParameters>]

Merge-TranslationCache -ClearCache [<CommonParameters>]

Merge-TranslationCache -PurgeFromCache [<CommonParameters>]
```
