# Now

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Restituisce la data e l'ora correnti del sistema come oggetto DateTime.

## Description

Fornisce un modo semplice per ottenere la data e l'ora correnti del sistema senza parametri. Restituisce un oggetto DateTime standard .NET che può essere utilizzato per calcoli di data/ora, formattazione e confronti.

## Syntax

```powershell
Now [<CommonParameters>]
```

## Examples

### Now Returns the current system date and time as a DateTime object

```powershell
Now
Returns the current system date and time as a DateTime object
```

### $timestamp = Now Stores the current date and time in a variable for later use

```powershell
$timestamp = Now
Stores the current date and time in a variable for later use
```

## Outputs

- `DateTime`
