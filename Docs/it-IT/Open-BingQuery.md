# Open-BingQuery

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** `bq`

## Synopsis

> Apre una query di ricerca Bing in un browser web.

## Description

Apre una o più query di ricerca Bing in un browser web, utilizzando impostazioni del browser e opzioni della riga di comando configurabili. Questa funzione fornisce un'interfaccia completa per eseguire ricerche Bing con ampie opzioni di personalizzazione del browser, inclusi posizionamento della finestra, selezione del monitor, navigazione privata e automazione della digitazione.

La funzione codifica automaticamente le query in formato URL e le formatta per l'API di ricerca Bing. Supporta tutti i browser principali tra cui Edge, Chrome e Firefox con funzionalità come navigazione privata, modalità applicazione, gestione precisa delle finestre e automazione della digitazione.

## Syntax

```powershell
Open-BingQuery -Queries <String[]> [[-Language] <String>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-Monitor <Int32>] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-BingQuery -Queries "PowerShell scripting" -Monitor 0

```powershell
Open-BingQuery -Queries "PowerShell scripting" -Monitor 0
```

Apre una ricerca Bing per "PowerShell scripting" sul monitor predefinito.

### bq "PowerShell scripting" -m 0

```powershell
bq "PowerShell scripting" -m 0
```

Apre una ricerca Bing utilizzando l'alias e i nomi dei parametri brevi.

### Open-BingQuery -Queries "machine learning" -Language "English" -Private

```powershell
Open-BingQuery -Queries "machine learning" -Language "English" -Private
```

Apre una sessione di navigazione privata per cercare "machine learning" con risultati in lingua inglese.

### "PowerShell", "automation", "scripting" | Open-BingQuery -All -Left

```powershell
"PowerShell", "automation", "scripting" | Open-BingQuery -All -Left
```

Cerca più termini tramite pipeline e apre i risultati in tutti i browser installati, posizionando le finestre sul lato sinistro dello schermo.
