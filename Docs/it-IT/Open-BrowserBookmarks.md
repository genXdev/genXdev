# Open-BrowserBookmarks

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `sites`

## Synopsis

> Apre i segnalibri del browser che corrispondono ai criteri di ricerca specificati.

## Description

Cerca segnalibri tra Microsoft Edge, Google Chrome e Mozilla Firefox in base alle query di ricerca fornite. Apre i segnalibri corrispondenti nel browser selezionato con impostazioni configurabili della finestra e modalità del browser.

Questa funzione fornisce un'interfaccia completa per trovare e aprire segnalibri del browser con opzioni avanzate di filtro e visualizzazione. Supporta più criteri di ricerca e può aprire i risultati in qualsiasi browser installato con ampia personalizzazione del posizionamento della finestra e del comportamento.

## Syntax

```powershell
Open-BrowserBookmarks [[-Queries] <String[]>] [[-Count] <Int32>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-Minimize] [-Monitor <Int32>] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PlayWright] [-Private] [-RestoreFocus] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-ShowWindow] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-BrowserBookmarks -Queries "github" -Edge -Count 5

```powershell
Open-BrowserBookmarks -Queries "github" -Edge -Count 5
```

Cerca i segnalibri contenenti "github" in Microsoft Edge e apre i primi 5 risultati nel browser predefinito.

### sites gh -e -c 5

```powershell
sites gh -e -c 5
```

Come sopra usando alias - cerca "gh" nei segnalibri di Edge e apre 5 risultati nel browser predefinito.

### Open-BrowserBookmarks -Queries "development", "tools" -Chrome -Firefox -Left -Count 10

```powershell
Open-BrowserBookmarks -Queries "development", "tools" -Chrome -Firefox -Left -Count 10
```

Cerca nei segnalibri di Chrome "development" e "tools", apre i primi 10 risultati in Firefox posizionato sul lato sinistro dello schermo.
