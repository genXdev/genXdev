# Open-SearchEngine

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** `q`

## Synopsis

> Apre una query di ricerca nel motore di ricerca specificato utilizzando un browser web.

## Description

Questa funzione fornisce un'interfaccia unificata per cercare attraverso più motori di ricerca inclusi Google, Bing, GitHub, Wikipedia, YouTube e altri. Supporta opzioni estese di configurazione del browser tra cui posizionamento della finestra, dimensioni, selezione del browser e preferenze linguistiche. La funzione indirizza dinamicamente le query alle funzioni specifiche del motore di ricerca appropriato.

## Syntax

```powershell
Open-SearchEngine -Queries <String[]> [[-EndPoint] <String>] [[-Language] <String>] [[-Monitor] <Int32>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-NewWindow] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SetForeground] [-SetRestored] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-SearchEngine -Queries "PowerShell cmdlets" -EndPoint "Google" -Language "English"

```powershell
Open-SearchEngine -Queries "PowerShell cmdlets" -EndPoint "Google" -Language "English"
```

Cerca "cmdlet di PowerShell" su Google con risultati in lingua inglese.

### q "GitHub PowerShell" Bing -Chrome -Private

```powershell
q "GitHub PowerShell" Bing -Chrome -Private
```

Utilizza l'alias 'q' per cercare "GitHub PowerShell" su Bing usando Chrome in modalità privata.
