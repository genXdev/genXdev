# Receive-RealTimeSpeechToText

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Converte l'input audio in tempo reale in testo utilizzando il modello Whisper AI.

## Description

Questo cmdlet acquisisce audio dal microfono o dal desktop e lo trascrive in testo in tempo reale utilizzando il modello Whisper AI. Supporta varie sorgenti audio, rilevamento del silenzio e multiple opzioni di configurazione per il riconoscimento vocale.

## Examples

### Example 1

```powershell
Receive-RealTimeSpeechToText
```

Utilizzo base con microfono.

### Example 2

```powershell
Receive-RealTimeSpeechToText -UseDesktopAudioCapture
```

Utilizzo della cattura audio del desktop.

### Example 3

```powershell
Receive-RealTimeSpeechToText -AudioDevice "Microphone*"
```

Utilizzo di un dispositivo audio specifico.

### Example 4

```powershell
Receive-RealTimeSpeechToText -MaxDurationOfSilence "00:00:05"
```

Con rilevamento del silenzio.
