# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Pronuncia la data corrente utilizzando la sintesi vocale.

## Description

Converte la data corrente in un formato in linguaggio naturale e la annuncia utilizzando il motore di sintesi vocale del sistema. La data viene pronunciata nel formato: "Oggi è [giorno della settimana], [mese] [giorno] [anno]"

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
