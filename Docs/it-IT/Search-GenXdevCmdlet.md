# Search-GenXdevCmdlet

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `searchcmdlet`

## Synopsis

> Cerca un cmdlet di GenXdev e lo apre facoltativamente in un IDE per la modifica.

## Description

Questa funzione cerca i cmdlet di GenXdev utilizzando un modello e apre facoltativamente il file sorgente del cmdlet trovato in Visual Studio Code o Visual Studio per la modifica. Recupera le informazioni sui cmdlet e fornisce scorciatoie da tastiera per navigare rapidamente alla definizione della funzione.

## Syntax

```powershell
Search-GenXdevCmdlet [[-CmdletName] <String>] [[-ModuleName] <String[]>] [-Code] [-EditCmdlet] [-FromScripts] [-NoLocal] [-OnlyPublished] [-VisualStudio] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CmdletName` | String | — | — | 0 | — | Modello di ricerca per filtrare i cmdlet 🌐 *Supports wildcards* |
| `-ModuleName` | String[] | — | ✅ (ByValue, ByPropertyName) | 1 | — | Nomi dei moduli GenXdev da cercare 🌐 *Supports wildcards* |
| `-NoLocal` | SwitchParameter | — | — | Named | — | Salta la ricerca nei percorsi dei moduli locali |
| `-OnlyPublished` | SwitchParameter | — | — | Named | — | Limita la ricerca ai soli percorsi dei moduli pubblicati |
| `-FromScripts` | SwitchParameter | — | — | Named | — | Cerca nei file script invece dei file modulo |
| `-Code` | SwitchParameter | — | — | Named | — | Apri il cmdlet trovato in Visual Studio Code |
| `-VisualStudio` | SwitchParameter | — | — | Named | — | Apri il cmdlet trovato in Visual Studio |
| `-EditCmdlet` | SwitchParameter | — | — | Named | — | Apre anche il file nell'editor dopo averlo trovato |

## Examples

### Search-GenXdevCmdlet -CmdletName "Get-*" -Code Opens first matching cmdlet starting with "Get-" in Visual Studio Code.

```powershell
Search-GenXdevCmdlet -CmdletName "Get-*" -Code
Opens first matching cmdlet starting with "Get-" in Visual Studio Code.
```

### searchcmdlet "Set-GitConfig" -EditCmdlet Searches for Set-GitConfig cmdlet and opens it for editing.

```powershell
searchcmdlet "Set-GitConfig" -EditCmdlet
Searches for Set-GitConfig cmdlet and opens it for editing.
```
