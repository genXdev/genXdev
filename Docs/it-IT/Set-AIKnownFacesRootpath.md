# Set-AIKnownFacesRootpath

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> Imposta la directory per i file di immagini dei volti utilizzati nelle operazioni di GenXdev.AI.

## Description

Questa funzione configura la directory globale dei volti utilizzata dal modulo GenXdev.AI per varie operazioni di riconoscimento facciale e AI. Le impostazioni possono essere memorizzate in modo persistente nelle preferenze (predefinito), solo nella sessione corrente (usando -SessionOnly), o cancellate dalla sessione (usando -ClearSession).

## Syntax

```powershell
Set-AIKnownFacesRootpath [[-FacesDirectory] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FacesDirectory` | String | — | — | 0 | — | Percorso della directory per i file immagine del volto |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Percorso del database per i file dei dati delle preferenze |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Utilizza impostazioni alternative memorizzate nella sessione per le preferenze dell'IA, come lingua, raccolte di immagini, ecc. |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Cancella le impostazioni alternative memorizzate nella sessione per le preferenze AI come Lingua, Raccolte immagini, ecc. |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Non utilizzare impostazioni alternative memorizzate nella sessione per le preferenze di IA come lingua, raccolte di immagini, ecc. |

## Examples

### Set-AIKnownFacesRootpath -FacesDirectory "C:\Faces"

```powershell
Set-AIKnownFacesRootpath -FacesDirectory "C:\Faces"
```

Imposta la directory dei volti in modo persistente nelle preferenze.

### Set-AIKnownFacesRootpath "C:\FacePictures"

```powershell
Set-AIKnownFacesRootpath "C:\FacePictures"
```

Imposta la directory dei volti in modo persistente nelle preferenze.

### Set-AIKnownFacesRootpath -FacesDirectory "C:\TempFaces" -SessionOnly

```powershell
Set-AIKnownFacesRootpath -FacesDirectory "C:\TempFaces" -SessionOnly
```

Imposta la directory dei volti solo per la sessione corrente (variabile globale).

### Set-AIKnownFacesRootpath -ClearSession

```powershell
Set-AIKnownFacesRootpath -ClearSession
```

Cancella l'impostazione della directory dei volti della sessione (variabile globale) senza influenzare le preferenze persistenti.
