# Set-AIMetaLanguage

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> Imposta la lingua predefinita e opzionalmente le directory delle immagini per le operazioni di metadata delle immagini di GenXdev.AI.

## Description

Questa funzione configura la lingua predefinita globale per le operazioni sui metadati delle immagini nel modulo GenXdev.AI. Opzionalmente, può anche impostare le directory globali delle immagini. Entrambe le impostazioni vengono persistenti nell'archivio delle preferenze del modulo per l'uso tra sessioni.

## Syntax

```powershell
Set-AIMetaLanguage [[-Language] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Language` | String | — | — | 0 | — | La lingua predefinita per le operazioni sui metadati delle immagini |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Percorso del database per i file dei dati delle preferenze |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Utilizza impostazioni alternative memorizzate nella sessione per le preferenze dell'IA, come lingua, raccolte di immagini, ecc. |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Cancella le impostazioni alternative memorizzate nella sessione per le preferenze AI come Lingua, Raccolte immagini, ecc. |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Non utilizzare impostazioni alternative memorizzate nella sessione per le preferenze di IA come lingua, raccolte di immagini, ecc. |

## Examples

### Set-AIMetaLanguage -Language "Spanish" -ImageDirectories @("C:\Images", "D:\Photos")

```powershell
Set-AIMetaLanguage -Language "Spanish" -ImageDirectories @("C:\Images", "D:\Photos")
```

Imposta in modo persistente le directory delle lingue e delle immagini nelle preferenze.

### Set-AIMetaLanguage "French"

```powershell
Set-AIMetaLanguage "French"
```

Imposta la lingua in modo persistente nelle preferenze.

### Set-AIMetaLanguage -Language "German" -SessionOnly

```powershell
Set-AIMetaLanguage -Language "German" -SessionOnly
```

Imposta la lingua solo per la sessione corrente (variabile globale).

### Set-AIMetaLanguage -ClearSession

```powershell
Set-AIMetaLanguage -ClearSession
```

Cancella l'impostazione della lingua della sessione (variabile globale) senza influenzare le preferenze persistenti.
