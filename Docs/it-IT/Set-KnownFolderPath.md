# Set-KnownFolderPath

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Modifica il percorso fisico di una cartella conosciuta di Windows.

## Description

Utilizza l'API Shell32 di Windows per riposizionare le cartelle di sistema come Documenti, Download, Desktop o altre cartelle note di Windows in una nuova posizione. La funzione convalida l'esistenza del percorso di destinazione, cerca il GUID univoco della cartella nel registro completo delle cartelle note e utilizza l'API SHSetKnownFolderPath per eseguire il riposizionamento. I casi d'uso comuni includono lo spostamento delle cartelle utente su un'unità diversa per la gestione dello spazio o l'organizzazione. Prestare attenzione quando si spostano cartelle critiche per il sistema, poiché ciò potrebbe influire sulla stabilità del sistema.

## Syntax

```powershell
Set-KnownFolderPath -KnownFolder <String> -Path <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-KnownFolder` | String | ✅ | — | 0 | — | Cartella nota da modificare |
| `-Path` | String | ✅ | — | 1 | — | Nuovo percorso per la cartella nota |

## Examples

### Set-KnownFolderPath -KnownFolder 'Documents' -Path 'D:\UserDocs'

```powershell
Set-KnownFolderPath -KnownFolder 'Documents' -Path 'D:\UserDocs'
```

### Set-KnownFolderPath Downloads 'E:\Downloads'

```powershell
Set-KnownFolderPath Downloads 'E:\Downloads'
```
