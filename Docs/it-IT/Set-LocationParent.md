# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> Passa alla directory padre e ne elenca il contenuto.

## Description

* Passa al livello di directory superiore nella gerarchia del file system.
* Visualizza il contenuto della nuova directory corrente.
* Fornisce un alias conveniente '..' per una navigazione rapida tra le directory.

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

Passa alla directory padre e ne elenca il contenuto.

### Example 2

```powershell
..
```

Utilizza l'alias per passare alla directory padre.
