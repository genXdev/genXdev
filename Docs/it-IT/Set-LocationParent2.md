# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> Passa su di due livelli di directory nella gerarchia del file system.

## Description

* Modifica la directory di lavoro corrente alla directory del nonno (due livelli superiori).
* Visualizza il contenuto della directory risultante.

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Passa alla directory madre e ne visualizza il contenuto.

### Example 2

```powershell
...
```

Utilizza l'alias '...' per la stessa operazione.
