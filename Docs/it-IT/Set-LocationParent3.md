# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....`

## Synopsis

> Passa su tre livelli di directory nella gerarchia del file system.

## Description

* Modifica la directory di lavoro corrente spostandosi verso l'alto di tre directory genitore nella gerarchia del filesystem.
* Visualizza il contenuto della directory risultante.

## Syntax

```powershell
Set-LocationParent3 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

Passa su tre livelli di directory a partire dalla posizione corrente.

### Example 2

```powershell
....
```

Utilizza l'alias per salire di tre livelli nella struttura delle directory.
