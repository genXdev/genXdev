# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> Sposta su cinque livelli di directory nella gerarchia del file system.

## Description

* Sposta la directory di lavoro corrente verso l'alto di cinque livelli.
* Mostra il contenuto della directory risultante.
* Fornisce un alias abbreviato conveniente '......' per una navigazione rapida.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Sposta su di cinque livelli di directory dalla posizione corrente.

### Example 2

```powershell
......
```

Utilizza l'alias per passare a cinque livelli di directory superiori.
