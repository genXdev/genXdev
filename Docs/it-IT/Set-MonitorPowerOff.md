# Set-MonitorPowerOff

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `poweroff`

## Synopsis

> Disabilita l'alimentazione di tutti i monitor collegati.

## Description

Utilizza chiamate API di Windows per inviare un segnale di spegnimento a tutti i monitor collegati. Questo equivale a premere il pulsante fisico di accensione sui monitor. I monitor entreranno in modalità di risparmio energetico ma possono essere riattivati muovendo il mouse o premendo un tasto.

## Syntax

```powershell
Set-MonitorPowerOff [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOff
```

Spegne tutti i monitor collegati.

### Example 2

```powershell
poweroff
```

Spegne tutti i monitor collegati utilizzando l'alias.
