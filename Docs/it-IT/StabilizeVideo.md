# StabilizeVideo

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** —

## Synopsis

> Stabilizza il file .mp4 più recente o specificato utilizzando FFmpeg + vid.stab (senza bordi neri).

## Description

Stabilizzazione vid.stab a due passaggi con optzoom=1 (zoom/ritaglio automatico per non mostrare mai i bordi) più leggera nitidezza. Funziona esattamente come il tuo cmdlet Save-Video — semplicissimo.

## Syntax

```powershell
StabilizeVideo [[-InputFile] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputFile` | String | — | — | 0 | — | The `-InputFile` parameter. |

## Examples

### Stabilize-Video                  # stabilizes the most recent .mp4     Stabilize-Video shaky.mp4         # specific file

```powershell
Stabilize-Video                  # stabilizes the most recent .mp4
    Stabilize-Video shaky.mp4         # specific file
```
