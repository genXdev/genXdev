# Start-VlcMediaPlayerNextInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcnext`

## Synopsis

> Porta VLC Media Player all'elemento successivo nella playlist corrente.

## Description

Questa funzione invia la pressione del tasto 'n' a VLC Media Player per passare alla traccia o all'elemento multimediale successivo nella playlist attualmente caricata. Mette a fuoco la finestra di VLC, invia il comando di avanzamento e ripristina il focus sulla console PowerShell. La funzione include il supporto ShouldProcess per i prompt di conferma quando necessario.

## Syntax

```powershell
Start-VlcMediaPlayerNextInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerNextInPlaylist

```powershell
Start-VlcMediaPlayerNextInPlaylist
```

### vlcnext

```powershell
vlcnext
```
