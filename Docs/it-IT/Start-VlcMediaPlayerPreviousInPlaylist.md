# Start-VlcMediaPlayerPreviousInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcprev`, `vlcback`

## Synopsis

> Passa all'elemento precedente nella playlist di VLC Media Player.

## Description

Questa funzione invia il comando del tasto 'p' a VLC Media Player per passare all'elemento precedente nella playlist corrente. La funzione supporta le operazioni WhatIf e ripristinerà lo stato attivo dopo aver inviato il comando.

## Syntax

```powershell
Start-VlcMediaPlayerPreviousInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerPreviousInPlaylist

```powershell
Start-VlcMediaPlayerPreviousInPlaylist
```

### vlcprev

```powershell
vlcprev
```

### vlcback

```powershell
vlcback
```
