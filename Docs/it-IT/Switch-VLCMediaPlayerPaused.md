# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> Alterna lo stato di pausa/riproduzione di VLC Media Player.

## Description

Questa funzione invia un tasto spazio a VLC Media Player per alternare tra stato in pausa e in riproduzione. Ripristina automaticamente il focus sulla finestra originale dopo aver inviato il comando del tasto.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

Commuta lo stato di pausa/riproduzione di VLC Media Player.

### vlcpause

```powershell
vlcpause
```

Usa l'alias per attivare/disattivare lo stato di pausa/riproduzione.

### vlcplay

```powershell
vlcplay
```

Utilizza l'alias alternativo per attivare/disattivare lo stato di pausa/riproduzione.
