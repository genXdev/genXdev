# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> Attiva/disattiva la modalità di ripetizione in VLC Media Player.

## Description

Questa funzione invia il comando del tasto 'r' a VLC Media Player per alternare tra diverse modalità di ripetizione (nessuna ripetizione, ripeti corrente, ripeti tutto). La funzione apre VLC se non è già in esecuzione e ripristina il focus alla finestra precedente dopo aver inviato il comando.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
