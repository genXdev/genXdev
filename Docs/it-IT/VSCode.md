# VSCode

> **Module:** GenXdev.Coding | **Type:** Function | **Aliases:** —

## Synopsis

> Apre uno o più file in Visual Studio Code.

## Description

Questa funzione prende i percorsi dei file e li apre in Visual Studio Code. Espande i percorsi e verifica l'esistenza del file prima di tentare di aprirlo. La funzione supporta sia percorsi di file diretti che input tramite pipeline, rendendola ideale per aprire rapidamente più file da ricerche nel terminale.

## Syntax

```powershell
VSCode -FilePath <String[]> [-Copilot] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String[] | ✅ | ✅ (ByValue) | 0 | — | Il percorso del file da aprire in VSCode |
| `-Copilot` | SwitchParameter | — | — | Named | — | Aggiungi file sorgente alla sessione di modifica di Copilot |

## Examples

### VSCode -FilePath "C:\path\to\file.txt" -Copilot

```powershell
VSCode -FilePath "C:\path\to\file.txt" -Copilot
```

### Get-ChildItem *.js -Recurse | VSCode

```powershell
Get-ChildItem *.js -Recurse | VSCode
```
