# WriteFileOutput

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Restituisce le informazioni sui file con nomi visualizzati con collegamenti ipertestuali per migliorare l'esperienza dell'utente.

## Description

Questa funzione elabora gli oggetti di input dei file e li restituisce in un formato intuitivo con nomi di file collegati ipertestualmente quando visualizzati nella console. Gestisce diversi tipi di input tra cui stringhe, oggetti FileInfo e altri tipi di oggetti. La funzione rileva automaticamente se l'output viene reindirizzato o acquisito in una pipeline e adatta il suo comportamento di conseguenza.

## Syntax

```powershell
WriteFileOutput -CallerInvocation <Object> -Input <Object> [-FullPaths] [-Prefix <String>] [-RelativeBasePath <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CallerInvocation` | Object | ✅ | ✅ (ByValue) | 0 | — | Le informazioni di chiamata dalla funzione chiamante |
| `-Input` | Object | ✅ | ✅ (ByValue) | 1 | — | L'oggetto di input da elaborare, che può essere un percorso file o un oggetto |
| `-Prefix` | String | — | — | Named | — | Un prefisso stringa opzionale da anteporre all'output visualizzato per fornire ulteriore contesto |
| `-RelativeBasePath` | String | — | — | Named | — | Percorso di base per generare percorsi di file relativi nell'output |
| `-FullPaths` | SwitchParameter | — | — | Named | — | Forza l'output a utilizzare percorsi assoluti completi invece di percorsi relativi |

## Examples

### WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"

```powershell
WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"
```

### Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_

```powershell
Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_
```
