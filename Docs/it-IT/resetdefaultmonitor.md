# resetdefaultmonitor

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Ripristina la configurazione predefinita del monitor secondario.

## Description

Questo script ripristina la configurazione predefinita del monitor secondario per il sistema,
impostando il monitor secondario al valore predefinito originale.
Ciò è utile per gli utenti che desiderano tornare alla precedente configurazione multi-monitor dopo aver utilizzato configurazioni affiancate.
Vedi anche: funzione 'sidebyside' per passare alla modalità affiancata per le nuove finestre.

## Syntax

```powershell
resetdefaultmonitor [<CommonParameters>]
```

## Examples

### secondscreen

```powershell
secondscreen
```

Ripristina la configurazione predefinita del monitor secondario per il sistema.
