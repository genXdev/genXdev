# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Imposta la configurazione predefinita del secondo monitor.

## Description

Imposta il comportamento predefinito per l'apertura delle finestre di GenXdev sul monitor secondario.
Ciò è utile per gli utenti con un singolo monitor o per coloro che preferiscono layout di finestre affiancate.
Vedere anche il cmdlet 'sidebyside' e 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Imposta le impostazioni predefinite per l'apertura delle finestre di GenXdev sul monitor secondario
