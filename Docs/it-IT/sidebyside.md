# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Imposta la configurazione predefinita affiancata.

## Description

Imposta il comportamento predefinito per l'apertura delle finestre di GenXdev in modo che siano affiancate a PowerShell.
Utile per utenti con un singolo monitor o che preferiscono layout di finestre affiancate.
Vedere anche il cmdlet 'secondscreen' e 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

Imposta le impostazioni predefinite per le finestre di GenXdev in modo che siano affiancate a PowerShell
