# Convert-DotNetTypeToLLMType

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> .NET 型名を LLM（言語モデル）型名に変換します。

## Description

.NET 型名を入力として受け取り、言語モデルで使用される対応する簡略化された型名を返します。一般的な.NET型を処理し、適切な型マッピングを提供します。

## Syntax

```powershell
Convert-DotNetTypeToLLMType -DotNetType <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DotNetType` | String | ✅ | — | 0 | — | LLM型名に変換する.NET型名 |

## Examples

### Example 1

```powershell
Convert-DotNetTypeToLLMType -DotNetType "System.String"
```

System.String 型をそれに対応する LLM の型に変換します。
