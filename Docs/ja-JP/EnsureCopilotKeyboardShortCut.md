# EnsureCopilotKeyboardShortCut

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Visual Studio Code で GitHub Copilot Chat のキーボード ショートカットを構成します。

## Description

この関数は、Visual Studio Code で GitHub Copilot Chat のファイル添付機能に適切なキーボードショートカット（Ctrl+Shift+Alt+F12）が設定されていることを保証します。

既存の Copilot 添付ショートカットをすべて削除し、現在の正しいコマンド（github.copilot.chat.attachFile）に置き換えます。

また、最大化されたパネルを切り替えるための Alt+`（バッククォート）ショートカットも追加します。

## Syntax

```powershell
EnsureCopilotKeyboardShortCut [<CommonParameters>]
```

## Examples

### EnsureCopilotKeyboardShortCut

```powershell
EnsureCopilotKeyboardShortCut
```
