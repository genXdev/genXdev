# EnsureNuGetAssembly

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> NuGet パッケージのキーまたは ID に基づいて .NET アセンブリをダウンロードし、読み込みます。

## Description

この関数は、指定されたNuGetパッケージアセンブリをダウンロードし、現在のPowerShellセッションに読み込むことを保証します。自動検出と読み込みをサポートし、構成オーバーライド用のJSONマニフェストへのフォールバックにより、スタブ依存関係（例：SQLite）などのエッジケースを処理します。インストールには、信頼性の高いパッケージ管理のためにdotnet CLIのみを使用し、グローバルなNuGetキャッシュから直接アセンブリを読み込み、コピーは行いません。遅延読み込みでは、処理を進める前に型が既に利用可能かどうかを確認します。

## Syntax

```powershell
EnsureNuGetAssembly -PackageKey <String> [-ConsentToThirdPartySoftwareInstallation] [-Description <String>] [-Destination <String>] [-ForceConsent] [-ForceLatest] [-ManifestPath <String>] [-Publisher <String>] [-TypeName <String>] [-Version <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-PackageKey` | String | ✅ | — | 0 | — | packages.json からのパッケージキー、または直接 NuGet パッケージ ID。 |
| `-ManifestPath` | String | — | — | Named | — | packages.json へのパス。省略した場合、モジュールルートがデフォルトになります。 |
| `-Version` | String | — | — | Named | — | 特定バージョン。省略した場合は、JSON内の最新バージョン、または最新版を使用します。 |
| `-TypeName` | String | — | — | Named | — | 読み込みを確認するための型名。 |
| `-ForceLatest` | SwitchParameter | — | — | Named | — | 正確なバージョンが失敗した場合、最新のバージョンにフォールバックします。 |
| `-Destination` | String | — | — | Named | — | カスタムインストール先。デフォルトはローカルの永続キャッシュまたはグローバルキャッシュ。 |
| `-Description` | String | — | — | Named | — | 同意のためのソフトウェアとその目的に関する任意の説明。 |
| `-Publisher` | String | — | — | Named | — | 同意を得るためのソフトウェアの任意の公開元またはベンダー。 |
| `-ForceConsent` | SwitchParameter | — | — | Named | — | 同意設定が設定されている場合でもプロンプトを強制します。 |
| `-ConsentToThirdPartySoftwareInstallation` | SwitchParameter | — | — | Named | — | サードパーティ製ソフトウェアのインストールを自動的に同意し、永続的なフラグを設定します。 |

## Examples

### EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core"

```powershell
EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core"
```

### EnsureNuGetAssembly -PackageKey "Microsoft.Playwright" -Version "1.54.0" -TypeName "Microsoft.Playwright.Playwright"

```powershell
EnsureNuGetAssembly -PackageKey "Microsoft.Playwright" -Version "1.54.0" -TypeName "Microsoft.Playwright.Playwright"
```

### EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core" -ConsentToThirdPartySoftwareInstallation

```powershell
EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core" -ConsentToThirdPartySoftwareInstallation
```
