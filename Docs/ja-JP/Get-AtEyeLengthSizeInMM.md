# Get-AtEyeLengthSizeInMM

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 腕の長さにある物体のおおよそのサイズ（mm）を計算します。

## Description

実際のサイズと距離から、平均的な成人の腕の長さ（約0.7メートル）で見た場合の物体の見かけの大きさを計算します。この計算では、角度サイズの小角近似を使用します。

## Syntax

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters <Double> -SizeInMeters <Double> [-EyeToArmLengthInMeters <Double>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | オブジェクトまでの距離（メートル）。 |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | オブジェクトの実際のサイズ（メートル単位）。 |
| `-EyeToArmLengthInMeters` | Double | — | — | Named | `0` | 腕の長さ（メートル単位）。既定値は0.7です。 |

## Examples

### Example 1

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters 10 -SizeInMeters 1
```

腕の長さでの見かけの大きさを計算します。これは、10メートルの距離にある1メートルの物体に対するものです。
