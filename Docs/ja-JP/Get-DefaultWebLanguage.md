# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> システムの現在の言語設定に基づいてデフォルトのウェブ言語キーを取得します。

## Description

現在のシステム言語とカルチャ設定を取得し、翻訳サービスで使用される対応するWeb言語辞書キーにマッピングします。

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

英語のシステムには「English」、オランダ語のシステムには「Dutch」などを返します。
