# Get-DefaultWebbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 現在のユーザーに対して設定されている既定のWebブラウザを返します。

## Description

* Windows レジストリを照会して、システムの既定の Web ブラウザに関する情報を取得します。
* ブラウザの名前、説明、アイコン パス、実行可能ファイル パスを含むハッシュテーブルを返します。
* この関数は、ユーザー設定とシステム全体のブラウザ登録の両方をチェックして、既定のブラウザを特定します。

## Syntax

```powershell
Get-DefaultWebbrowser [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebbrowser | Format-List
```

既定のブラウザに関する詳細情報を取得します。

### Example 2

```powershell
$browser = Get-DefaultWebbrowser
& $browser.Path https://www.github.com/
```

特定のURLでデフォルトブラウザを起動します。
