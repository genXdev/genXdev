# Get-DesktopScalingFactor

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 指定されたモニターのWindows表示スケーリング係数（DPI設定）を取得します。

## Description

* システム内のモニターに構成されている現在のWindows表示スケーリング係数を取得します。
* スケーリング係数はパーセンテージで表され、100が標準スケーリング（96 DPI）を表します。一般的な値は100、125、150、200です。
* モニターが指定されていない場合、プライマリディスプレイのスケーリング係数を返します。

## Syntax

```powershell
Get-DesktopScalingFactor [[-Monitor] <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | — | — | 0 | `0` | チェックするモニターインデックス（0 = プライマリモニター） |

## Examples

### Example 1

```powershell
Get-DesktopScalingFactor -Monitor 0
```

プライマリモニターのスケーリング係数のパーセンテージを返します。

### Example 2

```powershell
Get-DesktopScalingFactor 1
```

位置パラメーターを使用して、2番目のモニターのスケーリング係数のパーセンテージを返します。
