# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> テキスト読み上げエンジンが発話中の場合に true を返します。

## Description

デフォルトおよびカスタマイズされた音声合成エンジンの両方の状態を確認し、いずれかが現在発声中かどうかを判断します。発声中の場合はtrue、それ以外の場合はfalseを返します。

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

テキスト読み上げエンジンが発話中の場合に true を返します。

### Example 2

```powershell
iss
```

テキスト読み上げエンジンが発話中の場合にtrueを返します（エイリアス使用）。
