# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss

## Synopsis

> テキスト読み上げエンジンが発話中の場合、trueを返します。

## Description

デフォルトおよびカスタマイズされた音声合成エンジンの状態を確認し、いずれかが現在発話中かどうかを判定します。発話が進行中であればtrue、そうでなければfalseを返します。

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

テキスト読み上げエンジンが発話中の場合、trueを返します。

### Example 2

```powershell
iss
```

テキスト読み上げエンジンが発話中の場合にtrueを返します（エイリアスを使用）。
