# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> PowerShell ターミナルのメインウィンドウ用のウィンドウヘルパーオブジェクトを返します。

## Description

* 現在のPowerShellホストプロセスのメインウィンドウを表すWindowObjヘルパーオブジェクトを取得します。
* これにより、ターミナルウィンドウ自体の操作や対話が可能になります。

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

現在のPowerShellターミナルウィンドウのWindowObjを取得します。
