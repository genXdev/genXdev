# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Google検索でサポートされているすべての言語の逆引き辞書を返します。

## Description

* Google 検索でサポートされているすべての言語について、言語名とそれに対応する言語コードの辞書を返します。
* Webクエリや翻訳タスクにおける言語コードの参照に便利です。

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

言語名をそのコードにマッピングする辞書を取得します。
