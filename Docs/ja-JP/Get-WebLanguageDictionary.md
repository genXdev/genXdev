# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Google 検索でサポートされているすべての言語の逆引き辞書を返します。

## Description

* Google 検索でサポートされているすべての言語について、言語名を対応する言語コードにマッピングする辞書を返します。
* Web クエリや翻訳タスクでの言語コード検索に役立ちます。

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

言語名をそのコードにマッピングする辞書を取得します。
