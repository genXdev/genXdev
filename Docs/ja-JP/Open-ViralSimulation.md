# Open-ViralSimulation

> **Module:** GenXdev.Queries.Websites | **Type:** Function | **Aliases:** `viral

## Synopsis

> ウェブブラウザでウイルス感染シミュレーションゲームを開き、詳細な設定オプションを提供します。

## Description

https://viral.genxdev.net/ にあるウイルス拡散シミュレーションゲームを、指定されたウェブブラウザで、設定可能なウィンドウ設定、配置、動作オプションを使用して開きます。複数のブラウザ、プライベートブラウジング、アプリケーションモード、および包括的なウィンドウ管理機能をサポートしています。

この関数は、以下のような機能を備えたインタラクティブなウイルス拡散シミュレーション体験を提供します：
- 複数のブラウザ対応（Edge、Chrome、Firefox）
- ウィンドウの配置とサイズ制御
- 全画面モードとアプリケーションモードのオプション
- プライベートブラウジング機能
- キーボード自動化サポート
- マルチモニター対応


## Examples

### Open-ViralSimulation -Chrome -Private -NoFullScreen

```powershell
Open-ViralSimulation -Chrome -Private -NoFullScreen
```

Google Chromeのプライベートブラウジングモードで、全画面表示なしでウイルルシミュレーションを開きます。

### viral -ch -incognito -nfs

```powershell
viral -ch -incognito -nfs
```

エイリアスを使用してChromeでウィルスシミュレーションを開き、シークレットモードで、全画面表示はしない。
