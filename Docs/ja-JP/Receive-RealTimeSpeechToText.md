# Receive-RealTimeSpeechToText

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Whisper AIモデルを使用して、リアルタイムの音声入力をテキストに変換します。

## Description

このコマンドレットは、マイクまたはデスクトップからオーディオをキャプチャし、Whisper AIモデルを使用してリアルタイムでテキストに文字起こしします。さまざまなオーディオソース、無音検出、および音声認識のための複数の構成オプションをサポートしています。

## Examples

### Example 1

```powershell
Receive-RealTimeSpeechToText
```

マイクを使用した基本的な使用方法。

### Example 2

```powershell
Receive-RealTimeSpeechToText -UseDesktopAudioCapture
```

デスクトップオーディオキャプチャを使用しています。

### Example 3

```powershell
Receive-RealTimeSpeechToText -AudioDevice "Microphone*"
```

特定のオーディオデバイスを使用します。

### Example 4

```powershell
Receive-RealTimeSpeechToText -MaxDurationOfSilence "00:00:05"
```

無音検出機能付き。
