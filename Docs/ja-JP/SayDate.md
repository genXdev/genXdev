# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> 現在の日付をテキスト読み上げ合成で発声します。

## Description

現在の日付を自然言語形式に変換し、システムのテキスト読み上げエンジンを使用して発声します。日付は次の形式で読み上げられます：「今日は[曜日]、[月] [日] [年]です」

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
