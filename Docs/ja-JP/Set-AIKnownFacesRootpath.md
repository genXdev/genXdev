# Set-AIKnownFacesRootpath

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> GenXdev.AI の操作で使用される顔画像ファイルのディレクトリを設定します。

## Description

この関数は、GenXdev.AI モジュールがさまざまな顔認識および AI 操作で使用するグローバルな顔ディレクトリを構成します。設定は、永続的に保存する（デフォルト）、現在のセッションのみに保存する（-SessionOnly を使用）、またはセッションからクリアする（-ClearSession を使用）ことができます。

## Syntax

```powershell
Set-AIKnownFacesRootpath [[-FacesDirectory] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FacesDirectory` | String | — | — | 0 | — | 顔画像ファイルのディレクトリパス |
| `-PreferencesDatabasePath` | String | — | — | Named | — | ユーザー設定データファイルのデータベースパス |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | 言語、画像コレクションなどのAI設定のためにセッションに保存された代替設定を使用する |
| `-ClearSession` | SwitchParameter | — | — | Named | — | 言語や画像コレクションなどのAI設定に関して、セッションに保存されている代替設定をクリアします。 |
| `-SkipSession` | SwitchParameter | — | — | Named | — | AIの設定（言語、画像コレクションなど）のためにセッションに保存された代替設定を使用しない |

## Examples

### Set-AIKnownFacesRootpath -FacesDirectory "C:\Faces"

```powershell
Set-AIKnownFacesRootpath -FacesDirectory "C:\Faces"
```

設定に永続的に顔ディレクトリを設定します。

### Set-AIKnownFacesRootpath "C:\FacePictures"

```powershell
Set-AIKnownFacesRootpath "C:\FacePictures"
```

設定に永続的に顔ディレクトリを設定します。

### Set-AIKnownFacesRootpath -FacesDirectory "C:\TempFaces" -SessionOnly

```powershell
Set-AIKnownFacesRootpath -FacesDirectory "C:\TempFaces" -SessionOnly
```

現在のセッション（グローバル変数）のみの面ディレクトリを設定します。

### Set-AIKnownFacesRootpath -ClearSession

```powershell
Set-AIKnownFacesRootpath -ClearSession
```

永続的な設定に影響を与えずに、セッションの顔ディレクトリ設定（グローバル変数）をクリアします。
