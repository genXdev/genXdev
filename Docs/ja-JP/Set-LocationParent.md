# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> 現在の場所を親ディレクトリに変更し、その内容を一覧表示します。

## Description

* ファイルシステム階層内の1つ上のディレクトリに移動します。
* 新しいカレントディレクトリの内容を表示します。
* クイックディレクトリ移動のための便利な '..' エイリアスを提供します。

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

親ディレクトリに変更して、その内容を一覧表示します。

### Example 2

```powershell
..
```

エイリアスを使用して親ディレクトリに移動します。
