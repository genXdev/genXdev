# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...

## Synopsis

> ファイルシステムの階層で2つ上のディレクトリに移動します。

## Description

* カレントディレクトリを2階層上の親ディレクトリに変更します。
* 変更後のディレクトリの内容を表示します。

## Syntax

```powershell
Set-LocationParent2 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

親ディレクトリに移動して、その内容を表示します。

...
