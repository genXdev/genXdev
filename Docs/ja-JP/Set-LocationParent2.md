# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> ファイルシステム階層内で2つ上のディレクトリレベルに移動します。

## Description

* 現在の作業ディレクトリを2つ上の親ディレクトリに変更します。
* 結果のディレクトリの内容を表示します。

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

親ディレクトリの親ディレクトリに移動し、その内容を表示します。

### Example 2

```powershell
...
```

同じ操作に「...」エイリアスを使用します。
