# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....

## Synopsis

> ファイルシステム階層内で、３つ上のディレクトリに移動します。

## Description

* ファイルシステム階層内の親ディレクトリを3つ上に移動して、カレントワーキングディレクトリを変更します。
* 結果のディレクトリの内容を表示します。

## Syntax

```powershell
Set-LocationParent3 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

現在位置から3つ上のディレクトリに移動します。

### Example 2

```powershell
....
```

エイリアスを使用して3つ上のディレクトリに移動します。
