# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> ファイルシステム階層内で 4 つ上のディレクトリ レベルに移動します。

## Description

* ファイルシステムの現在の場所から4つ上のディレクトリ階層に移動します。
* 移動後、結果のディレクトリの内容を表示します。

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

現在の場所から4つ上のディレクトリ階層に移動します。

### Example 2

```powershell
.....
```

エイリアスを使用して、4つ上のディレクトリ階層へ移動します。
