# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....

## Synopsis

> ファイルシステム階層内で4つ上のディレクトリに移動します。

## Description

* ファイルシステム内の現在の場所から4つ上のディレクトリに移動します。
* 移動後、結果のディレクトリの内容を表示します。

## Syntax

```powershell
Set-LocationParent4 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

現在の場所から4つ上のディレクトリレベルに移動します。

### Example 2

```powershell
.....
```

エイリアスを使用して、4階層上のディレクトリに移動します。
