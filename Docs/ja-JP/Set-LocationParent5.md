# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> ファイルシステム階層内で5つ上のディレクトリレベルに移動します。

## Description

* 現在の作業ディレクトリを5つ上の親ディレクトリに変更します。
* 結果のディレクトリの内容を表示します。
* クイックナビゲーションのための便利な短縮エイリアス '......' を提供します。

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

現在の場所から5つ上のディレクトリ階層に移動します。

### Example 2

```powershell
......
```

エイリアスを使用して、5つ上のディレクトリ階層に移動します。
