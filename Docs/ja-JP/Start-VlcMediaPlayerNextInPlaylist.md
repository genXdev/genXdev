# Start-VlcMediaPlayerNextInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcnext`

## Synopsis

> VLC Media Playerを現在のプレイリストの次の項目に進めます。

## Description

この関数は、現在読み込まれているプレイリスト内の次のトラックまたはメディアアイテムにスキップするために、'n' キーストロークを VLC Media Player に送信します。VLC ウィンドウにフォーカスを当て、次のコマンドを送信し、PowerShell コンソールにフォーカスを戻します。この関数には、必要に応じて確認プロンプトを表示するための ShouldProcess サポートが含まれています。

## Syntax

```powershell
Start-VlcMediaPlayerNextInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerNextInPlaylist

```powershell
Start-VlcMediaPlayerNextInPlaylist
```

### vlcnext

```powershell
vlcnext
```
