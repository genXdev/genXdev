# Start-VlcMediaPlayerPreviousInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcprev`, `vlcback`

## Synopsis

> VLC Media Playerのプレイリストで前の項目に移動します。

## Description

この関数は、VLC Media Playerに「p」キーコマンドを送信して、現在のプレイリスト内の前のアイテムに移動します。この関数はWhatIf操作をサポートしており、コマンド送信後にフォーカスを復元します。

## Syntax

```powershell
Start-VlcMediaPlayerPreviousInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerPreviousInPlaylist

```powershell
Start-VlcMediaPlayerPreviousInPlaylist
```

### vlcprev

```powershell
vlcprev
```

### vlcback

```powershell
vlcback
```
