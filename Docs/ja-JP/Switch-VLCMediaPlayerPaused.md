# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> VLCメディアプレーヤーの一時停止/再生状態を切り替えます。

## Description

この関数は、VLC Media Playerにスペースキーを送信して、一時停止と再生の状態を切り替えます。キーコマンド送信後、自動的に元のウィンドウにフォーカスを戻します。

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

VLC メディアプレーヤーの一時停止/再生状態を切り替えます。

### vlcpause

```powershell
vlcpause
```

エイリアスを使用して、一時停止/再生状態を切り替えます。

### vlcplay

```powershell
vlcplay
```

代替のエイリアスを使用して、一時停止/再生状態を切り替えます。
