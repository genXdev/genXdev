# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> 指定されたスクリプトブロック文字列のインデントを変更した文字列を返します。

## Description

元のコードブロックのインデントを尊重しながら、スクリプトブロック文字列のインデントを変更します。

## Syntax

```powershell
alignScript [<CommonParameters>]
```
