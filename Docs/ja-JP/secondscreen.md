# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> 既定のセカンドモニターの構成を設定します。

## Description

GenXdev のウィンドウを開く際のデフォルト動作をセカンダリモニターに設定します。
これは、シングルモニターのユーザーや、ウィンドウを並べて表示したいユーザーに便利です。
関連コマンドレット: 'sidebyside' および 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

GenXdev のウィンドウをセカンダリモニターに開く既定値を設定します
