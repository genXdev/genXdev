# Enable-Screensaver

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> 구성된 Windows 화면 보호기를 시작합니다.

## Description

기본 화면 보호기 실행 파일(scrnsave.scr)을 /s 스위치와 함께 실행하여 Windows 시스템 화면 보호기를 즉시 활성화합니다.

## Syntax

```powershell
Enable-Screensaver [<CommonParameters>]
```

## Examples

### Enable-Screensaver

```powershell
Enable-Screensaver
```
