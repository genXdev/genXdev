# Get-AtEyeLengthSizeInMM

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 팔 길이에서 물체의 겉보기 크기를 mm 단위로 계산합니다.

## Description

평균 성인 팔 길이(약 0.7미터)에서 물체를 보았을 때 나타나는 크기를 실제 크기와 거리를 바탕으로 계산합니다. 계산은 각크기에 대한 작은 각도 근사를 사용합니다.

## Syntax

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters <Double> -SizeInMeters <Double> [-EyeToArmLengthInMeters <Double>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | 미터 단위의 개체까지의 거리입니다. |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | 객체의 실제 크기(미터 단위)입니다. |
| `-EyeToArmLengthInMeters` | Double | — | — | Named | `0` | 팔 길이 거리(미터)입니다. 기본값은 0.7입니다. |

## Examples

### Example 1

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters 10 -SizeInMeters 1
```

1미터 크기의 물체가 10미터 거리에 있을 때 팔 길이에서의 겉보기 크기를 계산합니다.
