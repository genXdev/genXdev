# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 시스템의 현재 언어 설정에 따라 기본 웹 언어 키를 가져옵니다.

## Description

현재 시스템 언어 및 문화 설정을 검색하여 번역 서비스에서 사용되는 해당 웹 언어 사전 키에 매핑합니다.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

"영어" 시스템의 경우 "English", 네덜란드어 시스템의 경우 "Dutch" 등을 반환합니다.
