# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss

## Synopsis

> 텍스트 음성 변환 엔진이 말하는 중이면 true를 반환합니다.

## Description

기본 음성 합성기와 사용자 정의 음성 합성기의 상태를 모두 확인하여 현재 말하기 중인지 확인합니다. 말하기가 진행 중이면 true를, 그렇지 않으면 false를 반환합니다.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

텍스트 음성 변환 엔진이 말하는 중이면 true를 반환합니다.

### Example 2

```powershell
iss
```

텍스트 음성 변환 엔진이 말하고 있는 경우 true를 반환합니다 (별칭 사용).
