# Get-LightTravelTimeByDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 빛이 거리를 이동하는 데 걸리는 시간을 계산합니다.

## Description

t = d / c를 사용하며, c는 매체에 따라 달라집니다.

## Syntax

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters <Double> [[-As] <String>] [<CommonParameters>]

Get-LightTravelTimeByDistance [[-SpeedOfLightInMetersPerSecond] <Double>] [<CommonParameters>]

Get-LightTravelTimeByDistance -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | 미터 단위의 거리 |
| `-SpeedOfLightInMetersPerSecond` | Double | — | — | 1 | `0` | 빛의 속도 (m/s, 기본값: 299792458) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | 빛이 통과하는 매질 *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | 시간에 대한 출력 단위 |

## Examples

### Example 1

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters 149597870700 -Medium "water" -As "minutes"
```

빛이 태양까지의 거리를 물 속에서 이동하는 데 걸리는 시간을 분 단위로 계산합니다.

### Example 2

```powershell
Get-LightTravelTimeByDistance 300000000 -SpeedOfLightInMetersPerSecond 225000000
```

빛이 300,000,000미터를 225,000,000m/s로 이동하는 시간을 계산합니다.
