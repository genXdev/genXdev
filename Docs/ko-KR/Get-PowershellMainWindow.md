# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> PowerShell 터미널의 주 창에 대한 창 도우미 개체를 반환합니다.

## Description

* 현재 PowerShell 호스트 프로세스의 기본 창을 나타내는 WindowObj 도우미 개체를 검색합니다.
* 이를 통해 터미널 창 자체를 조작하고 상호 작용할 수 있습니다.

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

현재 PowerShell 터미널 창에 대한 WindowObj를 검색합니다.
