# Get-SimularMovieTitles

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> 공통 속성을 기준으로 유사한 영화 제목을 찾습니다.

## Description

제공된 영화들을 분석하여 공통 속성을 찾고 10개의 유사한 영화 제목 목록을 반환합니다. 입력 영화들의 패턴과 테마를 식별하여 관련성 높은 추천을 제안하기 위해 AI를 사용합니다.
