# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Google 검색에서 지원하는 모든 언어에 대해 역방향 사전을 반환합니다.

## Description

* Google 검색에서 지원하는 모든 언어에 대해 언어 이름을 해당 언어 코드에 매핑하는 사전을 반환합니다.
* 웹 쿼리 및 번역 작업에서 언어 코드 조회에 유용합니다.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

언어 이름을 해당 코드에 매핑하는 사전을 검색합니다.
