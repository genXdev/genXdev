# Invoke-AIPowershellCommand

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `hint`

## Synopsis

> AI 지원을 사용하여 PowerShell 명령을 생성하고 실행합니다.

## Description

자연어 쿼리를 기반으로 PowerShell 명령을 생성하기 위해 AI 모델을 사용합니다. 이 함수는 명령을 직접 PowerShell 창으로 보내거나 클립보드에 복사할 수 있습니다. AI 모델을 활용하여 자연어를 해석하고 다양한 AI 백엔드에 대한 포괄적인 매개변수 지원으로 실행 가능한 PowerShell 명령으로 변환합니다.

## Syntax

```powershell
Invoke-AIPowershellCommand -PowershellCmdline <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-LLMQueryType <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-PowershellCmdline` | String | ✅ | — | 0 | — | 자연어 쿼리를 사용하여 명령을 생성합니다 |
| `-LLMQueryType` | String | — | — | Named | `'Coding'` | LLM 쿼리의 유형 |
| `-Model` | String | — | — | Named | — | AI 작업에 사용할 모델 식별자 또는 패턴 |
| `-ApiEndpoint` | String | — | — | Named | — | AI 작업을 위한 API 엔드포인트 URL |
| `-ApiKey` | String | — | — | Named | — | 인증된 AI 작업을 위한 API 키 |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | LLM이 JSON 스키마를 지원하지 않음을 나타냅니다. |

## Examples

### Invoke-AIPowershellCommand -Query "list all running processes"

```powershell
Invoke-AIPowershellCommand -Query "list all running processes"
```

실행 중인 프로세스를 나열하는 PowerShell 명령을 생성합니다.

### hint "list files modified today"

```powershell
hint "list files modified today"
```

별칭을 사용하여 오늘 수정된 파일을 찾는 명령을 생성합니다.

### Invoke-AIPowershellCommand -Query "stop service" -Clipboard

```powershell
Invoke-AIPowershellCommand -Query "stop service" -Clipboard
```

서비스를 중지하는 명령을 생성하여 클립보드에 복사합니다.
