# Move-ToRecycleBin

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 파일 및 디렉터리를 안전하게 Windows 휴지통으로 이동합니다.

## Description

* Windows Shell API를 사용하여 파일이나 디렉터리를 휴지통으로 안전하게 이동합니다.
  사용 중인 파일도 처리할 수 있습니다.
* 이 함수는 Shell.Application COM 개체를 사용하여 작업을 수행하므로
  적절한 재활용 동작과 실행 취소 기능을 보장합니다.

## Syntax

```powershell
Move-ToRecycleBin -Path <String[]> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String[] | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | 휴지통으로 이동할 경로를 지정합니다. |

## Examples

### Example 1

```powershell
Move-ToRecycleBin -Path "C:\temp\old-report.txt"
```

단일 파일을 휴지통으로 이동합니다.

### Example 2

```powershell
"file1.txt","file2.txt" | recycle
```

파이프라인 및 별칭을 사용하여 여러 파일을 휴지통으로 이동합니다.
