# New-ModuleMarkdownHelp

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> 모든 PowerShell 모듈에 대한 풍부한 Markdown 도움말 파일을 생성합니다.

## Description

PowerShell 모듈에 대한 포괄적인 Markdown 도움말 사이트를 생성합니다:
모든 메타데이터 섹션이 포함된 cmdlet당 하나의 .md 파일과 서브 모듈별 테이블이 각 cmdlet 파일에 링크된 README.md 인덱스를 생성합니다.

서브 모듈 검색은 이식 가능합니다: 스크립트 cmdlet의 경우 .psm1 점-소스 디렉터리, 컴파일된 cmdlet의 경우 .NET 네임스페이스, 서브 모듈 구조가 없는 모듈을 위한 계층적 폴백 체인을 사용합니다.

## Syntax

```powershell
New-ModuleMarkdownHelp -ModuleName <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-Force] [-Language <String>] [-LinkPrefix <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [-OutputPath <String>] [-SkipTranslation] [-TranslationInstructions <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String | ✅ | — | 0 | — | 도움말을 생성할 PowerShell 모듈의 이름 |
| `-OutputPath` | String | — | — | Named | — | .md 파일을 위한 사용자 지정 출력 디렉터리입니다. 기본값은 <moduleRoot>\Docs\<language>입니다. |
| `-Language` | String | — | — | Named | `'en-US'` | 생성된 도움말에 대한 BCP 47 언어 태그 (예: en-US, nl-NL, de-DE) |
| `-Force` | SwitchParameter | — | — | Named | — | 확인 메시지 없이 기존 .md 파일 덮어쓰기 |
| `-SkipTranslation` | SwitchParameter | — | — | Named | — | @{response=Skip LLM translation; keep help in source language} |
| `-Model` | String | — | — | Named | — | AI 번역에 사용할 모델 식별자 또는 패턴 |
| `-ApiEndpoint` | String | — | — | Named | — | AI 번역을 위한 API 엔드포인트 URL |
| `-ApiKey` | String | — | — | Named | — | 인증된 AI 번역을 위한 API 키 |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | LLM이 JSON 스키마를 지원하지 않음을 나타냅니다. |
| `-LinkPrefix` | String | — | — | Named | — | README 색인 링크의 URL 접두사 (예: https://github.com/org/repo/Docs/) |
| `-TranslationInstructions` | String | — | — | Named | — | Custom AI 번역 지침 |

## Examples

### New-ModuleMarkdownHelp -ModuleName 'Pester' -SkipTranslation

```powershell
New-ModuleMarkdownHelp -ModuleName 'Pester' -SkipTranslation
```

Pester cmdlet 당 하나의 .md와 README.md 인덱스가 포함된 Docs\ 폴더를 생성합니다.

### New-ModuleMarkdownHelp -ModuleName 'GenXdev' -Language 'nl-NL' -Force -Model 'deepseek-v4-pro' -ApiKey 'your-api-key' -ApiEndpoint 'https://api.deepseek.com/chat/completions'

```powershell
New-ModuleMarkdownHelp -ModuleName 'GenXdev' -Language 'nl-NL' -Force -Model 'deepseek-v4-pro' -ApiKey 'your-api-key' -ApiEndpoint 'https://api.deepseek.com/chat/completions'
```

네덜란드어로 번역된 마크다운 도움말을 생성하여 기존 파일을 덮어씁니다.

### New-ModuleMarkdownHelp -ModuleName 'GenXdev' -LinkPrefix `     'https://github.com/genXdev/genXdev/Docs/' -SkipTranslation

```powershell
New-ModuleMarkdownHelp -ModuleName 'GenXdev' -LinkPrefix `
    'https://github.com/genXdev/genXdev/Docs/' -SkipTranslation
```

README 색인에 절대 GitHub 링크가 포함된 도움말을 생성합니다.
