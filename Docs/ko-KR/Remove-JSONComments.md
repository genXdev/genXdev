# Remove-JSONComments

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> JSON 콘텐츠에서 주석을 제거합니다.

## Description

* JSON 콘텐츠를 처리하여 단일 행 및 여러 행 주석을 제거하면서 JSON 구조를 유지합니다.
* 구문 분석 전에 문서화 주석이 포함된 JSON 파일을 정리하는 데 유용합니다.
* 단일 행 주석(//)과 여러 행 주석(/* */)을 모두 지원합니다.

## Syntax

```powershell
Remove-JSONComments -Json <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Json` | String | ✅ | — | 0 | — | .DESCRIPTION
    Get-TempFiles 명령은 C:\Temp 디렉토리에서 모든 임시 파일(*.tmp)을 검색합니다.
.PARAMETER Path
    검색할 디렉토리 경로를 지정합니다.
.PARAMETER Recurse
    하위 디렉토리까지 검색하려면 이 스위치를 사용합니다.
.EXAMPLE
    Get-TempFiles -Path "D:\Logs" -Recurse
    D:\Logs 및 그 하위 디렉토리에서 모든 .tmp 파일을 검색합니다. |

## Examples

### Example 1

```powershell
$jsonContent = @'
{
    // This is a comment
    "name": "test", /* inline comment */
    "value": 123
}
'@ -split "`n"
Remove-JSONComments -Json $jsonContent
```

변수에 저장된 JSON 콘텐츠에서 주석을 제거합니다.

### Example 2

```powershell
$jsonContent | Remove-JSONComments
```

파이프라인에서 JSON 콘텐츠를 처리합니다.
