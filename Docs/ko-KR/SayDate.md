# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> 음성 합성을 사용하여 현재 날짜를 말합니다.

## Description

현재 날짜를 자연어 형식으로 변환하여 시스템의 텍스트 음성 변환 엔진을 사용해 발표합니다. 날짜는 "[요일], [월] [일] [년]입니다" 형식으로 말해집니다.

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
