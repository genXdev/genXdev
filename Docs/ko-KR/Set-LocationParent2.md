# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...

## Synopsis

> 파일 시스템 계층 구조에서 두 단계 위 디렉터리로 이동합니다.

## Description

* 현재 작업 디렉터리를 조부모 디렉터리(두 단계 위)로 변경합니다.
* 결과 디렉터리의 내용을 표시합니다.

## Syntax

```powershell
Set-LocationParent2 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

조부모 디렉터리로 변경하고 그 내용을 표시합니다.

```powershell
...
```
