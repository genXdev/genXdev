# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....

## Synopsis

> 파일 시스템 계층에서 세 디렉터리 수준 위로 이동합니다.

## Description

* 파일 시스템 계층 구조에서 상위 세 개의 부모 디렉터리로 이동하여 현재 작업 디렉터리를 변경합니다.
* 결과 디렉터리의 내용을 표시합니다.

## Syntax

```powershell
Set-LocationParent3 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

현재 위치에서 상위 세 개의 디렉터리로 이동합니다.

```powershell
....
```

별칭을 사용하여 세 디렉터리 레벨 위로 이동합니다.
