# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....

## Synopsis

> 파일 시스템 계층 구조에서 상위 네 디렉터리 수준으로 이동합니다.

## Description

* 현재 위치에서 파일 시스템의 상위 디렉터리로 네 단계 이동합니다.
* 이동 후, 결과 디렉터리의 내용을 표시합니다.

## Syntax

```powershell
Set-LocationParent4 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

현재 위치에서 상위 디렉터리 네 단계로 이동합니다.
