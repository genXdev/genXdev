# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......

## Synopsis

> 파일 시스템 계층 구조에서 다섯 단계 위로 이동합니다.

## Description

* 현재 작업 디렉토리를 상위 다섯 단계 부모 디렉토리로 변경합니다.
* 결과 디렉토리의 내용을 표시합니다.
* 빠른 탐색을 위한 편리한 약어 별칭 '......'을 제공합니다.

## Syntax

```powershell
Set-LocationParent5 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

현재 위치에서 상위 디렉터리 5단계로 이동합니다.

### Example 2

```powershell
......
```

별칭을 사용하여 다섯 단계 위의 디렉토리로 이동합니다.
