# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> 파일 시스템 계층 구조에서 상위 디렉터리로 5레벨 이동합니다.

## Description

* 현재 작업 디렉터리를 상위 다섯 단계 부모 디렉터리로 변경합니다.
* 결과 디렉터리의 내용을 표시합니다.
* 빠른 탐색을 위한 편리한 약어 별칭 '......'을 제공합니다.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

현재 위치에서 5개 상위 디렉터리로 이동합니다.

### Example 2

```powershell
......
```

별칭을 사용하여 5개 디렉터리 수준 위로 이동합니다.
