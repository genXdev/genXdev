# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> VLC 미디어 플레이어에서 반복 모드를 전환합니다.

## Description

이 함수는 VLC 미디어 플레이어에 'r' 키 명령을 보내 반복 모드(반복 없음, 현재 반복, 전체 반복)를 전환합니다. 함수는 VLC가 실행 중이지 않으면 VLC를 열고 명령을 보낸 후 이전 창에 포커스를 복원합니다.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
