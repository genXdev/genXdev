# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> 제공된 스크립트블록 문자열의 들여쓰기를 변경한 문자열을 반환합니다.

## Description

원래 코드 블록 들여쓰기를 유지하면서 스크립트블록 문자열의 들여쓰기를 변경합니다.

## Syntax

```powershell
alignScript [<CommonParameters>]
```
