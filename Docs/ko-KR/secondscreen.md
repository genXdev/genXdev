# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> 기본 보조 모니터 구성을 설정합니다.

## Description

GenXdev 창 열기의 기본 동작을 보조 모니터에서 수행하도록 설정합니다.
단일 모니터를 사용하거나 창을 나란히 배치하는 레이아웃을 선호하는 사용자에게 유용합니다.
cmdlet 'sidebyside' 및 'restoredefaultmonitor'도 참조하십시오.

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

GenXdev 창 열림이 보조 모니터에 표시되도록 기본값 설정
