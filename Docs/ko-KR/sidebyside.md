# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> 기본 나란히 보기 구성을 설정합니다.

## Description

GenXdev 창 열기의 기본 동작을 PowerShell과 나란히 표시하도록 설정합니다.
단일 모니터를 사용하거나 나란히 창 레이아웃을 선호하는 사용자에게 유용합니다.
'secondscreen' 및 'restoredefaultmonitor' cmdlet도 참조하세요.

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

GenXdev 창문 열림 기본값을 PowerShell과 나란히 표시하도록 설정
