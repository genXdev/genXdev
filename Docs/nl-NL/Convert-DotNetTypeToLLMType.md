# Convert-DotNetTypeToLLMType

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Converteert .NET-typenamen naar LLM (Language Model)-typenamen.

## Description

Neemt een .NET-typenaam als invoer en retourneert de bijbehorende vereenvoudigde typenaam die wordt gebruikt in taalmodellen. Het verwerkt veelvoorkomende .NET-typen en biedt geschikte type-toewijzingen.

## Syntax

```powershell
Convert-DotNetTypeToLLMType -DotNetType <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DotNetType` | String | ✅ | — | 0 | — | De .NET-typenaam om te converteren naar een LLM-typenaam |

## Examples

### Example 1

```powershell
Convert-DotNetTypeToLLMType -DotNetType "System.String"
```

Converteert een System.String-type naar het equivalente LLM-type.
