# Copy-IdenticalParamValues

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Kopieert parameterwaarden van gebonden parameters naar een nieuwe hashtable op basis van de mogelijke parameters van een andere functie.

## Description

* Deze cmdlet maakt een nieuwe hashtable aan die alleen de parameterwaarden bevat die overeenkomen met de parameters die zijn gedefinieerd in de opgegeven doelfunctie.
* De resulterende hashtable kan vervolgens worden gebruikt om de functie aan te roepen met behulp van PowerShell splatting.
* Schakelparameters worden alleen in het resultaat opgenomen als ze expliciet zijn opgegeven en ingesteld op $true in de gebonden parameters.
* Niet-aanwezige schakelparameters worden uitgesloten van het resultaat om de juiste parametersemantiek te behouden.

## Syntax

```powershell
Copy-IdenticalParamValues -BoundParameters <Object[]> -FunctionName <String> [[-DefaultValues] <Object>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-BoundParameters` | Object[] | ✅ | — | 0 | — | Bronparameters om van te kopiëren |
| `-FunctionName` | String | ✅ | — | 1 | — | Naam van de doelfunctie om parameters te filteren |
| `-DefaultValues` | Object | — | — | 2 | — | Standaardwaarden voor parameters |

## Examples

### Example 1

```powershell
function Test-Function {
    [CmdletBinding()]
    param(
        [Parameter(Mandatory = $true)]
        [string] $Path,
        [Parameter(Mandatory = $false)]
        [switch] $Recurse
    )

    $params = GenXdev\Copy-IdenticalParamValues `
        -BoundParameters $PSBoundParameters `
        -FunctionName 'Get-ChildItem'

    Get-ChildItem @params
}
```

Toont het kopiëren van gebonden parameters voor gebruik met splatting.
