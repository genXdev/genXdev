# EnsureNuGetAssembly

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Hiermee worden .NET-assembly's van NuGet-pakketten gedownload en geladen op basis van de pakketsleutel of -id.

## Description

Deze functie zorgt ervoor dat de opgegeven NuGet-pakketassemblages worden gedownload en geladen in de huidige PowerShell-sessie. Het ondersteunt automatische detectie en laden met een terugval naar een JSON-manifest voor configuratie-overschrijvingen, en behandelt randgevallen zoals stub-afhankelijkheden (bijv. SQLite). Installatie gebruikt uitsluitend de dotnet CLI voor betrouwbaar pakketbeheer en laadt assemblages rechtstreeks vanuit de globale NuGet-cache zonder ze te kopiëren. Lazy loading controleert of het type al beschikbaar is voordat het verdergaat.

## Syntax

```powershell
EnsureNuGetAssembly -PackageKey <String> [-ConsentToThirdPartySoftwareInstallation] [-Description <String>] [-Destination <String>] [-ForceConsent] [-ForceLatest] [-ManifestPath <String>] [-Publisher <String>] [-TypeName <String>] [-Version <String>] [<CommonParameters>]
```

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-PackageKey` | String | ✅ | — | 0 | — | Pakketcode uit packages.json of directe NuGet-pakket-id. |
| `-ManifestPath` | String | — | — | Named | — | Pad naar packages.json; standaard naar modulehoofdmap indien weggelaten. |
| `-Version` | String | — | — | Named | — | Specifieke versie; indien weggelaten, gebruik de hoogste uit JSON of de nieuwste. |
| `-TypeName` | String | — | — | Named | — | TypeName om het laden te verifiëren. |
| `-ForceLatest` | SwitchParameter | — | — | Named | — | Terugvallen naar de nieuwste versie als de exacte versie mislukt. |
| `-Destination` | String | — | — | Named | — | Aangepaste installatiedoelmap; standaard naar lokale persistente of globale cache. |
| `-Description` | String | — | — | Named | — | Optionele beschrijving van de software en het doel ervan voor toestemming. |
| `-Publisher` | String | — | — | Named | — | Optionele uitgever of leverancier van de software voor toestemming. |
| `-ForceConsent` | SwitchParameter | — | — | Named | — | Dwing een prompt af, zelfs als de voorkeur is ingesteld op toestemming. |
| `-ConsentToThirdPartySoftwareInstallation` | SwitchParameter | — | — | Named | — | Automatisch toestemming geven voor installatie van software van derden en de permanente vlag instellen. |

## Examples

### EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core"

```powershell
EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core"
```

### EnsureNuGetAssembly -PackageKey "Microsoft.Playwright" -Version "1.54.0" -TypeName "Microsoft.Playwright.Playwright"

```powershell
EnsureNuGetAssembly -PackageKey "Microsoft.Playwright" -Version "1.54.0" -TypeName "Microsoft.Playwright.Playwright"
```

```powershell
EnsureNuGetAssembly -PackageKey "Microsoft.Data.Sqlite.Core" -ConsentToThirdPartySoftwareInstallation
```
