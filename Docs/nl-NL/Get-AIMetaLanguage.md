# Get-AIMetaLanguage

> **Module:** GenXdev.AI.Queries | **Type:** Cmdlet | **Aliases:** `getimgmetalang`

## Synopsis

> Hiermee wordt de geconfigureerde standaardtaal voor metagegevensbewerkingen op afbeeldingen opgehaald.

## Description

* Met deze cmdlet wordt de standaardtaal opgehaald die door de GenXdev.AI-module wordt gebruikt voor metadata-bewerkingen van afbeeldingen.
* Er wordt eerst gecontroleerd op globale variabelen (tenzij SkipSession is opgegeven), daarna wordt teruggevallen op persistente voorkeuren en uiteindelijk op systeemstandaarden.

## Syntax

```powershell
Get-AIMetaLanguage [[-Language] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Language` | String | — | — | 0 | — | De standaardtaal voor bewerkingen van afbeeldingsmetadata |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Databasepad voor voorkeursgegevensbestanden |
| `-SessionOnly` | SwitchParameter | — | — | Named | `False` | Gebruik alternatieve instellingen opgeslagen in sessie voor AI-voorkeuren zoals taal, afbeeldingsverzamelingen, enz. |
| `-ClearSession` | SwitchParameter | — | — | Named | `False` | Wis de sessie-instelling (globale variabele) voordat u gaat ophalen |
| `-SkipSession` | SwitchParameter | — | — | Named | `False` | Gebruik geen alternatieve instellingen die in de sessie zijn opgeslagen voor AI-voorkeuren zoals taal, afbeeldingsverzamelingen, enz. |

## Examples

### Example 1

```powershell
Get-AIMetaLanguage
```

Haal de momenteel geconfigureerde taal op uit globale variabelen of voorkeuren.

### Example 2

```powershell
Get-AIMetaLanguage -SkipSession
```

Haal de geconfigureerde taal op uit persistente voorkeuren, waarbij sessie wordt genegeerd.

### Example 3

```powershell
Get-AIMetaLanguage -ClearSession
```

Wis de sessie-instelling en haalt de taal op uit persistente voorkeuren.
