# Get-ClipboardFiles

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `getclipfiles`, `gcbf`

## Synopsis

> Haalt bestanden op uit het Windows-klembord die zijn ingesteld voor bestandsbewerkingen zoals kopiëren/plakken.

## Description

* Deze functie haalt bestandspaden op uit het Windows-klembord die
  eerder waren ingesteld voor bestandsbewerkingen.
* Het verwerkt zowel STA- als MTA-threadmodi automatisch, wat compatibiliteit
  garandeert in verschillende PowerShell-uitvoeringscontexten.
* De functie valideert het bestaan van bestanden en retourneert alleen bestaande
  bestanden/mappen als objecten vergelijkbaar met Get-ChildItem of Get-Item output.

## Syntax

```powershell
Get-ClipboardFiles [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ClipboardFiles
```

Alle bestanden die momenteel in het klembord staan, ophalen als bestandssysteemobjecten.

### Example 2

```powershell
$clipboardFiles = Get-ClipboardFiles
$clipboardFiles | ForEach-Object { Write-Host $_.FullName }
```

Klembordbestanden ophalen en hun volledige paden weergeven.

### Example 3

```powershell
Get-ClipboardFiles | Where-Object { $_.Extension -eq '.txt' }
```

Alleen tekstbestanden ophalen van het klembord.
