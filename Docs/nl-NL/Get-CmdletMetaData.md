# Get-CmdletMetaData

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Haalt metagegevens op voor een opgegeven GenXdev-cmdlet, waarbij help-tekst optioneel naar een andere taal wordt vertaald.

## Description

Haalt uitgebreide metadata op en retourneert deze over een GenXdev cmdlet, inclusief
zijn synopsis, beschrijving, parameters, voorbeelden en andere helpinformatie.
Wanneer een doeltaal wordt opgegeven via de parameter -Language, kan de helptekst
worden vertaald met behulp van AI-gebaseerde vertaaldiensten. Aangepaste vertaal
instructies kunnen worden verstrekt om de vertaaluitvoer te verfijnen.

## Syntax

```powershell
Get-CmdletMetaData -Name <String> [[-Language] <String>] [[-TranslationInstructions] <String>] [[-Model] <String>] [[-ApiEndpoint] <String>] [[-ApiKey] <String>] [[-NoSupportForJsonSchema] <String>] [-SkipTranslation] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Name` | String | ✅ | — | 0 | — | De naam van de cmdlet om metadata op te halen |
| `-Language` | String | — | — | 1 | — | BCP 47-taalcode voor het vertalen van helptekst (bijv. nl-NL, de-DE). Weglaten om vertaling over te slaan. |
| `-TranslationInstructions` | String | — | — | 2 | — | Aangepaste instructies voor het AI-vertaalmodel. Overschrijft de standaard cmdlet-metadata-bewuste vertaalinstructies. |
| `-Model` | String | — | — | 3 | — | De modelidentificatie of het patroon dat moet worden gebruikt voor AI-vertalingen |
| `-ApiEndpoint` | String | — | — | 4 | — | De API-eindpunt-URL voor AI-vertalingen |
| `-ApiKey` | String | — | — | 5 | — | De API-sleutel voor geverifieerde AI-vertalingen |
| `-NoSupportForJsonSchema` | String | — | — | 6 | — | Geeft aan dat LLM geen ondersteuning heeft voor JSON-schema's |
| `-SkipTranslation` | SwitchParameter | — | — | Named | — | Sla LLM-gebaseerde vertaling over; bewaar helptekst in de brontaal, zelfs wanneer -Language is opgegeven. |

## Examples

### Get-CmdletMetaData -Name "Find-Item"

```powershell
Get-CmdletMetaData -Name "Find-Item"
```

Haalt metagegevens op voor de Find-Item-cmdlet in de standaardtaal.

### Get-CmdletMetaData -Name "Find-Item" -Language "nl-NL"

```powershell
Get-CmdletMetaData -Name "Find-Item" -Language "nl-NL"
```

Haalt metadata op voor de cmdlet Find-Item met Nederlandse vertalingen.
