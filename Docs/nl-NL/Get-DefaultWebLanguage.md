# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Haalt de standaard webtaalsleutel op op basis van de huidige taalinstellingen van het systeem.

## Description

Haalt de huidige systeemtaal- en cultuurinstellingen op en wijst deze toe aan de bijbehorende websleutel voor taalwoordenboeken die wordt gebruikt door vertaaldiensten.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Retourneert "Engels" voor een Engels systeem, "Nederlands" voor een Nederlands systeem, enz.
