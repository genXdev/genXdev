# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Haalt de standaard websleutel voor taal op, gebaseerd op de huidige taalinstellingen van het systeem.

## Description

* Haalt de huidige taal- en cultuurinstellingen van het systeem op en wijst deze toe aan de bijbehorende webtaalwoordenboeksleutel die wordt gebruikt door vertaaldiensten.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Geeft 'Engels' voor een Engels systeem, 'Nederlands' voor een Nederlands systeem, etc.
