# Get-ForegroundWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Hiermee wordt de greep van het momenteel actieve voorgrondvenster opgehaald.

## Description

* Met deze cmdlet wordt de vensterhendel (HWND) opgehaald van het venster dat
  momenteel op de voorgrond staat en de toetsenbordfocus heeft.
* Er worden Windows API-functies gebruikt via P/Invoke om te communiceren met de
  User32.dll-bibliotheek.

## Syntax

```powershell
Get-ForegroundWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ForegroundWindow
```

Haalt de IntPtr-handle van het actieve venster op en retourneert deze.

### Example 2

```powershell
$windowHandle = Get-ForegroundWindow
Write-Host "Active window handle: $windowHandle"
```

Slaat de handgreep van het voorgrondvenster op in een variabele en geeft deze weer.
