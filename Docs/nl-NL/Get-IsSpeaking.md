# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> Geeft true terug als de tekst-naar-spraak-engine aan het spreken is.

## Description

Controleert de status van zowel de standaard als aangepaste spraaksynthesizers om te bepalen of er momenteel wordt gesproken. Geeft 'waar' terug als er spraak gaande is, anders 'onwaar'.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Geeft true terug als de tekst-naar-spraak-engine aan het spreken is.

### Example 2

```powershell
iss
```

Retourneert waar als de tekst-naar-spraak-engine aan het spreken is (met alias).
