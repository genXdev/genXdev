# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss

## Synopsis

> Geeft true terug als de tekst-naar-spraak-engine aan het spreken is.

## Description

Controleert de status van zowel de standaard als de aangepaste spraaksynthesizers om te bepalen of een van beide momenteel aan het spreken is. Geeft true terug als er spraak bezig is, anders false.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Geeft true terug als de tekst-naar-spraak-engine aan het spreken is.

### Example 2

```powershell
iss
```

Retourneert waar als de tekst-naar-spraak-engine aan het spreken is (met behulp van alias).
