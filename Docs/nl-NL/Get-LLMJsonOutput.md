# Get-LLMJsonOutput

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> @{response=Uitpakken van geldige JSON uit LLM-antwoordtekst met behulp van best-effort heuristieken.}

## Description

Neemt onbewerkte LLM-respons tekst die markdown-codeblokken, commentaar of andere niet-JSON-inhoud kan bevatten en probeert schone, geldige JSON te extraheren. Maakt gebruik van een meerstappenstrategie: verwijdert eerst markdown-codeblokken, probeert vervolgens de volledige string als JSON te parseren en als dat mislukt, gebruikt het schema-gestuurde extractie gevolgd door heuristische terugvalopties. De functie verwerkt: - Markdown-codeblokken (...) - JSON-objecten ({...}) en arrays ([...]) - Strings, getallen, booleans en datum-tijdwaarden - Schema-gestuurde extractie met behulp van json_schema-typehints

## Syntax

```powershell
Get-LLMJsonOutput -Text <String> [[-ResponseFormat] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Text` | String | ✅ | — | 0 | — | @{response=Alsjeblieft, geef de tekst die je vertaald wilt hebben.} |
| `-ResponseFormat` | String | — | — | 1 | — | @{response=Deze cmdlet wordt alleen ondersteund op het Windows-platform.} |

## Examples

### Get-LLMJsonOutput -Tekst '

Get-LLMJsonOutput -Tekst '

```json
{"result": "hello"}
```

@{response=Extraheert: {"result": "hello"}}

### $schema = '{"type":"json_schema","json_schema":{"name":"r","schema":{"type":"object"}}}' Get-LLMJsonOutput -Text 'The answer is {"key": "value"}' -ResponseFormat $schema

```powershell
$schema = '{"type":"json_schema","json_schema":{"name":"r","schema":{"type":"object"}}}'
Get-LLMJsonOutput -Text 'The answer is {"key": "value"}' -ResponseFormat $schema
```

Extracteert: {"key": "value"}
