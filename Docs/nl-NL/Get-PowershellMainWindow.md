# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Retourneert een hulpobject voor het hoofdvenster van de PowerShell-terminal.

## Description

* Haalt een WindowObj-hulpobject op dat het hoofdvenster van het huidige PowerShell-hostproces vertegenwoordigt.
* Dit maakt manipulatie en interactie met het terminalvenster zelf mogelijk.

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

Haalt het WindowObj op voor het huidige PowerShell-terminalvenster.
