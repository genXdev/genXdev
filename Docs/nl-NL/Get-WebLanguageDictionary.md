# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Geeft een omgekeerd woordenboek terug voor alle talen die worden ondersteund door Google Zoeken.

## Description

* Retourneert een woordenboek dat taalcodes toewijst aan hun corresponderende taalnamen voor alle talen die worden ondersteund door Google Zoeken.
* Handig voor het opzoeken van taalcodes in webquery's en vertaaltaken.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

Haalt de woordenboektoewijzing van taalbeschrijvingen naar hun codes op.
