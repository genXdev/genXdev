# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Retourneert een omgekeerd woordenboek voor alle talen die worden ondersteund door Google Zoeken.

## Description

* Retourneert een woordenboek dat taalnamen toewijst aan de bijbehorende
  taalcodes voor alle talen die worden ondersteund door Google Zoeken.
* Nuttig voor het opzoeken van taalcodes in webquery's en vertaaltaken.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

Haalt de woordenboektoewijzing van taalnamen naar hun codes op.
