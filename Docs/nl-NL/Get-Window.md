# Get-Window

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Haalt vensterinformatie op voor opgegeven processen of vensterhandvatten.

## Description

* Haalt vensterinformatie op met behulp van procesnaam, ID of vensterhandvat.
* Retourneert WindowObj-objecten met details over de hoofdvensters van
  overeenkomende processen.
* Ondersteunt jokertekens bij zoeken op procesnaam.

## Syntax

```powershell
Get-Window [[-ProcessName] <String>] [-ProcessId <Int32>] [-WindowHandle <Int64>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ProcessName` | String | — | ✅ (ByPropertyName) | 0 | — | Naam van het proces om vensterinformatie voor op te halen 🌐 *Supports wildcards* |
| `-ProcessId` | Int32 | — | ✅ (ByPropertyName) | Named | `0` | ID van het proces waarvoor vensterinformatie moet worden opgehaald |
| `-WindowHandle` | Int64 | — | ✅ (ByPropertyName) | Named | `0` | Vensterhendel om informatie op te vragen |

## Examples

### Example 1

```powershell
Get-Window -ProcessName "notepad"
```

Haalt vensterinformatie op voor alle processen met de naam 'notepad'.

### Example 2

```powershell
Get-Window -ProcessId 1234
```

Haalt vensterinformatie op voor het proces met ID 1234.

### Example 3

```powershell
Get-Window -WindowHandle 45678
```

Haalt vensterinformatie op voor het venster met handle 45678.
