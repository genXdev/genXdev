# Invoke-GenXdevPSFormatter

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Formatteert PowerShell-scriptbestanden met behulp van de opmaakregels van PSScriptAnalyzer.

## Description

Deze functie past PowerShell-opmaakregels toe op scriptbestanden met behulp van de Invoke-Formatter-cmdlet van PSScriptAnalyzer. Het kan individuele bestanden verwerken of recursief meerdere bestanden in mappen opmaken. De functie gebruikt aanpasbare opmaakinstellingen en biedt gedetailleerde logging van het opmaakproces.

## Syntax

```powershell
Invoke-GenXdevPSFormatter -Path <String> [-Range <Int32[]>] [-Recurse] [-Settings <Object>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Hiermee geeft u het pad naar het scriptbestand dat moet worden opgemaakt. |
| `-Settings` | Object | — | — | Named | — | Een instellingen-hashtable of een pad naar een PowerShell-gegevensbestand (.psd1) dat de opmaakinstellingen bevat. |
| `-Range` | Int32[] | — | — | Named | — | Het bereik waarbinnen opmaak moet plaatsvinden als een array van vier gehele getallen: startregelnummer, startkolomnummer, eindregelnummer, eindkolomnummer. |
| `-Recurse` | SwitchParameter | — | — | Named | — | Bestanden recursief verwerken in submappen. |

## Examples

### Invoke-GenXdevPSFormatter -Path "C:\Scripts\MyScript.ps1"

```powershell
Invoke-GenXdevPSFormatter -Path "C:\Scripts\MyScript.ps1"
```

### Invoke-GenXdevPSFormatter -Path "C:\Scripts" -Recurse

```powershell
Invoke-GenXdevPSFormatter -Path "C:\Scripts" -Recurse
```

### "MyScript.ps1" | Invoke-GenXdevPSFormatter -Settings @{IncludeRules=@('PSUseCorrectCasing')}

```powershell
"MyScript.ps1" | Invoke-GenXdevPSFormatter -Settings @{IncludeRules=@('PSUseCorrectCasing')}
```
