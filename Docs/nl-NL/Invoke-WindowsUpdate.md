# Invoke-WindowsUpdate

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** `updatewindows`

## Synopsis

> Controleert of Windows up-to-date is en installeert optioneel beschikbare updates.

## Description

Deze functie controleert op Windows-updates, winget-pakketupdates en PowerShell-module-updates. Het kan beschikbare updates weergeven of deze automatisch installeren. De functie vereist beheerdersrechten om Windows-updates te installeren en kan optioneel het systeem opnieuw opstarten als updates een herstart vereisen.

## Syntax

```powershell
Invoke-WindowsUpdate [[-Criteria] <String>] [-AutoReboot] [-GroupByCategory] [-IncludeDrivers] [-Install] [-NoBanner] [-NoRebootCheck] [-ProcessOptionalUpdates] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Install` | SwitchParameter | — | — | Named | — | Automatisch beschikbare Windows-updates installeren |
| `-AutoReboot` | SwitchParameter | — | — | Named | — | Automatisch opnieuw opstarten indien updates een herstart vereisen |
| `-Criteria` | String | — | — | 0 | `"IsInstalled=0 and IsHidden=0 and AutoSelectOnWebSites=1"` | Aangepaste zoekcriteria voor Windows Update |
| `-IncludeDrivers` | SwitchParameter | — | — | Named | — | Stuurprogramma's opnemen in updatezoekopdracht |
| `-ProcessOptionalUpdates` | SwitchParameter | — | — | Named | — | Voeg optionele updates toe en vraag om elke te installeren |
| `-GroupByCategory` | SwitchParameter | — | — | Named | — | Groeperen en kleur uitvoeren op updatcategorie |
| `-NoBanner` | SwitchParameter | — | — | Named | — | Uitgang banner/status uitschakelen |
| `-NoRebootCheck` | SwitchParameter | — | — | Named | — | Controle/rapportage van herstartvereisten overslaan |

## Examples

### Invoke-WindowsUpdate

```powershell
Invoke-WindowsUpdate
```

Controleert op beschikbare Windows- en winget-updates zonder ze te installeren.

### Invoke-WindowsUpdate -Install

```powershell
Invoke-WindowsUpdate -Install
```

Installeert automatisch alle beschikbare Windows- en winget-updates.

### updatewindows -Install -AutoReboot

```powershell
updatewindows -Install -AutoReboot
```

Installeert alle updates en start automatisch opnieuw op indien nodig met behulp van de alias.

### Invoke-WindowsUpdate -GroupByCategory

```powershell
Invoke-WindowsUpdate -GroupByCategory
```

Geeft beschikbare updates weer, gegroepeerd per categorie voor een betere organisatie.

### Invoke-WindowsUpdate -Install -ProcessOptionalUpdates

```powershell
Invoke-WindowsUpdate -Install -ProcessOptionalUpdates
```

Installeert alle belangrijke updates automatisch en vraagt voor elke optionele update individueel om toestemming, waardoor selectieve installatie van optionele updates mogelijk is.

### Invoke-WindowsUpdate -IncludeDrivers -Criteria "IsInstalled=0"

```powershell
Invoke-WindowsUpdate -IncludeDrivers -Criteria "IsInstalled=0"
```

Controleert op updates, inclusief stuurprogramma's, met aangepaste zoekcriteria.
