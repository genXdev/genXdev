# Merge-TranslationCache

> **Module:** GenXdev.AI | **Type:** Function | **Aliases:** —

## Synopsis

> Beheert de persistente vertaalcache met gebufferde schijfschrijfbewerkingen per taal.

## Description

Onderhoudt een in-memory vertaalcache op module-niveau voor Get-TextTranslation.
De cache is opgesplitst in JSON-bestanden per taal onder
$env:LOCALAPPDATA\GenXdev.PowerShell\. Toevoegingen worden gebundeld —
schrijfbewerkingen naar schijf gebeuren alleen om de 100 mutaties per taal, of wanneer -PersistNow
wordt aangeroepen. -PersistNow wist ook de in-memory cache om onbeperkte groei
te voorkomen over verwerkingsbatches.

## Syntax

```powershell
Merge-TranslationCache [-GetCache] [<CommonParameters>]

Merge-TranslationCache -Key <String> -Language <String> -Value <String> [<CommonParameters>]

Merge-TranslationCache -PersistNow [<CommonParameters>]

Merge-TranslationCache -ClearCache [<CommonParameters>]

Merge-TranslationCache -PurgeFromCache [<CommonParameters>]
```
