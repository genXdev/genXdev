# Open-SearchEngine

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** `q`

## Synopsis

> Opent een zoekopdracht in de opgegeven zoekmachine met behulp van een webbrowser.

## Description

Deze functie biedt een uniforme interface om te zoeken in meerdere zoekmachines, waaronder Google, Bing, GitHub, Wikipedia, YouTube en andere. Het ondersteunt uitgebreide browserconfiguratie-opties, waaronder vensterpositionering, grootte, browserselectie en taalvoorkeuren. De functie routeert query's dynamisch naar de juiste zoekmachinespecifieke functies.

## Syntax

```powershell
Open-SearchEngine -Queries <String[]> [[-EndPoint] <String>] [[-Language] <String>] [[-Monitor] <Int32>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-NewWindow] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SetForeground] [-SetRestored] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-SearchEngine -Queries "PowerShell cmdlets" -EndPoint "Google" -Language "English"

```powershell
Open-SearchEngine -Queries "PowerShell cmdlets" -EndPoint "Google" -Language "English"
```

Zoekt naar 'PowerShell cmdlets' op Google met resultaten in de Engelse taal.

### q "GitHub PowerShell" Bing -Chrome -Private

```powershell
q "GitHub PowerShell" Bing -Chrome -Private
```

Gebruikt de alias 'q' om te zoeken naar "GitHub PowerShell" op Bing met Chrome in de privémodus.
