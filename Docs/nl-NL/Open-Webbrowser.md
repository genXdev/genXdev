# Open-Webbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `wb`

## Synopsis

> Opent URL's in een of meer browservensters met optionele positionering en opmaak.

## Description

Deze functie biedt een geavanceerde wrapper voor het starten van browsers met uitgebreide opties voor vensterpositionering, browserselectie en gedragsaanpassing. Het ondersteunt meerdere browsers waaronder Edge, Chrome en Firefox met functies zoals privé browsen, applicatiemodus en nauwkeurig vensterbeheer.

Belangrijke functies:
- Slimme browserdetectie en -selectie
- Vensterpositionering (links, rechts, boven, onder, gecentreerd, volledig scherm)
- Ondersteuning voor meerdere monitoren met automatische of handmatige monitorkeuze
- Ondersteuning voor privé-/incognitobrowsermodus
- Applicatiemodus voor afleidingsvrij browsen
- Opties voor het blokkeren van extensies en pop-ups
- Focusbeheer en venstermanipulatie
- Batch-URL-openen in meerdere browsers
- Toetsaanslagautomatisering naar browservensters

De functie kan automatisch systeemcapaciteiten detecteren en het gedrag daarop aanpassen. Voor browsers die niet op het systeem zijn geïnstalleerd, worden bewerkingen stil overgeslagen zonder fouten.

## Syntax

```powershell
Open-Webbrowser [[-Url] <String[]>] [[-Monitor] <Int32>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-ConsentToThirdPartySoftwareInstallation] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-ForceConsent] [-FullScreen] [-Headless] [-Height <Int32>] [-Input <String>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-PreferPlaywrightBrowser] [-Private] [-RestoreFocus] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### wb -PlayWright https://github.com

```powershell
wb -PlayWright https://github.com
```

Opent GitHub in een door Playwright beheerd Chromium-browser.

### Open-Webbrowser -Url "https://github.com"

```powershell
Open-Webbrowser -Url "https://github.com"
```

Opent GitHub in de standaardbrowser.

### Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left

```powershell
Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left
```

Opent Stack Overflow in de linkerhelft van monitor 1.

### wb "https://google.com" -m 0 -fs

```powershell
wb "https://google.com" -m 0 -fs
```

Opent Google in volledig scherm op de primaire monitor met behulp van aliassen.

### Open-Webbrowser -Chrome -Private -NewWindow

```powershell
Open-Webbrowser -Chrome -Private -NewWindow
```

Opent een nieuw Chrome-venster in de incognitomodus.

### "https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All

```powershell
"https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All
```

Opent meerdere URL's in alle geïnstalleerde browsers via pijplijn.

### Open-Webbrowser -Monitor 0 -Right

```powershell
Open-Webbrowser -Monitor 0 -Right
```

Plaatst een reeds geopend browservenster opnieuw naar de rechterkant van de primaire monitor.

### Open-Webbrowser -ApplicationMode -Url "https://app.example.com"

```powershell
Open-Webbrowser -ApplicationMode -Url "https://app.example.com"
```

Opent een webtoepassing in de app-modus zonder browserbesturingselementen.
