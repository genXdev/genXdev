# Remove-JSONComments

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Verwijdert opmerkingen uit JSON-inhoud.

## Description

* Verwerkt JSON-inhoud en verwijdert zowel enkele als multi-line
  opmerkingen, terwijl de JSON-structuur behouden blijft.
* Handig voor het opschonen van JSON-bestanden die documentatie-opmerkingen bevatten
  vóór het parseren.
* Ondersteunt zowel enkele opmerkingen (//) als multi-line opmerkingen (/* */).

## Syntax

```powershell
Remove-JSONComments -Json <String> [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$jsonContent = @'
{
    // This is a comment
    "name": "test", /* inline comment */
    "value": 123
}
'@ -split "`n"
Remove-JSONComments -Json $jsonContent
```

Verwijdert opmerkingen uit JSON-inhoud die is opgeslagen in een variabele.

### Example 2

```powershell
$jsonContent | Remove-JSONComments
```

Verwerkt JSON-inhoud uit de pijplijn.
