# Save-DesktopScreenShot

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Maakt een schermafbeelding van de opgegeven monitor en slaat deze op in het opgegeven pad.

## Description

* Maakt een screenshot van het bureaublad van de opgegeven monitor en slaat
  deze op als een afbeeldingsbestand.
* Ondersteunt meerdere monitoren en detecteert automatisch de juiste scherm
  grenzen en schaalfactoren voor hoge-DPI-schermen.

## Syntax

```powershell
Save-DesktopScreenShot -Monitor <Int32> -OutputPath <String> [-Quality <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | ✅ | ✅ (ByPropertyName) | 0 | `0` | Monitorindex om vast te leggen (0=primaire monitor, 1+=specifieke monitor, -1=alle monitors) |
| `-OutputPath` | String | ✅ | ✅ (ByPropertyName) | 1 | — | Volledig pad waar de schermafbeelding wordt opgeslagen |
| `-Quality` | Int32 | — | — | Named | `0` | JPEG-kwaliteitsniveau (1-100), alleen van toepassing op JPEG-indeling |

## Examples

### Example 1

```powershell
Save-DesktopScreenShot -Monitor 0 -OutputPath "C:\Screenshots\primary.png"
```

Maakt een schermafbeelding van de primaire monitor en slaat deze op als PNG.

### Example 2

```powershell
Save-DesktopScreenShot -Monitor 1 -OutputPath "C:\Screenshots\monitor2.jpg" -Quality 95
```

Legt een schermafbeelding vast van de tweede monitor met hoge kwaliteit JPEG.

### Example 3

```powershell
Save-DesktopScreenShot -Monitor -1 -OutputPath "C:\Screenshots\all_monitors.png"
```

Maakt van alle monitors één enkele screenshotafbeelding.
