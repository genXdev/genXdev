# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Spreekt de huidige datum uit met behulp van tekst-naar-spraaksynthese.

## Description

Converteert de huidige datum naar een natuurlijke taalindeling en kondigt deze aan met behulp van de tekst-naar-spraakengine van het systeem. De datum wordt uitgesproken in het formaat: 'Het is [dag van de week], [maand] [dag] [jaar]'

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
