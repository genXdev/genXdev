# Send-Key

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** `sendkeys`, `invokekeys`

## Synopsis

> Simuleert toetsaanslagen naar een venster of proces.

## Description

Deze functie verzendt toetsenbordinvoer naar een doelvenster of -proces met behulp van het Windows Script Host Shell-object. Het kan vensters targeten op procesnaam, proces-ID of vensterhandle. De functie ondersteunt speciale toetsensequenties, escape-tekens en diverse timingopties voor betrouwbare toetsbezorging.

## Syntax

```powershell
Send-Key -KeysToSend <String[]> [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [<CommonParameters>]

Send-Key [-ProcessName <String>] [<CommonParameters>]

Send-Key [-ProcessId <Int32>] [<CommonParameters>]

Send-Key [-WindowHandle <Int64>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-KeysToSend` | String[] | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | De toetsenbordinvoer om als een reeks tekenreeksen te verzenden |
| `-ProcessName` | String | — | ✅ (ByPropertyName) | Named | — | Naam van het proces waarnaar toetsen moeten worden verzonden (ondersteunt jokertekens) *(Parameter set: )* 🌐 *Supports wildcards* |
| `-ProcessId` | Int32 | — | ✅ (ByPropertyName) | Named | — | ID van het proces om toetsen naar te sturen *(Parameter set: )* |
| `-WindowHandle` | Int64 | — | ✅ (ByPropertyName) | Named | — | Vensterhandgreep om toetsen naar te sturen *(Parameter set: )* |
| `-SendKeyEscape` | SwitchParameter | — | — | Named | — | Onsnap aan invoertekst ontsnappingsbesturingstekens en modifiers |
| `-SendKeyHoldKeyboardFocus` | SwitchParameter | — | — | Named | — | Houd de toetsenbordfocus op het doelvenster na het verzenden van toetsen |
| `-SendKeyUseShiftEnter` | SwitchParameter | — | — | Named | — | Gebruik Shift+Enter in plaats van Enter voor nieuwe regels |
| `-SendKeyDelayMilliSeconds` | Int32 | — | — | Named | `10` | Vertraging tussen verschillende invoerreeksen in milliseconden |

## Examples

### Send-Key -KeysToSend "Hello World{ENTER}" -ProcessName "notepad" Sends text to Notepad followed by Enter key using process name targeting.

```powershell
Send-Key -KeysToSend "Hello World{ENTER}" -ProcessName "notepad"
Sends text to Notepad followed by Enter key using process name targeting.
```

### Send-Key "Special {F11} key" -SendKeyEscape -ProcessId 1234 Sends literal "{F11}" text rather than F11 key using process ID targeting.

```powershell
Send-Key "Special {F11} key" -SendKeyEscape -ProcessId 1234
Sends literal "{F11}" text rather than F11 key using process ID targeting.
```

### sendkeys "Line 1{ENTER}Line 2" -WindowHandle 123456 -SendKeyDelayMilliSeconds 50 Sends multi-line text with custom delay using window handle targeting.

```powershell
sendkeys "Line 1{ENTER}Line 2" -WindowHandle 123456 -SendKeyDelayMilliSeconds 50
Sends multi-line text with custom delay using window handle targeting.
```
