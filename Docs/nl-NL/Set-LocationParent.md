# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> Wijzigt de huidige locatie naar de bovenliggende map en geeft de inhoud ervan weer.

## Description

* Navigeert één mapniveau omhoog in de bestandssysteemhiërarchie.
* Toont de inhoud van de nieuwe huidige map.
* Biedt een handig '..'-alias voor snelle mapnavigatie.

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

Wijzigt de bovenliggende map en geeft de inhoud ervan weer.

### Example 2

```powershell
..
```

Gebruikt het alias om naar de bovenliggende map te gaan.
