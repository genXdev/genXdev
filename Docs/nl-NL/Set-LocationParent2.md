# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> Navigeert twee mapniveaus omhoog in de bestandssysteemhiërarchie.

## Description

* Wijzigt de huidige werkmap naar de bovenliggende map van de bovenliggende map (twee niveaus omhoog).
* Toont de inhoud van de resulterende map.

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Wijzigt naar de bovenliggende map van de bovenliggende map en geeft de inhoud ervan weer.

### Example 2

```powershell
...
```

Gebruikt het '...' alias voor dezelfde bewerking.
