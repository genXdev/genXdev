# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...

## Synopsis

> Navigeert twee niveaus omhoog in de bestandssysteemhiërarchie.

## Description

* Wijzigt de huidige werkmap naar de bovenliggende map van de bovenliggende map (twee niveaus omhoog).
* Toont de inhoud van de resulterende map.

## Syntax

```powershell
Set-LocationParent2 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Wijzigt de bovenliggende map van de grootouder en geeft de inhoud ervan weer.
