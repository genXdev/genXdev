# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....

## Synopsis

> Navigeert vier mappen omhoog in de bestandssysteemhiërarchie.

## Description

* Gaat vier directoryniveaus omhoog vanaf de huidige locatie in het bestandssysteem.
* Toont na navigatie de inhoud van de resulterende directory.

## Syntax

```powershell
Set-LocationParent4 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Navigeert vier directoryniveaus omhoog vanaf de huidige locatie.

### Example 2

```powershell
.....
```

Gebruikt het alias om vier directoryniveaus omhoog te navigeren.
