# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> Navigeert vier mappen omhoog in de bestandssysteemhiërarchie.

## Description

* Verplaatst vier mapniveaus omhoog vanaf de huidige locatie in het
  bestandssysteem.
* Na navigatie wordt de inhoud van de resulterende map weergegeven.

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Navigeert vier mapniveaus omhoog vanaf de huidige locatie.

### Example 2

```powershell
.....
```

Gebruikt het alias om vier mappeniveaus omhoog te navigeren.
