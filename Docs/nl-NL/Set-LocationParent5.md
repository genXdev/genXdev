# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......

## Synopsis

> Navigeert vijf niveaus omhoog in de bestandssysteemhiërarchie.

## Description

* Wijzigt de huidige werkmap door vijf bovenliggende mapniveaus omhoog te gaan.
* Toont de inhoud van de resulterende map.
* Biedt een handig alias '......' voor snelle navigatie.

## Syntax

```powershell
Set-LocationParent5 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Navigeert vijf niveaus omhoog in de directorystructuur vanaf de huidige locatie.

### Example 2

```powershell
......
```

Gebruikt het alias om vijf directoryniveaus omhoog te navigeren.
