# Start-VlcMediaPlayerNextInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcnext`

## Synopsis

> Schakelt VLC Media Player naar het volgende item in de huidige afspeellijst.

## Description

Deze functie stuurt de 'n'-toetsaanslag naar VLC Media Player om over te schakelen naar het volgende nummer of media-item in de momenteel geladen afspeellijst. Het richt het VLC-venster, verzendt de volgende opdracht en herstelt de focus naar de PowerShell-console. De functie bevat ShouldProcess-ondersteuning voor bevestigingsprompts indien nodig.

## Syntax

```powershell
Start-VlcMediaPlayerNextInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerNextInPlaylist

```powershell
Start-VlcMediaPlayerNextInPlaylist
```

### vlcnext

```powershell
vlcnext
```
