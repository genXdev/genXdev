# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> Schakelt de herhaalmodus in VLC Media Player.

## Description

Deze functie stuurt het 'r' toetscommando naar VLC Media Player om te schakelen tussen verschillende herhaalmodi (geen herhaling, herhaal huidige, herhaal alles). De functie opent VLC als het nog niet actief is en herstelt de focus naar het vorige venster na het verzenden van het commando.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
