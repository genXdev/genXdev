# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Retourneert een tekenreeks (met gewijzigde inspringing) van een opgegeven scriptblock-tekenreeks

## Description

Wijzigt de inspringing van een scriptbloktekenreeks met behoud van de oorspronkelijke codeblok-inspringingen

## Syntax

```powershell
alignScript [<CommonParameters>]
```
