# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Stelt de standaardconfiguratie voor het tweede beeldscherm in.

## Description

Stelt het standaardgedrag in voor GenXdev-vensteropeningen op het tweede scherm.
Dit is handig voor gebruikers met één monitor of degenen die de voorkeur geven aan naast elkaar geplaatste vensterlay-outs.
Zie ook cmdlet 'sidebyside' en 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Stelt standaardwaarden in voor GenXdev-vensteropeningen op de secundaire monitor
