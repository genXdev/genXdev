# Assert-GenXdevDependencyUsage

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `checkgenxdevdependencies`

## Synopsis

> Sprawdza poprawność użycia zależności między modułami GenXdev, aby zapewnić zachowanie właściwej hierarchii modułów.

## Description

Ta funkcja analizuje moduły GenXdev, aby upewnić się, że przestrzegają one poprawnej hierarchii zależności. Sprawdza, czy moduły odwołują się tylko do zależności wymienionych w ich manifeście RequiredModules, oraz zapobiega cyklicznym zależnościom poprzez walidację, że moduły nie odwołują się do modułów pojawiających się później w łańcuchu zależności.

## Syntax

```powershell
Assert-GenXdevDependencyUsage [[-ModuleName] <String[]>] [-FromScripts] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String[] | — | ✅ (ByValue, ByPropertyName) | 1 | `@('GenXdev*')` | Filtr do zastosowania na nazwy modułów 🌐 *Supports wildcards* |
| `-FromScripts` | SwitchParameter | — | — | Named | — | Wyszukaj w plikach skryptów, a nie w plikach modułów |

## Examples

### Assert-GenXdevDependencyUsage -ModuleName "GenXdev.Coding"

```powershell
Assert-GenXdevDependencyUsage -ModuleName "GenXdev.Coding"
```

### checkgenxdevdependencies "GenXdev*" -FromScripts

```powershell
checkgenxdevdependencies "GenXdev*" -FromScripts
```
