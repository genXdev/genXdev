# Convert-DotNetTypeToLLMType

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Konwertuje nazwy typów .NET na nazwy typów LLM (Model Językowy).

## Description

Przyjmuje nazwę typu .NET jako dane wejściowe i zwraca odpowiadającą jej uproszczoną nazwę typu używaną w modelach językowych. Obsługuje popularne typy .NET i zapewnia odpowiednie mapowanie typów.

## Syntax

```powershell
Convert-DotNetTypeToLLMType -DotNetType <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DotNetType` | String | ✅ | — | 0 | — | Nazwa typu .NET do przekonwertowania na nazwę typu LLM |

## Examples

### Example 1

```powershell
Convert-DotNetTypeToLLMType -DotNetType "System.String"
```

Konwertuje typ System.String na jego odpowiednik LLM.
