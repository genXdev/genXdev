# Convert-PhysicsUnit

> **Module:** GenXdev.Helpers.Physics | **Type:** Function | **Aliases:** —

## Synopsis

> Konwertuje wartość z jednej jednostki fizycznej na inną w ramach tej samej kategorii.

## Description

Ta funkcja konwertuje wielkości fizyczne między zgodnymi jednostkami w kategoriach takich jak długość, czas, prędkość itp. Wnioskuje kategorię na podstawie jednostek i zgłasza błąd dla niezgodnych konwersji.

## Syntax

```powershell
Convert-PhysicsUnit -Value <Double> -FromUnit <String> -ToUnit <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Value` | Double | ✅ | — | 0 | — | Wartość liczbowa do przekonwertowania |
| `-FromUnit` | String | ✅ | — | 1 | — | Jednostka wartości wejściowej |
| `-ToUnit` | String | ✅ | — | 2 | — | Żądana jednostka wyjściowa |

## Examples

### Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"

```powershell
Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"
```

Konwertuje 100 metrów na stopy.

### Convert-PhysicsUnit 10 "seconds" "minutes"

```powershell
Convert-PhysicsUnit 10 "seconds" "minutes"
```

Konwertuje 10 sekund na minuty za pomocą parametrów pozycyjnych.
