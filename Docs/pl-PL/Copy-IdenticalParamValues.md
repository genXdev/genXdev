# Copy-IdenticalParamValues

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Kopiuje wartości parametrów z powiązanych parametrów do nowej tabeli mieszającej na podstawie możliwych parametrów innej funkcji.

## Description

* To polecenie cmdlet tworzy nową tablicę mieszającą zawierającą tylko wartości parametrów,
  które pasują do parametrów zdefiniowanych w określonej funkcji docelowej.
* Wynikową tablicę mieszającą można następnie użyć do wywołania funkcji za pomocą
  rozwijania parametrów (splatting) w PowerShell.
* Parametry przełączników są uwzględniane w wyniku tylko wtedy, gdy zostały jawnie
  podane i ustawione na $true w powiązanych parametrach.
* Nieobecne parametry przełączników są wykluczane z wyniku, aby zachować
  odpowiednią semantykę parametrów.

## Syntax

```powershell
Copy-IdenticalParamValues -BoundParameters <Object[]> -FunctionName <String> [[-DefaultValues] <Object>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-BoundParameters` | Object[] | ✅ | — | 0 | — | Parametry źródłowe wiązane do kopiowania |
| `-FunctionName` | String | ✅ | — | 1 | — | Nazwa funkcji docelowej do filtrowania parametrów |
| `-DefaultValues` | Object | — | — | 2 | — | Wartości domyślne dla parametrów |

## Examples

### Example 1

```powershell
function Test-Function {
    [CmdletBinding()]
    param(
        [Parameter(Mandatory = $true)]
        [string] $Path,
        [Parameter(Mandatory = $false)]
        [switch] $Recurse
    )

    $params = GenXdev\Copy-IdenticalParamValues `
        -BoundParameters $PSBoundParameters `
        -FunctionName 'Get-ChildItem'

    Get-ChildItem @params
}
```

Demonstruje kopiowanie powiązanych parametrów do użycia z rozwijaniem.
