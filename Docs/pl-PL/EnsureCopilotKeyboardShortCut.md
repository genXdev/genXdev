# EnsureCopilotKeyboardShortCut

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Konfiguruje skróty klawiaturowe GitHub Copilot Chat w Visual Studio Code.

## Description

Ta funkcja zapewnia, że funkcja dołączania plików w GitHub Copilot Chat ma odpowiedni skrót klawiaturowy (Ctrl+Shift+Alt+F12) skonfigurowany w Visual Studio Code.

Usunie ona wszystkie istniejące skróty Copilot do dołączania i zastąpi je bieżącym poprawnym poleceniem (github.copilot.chat.attachFile).

Dodaje również skrót Alt+` (backtick) do przełączania maksymalizowanego panelu.

## Syntax

```powershell
EnsureCopilotKeyboardShortCut [<CommonParameters>]
```

## Examples

### EnsureCopilotKeyboardShortCut

```powershell
EnsureCopilotKeyboardShortCut
```
