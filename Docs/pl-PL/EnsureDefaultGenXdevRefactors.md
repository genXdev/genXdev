# EnsureDefaultGenXdevRefactors

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Zapewnia dostępność domyślnych definicji refaktoryzacji GenXdev.

## Description

Ta funkcja tworzy i utrzymuje domyślne definicje refaktoringu dla modułów PowerShell GenXdev. Konfiguruje refaktory do dokumentacji i formatowania, konwersji do C# oraz monitów o zgodę na instalację.

## Syntax

```powershell
EnsureDefaultGenXdevRefactors [-Force] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Force` | SwitchParameter | — | — | Named | — | Wymusza odtworzenie istniejących definicji refaktoryzacji |

## Examples

### EnsureDefaultGenXdevRefactors

```powershell
EnsureDefaultGenXdevRefactors
```

### EnsureDefaultGenXdevRefactors -Force

```powershell
EnsureDefaultGenXdevRefactors -Force
```
