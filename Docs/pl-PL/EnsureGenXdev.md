# EnsureGenXdev

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Zapewnia, że wszystkie moduły GenXdev są prawidłowo załadowane, wywołując wszystkie polecenia cmdlet Ensure*.

## Description

Ta funkcja pobiera wszystkie polecenia cmdlet GenXdev, które zaczynają się od „Ensure” lub „Optimize-Ensure” (w przypadku poleceń cmdlet w C#) i wykonuje każde z nich, aby zagwarantować, że wszystkie wymagane moduły GenXdev i zależności są prawidłowo załadowane i dostępne do użycia. Wszelkie błędy podczas wykonywania są przechwytywane i wyświetlane jako komunikaty informacyjne.

Opcjonalnie może również pobrać i załadować wszystkie pakiety NuGet zdefiniowane w pliku manifestu packages.json.

## Syntax

```powershell
EnsureGenXdev [-ConsentToThirdPartySoftwareInstallation] [-DownloadAllNugetPackages] [-Force] [-ForceConsent] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Force` | SwitchParameter | — | — | Named | — | Wymusza wykonanie operacji upewnienia, nawet jeśli wydaje się, że zostały już zakończone |
| `-DownloadAllNugetPackages` | SwitchParameter | — | — | Named | — | Pobiera i ładuje wszystkie pakiety NuGet zdefiniowane w pliku manifestu packages.json |
| `-ForceConsent` | SwitchParameter | — | — | Named | — | Wymuś monit o zgodę, nawet jeśli preferencja jest ustawiona na instalację oprogramowania firm trzecich. |
| `-ConsentToThirdPartySoftwareInstallation` | SwitchParameter | — | — | Named | — | Automatycznie wyrażaj zgodę na instalację oprogramowania firm trzecich i ustaw trwałą flagę dla wszystkich pakietów. |

## Examples

### EnsureGenXdev

```powershell
EnsureGenXdev
```

To polecenie uruchamia wszystkie dostępne polecenia cmdlet Ensure* w celu zainicjowania środowiska GenXdev.

### EnsureGenXdev -DownloadAllNugetPackages

```powershell
EnsureGenXdev -DownloadAllNugetPackages
```

To polecenie uruchamia wszystkie polecenia cmdlet Ensure*, a także pobiera i ładuje wszystkie pakiety NuGet zdefiniowane w pliku manifestu packages.json.

### EnsureGenXdev -DownloadAllNugetPackages -ConsentToThirdPartySoftwareInstallation

```powershell
EnsureGenXdev -DownloadAllNugetPackages -ConsentToThirdPartySoftwareInstallation
```

To polecenie uruchamia wszystkie polecenia cmdlet Ensure* i pobiera pakiety NuGet.
