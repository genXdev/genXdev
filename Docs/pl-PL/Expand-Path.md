# Expand-Path

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** `ep`

## Synopsis

> Rozwija dowolne odwołanie do pliku w pełną ścieżkę.

## Description

Rozwija dowolne odwołanie do pliku na pełną ścieżkę, względem bieżącego katalogu użytkownika. Opcjonalnie może zapewnić, że katalogi lub pliki istnieją.

## Syntax

```powershell
Expand-Path -FilePath <String> [-CreateDirectory] [-CreateFile] [-DeleteExistingFile] [-DirectoryMustExist] [-FileMustExist] [-ForceDrive <Char>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Ścieżka do rozwinięcia |
| `-CreateDirectory` | SwitchParameter | — | — | Named | — | Utworzy katalog, jeśli nie istnieje |
| `-CreateFile` | SwitchParameter | — | — | Named | — | Utworzy pusty plik, jeśli nie istnieje |
| `-DeleteExistingFile` | SwitchParameter | — | — | Named | — | Usunie plik, jeśli już istnieje |
| `-ForceDrive` | Char | — | — | Named | `'*'` | Wymusi użycie określonego dysku |
| `-FileMustExist` | SwitchParameter | — | — | Named | — | Zgłosi błąd, jeśli plik nie istnieje |
| `-DirectoryMustExist` | SwitchParameter | — | — | Named | — | Wyrzuci błąd, jeśli katalog nie istnieje |

## Examples

### Expand-Path -FilePath ".\myfile.txt" -CreateFile

```powershell
Expand-Path -FilePath ".\myfile.txt" -CreateFile
```

### ep ~\documents\test.txt -CreateFile

```powershell
ep ~\documents\test.txt -CreateFile
```
