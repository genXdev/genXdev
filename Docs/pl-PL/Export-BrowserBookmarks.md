# Export-BrowserBookmarks

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Eksportuje zakładki przeglądarki do pliku JSON.

## Description

Polecenie cmdlet Export-BrowserBookmark eksportuje zakładki z określonej przeglądarki internetowej (Microsoft Edge, Google Chrome lub Mozilla Firefox) do pliku JSON. Jednocześnie można określić tylko jeden typ przeglądarki. Zakładki są eksportowane z pełnym zachowaniem ich struktury i metadanych.

## Syntax

```powershell
Export-BrowserBookmarks -OutputFile <String> [-Chrome] [-Edge] [<CommonParameters>]

Export-BrowserBookmarks [-Firefox] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-OutputFile` | String | ✅ | — | 0 | — | Ścieżka do pliku JSON, w którym zostaną zapisane zakładki |
| `-Chrome` | SwitchParameter | — | — | Named | — | Eksportowanie zakładek z Google Chrome |
| `-Edge` | SwitchParameter | — | — | Named | — | Eksportuj zakładki z przeglądarki Microsoft Edge |
| `-Firefox` | SwitchParameter | — | — | Named | — | Eksport zakładek z przeglądarki Mozilla Firefox *(Parameter set: )* |

## Examples

### Export-BrowserBookmarks -OutputFile "C:\MyBookmarks.json" -Edge

```powershell
Export-BrowserBookmarks -OutputFile "C:\MyBookmarks.json" -Edge
```

### Export-BrowserBookmarks "C:\MyBookmarks.json" -Chrome

```powershell
Export-BrowserBookmarks "C:\MyBookmarks.json" -Chrome
```
