# Get-ApparentSizeAtArmLength

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza pozorny rozmiar obiektu na wyciągnięcie ręki.

## Description

Oblicza rozmiar pozorny z użyciem przybliżenia małego kąta.

## Syntax

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters <Double> -SizeInMeters <Double> [[-ArmLengthInMeters] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | Odległość do obiektu w metrach |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | Rzeczywisty rozmiar obiektu w metrach |
| `-ArmLengthInMeters` | Double | — | — | 2 | `0` | Długość ramienia w metrach (domyślnie: 0,7) |
| `-As` | String | — | — | 3 | — | Jednostka rozmiaru wyniku |

## Examples

### Example 1

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters 10 -SizeInMeters 1 -As "centimeters"
```

Oblicza pozorny rozmiar obiektu o długości 1 metra znajdującego się w odległości 10 metrów.

### Example 2

```powershell
Get-ApparentSizeAtArmLength 10 1
```

Oblicza rozmiar pozorny przy użyciu parametrów pozycyjnych.
