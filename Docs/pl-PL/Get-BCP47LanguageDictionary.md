# Get-BCP47LanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Zwraca słownik wszystkich zainstalowanych tagów językowych BCP 47 oraz ich nazw wyświetlanych.

## Description

```text
* Tworzy słownik mapujący kody kulturowe BCP 47 (np. en-US, nl-NL,
  de-DE) na nie same oraz ich angielskie nazwy wyświetlane na kody.
* Używa API .NET CultureInfo do wyliczenia wszystkich konkretnych kultur
  zainstalowanych w systemie, co czyni go autorytatywnym źródłem
  prawdy dla nazw folderów językowych pomocy PowerShell i docelowych języków tłumaczenia.
* Klucze słownika obejmują:
  - Pełne tagi BCP 47 (np. "en-US", "nl-NL", "zh-Hans-CN")
  - Angielskie nazwy wyświetlane (np. "English (United States)",
    "Dutch (Netherlands)")
  - Proste nazwy języków mapowane na ich najczęstszy region
    (np. "English" -> "en-US", "Dutch" -> "nl-NL")
```
## Syntax

```powershell
Get-BCP47LanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-BCP47LanguageDictionary
```

Pobierz pełny słownik języka BCP 47.

### Example 2

```powershell
$dict = Get-BCP47LanguageDictionary
$dict["nl-NL"]  # returns "nl-NL"
$dict["Dutch"]  # returns "nl-NL"
```

Weryfikuj i rozstrzygaj kod języka dla generowania pomocy.
