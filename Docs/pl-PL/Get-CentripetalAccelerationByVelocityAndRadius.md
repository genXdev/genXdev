# Get-CentripetalAccelerationByVelocityAndRadius

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza przyspieszenie dośrodkowe.

## Description

Używa a = v² / r.

## Syntax

```powershell
Get-CentripetalAccelerationByVelocityAndRadius -VelocityInMetersPerSecond <Double> -RadiusInMeters <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-VelocityInMetersPerSecond` | Double | ✅ | — | 0 | `0` | Prędkość w m/s |
| `-RadiusInMeters` | Double | ✅ | — | 1 | `0` | Promień w metrach |
| `-As` | String | — | — | 2 | — | Jednostka wyjściowa dla przyspieszenia |

## Examples

### Example 1

```powershell
Get-CentripetalAccelerationByVelocityAndRadius -VelocityInMetersPerSecond 10 -RadiusInMeters 5 -As "g"
```

Oblicza przyspieszenie dośrodkowe dla prędkości 10 m/s i promienia 5 m, wynik w jednostkach g.

### Example 2

```powershell
Get-CentripetalAccelerationByVelocityAndRadius 20 10
```

Oblicza przyspieszenie dośrodkowe dla prędkości 20 m/s i promienia 10 m przy użyciu parametrów pozycyjnych.
