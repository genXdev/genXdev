# Get-ChildProcesses

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Pobiera wszystkie procesy będące potomkami bieżącego procesu PowerShell.

## Description

* Sprawdza wszystkie uruchomione procesy i identyfikuje te, które mają
  bieżący proces PowerShell jako przodka w łańcuchu procesów nadrzędnych.
* Obejmuje to zarówno bezpośrednie procesy podrzędne, jak i ich potomków
  (wnuki itp.).

## Syntax

```powershell
Get-ChildProcesses [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ChildProcesses
```

Pobierz wszystkie procesy potomne bieżącej sesji programu PowerShell.

### Example 2

```powershell
Get-ChildProcesses -Verbose
```

Pobierz procesy potomne z pełnymi informacjami o szczegółowym wykrywaniu procesów.
