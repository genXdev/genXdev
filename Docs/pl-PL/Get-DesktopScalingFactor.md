# Get-DesktopScalingFactor

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Pobiera współczynnik skalowania wyświetlania systemu Windows (ustawienie DPI) dla określonego monitora.

## Description

* Pobiera bieżący współczynnik skalowania wyświetlania systemu Windows skonfigurowany dla monitora w systemie.
* Współczynnik skalowania jest wyrażony jako procent, gdzie 100 oznacza standardowe skalowanie (96 DPI). Typowe wartości to 100, 125, 150 i 200.
* Jeśli nie określono monitora, zwracany jest współczynnik skalowania dla głównego wyświetlacza.

## Syntax

```powershell
Get-DesktopScalingFactor [[-Monitor] <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | — | — | 0 | `0` | Indeks monitora do sprawdzenia (0 = monitor główny) |

## Examples

### Example 1

```powershell
Get-DesktopScalingFactor -Monitor 0
```

Zwraca procentowy współczynnik skalowania dla głównego monitora.

### Example 2

```powershell
Get-DesktopScalingFactor 1
```

Zwraca wartość procentową skalowania dla drugiego monitora przy użyciu parametru pozycyjnego.

## Parameter Details

### `-Monitor <Int32>`

> Indeks monitora do sprawdzenia (0 = monitor główny)

|:---|:---|

| **Position?** | 0 |
