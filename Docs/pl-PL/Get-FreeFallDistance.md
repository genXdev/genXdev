# Get-FreeFallDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza odległość przebytą podczas swobodnego spadania w danym czasie.

## Description

Oblicza odległość spadania podczas swobodnego spadania przy użyciu metody numerycznej uwzględniającej opór powietrza i prędkość graniczną.

## Syntax

```powershell
Get-FreeFallDistance -DurationInSeconds <Double> [[-TerminalVelocityInMetersPerSecond] <Double>] [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | Czas trwania upadku w sekundach |
| `-TerminalVelocityInMetersPerSecond` | Double | — | — | 1 | `0` | Prędkość terminalna w metrach na sekundę (domyślnie: 53) |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Przyspieszenie grawitacyjne w m/s² (domyślnie: 9,81) |
| `-As` | String | — | — | 3 | — | Jednostka dla odległości wyjścia |

## Examples

### Example 1

```powershell
Get-FreeFallDistance -DurationInSeconds 10 -TerminalVelocityInMetersPerSecond 53 -As "feet"
```

Oblicza odległość spadku w ciągu 10 sekund w stopach.

### Example 2

```powershell
Get-FreeFallDistance 5
```

Oblicza odległość w ciągu 5 sekund w metrach.
