# Get-FreeFallHeight

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza wysokość spadku swobodnego dla zadanego czasu trwania.

## Description

Oblicza odległość spadania podczas swobodnego spadania przy użyciu metody numerycznej uwzględniającej opór powietrza i prędkość graniczną.

## Syntax

```powershell
Get-FreeFallHeight -DurationInSeconds <Double> [[-TerminalVelocityInMs] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | Czas trwania upadku w sekundach |
| `-TerminalVelocityInMs` | Double | — | — | 1 | `0` | Prędkość terminalna w metrach na sekundę (domyślnie: 53 m/s dla człowieka) |
| `-As` | String | — | — | 2 | — | Jednostka dla wysokości wyjściowej |

## Examples

### Example 1

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -TerminalVelocityInMs 53
```

Oblicza wysokość spadku w ciągu 10 sekund przy domyślnej prędkości granicznej człowieka.

### Example 2

```powershell
Get-FreeFallHeight 5
```

Oblicza wysokość spadku w ciągu 5 sekund przy użyciu parametru pozycyjnego i domyślnej prędkości terminalnej.

### Example 3

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -As "feet"
```

Oblicza wysokość spadku w ciągu 10 sekund i zwraca wynik w stopach.
