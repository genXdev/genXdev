# Get-FreeFallTime

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza czas potrzebny do swobodnego spadku obiektu z danej wysokości podczas swobodnego spadania.

## Description

Oblicza czas potrzebny do upadku obiektu z określonej wysokości przy użyciu metody numerycznej uwzględniającej opór powietrza i prędkość graniczną. Obliczenia wykorzystują małe kroki czasowe, aby dokładnie modelować fizykę spadających obiektów z realistycznymi ograniczeniami prędkości granicznej.

## Syntax

```powershell
Get-FreeFallTime -HeightInMeters <Double> [[-TerminalVelocityInMs] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-HeightInMeters` | Double | ✅ | — | 0 | `0` | Wysokość upadku w metrach |
| `-TerminalVelocityInMs` | Double | — | — | 1 | `0` | Prędkość terminalna w metrach na sekundę (domyślnie: 53 m/s dla człowieka) |
| `-As` | String | — | — | 2 | — | Jednostka czasu wyjściowego |

## Examples

### Example 1

```powershell
Get-FreeFallTime -HeightInMeters 100 -TerminalVelocityInMs 53
```

Oblicza czas potrzebny na spadek z wysokości 100 metrów przy domyślnej prędkości granicznej człowieka.

### Example 2

```powershell
Get-FreeFallTime 50
```

Oblicza czas potrzebny do spadnięcia z wysokości 50 metrów przy użyciu parametru pozycyjnego i domyślnej prędkości końcowej.

### Example 3

```powershell
Get-FreeFallTime -HeightInMeters 100 -As "minutes"
```

Oblicza czas potrzebny na upadek z wysokości 100 metrów i zwraca wynik w minutach.
