# Get-GenXDevModule

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Pobiera wszystkie moduły GenXDev z określonej ścieżki.

## Description

Ta funkcja przeszukuje strukturę katalogów w poszukiwaniu modułów GenXdev, wykluczając te z '.local' w nazwie. Dla każdego znalezionego prawidłowego modułu zwraca najnowszy folder wersji (1.x), który zawiera prawidłowy plik manifestu modułu (.psd1).

## Syntax

```powershell
Get-GenXDevModule [[-Path] <String>] [-IncludeLocal] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String | — | — | 0 | — | Ścieżka katalogu głównego do wyszukiwania modułów GenXdev |
| `-IncludeLocal` | SwitchParameter | — | — | Named | — | The `-IncludeLocal` parameter. |

## Examples

### Get-GenXDevModule -Path "C:\PowerShell\Modules"

```powershell
Get-GenXDevModule -Path "C:\PowerShell\Modules"
```

### Get-GenXDevModule "C:\PowerShell\Modules"

```powershell
Get-GenXDevModule "C:\PowerShell\Modules"
```
