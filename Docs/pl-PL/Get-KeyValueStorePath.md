# Get-KeyValueStorePath

> **Module:** GenXdev.Data.KeyValueStore | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Pobiera ścieżkę pliku dla magazynu klucz-wartość.

## Description

* Tworzy ścieżkę pliku dla magazynu klucz-wartość na podstawie
  klucza synchronizacji, nazwy magazynu i katalogu bazowego.

## Syntax

```powershell
Get-KeyValueStorePath -SynchronizationKey <String> -StoreName <String> [-BasePath <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-SynchronizationKey` | String | ✅ | — | 0 | — | Klucz synchronizacji używany do identyfikacji magazynu |
| `-StoreName` | String | ✅ | — | 1 | — | Nazwa magazynu klucz-wartość |
| `-BasePath` | String | — | — | Named | — | Podstawowa ścieżka katalogu dla plików magazynu |

## Examples

### Example 1

```powershell
Get-KeyValueStorePath -SynchronizationKey "Local" -StoreName "MyStore"
```

Pobierz ścieżkę do lokalnego magazynu klucz-wartość.
