# Get-KineticEnergyByMassAndVelocity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza energię kinetyczną.

## Description

Używa KE = 1/2 m v².

## Syntax

```powershell
Get-KineticEnergyByMassAndVelocity -MassInKilograms <Double> -VelocityInMetersPerSecond <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Masa w kg |
| `-VelocityInMetersPerSecond` | Double | ✅ | — | 1 | `0` | Prędkość w m/s |
| `-As` | String | — | — | 2 | — | Jednostka wyjściowa dla energii |

## Examples

### Example 1

```powershell
Get-KineticEnergyByMassAndVelocity -MassInKilograms 10 -VelocityInMetersPerSecond 5 -As "calories"
```

Oblicza energię kinetyczną dla masy 10 kg poruszającej się z prędkością 5 m/s i podaje wynik w kaloriach.

### Example 2

```powershell
Get-KineticEnergyByMassAndVelocity 5 10
```

Oblicza energię kinetyczną dla masy 5 kg poruszającej się z prędkością 10 m/s przy użyciu parametrów pozycyjnych.
