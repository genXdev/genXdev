# Get-ModuleCmdletMetaData

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Pobiera metadane dla wszystkich poleceń cmdlet w module PowerShell.

## Description

Pobiera pełne metadane polecenia cmdlet (streszczenie, opis, parametry, przykłady, dane wyjściowe i aliasy) dla każdego polecenia cmdlet w określonym module, dodając właściwości SubModuleName i CmdletType do każdego wyniku.

Przypisanie podmodułu wykorzystuje dwie niezależne ścieżki:

- Polecenia cmdlet skryptu (.ps1): plik źródłowy dopasowany do mapowań katalogów dot-source .psm1 (Functions\<NazwaPodmodułu>\FileName.ps1).
- Polecenia cmdlet skompilowane (.dll): przestrzeń nazw z ImplementationType.Namespace (np. GenXdev.FileSystem).

## Syntax

```powershell
Get-ModuleCmdletMetaData -ModuleName <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-Language <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [-SkipTranslation] [-TranslationInstructions <String>] [<CommonParameters>]
```

## Examples

### Get-ModuleCmdletMetaData -ModuleName 'GenXdev'

```powershell
Get-ModuleCmdletMetaData -ModuleName 'GenXdev'
```

Zwraca metadane dla wszystkich poleceń cmdlet w module GenXdev.

### Get-ModuleCmdletMetaData -ModuleName 'GenXdev' -Language 'nl-NL'

```powershell
Get-ModuleCmdletMetaData -ModuleName 'GenXdev' -Language 'nl-NL'
```

Zwraca przetłumaczone na polski metadane dla wszystkich poleceń cmdlet GenXdev.
