# Get-MpCmdRunPath

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Pobiera ścieżkę do pliku wykonywalnego Windows Defender MpCmdRun.exe.

## Description

* Lokalizuje i zwraca pełną ścieżkę do narzędzia wiersza poleceń programu Windows Defender (MpCmdRun.exe).
* Polecenie cmdlet sprawdza standardową lokalizację instalacji w folderze Program Files i zapewnia odpowiednią obsługę błędów, jeśli plik nie zostanie znaleziony.

## Syntax

```powershell
Get-MpCmdRunPath [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$defenderPath = Get-MpCmdRunPath
```

Pobiera pełną ścieżkę do narzędzia wiersza poleceń programu Windows Defender.
