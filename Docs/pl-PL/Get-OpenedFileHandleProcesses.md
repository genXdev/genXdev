# Get-OpenedFileHandleProcesses

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Pobiera procesy, które mają otwarte uchwyty plików do określonych plików.

## Description

* Używa narzędzia handle.exe z pakietu Sysinternals do identyfikacji procesów, które mają obecnie otwarte uchwyty do jednego lub więcej określonych plików.
* Przydatne do określenia, które procesy uniemożliwiają operacje na plikach, takie jak usuwanie lub modyfikacja.
* Wymaga, aby handle.exe z pakietu Sysinternals był zainstalowany i dostępny w ścieżce systemowej.

## Syntax

```powershell
Get-OpenedFileHandleProcesses -FilePath <String[]> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String[] | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Ścieżka do pliku(ów) do sprawdzenia pod kątem otwartych uchwytów |

## Examples

### Example 1

```powershell
Get-OpenedFileHandleProcesses -FilePath "C:\temp\example.txt"
```

Identyfikuje wszystkie procesy, które mają otwarte uchwyty do określonego pliku.

### Example 2

```powershell
"file1.txt", "file2.txt" | Get-OpenedFileHandleProcesses
```

Sprawdza wiele plików za pomocą danych wejściowych potoku pod kątem procesów z otwartymi uchwytami.
