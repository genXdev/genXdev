# Get-PlaywrightSessionReference

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Pobiera odwołanie do bieżącej sesji przeglądarki Playwright.

## Description

Zwraca tablicę mieszającą zawierającą dane strony Playwright, kontekstu przeglądarki i sesji dla bieżącej strony przeglądarki. To odwołanie może być używane z poleceniem Invoke-WebbrowserEvaluation -ByReference w celu wskazania konkretnej strony.

Typ przeglądarki można określić za pomocą przełączników -Edge, -Chrome, -Chromium, -Firefox lub -Webkit. Jeśli nie określono żadnego przełącznika, preferowany jest Chromium.

## Syntax

```powershell
Get-PlaywrightSessionReference [-Chrome] [-Chromium] [-Edge] [-Firefox] [-Webkit] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Edge` | SwitchParameter | — | — | Named | — | Preferuj sesję przeglądarki Microsoft Edge |
| `-Chrome` | SwitchParameter | — | — | Named | — | Preferuj sesję przeglądarki Google Chrome |
| `-Chromium` | SwitchParameter | — | — | Named | — | Preferuj Microsoft Edge lub Google Chrome, w zależności od tego, która przeglądarka jest domyślna |
| `-Firefox` | SwitchParameter | — | — | Named | — | Preferuj sesję przeglądarki Firefox |
| `-Webkit` | SwitchParameter | — | — | Named | — | Preferuj sesję przeglądarki WebKit |

## Examples

### Get a reference to the current browser session $sessionRef = Get-PlaywrightSessionReference

```powershell
Get a reference to the current browser session
$sessionRef = Get-PlaywrightSessionReference
```

### Store the reference and use it later to execute JavaScript $ref = Get-PlaywrightSessionReference Invoke-WebbrowserEvaluation "document.title" -ByReference $ref

```powershell
Store the reference and use it later to execute JavaScript
$ref = Get-PlaywrightSessionReference
Invoke-WebbrowserEvaluation "document.title" -ByReference $ref
```
