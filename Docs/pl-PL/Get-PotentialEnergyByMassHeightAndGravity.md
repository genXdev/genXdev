# Get-PotentialEnergyByMassHeightAndGravity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza energię potencjalną grawitacji.

## Description

Używa PE = m g h.

## Syntax

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms <Double> -HeightInMeters <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Masa w kg |
| `-HeightInMeters` | Double | ✅ | — | 1 | `0` | Wysokość w metrach |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Grawitacja w m/s² (domyślnie: 9,81) |
| `-As` | String | — | — | 3 | — | Jednostka wyjściowa dla energii |

## Examples

### Example 1

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms 10 -HeightInMeters 5 -As "calories"
```

Oblicza energię potencjalną grawitacji dla masy 10 kg na wysokości 5 metrów, podając wynik w kaloriach.

### Example 2

```powershell
Get-PotentialEnergyByMassHeightAndGravity 5 10
```

Pokazuje użycie parametrów pozycyjnych do obliczania energii potencjalnej.
