# Get-ProjectileRangeByInitialSpeedAndAngle

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza zasięg pocisku.

## Description

Używa wzoru R = (v² sin(2θ)) / g dla idealnego rzutu ukośnego.

## Syntax

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle -InitialSpeedInMetersPerSecond <Double> -AngleInDegrees <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InitialSpeedInMetersPerSecond` | Double | ✅ | — | 0 | `0` | Prędkość początkowa w m/s |
| `-AngleInDegrees` | Double | ✅ | — | 1 | `0` | Kąt wyrzutu w stopniach |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Grawitacja w m/s² (domyślnie: 9,81) |
| `-As` | String | — | — | 3 | — | Jednostka wyniku dla zakresu |

## Examples

### Example 1

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle -InitialSpeedInMetersPerSecond 20 -AngleInDegrees 45 -As "feet"
```

Oblicza zasięg przy użyciu wzoru na ruch pocisku.

### Example 2

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle 30 30
```

Oblicza zakres z parametrami pozycyjnymi.
