# Get-RefractionAngleByIncidentAngleAndIndices

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza kąt załamania za pomocą prawa Snelliusa.

## Description

Używa θ2 = arcsin((n1 / n2) sin θ1).

## Syntax

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees <Double> -IndexOfRefraction1 <Double> -IndexOfRefraction2 <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-IncidentAngleInDegrees` | Double | ✅ | — | 0 | `0` | Kąt padania w stopniach |
| `-IndexOfRefraction1` | Double | ✅ | — | 1 | `0` | Współczynnik załamania pierwszego ośrodka |
| `-IndexOfRefraction2` | Double | ✅ | — | 2 | `0` | Współczynnik załamania drugiego ośrodka |
| `-As` | String | — | — | 3 | — | Jednostka wyjściowa dla kąta |

## Examples

### Example 1

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees 30 -IndexOfRefraction1 1 -IndexOfRefraction2 1.33 -As "radians"
```

Oblicza kąt załamania, gdy światło przechodzi z powietrza (n=1) do wody (n=1,33) pod kątem padania 30 stopni.

### Example 2

```powershell
Get-RefractionAngleByIncidentAngleAndIndices 45 1 1.5
```

Oblicza kąt załamania, gdy światło przechodzi z powietrza (n=1) do szkła (n=1,5) pod kątem padania 45 stopni.
