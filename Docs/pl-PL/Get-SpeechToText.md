# Get-SpeechToText

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Konwertuje pliki audio na tekst za pomocą modelu rozpoznawania mowy Whisper od OpenAI.

## Description

Przetwarza pliki audio i konwertuje mowę na tekst przy użyciu biblioteki Whisper.NET, która implementuje system automatycznego rozpoznawania mowy (ASR) Whisper od OpenAI. Obsługuje wiele języków, możliwości tłumaczenia oraz różne ustawienia jakości transkrypcji.

## Examples

### Example 1

```powershell
Get-SpeechToText -Input "C:\audio\recording.wav"
```

Transkrybuje plik audio na tekst przy użyciu domyślnych ustawień.

### Example 2

```powershell
Get-ChildItem "C:\audio\*.wav" | Get-SpeechToText
```

Transkrybuje wszystkie pliki WAV w katalogu.

### Example 3

```powershell
Get-SpeechToText -Input "audio.mp3" -LanguageIn "es" -WithTranslate
```

Transkrybuje hiszpańskie audio i tłumaczy je na angielski.

### Example 4

```powershell
Get-SpeechToText -Input "recording.wav" -Passthru -WithTokenTimestamps
```

Zwraca obiekty SegmentData z precyzyjnymi informacjami o czasie.
