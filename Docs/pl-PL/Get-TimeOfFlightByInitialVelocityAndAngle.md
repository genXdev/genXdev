# Get-TimeOfFlightByInitialVelocityAndAngle

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza czas lotu pocisku.

## Description

Używa T = (2 v sinθ) / g dla ruchu idealnego.

## Syntax

```powershell
Get-TimeOfFlightByInitialVelocityAndAngle -InitialVelocityInMetersPerSecond <Double> -AngleInDegrees <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InitialVelocityInMetersPerSecond` | Double | ✅ | — | 0 | `0` | Prędkość początkowa w m/s |
| `-AngleInDegrees` | Double | ✅ | — | 1 | `0` | Kąt wyrzutu w stopniach |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Grawitacja w m/s² (domyślnie: 9,81) |
| `-As` | String | — | — | 3 | — | Jednostka czasu wyjścia |

## Examples

### Example 1

```powershell
Get-TimeOfFlightByInitialVelocityAndAngle -InitialVelocityInMetersPerSecond 20 -AngleInDegrees 45 -As "minutes"
```

Oblicza czas lotu pocisku wystrzelonego z prędkością 20 m/s pod kątem 45 stopni, podawany w minutach.

### Example 2

```powershell
Get-TimeOfFlightByInitialVelocityAndAngle 30 30
```

Oblicza czas lotu dla pocisku wystrzelonego z prędkością 30 m/s pod kątem 30 stopni.
