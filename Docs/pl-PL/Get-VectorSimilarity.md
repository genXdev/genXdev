# Get-VectorSimilarity

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Oblicza podobieństwo cosinusowe między dwoma wektorami.

## Description

Przyjmuje dwa numeryczne wektory (tablice) jako dane wejściowe i oblicza ich podobieństwo cosinusowe. Wynik wskazuje, jak blisko powiązane są wektory, gdzie 0 oznacza całkowite niepodobieństwo, a 1 identyczność.

## Syntax

```powershell
Get-VectorSimilarity -Vector1 <Double[]> -Vector2 <Double[]> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Vector1` | Double[] | ✅ | — | 0 | — | Pierwsza tablica wektorowa liczb |
| `-Vector2` | Double[] | ✅ | — | 1 | — | Druga tablica wektorowa liczb |

## Examples

### Example 1

```powershell
$v1 = @(0.12, -0.45, 0.89)
$v2 = @(0.15, -0.40, 0.92)
Get-VectorSimilarity -Vector1 $v1 -Vector2 $v2
```

Zwraca około 0,998, wskazując na wysokie podobieństwo.
