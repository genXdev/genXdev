# Invoke-ImageScenesUpdate

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `imagescenedetection`

## Synopsis

> Aktualizuje metadane klasyfikacji scen dla plików obrazów w określonym katalogu.

## Description

Ta funkcja przetwarza obrazy w określonym katalogu w celu klasyfikacji scen za pomocą sztucznej inteligencji. Tworzy pliki metadanych JSON zawierające klasyfikacje scen, wyniki pewności i etykiety. Funkcja obsługuje przetwarzanie wsadowe z konfigurowalnymi progami pewności i może opcjonalnie pomijać istniejące pliki metadanych lub ponawiać wcześniej nieudane klasyfikacje.

## Syntax

```powershell
Invoke-ImageScenesUpdate [[-ImageDirectories] <String[]>] [-ApiKey <String>] [-AutoUpdateFaces] [-ClearSession] [-ConfidenceThreshold <Double>] [-ContainerName <String>] [-FacesDirectory <String>] [-Force] [-HealthCheckInterval <Int32>] [-HealthCheckTimeout <Int32>] [-ImageName <String>] [-Language <String>] [-Model <String>] [-NoDockerInitialize] [-NoSupportForJsonSchema <String>] [-OnlyNew] [-PassThru] [-PreferencesDatabasePath <String>] [-Recurse] [-RetryFailed] [-ServicePort <Int32>] [-SessionOnly] [-ShowWindow] [-SkipSession] [-TimeoutSecond <Int32>] [-UseGPU] [-VolumeName <String>] [<CommonParameters>]
```

## Examples

### Invoke-ImageScenesUpdate -ImageDirectories @("C:\Photos", "D:\Pictures") -Recurse

```powershell
Invoke-ImageScenesUpdate -ImageDirectories @("C:\Photos", "D:\Pictures") -Recurse
```

Przetwarza wszystkie obrazy w C:\Photos i D:\Pictures oraz podkatalogach pod kątem klasyfikacji scen.

### scenerecognition @("C:\Photos", "C:\Archive") -RetryFailed -OnlyNew

```powershell
scenerecognition @("C:\Photos", "C:\Archive") -RetryFailed -OnlyNew
```

Używa aliasu do ponowienia klasyfikacji nieudanych oraz przetwarzania tylko nowych obrazów w wielu katalogach.

### Invoke-ImageScenesUpdate -ImageDirectories ".\MyImages" -Force -UseGPU

```powershell
Invoke-ImageScenesUpdate -ImageDirectories ".\MyImages" -Force -UseGPU
```

Wymusza przebudowę kontenera i używa akceleracji GPU w celu szybszego przetwarzania.

### Invoke-ImageScenesUpdate -ImageDirectories "C:\Photos" -ConfidenceThreshold 0.6 -Recurse

```powershell
Invoke-ImageScenesUpdate -ImageDirectories "C:\Photos" -ConfidenceThreshold 0.6 -Recurse
```

Przetwarza wszystkie obrazy rekurencyjnie i przechowuje tylko klasyfikacje scen z ufnością >= 60%.
