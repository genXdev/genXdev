# Open-StackOverflowQuery

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** `qso`

## Synopsis

> Otwiera zapytania wyszukiwania Stack Overflow w przeglądarce internetowej.

## Description

Otwiera jedno lub wiele zapytań Stack Overflow w przeglądarce internetowej z zaawansowanymi opcjami konfiguracji przeglądarki i wyświetlania. Ta funkcja zapewnia zaawansowane opakowanie wokół funkcji wyszukiwania Stack Overflow z rozbudowanymi opcjami pozycjonowania okna, wyboru przeglądarki i dostosowania zachowania.

Kluczowe funkcje:
- Obsługa wielu zapytań wyszukiwania z wejściem potokowym
- Inteligentne wykrywanie i wybór przeglądarki (Edge, Chrome, Firefox, wszystkie przeglądarki)
- Zaawansowane pozycjonowanie okna (lewo, prawo, góra, dół, wyśrodkowane, pełny ekran)
- Obsługa wielu monitorów z automatycznym lub ręcznym wyborem monitora
- Obsługa trybu prywatnego/incognito
- Tryb aplikacji dla przeglądania bez rozpraszania
- Obsługa lokalizacji językowej dla międzynarodowych wyników wyszukiwania
- Opcje blokowania rozszerzeń i wyskakujących okien
- Zarządzanie fokusem i manipulacja oknem
- Automatyzacja naciśnięć klawiszy w oknach przeglądarki
- Opcje zwracania adresów URL dla przepływów automatyzacji

Funkcja automatycznie wykrywa możliwości systemu i odpowiednio dostosowuje zachowanie. Dla przeglądarek niezainstalowanych w systemie operacje są cicho pomijane bez błędów.

## Syntax

```powershell
Open-StackOverflowQuery -Queries <String[]> [[-Language] <String>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-Monitor <Int32>] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-StackOverflowQuery -Queries "powershell array" -Monitor 0

```powershell
Open-StackOverflowQuery -Queries "powershell array" -Monitor 0
```

Otwiera wyszukiwanie w Stack Overflow dla "powershell array" na głównym monitorze.

### qso "powershell array" -mon 0

```powershell
qso "powershell array" -mon 0
```

Otwiera wyszukiwanie Stack Overflow przy użyciu aliasu z pozycjonowaniem monitora.

### "powershell", "array manipulation" | Open-StackOverflowQuery -Language "English" -Chrome

```powershell
"powershell", "array manipulation" | Open-StackOverflowQuery -Language "English" -Chrome
```

Otwiera wiele wyszukiwań Stack Overflow w Chrome z preferencją języka angielskiego.

### Open-StackOverflowQuery -Queries "c# linq" -ReturnURL

```powershell
Open-StackOverflowQuery -Queries "c# linq" -ReturnURL
```

Zwraca adres URL wyszukiwania Stack Overflow bez otwierania przeglądarki.
