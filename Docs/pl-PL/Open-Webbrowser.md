# Open-Webbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `wb`

## Synopsis

> Otwiera adresy URL w jednym lub więcej oknach przeglądarki z opcjonalnym pozycjonowaniem i stylowaniem.

## Description

Ta funkcja zapewnia zaawansowaną otoczkę wokół uruchamiania przeglądarki z rozbudowanymi opcjami pozycjonowania okna, wyboru przeglądarki i dostosowywania zachowania. Obsługuje wiele przeglądarek, w tym Edge, Chrome i Firefox, z funkcjami takimi jak tryb prywatny, tryb aplikacji i precyzyjne zarządzanie oknami.

Kluczowe funkcje:
- Inteligentne wykrywanie i wybór przeglądarki
- Pozycjonowanie okna (lewo, prawo, góra, dół, wyśrodkowane, pełny ekran)
- Obsługa wielu monitorów z automatycznym lub ręcznym wyborem monitora
- Obsługa trybu prywatnego/incognito
- Tryb aplikacji do przeglądania bez rozpraszania uwagi
- Opcje blokowania rozszerzeń i wyskakujących okien
- Zarządzanie fokusem i manipulowanie oknem
- Otwieranie partii adresów URL w wielu przeglądarkach
- Automatyzacja naciśnięć klawiszy w oknach przeglądarki

Funkcja może automatycznie wykrywać możliwości systemu i odpowiednio dostosowywać zachowanie. Dla przeglądarek niezainstalowanych w systemie operacje są po cichu pomijane bez błędów.

## Examples

### wb -PlayWright https://github.com

```powershell
wb -PlayWright https://github.com
```

Otwiera GitHub w przeglądarce Chromium zarządzanej przez Playwright.

### Open-Webbrowser -Url "https://github.com"

```powershell
Open-Webbrowser -Url "https://github.com"
```

Otwiera GitHub w domyślnej przeglądarce.

### Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left

```powershell
Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left
```

Otwiera Stack Overflow w lewej połowie monitora 1.

### wb "https://google.com" -m 0 -fs

```powershell
wb "https://google.com" -m 0 -fs
```

Otwiera Google w trybie pełnoekranowym na głównym monitorze przy użyciu aliasów.

### Open-Webbrowser -Chrome -Private -NewWindow

```powershell
Open-Webbrowser -Chrome -Private -NewWindow
```

Otwiera nowe okno Chrome w trybie incognito.

### "https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All

```powershell
"https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All
```

Otwiera wiele adresów URL we wszystkich zainstalowanych przeglądarkach za pomocą potoku.

### Open-Webbrowser -Monitor 0 -Right

```powershell
Open-Webbrowser -Monitor 0 -Right
```

Przesuwa już otwarte okno przeglądarki na prawą stronę głównego monitora.

### Open-Webbrowser -ApplicationMode -Url "https://app.example.com"

```powershell
Open-Webbrowser -ApplicationMode -Url "https://app.example.com"
```

Otwiera aplikację internetową w trybie aplikacji bez kontrolek przeglądarki.
