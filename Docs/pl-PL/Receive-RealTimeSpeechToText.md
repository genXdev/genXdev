# Receive-RealTimeSpeechToText

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Konwertuje dźwięk w czasie rzeczywistym na tekst za pomocą modelu Whisper AI.

## Description

To polecenie cmdlet przechwytuje dźwięk z mikrofonu lub pulpitu i transkrybuje go na tekst w czasie rzeczywistym przy użyciu modelu Whisper AI. Obsługuje różne źródła dźwięku, wykrywanie ciszy oraz wiele opcji konfiguracyjnych rozpoznawania mowy.

## Examples

### Example 1

```powershell
Receive-RealTimeSpeechToText
```

Podstawowe użycie z mikrofonem.

### Example 2

```powershell
Receive-RealTimeSpeechToText -UseDesktopAudioCapture
```

Używając przechwytywania dźwięku z pulpitu.

### Example 3

```powershell
Receive-RealTimeSpeechToText -AudioDevice "Microphone*"
```

Używanie określonego urządzenia audio.

### Example 4

```powershell
Receive-RealTimeSpeechToText -MaxDurationOfSilence "00:00:05"
```

Z wykrywaniem ciszy.
