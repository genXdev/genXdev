# Remove-AllItems

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** `sdel`

## Synopsis

> Rekurencyjnie usuwa całą zawartość katalogu z zaawansowaną obsługą błędów.

## Description

Bezpiecznie usuwa wszystkie pliki i podkatalogi w określonym katalogu, stosując strategię usuwania w odwrotnej kolejności, aby obsłużyć głębokie ścieżki. Obejmuje obsługę WhatIf, rejestrowanie szczegółowe i metody awaryjne usuwania zablokowanych plików.

## Syntax

```powershell
Remove-AllItems -Path <String> [-DeleteFolder] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Ścieżka katalogu do wyczyszczenia |
| `-DeleteFolder` | SwitchParameter | — | — | Named | — | Usuń także folder główny dostarczony z parametrem Path |

## Examples

### Remove-AllItems -Path "C:\Temp\BuildOutput" -DeleteFolder -Verbose

```powershell
Remove-AllItems -Path "C:\Temp\BuildOutput" -DeleteFolder -Verbose
```

### sdel ".\temp" -DeleteFolder

```powershell
sdel ".\temp" -DeleteFolder
```
