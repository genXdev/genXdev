# Save-DesktopScreenShot

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Przechwytuje zrzut ekranu określonego monitora i zapisuje go do określonej ścieżki.

## Description

* Wykonuje zrzut ekranu pulpitu z określonego monitora i zapisuje go jako plik obrazu.
* Obsługuje wiele monitorów i automatycznie wykrywa prawidłowe granice ekranu oraz współczynniki skalowania dla wyświetlaczy o wysokiej rozdzielczości (HiDPI).

## Syntax

```powershell
Save-DesktopScreenShot -Monitor <Int32> -OutputPath <String> [-Quality <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | ✅ | ✅ (ByPropertyName) | 0 | `0` | Indeks monitora do przechwycenia (0=podstawowy, 1+=określony monitor, -1=wszystkie monitory) |
| `-OutputPath` | String | ✅ | ✅ (ByPropertyName) | 1 | — | Pełna ścieżka, pod którą zostanie zapisany zrzut ekranu |
| `-Quality` | Int32 | — | — | Named | `0` | Poziom jakości JPEG (1-100), dotyczy tylko formatu JPEG |

## Examples

### Example 1

```powershell
Save-DesktopScreenShot -Monitor 0 -OutputPath "C:\Screenshots\primary.png"
```

Przechwytuje zrzut ekranu głównego monitora i zapisuje go jako PNG.

### Example 2

```powershell
Save-DesktopScreenShot -Monitor 1 -OutputPath "C:\Screenshots\monitor2.jpg" -Quality 95
```

Tworzy zrzut ekranu drugiego monitora w wysokiej jakości JPEG.

### Example 3

```powershell
Save-DesktopScreenShot -Monitor -1 -OutputPath "C:\Screenshots\all_monitors.png"
```

Przechwytuje wszystkie monitory połączone w jeden zrzut ekranu.
