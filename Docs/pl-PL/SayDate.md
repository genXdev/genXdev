# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Wypowiada bieżącą datę przy użyciu syntezy mowy.

## Description

Konwertuje bieżącą datę na format języka naturalnego i ogłasza ją za pomocą systemowego silnika zamiany tekstu na mowę. Data jest wypowiadana w formacie: "Jest [dzień tygodnia], [miesiąc] [dzień] [rok]"

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
