# Search-GenXdevCmdlet

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `searchcmdlet`

## Synopsis

> Wyszukuje polecenia cmdlet GenXdev i opcjonalnie otwiera je w IDE do edycji.

## Description

Ta funkcja wyszukuje polecenia cmdlet GenXdev według wzorca i opcjonalnie otwiera plik źródłowy znalezionego polecenia cmdlet w programie Visual Studio Code lub Visual Studio w celu edycji. Pobiera informacje o poleceniu cmdlet i udostępnia skróty klawiaturowe umożliwiające szybkie przejście do definicji funkcji.

## Syntax

```powershell
Search-GenXdevCmdlet [[-CmdletName] <String>] [[-ModuleName] <String[]>] [-Code] [-EditCmdlet] [-FromScripts] [-NoLocal] [-OnlyPublished] [-VisualStudio] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CmdletName` | String | — | — | 0 | — | Wzorzec wyszukiwania do filtrowania poleceń cmdlet 🌐 *Supports wildcards* |
| `-ModuleName` | String[] | — | ✅ (ByValue, ByPropertyName) | 1 | — | Nazwy modułów GenXdev do wyszukania 🌐 *Supports wildcards* |
| `-NoLocal` | SwitchParameter | — | — | Named | — | Pomiń wyszukiwanie w lokalnych ścieżkach modułów |
| `-OnlyPublished` | SwitchParameter | — | — | Named | — | Ogranicz wyszukiwanie tylko do opublikowanych ścieżek modułów |
| `-FromScripts` | SwitchParameter | — | — | Named | — | Wyszukaj w plikach skryptów, a nie w plikach modułów |
| `-Code` | SwitchParameter | — | — | Named | — | Otwórz znalezione polecenie cmdlet w Visual Studio Code |
| `-VisualStudio` | SwitchParameter | — | — | Named | — | Otwórz znalezioną cmdlet w programie Visual Studio |
| `-EditCmdlet` | SwitchParameter | — | — | Named | — | Otwiera również plik w edytorze po znalezieniu |

## Examples

### Search-GenXdevCmdlet -CmdletName "Get-*" -Code Opens first matching cmdlet starting with "Get-" in Visual Studio Code.

```powershell
Search-GenXdevCmdlet -CmdletName "Get-*" -Code
Opens first matching cmdlet starting with "Get-" in Visual Studio Code.
```

### searchcmdlet "Set-GitConfig" -EditCmdlet Searches for Set-GitConfig cmdlet and opens it for editing.

```powershell
searchcmdlet "Set-GitConfig" -EditCmdlet
Searches for Set-GitConfig cmdlet and opens it for editing.
```
