# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> Przechodzi o dwa poziomy w górę w hierarchii systemu plików.

## Description

* Zmienia bieżący katalog roboczy na katalog nadrzędny (dwa poziomy wyżej).
* Wyświetla zawartość wynikowego katalogu.

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Przechodzi do katalogu nadrzędnego i wyświetla jego zawartość.

### Example 2

```powershell
...
```

Używa aliasu '...' dla tej samej operacji.
