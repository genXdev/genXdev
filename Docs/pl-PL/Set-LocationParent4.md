# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> Przechodzi o cztery poziomy w górę w hierarchii systemu plików.

## Description

* Przesuwa się o cztery poziomy w górę w systemie plików z bieżącej lokalizacji.
* Po nawigacji wyświetla zawartość wynikowego katalogu.

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Przechodzi o cztery poziomy katalogów w górę z bieżącej lokalizacji.

### Example 2

```powershell
.....
```

Używa aliasu, aby przejść o cztery poziomy katalogów w górę.
