# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> Przechodzi o pięć poziomów katalogów w górę w hierarchii systemu plików.

## Description

* Zmienia bieżący katalog roboczy, przechodząc o pięć poziomów w górę w hierarchii katalogów.
* Wyświetla zawartość wynikowego katalogu.
* Udostępnia wygodny alias skrócony '......' dla szybkiej nawigacji.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Przechodzi pięć poziomów katalogów w górę od bieżącej lokalizacji.

### Example 2

```powershell
......
```

Używa aliasu, aby przejść w górę o pięć poziomów katalogów.
