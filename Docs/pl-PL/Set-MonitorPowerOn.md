# Set-MonitorPowerOn

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `wakemonitor`, `monitoroff`

## Synopsis

> Włącza zasilanie monitora.

## Description

Używa interfejsu Windows API przez GenXdev.Helpers.WindowObj, aby wybudzić monitor ze stanu uśpienia/wyłączenia. Jest to przydatne w skryptach automatyzacji, które muszą upewnić się, że monitor jest włączony.

## Syntax

```powershell
Set-MonitorPowerOn [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOn
```

Włącza zasilanie monitora.

### Example 2

```powershell
wake-monitor
```

Włącza zasilanie monitora przy użyciu aliasu.
