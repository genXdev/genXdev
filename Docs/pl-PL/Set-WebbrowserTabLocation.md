# Set-WebbrowserTabLocation

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `lt`, `Nav`

## Synopsis

> Nawiguje bieżącą kartę przeglądarki do określonego adresu URL.

## Description

Ustawia lokalizację (URL) aktualnie wybranej karty przeglądarki internetowej. Obsługuje zarówno przeglądarki Edge, jak i Chrome za pomocą opcjonalnych przełączników. Nawigacja obejmuje walidację adresu URL i zapewnia prawidłowe ładowanie strony poprzez operacje asynchroniczne.

## Syntax

```powershell
Set-WebbrowserTabLocation -Url <String> [-ByReference <PSObject>] [-Chromium] [-Firefox] [-NoAutoSelectTab] [-Page <Object>] [-Webkit] [<CommonParameters>]

Set-WebbrowserTabLocation [-Edge] [<CommonParameters>]

Set-WebbrowserTabLocation [-Chrome] [<CommonParameters>]
```

| Name | Type | Required | Pipeline | Position | Default | Description |

| `-Edge` | SwitchParameter | — | — | Named | — | Nawiguj przy użyciu przeglądarki Microsoft Edge *(Parameter set: )* |
| `-Chrome` | SwitchParameter | — | — | Named | — | Nawiguj za pomocą przeglądarki Google Chrome *(Parameter set: )* |
| `-Chromium` | SwitchParameter | — | — | Named | — | Nawiguj przy użyciu Microsoft Edge lub Google Chrome, w zależności od tego, która przeglądarka jest domyślna |

### Set-WebbrowserTabLocation -Url "https://github.com/microsoft" -Edge

```powershell
Set-WebbrowserTabLocation -Url "https://github.com/microsoft" -Edge
```
