# Start-VlcMediaPlayerPreviousInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcprev`, `vlcback`

## Synopsis

> Przechodzi do poprzedniego elementu na liście odtwarzania VLC Media Player.

## Description

Ta funkcja wysyła polecenie klawisza 'p' do odtwarzacza multimedialnego VLC, aby przejść do poprzedniego elementu na bieżącej liście odtwarzania. Funkcja obsługuje operacje WhatIf i po wysłaniu polecenia przywraca fokus.

## Syntax

```powershell
Start-VlcMediaPlayerPreviousInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerPreviousInPlaylist

```powershell
Start-VlcMediaPlayerPreviousInPlaylist
```

### vlcprev

```powershell
vlcprev
```

### vlcback

```powershell
vlcback
```
