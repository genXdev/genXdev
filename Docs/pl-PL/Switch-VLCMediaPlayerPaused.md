# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> Przełącza stan pauzy/odtwarzania odtwarzacza VLC Media Player.

## Description

Funkcja ta wysyła klawisz spacji do odtwarzacza VLC Media Player, aby przełączać między stanem wstrzymania a odtwarzania. Po wysłaniu polecenia klawisza automatycznie przywraca fokus do oryginalnego okna.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

Przełącza stan pauzy/odtwarzania odtwarzacza VLC Media Player.

### vlcpause

```powershell
vlcpause
```

Używa aliasu do przełączania stanu pauzy/odtwarzania.

### vlcplay

```powershell
vlcplay
```

Używa alternatywnego aliasu do przełączania stanu pauzy/odtwarzania.
