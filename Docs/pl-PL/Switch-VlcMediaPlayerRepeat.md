# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> Przełącza tryb powtarzania w odtwarzaczu multimedialnym VLC.

## Description

Ta funkcja wysyła polecenie klawisza 'r' do odtwarzacza VLC Media Player, aby przełączać między różnymi trybami powtarzania (brak powtarzania, powtarzanie bieżącego, powtarzanie wszystkich). Funkcja uruchamia VLC, jeśli nie jest jeszcze uruchomiony, i przywraca fokus do poprzedniego okna po wysłaniu polecenia.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
