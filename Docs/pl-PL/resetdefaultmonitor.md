# resetdefaultmonitor

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Przywraca domyślną konfigurację monitora pomocniczego.

## Description

Ten skrypt przywraca domyślną konfigurację drugiego monitora dla systemu, ustawiając drugi monitor na pierwotną wartość domyślną. Jest to przydatne dla użytkowników, którzy chcą powrócić do poprzedniej konfiguracji wielu monitorów po użyciu układów obok siebie. Zobacz także: funkcja 'sidebyside' do przełączania na tryb równoległy dla nowych okien.

## Syntax

```powershell
resetdefaultmonitor [<CommonParameters>]
```

## Examples

### secondscreen

```powershell
secondscreen
```

Przywraca domyślną konfigurację monitora dodatkowego dla systemu.
