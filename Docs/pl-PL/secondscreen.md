# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Ustawia domyślną konfigurację drugiego monitora.

## Description

Ustawia domyślne zachowanie otwierania okien GenXdev na monitorze pomocniczym.
Jest to przydatne dla użytkowników z jednym monitorem lub preferujących układ okien obok siebie.
Zobacz także polecenia cmdlet 'sidebyside' i 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Ustawia domyślne otwieranie okien GenXdev na drugim monitorze
