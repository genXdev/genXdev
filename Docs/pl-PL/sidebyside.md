# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Ustawia domyślną konfigurację side-by-side.

## Description

Ustawia domyślne zachowanie otwierania okien GenXdev obok siebie z programem PowerShell. Jest to przydatne dla użytkowników z jednym monitorem lub tych, którzy preferują układ okien obok siebie. Zobacz także polecenia cmdlet 'secondscreen' i 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

Ustawia domyślne zachowanie otwierania okien GenXdev obok siebie z PowerShell
