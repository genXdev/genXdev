# Assert-GenXdevDependencyUsage

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `checkgenxdevdependencies`

## Synopsis

> Valida o uso de dependências nos módulos do GenXdev para garantir que a hierarquia adequada de módulos seja mantida.

## Description

Esta função analisa os módulos GenXdev para garantir que sigam a hierarquia correta de dependências. Ela verifica se os módulos apenas referenciam dependências que estão listadas em seu manifesto RequiredModules e previne dependências circulares ao validar que os módulos não referenciam módulos que vêm depois na cadeia de dependências.

## Syntax

```powershell
Assert-GenXdevDependencyUsage [[-ModuleName] <String[]>] [-FromScripts] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String[] | — | ✅ (ByValue, ByPropertyName) | 1 | `@('GenXdev*')` | Filtro para aplicar aos nomes de módulo 🌐 *Supports wildcards* |
| `-FromScripts` | SwitchParameter | — | — | Named | — | Pesquisar em arquivos de script em vez de arquivos de módulo |

## Examples

### Assert-GenXdevDependencyUsage -ModuleName "GenXdev.Coding"

```powershell
Assert-GenXdevDependencyUsage -ModuleName "GenXdev.Coding"
```

### checkgenxdevdependencies "GenXdev*" -FromScripts

```powershell
checkgenxdevdependencies "GenXdev*" -FromScripts
```
