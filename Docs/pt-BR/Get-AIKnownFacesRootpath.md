# Get-AIKnownFacesRootpath

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> Obtém o diretório configurado para arquivos de imagem de rosto usados nas operações do GenXdev.AI.

## Description

Esta função recupera o diretório global de faces usado pelo módulo GenXdev.AI para várias operações de reconhecimento facial e IA. Ela verifica primeiro as variáveis Globais (a menos que SkipSession seja especificado), então recorre às preferências persistentes e, finalmente, usa os padrões do sistema.

## Syntax

```powershell
Get-AIKnownFacesRootpath [[-FacesDirectory] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FacesDirectory` | String | — | — | 0 | — | Caminho do diretório para os arquivos de imagem de rosto |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Use configurações alternativas armazenadas na sessão para preferências de IA como Idioma, Coleções de imagens, etc. |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Limpe a configuração da sessão (variável global) antes de recuperar |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Caminho do banco de dados para arquivos de dados de preferência |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Não use configurações alternativas armazenadas na sessão para preferências de IA como Idioma, Coleções de imagens, etc. |

## Examples

### Get-AIKnownFacesRootpath

```powershell
Get-AIKnownFacesRootpath
```

Obtém o diretório de rostos atualmente configurado a partir de variáveis globais ou preferências.

### Get-AIKnownFacesRootpath -SkipSession

```powershell
Get-AIKnownFacesRootpath -SkipSession
```

Obtém o diretório de faces configurado apenas das preferências persistentes, ignorando qualquer configuração de sessão.

### Get-AIKnownFacesRootpath -ClearSession

```powershell
Get-AIKnownFacesRootpath -ClearSession
```

Limpa a configuração do diretório de sessões e obtém o diretório das preferências persistentes.

### Get-AIKnownFacesRootpath "C:\MyFaces"

```powershell
Get-AIKnownFacesRootpath "C:\MyFaces"
```

Retorna o diretório especificado após expandir o caminho.
