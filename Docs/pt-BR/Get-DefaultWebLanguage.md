# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Obtém a chave de idioma padrão da Web com base nas configurações de idioma atuais do sistema.

## Description

* Recupera as configurações atuais de idioma e cultura do sistema e as mapeia
  para a chave correspondente do dicionário de idiomas da web usada pelos
  serviços de tradução.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Retorna "Inglês" para um sistema inglês, "Holandês" para um sistema holandês, etc.
