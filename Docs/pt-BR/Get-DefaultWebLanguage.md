# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Obtém a chave de idioma web padrão com base nas configurações atuais de idioma do sistema.

## Description

* Recupera as configurações atuais de idioma e cultura do sistema e as mapeia
  para a chave do dicionário de idiomas da web correspondente usada pelos serviços
  de tradução.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Retorna "Inglês" para um sistema em inglês, "Holandês" para um sistema em holandês, etc.
