# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss

## Synopsis

> Retorna verdadeiro se o mecanismo de texto para fala estiver falando.

## Description

Verifica o estado dos sintetizadores de fala padrão e personalizados para determinar se algum está falando no momento. Retorna verdadeiro se a fala estiver em andamento, falso caso contrário.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Retorna verdadeiro se o mecanismo de texto para fala estiver falando.

### Example 2

```powershell
iss
```

Retorna verdadeiro se o mecanismo de texto para fala estiver falando (usando alias).
