# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Retorna um objeto auxiliar de janela para a janela principal do terminal do PowerShell.

## Description

* Recupera um objeto auxiliar WindowObj que representa a janela principal do
  processo host atual do PowerShell.
* Isso permite manipulação e interação com a própria janela do terminal.

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

Recupera o objeto WindowObj para a janela de terminal atual do PowerShell.
