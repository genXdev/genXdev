# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Retorna um dicionário reverso para todos os idiomas suportados pela Pesquisa Google.

## Description

* Retorna um dicionário mapeando nomes de idiomas para seus códigos
  de idioma correspondentes para todos os idiomas suportados pela Pesquisa Google.
* Útil para consultas de códigos de idioma em pesquisas web e tarefas de tradução.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

Recupera o mapeamento de nomes de idiomas para seus códigos.
