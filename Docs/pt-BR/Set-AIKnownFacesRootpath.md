# Set-AIKnownFacesRootpath

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> Define o diretório para arquivos de imagem facial usados nas operações da GenXdev.AI.

## Description

Esta função configura o diretório global de faces usado pelo módulo GenXdev.AI para várias operações de reconhecimento facial e IA. As configurações podem ser armazenadas de forma persistente nas preferências (padrão), apenas na sessão atual (usando -SessionOnly) ou removidas da sessão (usando -ClearSession).

## Syntax

```powershell
Set-AIKnownFacesRootpath [[-FacesDirectory] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FacesDirectory` | String | — | — | 0 | — | Caminho do diretório para os arquivos de imagem de rosto |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Caminho do banco de dados para arquivos de dados de preferência |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Use configurações alternativas armazenadas na sessão para preferências de IA como Idioma, Coleções de imagens, etc. |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Limpe as configurações alternativas armazenadas na sessão para preferências de IA como Idioma, coleções de Imagens, etc. |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Não use configurações alternativas armazenadas na sessão para preferências de IA como Idioma, Coleções de imagens, etc. |

## Examples

### Set-AIKnownFacesRootpath -FacesDirectory "C:\Faces"

```powershell
Set-AIKnownFacesRootpath -FacesDirectory "C:\Faces"
```

Define o diretório de faces de forma persistente nas preferências.

### Set-AIKnownFacesRootpath "C:\FacePictures"

```powershell
Set-AIKnownFacesRootpath "C:\FacePictures"
```

Define o diretório de faces de forma persistente nas preferências.

### Set-AIKnownFacesRootpath -FacesDirectory "C:\TempFaces" -SessionOnly

```powershell
Set-AIKnownFacesRootpath -FacesDirectory "C:\TempFaces" -SessionOnly
```

Define o diretório de faces apenas para a sessão atual (variável global).

### Set-AIKnownFacesRootpath -ClearSession

```powershell
Set-AIKnownFacesRootpath -ClearSession
```

Limpa a configuração do diretório de faces da sessão (variável Global) sem afetar as preferências persistentes.
