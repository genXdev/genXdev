# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> Navega para cima dois níveis de diretório na hierarquia do sistema de arquivos.

## Description

* Altera o diretório de trabalho atual para o diretório avô
  (dois níveis acima).
* Exibe o conteúdo do diretório resultante.

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Altera para o diretório pai e exibe seu conteúdo.

### Example 2

```powershell
...
```

Usa o alias '...' para a mesma operação.
