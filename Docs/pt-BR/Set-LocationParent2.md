# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...

## Synopsis

> Navega dois níveis acima na hierarquia do sistema de arquivos.

## Description

* Altera o diretório de trabalho atual para o diretório avô (dois níveis acima).
* Exibe o conteúdo do diretório resultante.

## Syntax

```powershell
Set-LocationParent2 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Altera para o diretório avô e exibe seu conteúdo.
