# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....

## Synopsis

> Navega três níveis de diretório acima na hierarquia do sistema de arquivos.

## Description

* Altera o diretório de trabalho atual subindo três níveis
  na hierarquia do sistema de arquivos.
* Exibe o conteúdo do diretório resultante.

## Syntax

```powershell
Set-LocationParent3 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

Navega três níveis de diretório acima da localização atual.

```powershell
....
```
