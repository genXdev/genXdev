# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> Navega quatro níveis de diretório acima na hierarquia do sistema de arquivos.

## Description

* Move quatro níveis de diretório acima da localização atual no sistema de arquivos.
* Após a navegação, exibe o conteúdo do diretório resultante.

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Navega quatro níveis de diretório para cima a partir da localização atual.

### Example 2

```powershell
.....
```

Usa o alias para navegar quatro níveis de diretório acima.
