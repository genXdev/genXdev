# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....

## Synopsis

> Navega quatro níveis de diretório acima na hierarquia do sistema de arquivos.

## Description

* Sobe quatro níveis de diretório a partir do local atual no sistema de arquivos.
* Após a navegação, exibe o conteúdo do diretório resultante.

## Syntax

```powershell
Set-LocationParent4 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Navega quatro níveis de diretório acima a partir da localização atual.

```powershell
.....
```

Usa o alias para navegar quatro níveis de diretório acima.
