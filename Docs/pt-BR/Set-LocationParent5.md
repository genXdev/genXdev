# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> Navega cinco níveis de diretório acima na hierarquia do sistema de arquivos.

## Description

* Altera o diretório de trabalho atual subindo cinco níveis pai
  de diretório.
* Exibe o conteúdo do diretório resultante.
* Fornece um alias de atalho conveniente '......' para navegação rápida.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Navega cinco níveis de diretório acima do local atual.

### Example 2

```powershell
......
```

Usa o alias para navegar cinco níveis de diretório acima.
