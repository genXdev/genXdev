# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......

## Synopsis

> Navega cinco níveis de diretório acima na hierarquia do sistema de arquivos.

## Description

* Altera o diretório de trabalho atual subindo cinco níveis de diretório pai.
* Exibe o conteúdo do diretório resultante.
* Fornece um alias abreviado conveniente '......' para navegação rápida.

## Syntax

```powershell
Set-LocationParent5 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Navega cinco níveis de diretório acima a partir da localização atual.

### Example 2

```powershell
......
```

Usa o alias para navegar cinco níveis de diretório acima.
