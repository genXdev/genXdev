# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> Alterna o estado de pausa/reprodução do VLC Media Player.

## Description

Esta função envia uma tecla de espaço para o VLC Media Player para alternar entre os estados pausado e reproduzindo. Ela automaticamente restaura o foco para a janela original após enviar o comando de tecla.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

Alterna o estado de pausa/reprodução do VLC Media Player.

### vlcpause

```powershell
vlcpause
```

Usa o alias para alternar o estado de pausa/reprodução.

### vlcplay

```powershell
vlcplay
```

Usa o alias alternativo para alternar entre os estados de pausa/reprodução.
