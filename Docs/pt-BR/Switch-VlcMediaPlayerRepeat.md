# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> Alterna o modo de repetição no VLC Media Player.

## Description

Esta função envia o comando da tecla 'r' para o VLC Media Player para alternar entre diferentes modos de repetição (sem repetição, repetir atual, repetir tudo). A função abre o VLC se ele não estiver em execução e restaura o foco para a janela anterior após enviar o comando.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
