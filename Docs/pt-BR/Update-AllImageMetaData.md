# Update-AllImageMetaData

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `updateallimages`

## Synopsis

> Atualiza em lote palavras-chave, rostos, objetos e cenas de imagens em vários diretórios do sistema.

## Description

Esta função processa sistematicamente imagens em vários diretórios do sistema para atualizar suas palavras-chave, dados de reconhecimento facial, dados de detecção de objetos e dados de classificação de cena usando serviços de IA. Ela abrange pastas de armazenamento de mídia, arquivos do sistema, downloads, OneDrive e fotos pessoais.

A função processa imagens percorrendo cada diretório e processando arquivos individualmente. As funções do DeepStack (faces, objetos, cenas) são executadas primeiro, seguidas pela geração de palavras-chave e descrição.

Isso permite saída de dados estruturados para operações de pipeline como:
Update-AllImageMetaData | Export-ImageIndex

## Syntax

```powershell
Update-AllImageMetaData -ImageDirectories <String[]> [-ApiEndpoint <String>] [-ApiKey <String>] [-AutoUpdateFaces] [-ClearSession] [-ConfidenceThreshold <Double>] [-ContainerName <String>] [-FacesDirectory <String>] [-Force] [-HealthCheckInterval <Int32>] [-HealthCheckTimeout <Int32>] [-ImageName <String>] [-Language <String>] [-Model <String>] [-NoDockerInitialize] [-NoRecurse] [-NoSupportForJsonSchema <String>] [-PassThru] [-PreferencesDatabasePath <String>] [-RedoAll] [-RetryFailed] [-ServicePort <Int32>] [-SessionOnly] [-SkipSession] [-TimeoutSeconds <Int32>] [-UseGPU] [-VolumeName <String>] [<CommonParameters>]
```
