# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Retorna uma string (com indentação alterada) de uma cadeia de scriptblock fornecida

## Description

Altera o recuo de uma string de bloco de script respeitando os recuos originais do bloco de código

## Syntax

```powershell
alignScript [<CommonParameters>]
```
