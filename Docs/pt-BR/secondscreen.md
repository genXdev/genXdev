# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Define a configuração padrão do segundo monitor.

## Description

Define o comportamento padrão para aberturas de janelas do GenXdev no monitor secundário.
Isso é útil para usuários com um único monitor ou que preferem layouts de janelas lado a lado.
Veja também o cmdlet 'sidebyside' e 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Define os padrões para que as aberturas de janelas do GenXdev sejam no monitor secundário
