# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Define a configuração lado a lado padrão.

## Description

Define o comportamento padrão das aberturas de janela do GenXdev para serem lado a lado com o PowerShell.
Isto é útil para usuários com um único monitor ou que preferem layouts de janela lado a lado.
Consulte também o cmdlet 'secondscreen' e 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

Define padrões para que as janelas do GenXdev sejam abertas lado a lado com o PowerShell
