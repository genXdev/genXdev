# Convert-PhysicsUnit

> **Module:** GenXdev.Helpers.Physics | **Type:** Function | **Aliases:** —

## Synopsis

> Преобразует значение из одной физической единицы измерения в другую в рамках одной категории.

## Description

Эта функция преобразует физические величины между совместимыми единицами в таких категориях, как длина, время, скорость и т.д. Она определяет категорию на основе единиц и выдает ошибку для несовместимых преобразований.

## Syntax

```powershell
Convert-PhysicsUnit -Value <Double> -FromUnit <String> -ToUnit <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Value` | Double | ✅ | — | 0 | — | Числовое значение для преобразования |
| `-FromUnit` | String | ✅ | — | 1 | — | Единица измерения входного значения |
| `-ToUnit` | String | ✅ | — | 2 | — | Желаемая единица вывода |

## Examples

### Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"

```powershell
Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"
```

Преобразует 100 метров в футы.

### Convert-PhysicsUnit 10 "seconds" "minutes"

```powershell
Convert-PhysicsUnit 10 "seconds" "minutes"
```

Преобразует 10 секунд в минуты с помощью позиционных параметров.
