# Get-AIKnownFacesRootpath

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> Получает настроенную директорию для файлов изображений лиц, используемых в операциях GenXdev.AI.

## Description

Эта функция получает глобальный каталог лиц, используемый модулем GenXdev.AI для различных операций распознавания лиц и ИИ. Она сначала проверяет глобальные переменные (если не указан параметр SkipSession), затем обращается к постоянным настройкам и, наконец, использует системные значения по умолчанию.

## Syntax

```powershell
Get-AIKnownFacesRootpath [[-FacesDirectory] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FacesDirectory` | String | — | — | 0 | — | Путь к каталогу с файлами изображений лиц |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Используйте альтернативные настройки, сохраненные в сеансе, для предпочтений ИИ, таких как язык, коллекции изображений и т. д. |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Очистите параметры сеанса (глобальную переменную) перед получением |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Путь к базе данных для файлов данных предпочтений |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Не используйте альтернативные настройки, сохраненные в сеансе, для предпочтений ИИ, таких как язык, коллекции изображений и т.д. |

## Examples

### Get-AIKnownFacesRootpath

```powershell
Get-AIKnownFacesRootpath
```

Получает текущий каталог лиц, настроенный в глобальных переменных или параметрах.

### Get-AIKnownFacesRootpath -SkipSession

```powershell
Get-AIKnownFacesRootpath -SkipSession
```

Получает настроенный каталог лиц только из постоянных предпочтений, игнорируя любые параметры сеанса.

### Get-AIKnownFacesRootpath -ClearSession

```powershell
Get-AIKnownFacesRootpath -ClearSession
```

Очищает настройку каталога сеансовых лиц, а затем получает каталог из постоянных параметров.

### Get-AIKnownFacesRootpath "C:\MyFaces"

```powershell
Get-AIKnownFacesRootpath "C:\MyFaces"
```

Возвращает указанный каталог после расширения пути.
