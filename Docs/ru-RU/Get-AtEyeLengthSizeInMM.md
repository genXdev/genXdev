# Get-AtEyeLengthSizeInMM

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Рассчитывает видимый размер объекта на расстоянии вытянутой руки в мм.

## Description

Вычисляет размер, который объект будет казаться, если смотреть с расстояния средней длины руки взрослого человека (приблизительно 0,7 метра), учитывая его фактический размер и расстояние. Расчет использует приближение малого угла для углового размера.

## Syntax

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters <Double> -SizeInMeters <Double> [-EyeToArmLengthInMeters <Double>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | Расстояние до объекта в метрах. |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | Фактический размер объекта в метрах. |
| `-EyeToArmLengthInMeters` | Double | — | — | Named | `0` | Расстояние на расстоянии вытянутой руки в метрах. Значение по умолчанию — 0.7. |

## Examples

### Example 1

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters 10 -SizeInMeters 1
```

Вычисляет кажущийся размер на расстоянии вытянутой руки для объекта размером 1 метр на расстоянии 10 метров.
