# Get-CurrentFocusedProcess

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Извлекает объект процесса окна, которое в данный момент имеет фокус ввода с клавиатуры.

## Description

* Этот командлет использует вызовы Windows API через P/Invoke для идентификации и возврата объекта Process, связанного с текущим активным окном.
* Он использует функции User32.dll GetForegroundWindow и GetWindowThreadProcessId для определения того, какое окно имеет фокус, и получения его идентификатора процесса.

## Syntax

```powershell
Get-CurrentFocusedProcess [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$focusedProcess = Get-CurrentFocusedProcess
Write-Host "Active window process: $($focusedProcess.ProcessName)"
```

Получает объект процесса для окна, которое в данный момент имеет фокус ввода с клавиатуры.
