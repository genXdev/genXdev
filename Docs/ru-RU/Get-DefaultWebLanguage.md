# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Получает ключ языка веб-интерфейса по умолчанию на основе текущих языковых настроек системы.

## Description

* Получает текущие настройки языка и культуры системы и сопоставляет их с соответствующим ключом словаря веб-языка, используемого службами перевода.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Возвращает "English" для английской системы, "Dutch" для голландской системы и т.д.
