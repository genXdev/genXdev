# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> Возвращает true, если модуль синтеза речи воспроизводит речь.

## Description

Проверяет состояние как стандартного, так и настроенного синтезатора речи, чтобы определить, говорит ли один из них в данный момент. Возвращает true, если речь выполняется, в противном случае false.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Возвращает true, если модуль синтеза речи воспроизводит речь.

### Example 2

```powershell
iss
```

Возвращает true, если модуль синтеза речи говорит (с использованием псевдонима).
