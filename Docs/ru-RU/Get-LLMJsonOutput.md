# Get-LLMJsonOutput

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Извлекает допустимый JSON из текста ответа LLM, используя эвристику наилучшего усилия.

## Description

Принимает необработанный текст ответа LLM, который может содержать обрамление разметки, комментарии или другой не-JSON контент, и пытается извлечь из него чистый, валидный JSON. Использует многошаговую стратегию: сначала удаляет обрамление разметки, затем пытается разобрать всю строку как JSON, и если это не удаётся, использует извлечение на основе схемы с последующими эвристическими запасными вариантами.

Функция обрабатывает:
- Обрамление кода разметки (...)
- JSON объекты ({...}) и массивы ([...])
- Строки, числа, булевы значения и значения даты-времени
- Извлечение на основе схемы с использованием подсказок типа json_schema

## Syntax

```powershell
Get-LLMJsonOutput -Text <String> [[-ResponseFormat] <String>] [<CommonParameters>]
```

## Examples

### @{response=Get-LLMJsonOutput -Text '}

@{response=Get-LLMJsonOutput -Text '}

```json
{"result": "hello"}
```

@{result=hello}

### $schema = '{"type":"json_schema","json_schema":{"name":"r","schema":{"type":"object"}}}' Get-LLMJsonOutput -Text 'The answer is {"key": "value"}' -ResponseFormat $schema

```powershell
$schema = '{"type":"json_schema","json_schema":{"name":"r","schema":{"type":"object"}}}'
Get-LLMJsonOutput -Text 'The answer is {"key": "value"}' -ResponseFormat $schema
```

Извлекает: {"key": "value"}
