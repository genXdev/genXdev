# Get-MediaFileCreationDate

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** —

## Synopsis

> Извлекает дату создания для медиа и других файлов, используя наилучший возможный подход.

## Description

Предпринимает несколько стратегий для определения точной даты создания или захвата указанного файла. Стратегии включают чтение метаданных EXIF изображения, анализ информации о дате/времени из имен файлов и возврат к времени последней записи файла, когда нет другой надежной информации. Функция всегда возвращает [DateTime]; если дату определить невозможно, она возвращает [DateTime]::MinValue.

## Syntax

```powershell
Get-MediaFileCreationDate -FilePath <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | The `-FilePath` parameter. |

## Examples
