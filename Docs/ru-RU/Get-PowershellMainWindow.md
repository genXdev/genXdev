# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Возвращает вспомогательный объект окна для главного окна терминала PowerShell.

## Description

* Получает объект-помощник WindowObj, представляющий главное окно текущего процесса узла PowerShell.
* Это позволяет манипулировать и взаимодействовать с самим окном терминала.

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

Возвращает WindowObj для текущего окна терминала PowerShell.
