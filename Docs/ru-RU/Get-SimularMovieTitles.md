# Get-SimularMovieTitles

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> Находит похожие названия фильмов на основе общих свойств.

## Description

Анализирует предоставленные фильмы для поиска общих свойств и возвращает список из 10 похожих названий фильмов. Использует ИИ для выявления закономерностей и тем среди входящих фильмов, чтобы предложить релевантные рекомендации.
