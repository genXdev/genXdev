# Get-SoundTravelDistanceByTime

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Вычисляет расстояние, которое звук проходит за заданное время.

## Description

Использует формулу расстояние = скорость * время, со скоростью звука в воздухе по умолчанию.

## Syntax

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds <Double> [[-As] <String>] [<CommonParameters>]

Get-SoundTravelDistanceByTime [[-SpeedOfSoundInMetersPerSecond] <Double>] [<CommonParameters>]

Get-SoundTravelDistanceByTime -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-TimeInSeconds` | Double | ✅ | — | 0 | `0` | Время в секундах |
| `-SpeedOfSoundInMetersPerSecond` | Double | — | — | 1 | `0` | Скорость звука в м/с (по умолчанию: 343) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | Среда, через которую распространяется звук *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | Единица измерения выходного расстояния |

## Examples

### Example 1

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds 5 -Medium "water" -As "kilometers"
```

Вычисляет, как далеко звук распространяется в воде за 5 секунд, и преобразует результат в километры.

### Example 2

```powershell
Get-SoundTravelDistanceByTime 10 -SpeedOfSoundInMetersPerSecond 1480
```

Вычисляет расстояние, используя определенную скорость звука.
