# Get-VectorSimilarity

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Вычисляет косинусное сходство между двумя векторами.

## Description

Принимает на вход два числовых вектора (массива) и вычисляет их косинусную близость. Результат показывает, насколько векторы связаны: 0 означает полную несхожесть, а 1 — идентичность.

## Syntax

```powershell
Get-VectorSimilarity -Vector1 <Double[]> -Vector2 <Double[]> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Vector1` | Double[] | ✅ | — | 0 | — | Первый векторный массив чисел |
| `-Vector2` | Double[] | ✅ | — | 1 | — | Второй векторный массив чисел |

## Examples

### Example 1

```powershell
$v1 = @(0.12, -0.45, 0.89)
$v2 = @(0.15, -0.40, 0.92)
Get-VectorSimilarity -Vector1 $v1 -Vector2 $v2
```

Возвращает примерно 0,998, что указывает на высокую степень схожести.
