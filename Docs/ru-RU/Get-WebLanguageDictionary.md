# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Возвращает обратный словарь для всех языков, поддерживаемых поиском Google.

## Description

* Возвращает словарь, отображающий названия языков на соответствующие им коды языков для всех языков, поддерживаемых Google Поиском.
* Полезен для поиска кодов языков в веб-запросах и задачах перевода.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

Возвращает словарь, сопоставляющий названия языков с их кодами.
