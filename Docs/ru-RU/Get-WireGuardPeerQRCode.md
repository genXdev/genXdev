# Get-WireGuardPeerQRCode

> **Module:** GenXdev.Windows.WireGuard | **Type:** Function | **Aliases:** —

## Synopsis

> Генерирует QR-код для конфигурации однорангового узла WireGuard VPN.

## Description

Эта функция генерирует QR-код для конфигурации пира WireGuard VPN, который можно отсканировать с мобильных устройств для простой настройки. QR-код отображается в консоли и может использоваться для быстрой настройки клиентов WireGuard на смартфонах и планшетах. Функция взаимодействует с Docker-контейнером linuxserver/wireguard для генерации QR-кодов для конфигураций пиров.

## Syntax

```powershell
Get-WireGuardPeerQRCode -PeerName <String> [-ContainerName <String>] [-Force] [-HealthCheckInterval <Int32>] [-HealthCheckTimeout <Int32>] [-ImageName <String>] [-NoDockerInitialize] [-PGID <String>] [-PUID <String>] [-ServicePort <Int32>] [-TimeZone <String>] [-VolumeName <String>] [<CommonParameters>]
```
