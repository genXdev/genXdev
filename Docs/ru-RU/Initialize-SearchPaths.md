# Initialize-SearchPaths

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Инициализирует и настраивает пути поиска системы для управления пакетами.

## Description

* Формирует полный список путей поиска, объединяя стандартные системные расположения, пути Chocolatey, пути инструментов разработки и пути пользовательских пакетов.
* Обновляет системную переменную среды PATH, используя эти объединенные пути.

## Syntax

```powershell
Initialize-SearchPaths [[-WorkspaceFolder] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-WorkspaceFolder` | String | — | — | 0 | — | Путь к папке рабочей области, используемый для модулей Node и путей PowerShell. |

## Examples

### Example 1

```powershell
Initialize-SearchPaths
```

Инициализирует пути поиска, используя папку рабочей области по умолчанию.

### Example 2

```powershell
Initialize-SearchPaths -WorkspaceFolder "C:\workspace"
```

Инициализирует пути поиска с использованием определенной папки рабочей области.
