# Move-ToRecycleBin

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Безопасно перемещает файлы и каталоги в корзину Windows.

## Description

* Безопасно перемещает файлы или папки в корзину с помощью Windows Shell API, даже если они в данный момент используются.
* Функция использует COM-объект Shell.Application для выполнения операции, обеспечивая правильное поведение корзины и возможность отмены.

## Syntax

```powershell
Move-ToRecycleBin -Path <String[]> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String[] | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Укажите путь(и) для перемещения в корзину |

## Examples

### Example 1

```powershell
Move-ToRecycleBin -Path "C:\temp\old-report.txt"
```

Переместить один файл в корзину.

### Example 2

```powershell
"file1.txt","file2.txt" | recycle
```

Перемещение нескольких файлов в корзину с помощью конвейера и псевдонима.
