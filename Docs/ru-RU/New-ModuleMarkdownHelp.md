# New-ModuleMarkdownHelp

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Создает расширенные файлы справки в формате Markdown для любого модуля PowerShell.

## Description

Создает комплексный справочный сайт Markdown для любого модуля PowerShell:
один .md файл на каждый командлет со всеми разделами метаданных, а также README.md
индекс с таблицами по субмодулям, ссылающимися на каждый файл командлета.

Обнаружение субмодулей переносимо: каталоги с точечным импортом .psm1 для
скриптовых командлетов, пространства имен .NET для скомпилированных командлетов, с многоуровневой
цепочкой резервных вариантов для модулей без структуры субмодулей.

## Syntax

```powershell
New-ModuleMarkdownHelp -ModuleName <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-Force] [-Language <String>] [-LinkPrefix <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [-OutputPath <String>] [-SkipTranslation] [-TranslationInstructions <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String | ✅ | — | 0 | — | Имя модуля PowerShell, для которого необходимо создать справку |
| `-OutputPath` | String | — | — | Named | — | Пользовательский выходной каталог для файлов .md. По умолчанию — <moduleRoot>\Docs\<language>. |
| `-Language` | String | — | — | Named | `'en-US'` | Языковой тег BCP 47 для создаваемой справки (например, en-US, nl-NL, de-DE) |
| `-Force` | SwitchParameter | — | — | Named | — | Перезаписывать существующие .md файлы без запроса |
| `-SkipTranslation` | SwitchParameter | — | — | Named | — | @{response=Skip LLM translation; keep help in source language} |
| `-Model` | String | — | — | Named | — | Идентификатор или шаблон модели для использования в AI-переводах |
| `-ApiEndpoint` | String | — | — | Named | — | URL-адрес конечной точки API для AI-переводов |
| `-ApiKey` | String | — | — | Named | — | API-ключ для аутентифицированных AI-переводов |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | Указывает, что LLM не поддерживает JSON-схемы |
| `-LinkPrefix` | String | — | — | Named | — | Префикс URL для ссылок README index (например, https://github.com/org/repo/Docs/) |
| `-TranslationInstructions` | String | — | — | Named | — | Это текст для перевода, который должен быть преобразован. |

## Examples

### New-ModuleMarkdownHelp -ModuleName 'Pester' -SkipTranslation

```powershell
New-ModuleMarkdownHelp -ModuleName 'Pester' -SkipTranslation
```

Создает папку Docs\ с одним .md файлом на каждый командлет Pester + файл README.md индекс.

### New-ModuleMarkdownHelp -ModuleName 'GenXdev' -Language 'nl-NL' -Force -Model 'deepseek-v4-pro' -ApiKey 'your-api-key' -ApiEndpoint 'https://api.deepseek.com/chat/completions'

```powershell
New-ModuleMarkdownHelp -ModuleName 'GenXdev' -Language 'nl-NL' -Force -Model 'deepseek-v4-pro' -ApiKey 'your-api-key' -ApiEndpoint 'https://api.deepseek.com/chat/completions'
```

Создает справку в формате Markdown с переводом на русский язык, перезаписывая существующие файлы.

### New-ModuleMarkdownHelp -ModuleName 'GenXdev' -LinkPrefix `     'https://github.com/genXdev/genXdev/Docs/' -SkipTranslation

```powershell
New-ModuleMarkdownHelp -ModuleName 'GenXdev' -LinkPrefix `
    'https://github.com/genXdev/genXdev/Docs/' -SkipTranslation
```

Создает справку с абсолютными ссылками на GitHub в индексе README.
