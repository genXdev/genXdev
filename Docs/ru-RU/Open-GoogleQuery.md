# Open-GoogleQuery

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** `q

## Synopsis

> Открывает запросы Google в веб-браузере с настраиваемыми параметрами и обширными возможностями настройки.

## Description

Открывает один или несколько запросов Google в веб-браузере с всесторонней поддержкой множества языков, выбора монитора, позиционирования окна, выбора браузера и автоматизации клавиатуры. Функция предоставляет многофункциональный интерфейс для выполнения поисковых запросов Google с точным контролем над поведением и внешним видом браузера.

Ключевые возможности включают:
- Поддержка 100+ языков с помощью языковой фильтрации Google
- Поддержка нескольких мониторов с автоматическим или ручным выбором монитора
- Позиционирование окна (слева, справа, сверху, снизу, по центру, полноэкранный режим)
- Поддержка режима приватного/инкогнито просмотра
- Режим приложения для работы без отвлекающих факторов
- Возможности блокировки расширений и всплывающих окон
- Управление фокусом и манипуляции с окнами
- Пакетная обработка запросов с поддержкой конвейера
- Автоматизация нажатий клавиш для окон браузера
- Интеллектуальная передача параметров в нижележащие функции браузера

Функция автоматически форматирует поисковые запросы для поисковой системы Google и применяет языковую фильтрацию, если указано. Все возможности позиционирования и управления браузером наследуются от нижележащей функции Open-Webbrowser.

## Syntax

```powershell
Open-GoogleQuery -Queries <String[]> [[-Language] <String>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-Monitor <Int32>] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-GoogleQuery -Queries "PowerShell scripting" -Language "English" -Monitor 0

```powershell
Open-GoogleQuery -Queries "PowerShell scripting" -Language "English" -Monitor 0
```

Открывает поиск Google по запросу "PowerShell scripting" на английском языке на основном мониторе.

### q "machine learning algorithms" -m 2 -fs

```powershell
q "machine learning algorithms" -m 2 -fs
```

Открывает поиск Google для "алгоритмов машинного обучения" в полноэкранном режиме на мониторе 2, используя псевдонимы.

### "PowerShell", "Python", "JavaScript" | Open-GoogleQuery -Language "English" -Chrome

```powershell
"PowerShell", "Python", "JavaScript" | Open-GoogleQuery -Language "English" -Chrome
```

Открывает несколько поисковых запросов Google в Chrome через конвейерный ввод, каждый на английском языке.

### Open-GoogleQuery -Queries "artificial intelligence" -Private -NewWindow

```powershell
Open-GoogleQuery -Queries "artificial intelligence" -Private -NewWindow
```

Opens a private browsing search for "artificial intelligence" in a new window.

### Open-GoogleQuery -Queries "web development" -ApplicationMode -Centered

```powershell
Open-GoogleQuery -Queries "web development" -ApplicationMode -Centered
```

Открывает поиск в режиме приложения (без элементов управления браузера), центрированный на экране.
