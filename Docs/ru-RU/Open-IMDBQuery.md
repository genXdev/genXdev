# Open-IMDBQuery

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** `imdb`

## Synopsis

> Открывает поисковый запрос IMDB в веб-браузере.

## Description

<details>
<summary><b>Expand description</b></summary>

Открывает запрос «Internet Movie Database» в веб-браузере с расширенными параметрами настройки. Эта функция предоставляет удобную обертку для функционала поиска IMDB, позволяя пользователям искать фильмы, телешоу, актеров и другой развлекательный контент непосредственно из PowerShell.

Функция принимает поисковые запросы и формирует правильные URL-адреса поиска IMDB, затем открывает их в указанном веб-браузере с настраиваемым позиционированием окна, выбором браузера и параметрами отображения. Она поддерживает все основные браузеры, включая Edge, Chrome и Firefox, с опциями приватного просмотра, полноэкранного режима и управления окнами.

Ключевые возможности включают поддержку нескольких мониторов, локализацию языка, автоматизацию клавиатуры и полные параметры конфигурации браузера. Функция может обрабатывать несколько поисковых запросов одновременно и предоставляет возможности как для интерактивного использования, так и для автоматизированных рабочих процессов.

## Examples

### Open-IMDBQuery -Queries "The Matrix" -Monitor 0

```powershell
Open-IMDBQuery -Queries "The Matrix" -Monitor 0
```

Открывает поиск IMDB для "Матрицы" на основном мониторе.

### imdb "The Matrix" -m 0

```powershell
imdb "The Matrix" -m 0
```

Открывает поиск IMDB для «Матрицы» с использованием псевдонима и сокращенных параметров.

### "Inception", "Interstellar" | Open-IMDBQuery -Language "French" -Chrome

```powershell
"Inception", "Interstellar" | Open-IMDBQuery -Language "French" -Chrome
```

Выполняет поиск нескольких фильмов на IMDB с французскими результатами в Chrome.

### Open-IMDBQuery -Queries "Tom Hanks" -FullScreen -RestoreFocus

```powershell
Open-IMDBQuery -Queries "Tom Hanks" -FullScreen -RestoreFocus
```

Выполняет поиск Тома Хэнкса на IMDB в полноэкранном режиме, затем возвращает фокус в PowerShell.
