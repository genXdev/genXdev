# Open-ViralSimulation

> **Module:** GenXdev.Queries.Websites | **Type:** Function | **Aliases:** `viral`

## Synopsis

> Открывает симуляцию вирусной эпидемии в веб-браузере с расширенными настройками.

## Description

Открывает симуляцию вирусной игры по адресу https://viral.genxdev.net/ в указанном веб-браузере с настраиваемыми параметрами окна, позиционирования и поведения. Поддерживает несколько браузеров, режим приватного просмотра, режим приложения и расширенные функции управления окнами.

Функция обеспечивает интерактивный опыт симуляции вируса с такими возможностями, как:
- Поддержка нескольких браузеров (Edge, Chrome, Firefox)
- Настройка позиции и размера окна
- Режимы полноэкранного и приложения
- Возможности приватного просмотра
- Поддержка автоматизации клавиатуры
- Поддержка нескольких мониторов


## Examples

### Open-ViralSimulation -Chrome -Private -NoFullScreen

```powershell
Open-ViralSimulation -Chrome -Private -NoFullScreen
```

Открывает симуляцию вируса в Google Chrome в режиме приватного просмотра без полноэкранного режима.

### viral -ch -incognito -nfs

```powershell
viral -ch -incognito -nfs
```

Открывает вирусную симуляцию с использованием псевдонимов в Chrome, в режиме инкогнито и без полноэкранного режима.
