# Open-Webbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `wb`

## Synopsis

> Открывает URL-адреса в одном или нескольких окнах браузера с возможностью задания положения и стиля.

## Description

Эта функция предоставляет расширенную обёртку для запуска браузера с обширными возможностями позиционирования окна, выбора браузера и настройки поведения. Она поддерживает несколько браузеров, включая Edge, Chrome и Firefox, с такими функциями, как приватный просмотр, режим приложения и точное управление окнами.

Ключевые особенности:
- Интеллектуальное обнаружение и выбор браузера
- Позиционирование окна (слева, справа, сверху, снизу, по центру, полноэкранный режим)
- Поддержка нескольких мониторов с автоматическим или ручным выбором монитора
- Поддержка приватного/инкогнито режима
- Режим приложения для просмотра без отвлечений
- Опции блокировки расширений и всплывающих окон
- Управление фокусом и манипуляции с окнами
- Пакетное открытие URL-адресов в нескольких браузерах
- Автоматизация нажатий клавиш в окнах браузера

Функция может автоматически определять возможности системы и соответствующим образом настраивать поведение. Для браузеров, не установленных в системе, операции молча пропускаются без ошибок.

## Examples

### wb -PlayWright https://github.com

```powershell
wb -PlayWright https://github.com
```

Открывает GitHub в управляемом Playwright браузере Chromium.

### Open-Webbrowser -Url "https://github.com"

```powershell
Open-Webbrowser -Url "https://github.com"
```

Открывает GitHub в браузере по умолчанию.

### Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left

```powershell
Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left
```

Открывает Stack Overflow в левой половине монитора 1.

### wb "https://google.com" -m 0 -fs

```powershell
wb "https://google.com" -m 0 -fs
```

Открывает Google в полноэкранном режиме на основном мониторе с помощью псевдонимов.

### Open-Webbrowser -Chrome -Private -NewWindow

```powershell
Open-Webbrowser -Chrome -Private -NewWindow
```

Открывает новое окно Chrome в режиме инкогнито.

### "https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All

```powershell
"https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All
```

Открывает несколько URL-адресов во всех установленных браузерах с помощью конвейера.

### Open-Webbrowser -Monitor 0 -Right

```powershell
Open-Webbrowser -Monitor 0 -Right
```

Перемещает уже открытое окно браузера в правую сторону основного монитора.

### Open-Webbrowser -ApplicationMode -Url "https://app.example.com"

```powershell
Open-Webbrowser -ApplicationMode -Url "https://app.example.com"
```

Открывает веб-приложение в режиме приложения без элементов управления браузера.
