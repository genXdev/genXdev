# Remove-DoubleEmptyLines

> **Module:** GenXdev.Coding.Templating | **Type:** Function | **Aliases:** —

## Synopsis

> Удаляет двойные (последовательные) пустые строки из строки кода, при необходимости переформатируя вывод.

## Description

Очищает исходный код или текст, удаляя последовательные пустые строки, заменяя их одной пустой строкой. Это улучшает читаемость и соответствует общим правилам форматирования кода. При указании параметра -Reformat применяются дополнительные правила форматирования для нормализации структуры кода.

## Syntax

```powershell
Remove-DoubleEmptyLines -code <String> [-Reformat] [<CommonParameters>]
```

## Examples

### $cleanCode = $sourceCode | Remove-DoubleEmptyLines

```powershell
$cleanCode = $sourceCode | Remove-DoubleEmptyLines
```

Пропускает исходный код через функцию для удаления подряд идущих пустых строк.

### Remove-DoubleEmptyLines -code $sourceCode -Reformat

```powershell
Remove-DoubleEmptyLines -code $sourceCode -Reformat
```

Удаляет двойные пустые строки и применяет дополнительное форматирование.
