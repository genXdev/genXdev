# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Произносит текущую дату с помощью синтеза речи.

## Description

Преобразует текущую дату в формат естественного языка и объявляет её с помощью системы преобразования текста в речь. Дата произносится в формате: «Сегодня [день недели], [месяц] [число] [год]»

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
