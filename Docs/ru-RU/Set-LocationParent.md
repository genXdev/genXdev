# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> Перемещает текущую папку в родительский каталог и выводит его содержимое.

## Description

* Переходит на один уровень вверх в иерархии файловой системы.
* Отображает содержимое нового текущего каталога.
* Предоставляет удобный псевдоним '..' для быстрой навигации по каталогам.

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

Переходит в родительский каталог и выводит его содержимое.

### Example 2

```powershell
..
```

Использует псевдоним для перехода в родительский каталог.
