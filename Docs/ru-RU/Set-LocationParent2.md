# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...

## Synopsis

> Перемещается на два уровня вверх по иерархии файловой системы.

## Description

* Изменяет текущий рабочий каталог на каталог прародителя (на два уровня вверх).
* Отображает содержимое результирующего каталога.

## Syntax

```powershell
Set-LocationParent2 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Переходит в родительский каталог и отображает его содержимое.
