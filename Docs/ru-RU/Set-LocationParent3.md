# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....

## Synopsis

> Поднимается на три уровня вверх по иерархии файловой системы.

## Description

* Изменяет текущий рабочий каталог, перемещаясь на три уровня вверх по иерархии файловой системы.
* Отображает содержимое полученного каталога.

## Syntax

```powershell
Set-LocationParent3 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

Поднимается на три уровня выше текущего местоположения.
