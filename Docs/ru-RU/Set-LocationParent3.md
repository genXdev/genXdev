# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....`

## Synopsis

> Перемещается на три уровня вверх по иерархии файловой системы.

## Description

* Изменяет текущий рабочий каталог, перемещаясь на три уровня вверх по иерархии файловой системы.
* Отображает содержимое полученного каталога.

## Syntax

```powershell
Set-LocationParent3 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

Перемещает на три уровня каталогов вверх от текущего расположения.

```powershell
....
```

Использует псевдоним для перехода на три уровня каталогов вверх.
