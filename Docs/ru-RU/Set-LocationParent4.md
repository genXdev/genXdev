# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....

## Synopsis

> Перемещается на четыре уровня вверх по иерархии файловой системы.

## Description

* Перемещается на четыре уровня вверх от текущей позиции в файловой системе.
* После навигации отображает содержимое полученного каталога.

## Syntax

```powershell
Set-LocationParent4 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Перемещается на четыре уровня каталогов вверх от текущего местоположения.
