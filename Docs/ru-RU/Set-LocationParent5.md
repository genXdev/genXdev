# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> Перемещается на пять уровней вверх по иерархии файловой системы.

## Description

* Изменяет текущий рабочий каталог, перемещаясь на пять уровней выше.
* Отображает содержимое результирующего каталога.
* Предоставляет удобный псевдоним '......' для быстрой навигации.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Перемещение на пять уровней каталогов вверх от текущего местоположения.

### Example 2

```powershell
......
```

Использует псевдоним для перемещения на пять уровней вверх по каталогу.
