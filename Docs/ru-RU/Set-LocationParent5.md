# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......

## Synopsis

> Перемещается на пять уровней вверх по иерархии файловой системы.

## Description

* Изменяет текущую рабочую директорию, перемещаясь на пять родительских уровней вверх.
* Отображает содержимое полученной директории.
* Предоставляет удобный сокращённый псевдоним '......' для быстрой навигации.

## Syntax

```powershell
Set-LocationParent5 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Поднимается на пять уровней каталогов вверх от текущего местоположения.

```powershell
......
```

Использует псевдоним для поднятия на пять уровней каталогов вверх.
