# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> Переключает состояние паузы/воспроизведения медиаплеера VLC.

## Description

Эта функция отправляет пробел в VLC Media Player для переключения между состояниями паузы и воспроизведения. После отправки команды она автоматически возвращает фокус исходному окну.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

Переключает состояние паузы/воспроизведения медиаплеера VLC.

### vlcpause

```powershell
vlcpause
```

Использует псевдоним для переключения состояния паузы/воспроизведения.

### vlcplay

```powershell
vlcplay
```

Использует альтернативный псевдоним для переключения состояния паузы/воспроизведения.
