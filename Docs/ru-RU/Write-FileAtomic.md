# Write-FileAtomic

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Атомарно записывает байты в файл для предотвращения повреждения.

## Description

* Атомарная запись: используется стратегия временный файл + переименование, чтобы целевой файл никогда не оставался в повреждённом состоянии, если процесс был прерван.
* Логика повторных попыток: повторяет запись до MaxRetries раз с задержкой RetryDelayMs миллисекунд между попытками.
* Поддержка антидребезга: когда DebounceMs > 0, быстрые последовательные записи в один и тот же файл объединяются — записывается только последний полезный набор данных, когда файл не трогали в течение DebounceMs мс.
* Создание каталогов: автоматически создаёт родительские каталоги, если их не существует.

## Syntax

```powershell
Write-FileAtomic -FilePath <String> [[-Data] <Byte[]>] [-DebounceMs <Int32>] [-MaxRetries <Int32>] [-RetryDelayMs <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | ✅ | — | 0 | — | Путь к файлу для записи. |
| `-Data` | Byte[] | — | — | 1 | — | Массив байтов для записи в файл. |
| `-MaxRetries` | Int32 | — | — | Named | `0` | Максимальное количество повторных попыток для атомарной операции записи. По умолчанию: 4. |
| `-RetryDelayMs` | Int32 | — | — | Named | `0` | Задержка в миллисекундах между повторными попытками. По умолчанию 200. |
| `-DebounceMs` | Int32 | — | — | Named | `0` | Если больше 0, подавляет частые записи в этот файл. По умолчанию 0 (отключено). |

## Examples

### Example 1

```powershell
Write-FileAtomic -FilePath "data.bin" -Data $bytes
```

Атомарно записывает массив байтов в data.bin с настройками повторных попыток по умолчанию.

### Example 2

```powershell
# Debounce: only the LAST write within 5 seconds actually hits disk
1..100 | ForEach-Object {
    Write-FileAtomic -FilePath "state.bin" -Data $_ `
        -DebounceMs 5000
}
Start-Sleep -Seconds 6
```

Быстро записывает 100 раз, но только последний полезный груз (100) сохраняется на диске после 5-секундного периода затишья.

### Example 3

```powershell
Write-FileAtomic -FilePath "cache.dat" -Data $buffer `
    -MaxRetries 10 -RetryDelayMs 500
```

Выполняет запись с пользовательскими повторными попытками: до 10 попыток с интервалом 500 мс между ними.
