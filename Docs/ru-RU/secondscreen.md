# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Устанавливает конфигурацию второго монитора по умолчанию.

## Description

Задает поведение по умолчанию для открытия окон GenXdev на дополнительном мониторе.
Это полезно для пользователей с одним монитором или тех, кто предпочитает расположение окон рядом.
См. также командлеты 'sidebyside' и 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Устанавливает значения по умолчанию для открытия окон GenXdev на втором мониторе
