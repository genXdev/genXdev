# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Устанавливает конфигурацию "рядом" по умолчанию.

## Description

Устанавливает поведение по умолчанию для открытия окон GenXdev рядом с PowerShell.
Это полезно для пользователей с одним монитором или тех, кто предпочитает боковое расположение окон.
См. также командлеты 'secondscreen' и 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

Устанавливает параметры по умолчанию для открытия окон GenXdev рядом с PowerShell
