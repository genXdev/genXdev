# Assert-GenXdevDependencyUsage

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `checkgenxdevdependencies`

## Synopsis

> Validerar användning av beroenden mellan GenXdev-moduler för att säkerställa att korrekt modulhierarki upprätthålls.

## Description

Denna funktion analyserar GenXdev-moduler för att säkerställa att de följer korrekt beroendehierarki. Den kontrollerar att moduler endast refererar till beroenden som är listade i deras RequiredModules-manifest, och förhindrar cirkulära beroenden genom att validera att moduler inte refererar till moduler som kommer senare i beroendekedjan.

## Syntax

```powershell
Assert-GenXdevDependencyUsage [[-ModuleName] <String[]>] [-FromScripts] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String[] | — | ✅ (ByValue, ByPropertyName) | 1 | `@('GenXdev*')` | Filter för att tillämpa på modulnamn 🌐 *Supports wildcards* |
| `-FromScripts` | SwitchParameter | — | — | Named | — | Sök i skriptfiler istället för modulfiler |

## Examples

### Assert-GenXdevDependencyUsage -ModuleName "GenXdev.Coding"

```powershell
Assert-GenXdevDependencyUsage -ModuleName "GenXdev.Coding"
```

### checkgenxdevdependencies "GenXdev*" -FromScripts

```powershell
checkgenxdevdependencies "GenXdev*" -FromScripts
```
