# Close-WebbrowserTab

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `ct`, `CloseTab`

## Synopsis

> Stänger den för närvarande valda webbläsarfliken.

## Description

Stänger den för närvarande valda webbläsarfliken med hjälp av ChromeDrivers CloseAsync()-metod. Om ingen flik är vald för tillfället, försöker funktionen automatiskt att välja den senast använda fliken innan den stängs.

## Syntax

```powershell
Close-WebbrowserTab [-Chrome] [-Chromium] [-Edge] [-Firefox] [-Webkit] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Edge` | SwitchParameter | — | — | Named | — | Navigera med webbläsaren Microsoft Edge |
| `-Chrome` | SwitchParameter | — | — | Named | — | Navigera med webbläsaren Google Chrome |
| `-Chromium` | SwitchParameter | — | — | Named | — | Öppnas i Microsoft Edge eller Google Chrome, beroende på vilken standardwebbläsaren är |
| `-Firefox` | SwitchParameter | — | — | Named | — | Öppnas i Firefox |
| `-Webkit` | SwitchParameter | — | — | Named | — | Använder den Playwright-hanterade WebKit-webbläsaren |

## Examples

### Close-WebbrowserTab Closes the currently active browser tab

```powershell
Close-WebbrowserTab
Closes the currently active browser tab
```

### ct Uses the alias to close the currently active browser tab

```powershell
ct
Uses the alias to close the currently active browser tab
```
