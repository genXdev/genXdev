# Convert-PhysicsUnit

> **Module:** GenXdev.Helpers.Physics | **Type:** Function | **Aliases:** —

## Synopsis

> Konverterar ett värde från en fysikenhet till en annan inom samma kategori.

## Description

Den här funktionen konverterar fysikaliska storheter mellan kompatibla enheter inom kategorier som längd, tid, hastighet osv. Den härleder kategorin från enheterna och ger ett felmeddelande vid inkompatibla konverteringar.

## Syntax

```powershell
Convert-PhysicsUnit -Value <Double> -FromUnit <String> -ToUnit <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Value` | Double | ✅ | — | 0 | — | Det numeriska värdet att konvertera |
| `-FromUnit` | String | ✅ | — | 1 | — | Enheten för inmatningsvärdet |
| `-ToUnit` | String | ✅ | — | 2 | — | Den önskade utmatningsenheten |

## Examples

### Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"

```powershell
Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"
```

Konverterar 100 meter till fot.

### Convert-PhysicsUnit 10 "seconds" "minutes"

```powershell
Convert-PhysicsUnit 10 "seconds" "minutes"
```

Konverterar 10 sekunder till minuter med hjälp av positionsparametrar.
