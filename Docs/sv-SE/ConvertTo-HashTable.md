# ConvertTo-HashTable

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Konverterar en PSCustomObject till en HashTable rekursivt.

## Description

* Den här funktionen konverterar en PSCustomObject och alla dess kapslade PSCustomObject-egenskaper till HashTables.
* Den hanterar arrayer och andra samlingstyper genom att bearbeta varje element rekursivt.

## Syntax

```powershell
ConvertTo-HashTable -InputObject <Object> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputObject` | Object | ✅ | ✅ (ByValue) | 0 | — | PSCustomObject som ska konverteras till en HashTable |

## Examples

### Example 1

```powershell
$object = [PSCustomObject]@{
    Name = "John"
    Age = 30
    Details = [PSCustomObject]@{
        City = "New York"
    }
}
$hashTable = ConvertTo-HashTable -InputObject $object
```

Konvertera ett PSCustomObject till en HashTable med kapslade egenskaper.
