# Copy-IdenticalParamValues

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Kopierar parametervärden från bundna parametrar till en ny hashtabell baserat på en annan funktions möjliga parametrar.

## Description

Denna cmdlet skapar en ny hashtabell som endast innehåller parametervärdena som matchar de parametrar som definierats i den angivna målfunktionen. Den resulterande hashtabellen kan sedan användas för att anropa funktionen med PowerShell-splatting. Växelparametrar inkluderas endast i resultatet om de uttryckligen angavs och sattes till $true i de bundna parametrarna. Icke-närvarande växelparametrar exkluderas från resultatet för att upprätthålla korrekt parametersemantik.

## Syntax

```powershell
Copy-IdenticalParamValues -BoundParameters <Object[]> -FunctionName <String> [[-DefaultValues] <Object>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-BoundParameters` | Object[] | ✅ | — | 0 | — | Källbundna parametrar att kopiera från |
| `-FunctionName` | String | ✅ | — | 1 | — | Målfuktionens namn för att filtrera parametrar |
| `-DefaultValues` | Object | — | — | 2 | — | Standardvärden för parametrar |

## Examples

### Example 1

```powershell
function Test-Function {
    [CmdletBinding()]
    param(
        [Parameter(Mandatory = $true)]
        [string] $Path,
        [Parameter(Mandatory = $false)]
        [switch] $Recurse
    )

    $params = GenXdev\Copy-IdenticalParamValues `
        -BoundParameters $PSBoundParameters `
        -FunctionName 'Get-ChildItem'

    Get-ChildItem @params
}
```

@{response=Demonstrera kopiering av bundna parametrar för användning med splatting.}
