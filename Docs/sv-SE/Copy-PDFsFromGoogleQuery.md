# Copy-PDFsFromGoogleQuery

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Laddar ner PDF-filer som hittats via Googles sökresultat.

## Description

Utför en Google-sökning i den tidigare valda webbläsarfliken och laddar ner
alla funna PDF-filer till den aktuella katalogen. Stöder flera sökningar och
språkfiltrering.

## Syntax

```powershell
Copy-PDFsFromGoogleQuery -Queries <String[]> [[-Max] <Int32>] [[-Language] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Queries` | String[] | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Söktermerna för att fråga Google efter PDF-filer |
| `-Max` | Int32 | — | — | 1 | `200` | Maximalt antal resultat att hämta (standard: 200) |
| `-Language` | String | — | — | 2 | — | Valfritt språkfilter för sökresultat |

## Examples

### Open-Webbrowser Select-WebbrowserTab $null = New-Item -ItemType Directory -Name pdfs Set-Location pdfs Copy-PDFsFromGoogleQuery "scientific paper co2" -Max 50 -Language "English"

```powershell
Open-Webbrowser
Select-WebbrowserTab
$null = New-Item -ItemType Directory -Name pdfs
Set-Location pdfs
Copy-PDFsFromGoogleQuery "scientific paper co2" -Max 50 -Language "English"
```
