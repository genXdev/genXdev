# CurrentUserHasElevatedRights

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Kontrollerar om den aktuella användaren har förhöjda rättigheter.

## Description

Avgör om den aktuella Windows-användaren har administratörs- eller säkerhetskopieringsoperatörsbehörigheter genom att kontrollera deras säkerhetshuvudmansroller. Returnerar true om användaren har förhöjda rättigheter, annars false. Implementerar robust felhantering för säkerhets- och åtkomstrelaterade problem.

## Syntax

```powershell
CurrentUserHasElevatedRights [<CommonParameters>]
```

## Examples

### $hasRights = CurrentUserHasElevatedRights ##############################################################################

```powershell
$hasRights = CurrentUserHasElevatedRights
##############################################################################
```

## Outputs

- `Boolean
Returns true if user has admin or backup operator rights, false otherwise.`
- `Boolean`
