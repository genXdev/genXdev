# EnsureGenXdev

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Säkerställer att alla GenXdev-moduler laddas korrekt genom att anropa alla Ensure*-cmdletar.

## Description

Denna funktion hämtar alla GenXdev-cmdletar som börjar med "Ensure" eller "Optimize-Ensure" (för c#-cmdletar) och kör varje sådan för att garantera att alla nödvändiga GenXdev-moduler och beroenden är korrekt laddade och tillgängliga för användning. Eventuella fel under körningen fångas upp och visas som informationsmeddelanden.

Alternativt kan den också ladda ner och ladda alla NuGet-paket som definieras i manifestfilen packages.json.

## Syntax

```powershell
EnsureGenXdev [-ConsentToThirdPartySoftwareInstallation] [-DownloadAllNugetPackages] [-Force] [-ForceConsent] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Force` | SwitchParameter | — | — | Named | — | Tvingar utförandet av säkerhetsåtgärder även om de verkar redan vara slutförda. |
| `-DownloadAllNugetPackages` | SwitchParameter | — | — | Named | — | Laddar ner och läser in alla NuGet-paket som definierats i manifestfilen packages.json |
| `-ForceConsent` | SwitchParameter | — | — | Named | — | Tvinga en samtyckesprompt även om inställningen är inställd för installation av tredjepartsprogramvara. |
| `-ConsentToThirdPartySoftwareInstallation` | SwitchParameter | — | — | Named | — | Godkänn automatiskt installation av tredjepartsprogramvara och ange permanent flagga för alla paket. |

## Examples

### EnsureGenXdev

```powershell
EnsureGenXdev
```

Det här kommandot kör alla tillgängliga Ensure* cmdletar för att initiera GenXdev-miljön.

### EnsureGenXdev -DownloadAllNugetPackages

```powershell
EnsureGenXdev -DownloadAllNugetPackages
```

Detta kommando kör alla cmdlets av typen Ensure* och laddar även ner samt inlästa alla NuGet-paket som definierats i manifestfilen packages.json.

### EnsureGenXdev -DownloadAllNugetPackages -ConsentToThirdPartySoftwareInstallation

```powershell
EnsureGenXdev -DownloadAllNugetPackages -ConsentToThirdPartySoftwareInstallation
```

Detta kommando kör alla Ensure* cmdlets och laddar ner NuGet-paket.
