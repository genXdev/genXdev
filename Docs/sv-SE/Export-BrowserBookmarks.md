# Export-BrowserBookmarks

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Exporterar webbläsarbokmärken till en JSON-fil.

## Description

Cmdleten Export-BrowserBookmarks exporterar bokmärken från en angiven webbläsare (Microsoft Edge, Google Chrome eller Mozilla Firefox) till en JSON-fil. Endast en webbläsartyp kan anges åt gången. Bokmärkena exporteras med fullständig bevarande av deras struktur och metadata.

## Syntax

```powershell
Export-BrowserBookmarks -OutputFile <String> [-Chrome] [-Edge] [<CommonParameters>]

Export-BrowserBookmarks [-Firefox] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-OutputFile` | String | ✅ | — | 0 | — | Sökväg till JSON-filen där bokmärken kommer att sparas |
| `-Chrome` | SwitchParameter | — | — | Named | — | Exportera bokmärken från Google Chrome |
| `-Edge` | SwitchParameter | — | — | Named | — | Exportera bokmärken från Microsoft Edge |
| `-Firefox` | SwitchParameter | — | — | Named | — | Exportera bokmärken från Mozilla Firefox *(Parameter set: )* |

## Examples

### Export-BrowserBookmarks -OutputFile "C:\MyBookmarks.json" -Edge

```powershell
Export-BrowserBookmarks -OutputFile "C:\MyBookmarks.json" -Edge
```

### Export-BrowserBookmarks "C:\MyBookmarks.json" -Chrome

```powershell
Export-BrowserBookmarks "C:\MyBookmarks.json" -Chrome
```
