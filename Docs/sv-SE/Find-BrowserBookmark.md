# Find-BrowserBookmark

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `bookmarks`

## Synopsis

> Hittar bokmärken från en eller flera webbläsare.

## Description

Söker igenom bokmärken från Microsoft Edge, Google Chrome eller Mozilla Firefox.
Returnerar bokmärken som matchar en eller flera sökfrågor i deras namn, URL eller
mappsökväg. Om inga frågor anges returneras alla bokmärken från de valda
webbläsarna.

## Syntax

```powershell
Find-BrowserBookmark [[-Queries] <String[]>] [-Chrome] [-Count <Int32>] [-Edge] [-Firefox] [-PassThru] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Queries` | String[] | — | ✅ (ByValue, ByPropertyName) | 0 | — | Söktermer för att hitta matchande bokmärken 🌐 *Supports wildcards* |
| `-Edge` | SwitchParameter | — | — | Named | — | Sök i Microsoft Edge-bokmärken |
| `-Chrome` | SwitchParameter | — | — | Named | — | Sök i Googles Chrome-bokmärken |
| `-Firefox` | SwitchParameter | — | — | Named | — | Sök i Firefox bokmärken |
| `-Count` | Int32 | — | — | Named | `99999999` | Maximalt antal resultat att returnera |
| `-PassThru` | SwitchParameter | — | — | Named | — | Returnera bokmärkesobjekt istället för bara URL:er |

## Examples

### Find-BrowserBookmark -Query "github" -Edge -Chrome -Count 10 Searches Edge and Chrome bookmarks for "github", returns first 10 URLs

```powershell
Find-BrowserBookmark -Query "github" -Edge -Chrome -Count 10
Searches Edge and Chrome bookmarks for "github", returns first 10 URLs
```

### bookmarks powershell -e -ff -PassThru Searches Edge and Firefox bookmarks for "powershell", returns full objects

```powershell
bookmarks powershell -e -ff -PassThru
Searches Edge and Firefox bookmarks for "powershell", returns full objects
```
