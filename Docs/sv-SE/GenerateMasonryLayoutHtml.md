# GenerateMasonryLayoutHtml

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Genererar en responsiv HTML-galleri med murverkslayout från bilddata.

## Description

Skapar ett interaktivt HTML-galleri med responsivt murverksrutnätslayout för att visa bilder. Funktioner inkluderar:
- Responsiv rutnätslayout som anpassar sig efter skärmstorlek
- Bildverktygstips som visar beskrivningar och nyckelord
- Klicka-för-att-kopiera bildsökväg-funktionalitet
- Ren modern stil med svävareffekter

## Syntax

```powershell
GenerateMasonryLayoutHtml -Images <Object> [[-FilePath] <String>] [-AutoAnimateRectangles] [-AutoScrollPixelsPerSecond <Int32>] [-CanDelete] [-CanEdit] [-Description <String>] [-EmbedImages] [-ImageUrlPrefix <String>] [-MaxPrintImages <Int32>] [-PageSize <Int32>] [-RootMargin <String>] [-ShowOnlyPictures] [-SingleColumnMode] [-Threshold <Double>] [-Title <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Images` | Object | ✅ | — | 0 | — | Matris av bildobjekt med sökväg, nyckelord och beskrivning |
| `-FilePath` | String | — | — | 1 | `$null` | Utdata sökväg för den genererade HTML-filen |
| `-Title` | String | — | — | Named | `'Photo Gallery'` | Titel för galleriet |
| `-Description` | String | — | — | Named | `'Hover over images to see face recognition, object detection, and scene classification data'` | Beskrivning för galleriet |
| `-CanEdit` | SwitchParameter | — | — | Named | `$false` | Huruvida redigering är aktiverad |
| `-CanDelete` | SwitchParameter | — | — | Named | `$false` | Om borttagning är aktiverad |
| `-EmbedImages` | SwitchParameter | — | — | Named | `$false` | Bädda in bilder som base64-data-URL:er istället för file://-URL:er för bättre portabilitet. |
| `-ShowOnlyPictures` | SwitchParameter | — | — | Named | — | Visa endast bilder i en rundad rektangel, ingen text under. |
| `-AutoScrollPixelsPerSecond` | Int32 | — | — | Named | `$null` | Bläddra automatiskt sidan med detta antal pixlar per sekund (null för att avaktivera) |
| `-AutoAnimateRectangles` | SwitchParameter | — | — | Named | — | Animera rektanglar (objekt/ytor) i synligt område, cykla var 300:e ms |
| `-SingleColumnMode` | SwitchParameter | — | — | Named | `$false` | Tvinga enkelkolumnslayout (centrerad, 1/3 skärmbredd) |
| `-ImageUrlPrefix` | String | — | — | Named | `''` | Prefix som ska läggas till före varje bildsökväg (t.ex. för fjärr-URL:er) |
| `-PageSize` | Int32 | — | — | Named | `20` | Antal bilder att ladda per sida (för dynamisk laddning) |
| `-MaxPrintImages` | Int32 | — | — | Named | `50` | Maximalt antal bilder att ladda för utskriftsläge |
| `-RootMargin` | String | — | — | Named | `'1200px'` | rootMargin för IntersectionObserver för oändlig scroll-utlösare (t.ex. "1200px") |
| `-Threshold` | Double | — | — | Named | `0.1` | Tröskelvärde för IntersectionObserver för oändlig scrollutlösare |

## Examples

### Create gallery from image array and save to file $images = @(     @{         path = "C:\photos\sunset.jpg"         keywords = @("nature", "sunset", "landscape")         description = @{             short_description = "Mountain sunset"             long_description = "Beautiful sunset over mountain range"         }     } ) GenerateMasonryLayoutHtml -Images $images -FilePath "C:\output\gallery.html"

```powershell

    @{
        path = "C:\photos\sunset.jpg"
        keywords = @("nature", "sunset", "landscape")
        description = @{
            short_description = "Mountain sunset"
            long_description = "Beautiful sunset over mountain range"
        }
    }
)
GenerateMasonryLayoutHtml -Images $images -FilePath "C:\output\gallery.html"
```
