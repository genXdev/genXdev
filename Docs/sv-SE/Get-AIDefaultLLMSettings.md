# Get-AIDefaultLLMSettings

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Hämtar alla tillgängliga standardkonfigurationer för LLM-inställningar för AI-operationer i GenXdev.AI.

## Description

@{response=Hämtar den fullständiga uppsättningen standardinställningar för Large Language Model (LLM) som har konfigurerats för AI-operationer. Resultaten kan filtreras efter frågetyp, modellidentifierare, API-slutpunkt eller API-nyckel. Stöder både sessionsbaserad och beständig preferensbaserad hämtning, med alternativ för att rensa sessionstillstånd eller helt kringgå det för att läsa direkt från lagrade preferenser.}

## Syntax

```powershell
Get-AIDefaultLLMSettings [[-LLMQueryType] <String>] [-ApiEndpoint <String>] [-ApiKey <String>] [-ClearSession] [-Model <String>] [-NoSupportForJsonSchema <String>] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-LLMQueryType` | String | — | — | 0 | `'SimpleIntelligence'` | Typen av LLM-fråga för att hämta inställningar för |
| `-Model` | String | — | — | Named | — | Filtrera konfigurationer efter modellidentifierare eller mönster |
| `-ApiEndpoint` | String | — | — | Named | — | Filtrera konfigurationer efter API-slutpunkts-URL |
| `-ApiKey` | String | — | — | Named | — | Filtrera konfigurationer efter API-nyckel |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | Anger att LLM inte har stöd för JSON-scheman |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Använd alternativa inställningar som lagras i sessionen för AI-inställningar som språk, bildsamlingar, etc. |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Rensa sessionsinställningen (global variabel) innan hämtning |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Databassökväg för inställningsdatafiler |
| `-SkipSession` | SwitchParameter | — | — | Named | — | Hoppa över sessionsinställningar och hämta endast från inställningar eller standardvärden |

## Examples

### Get-AIDefaultLLMSettings -LLMQueryType "Coding"

```powershell
Get-AIDefaultLLMSettings -LLMQueryType "Coding"
```

Hämtar alla tillgängliga standardkonfigurationer för kodfrågetypen.
