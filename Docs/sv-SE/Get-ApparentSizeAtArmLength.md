# Get-ApparentSizeAtArmLength

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Beräknar den skenbara storleken av ett objekt på armlängds avstånd.

## Description

Beräknar den skenbara storleken med hjälp av småvinkelapproximation.

## Syntax

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters <Double> -SizeInMeters <Double> [[-ArmLengthInMeters] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | Avståndet till objektet i meter |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | Den faktiska storleken på objektet i meter |
| `-ArmLengthInMeters` | Double | — | — | 2 | `0` | Armlängden i meter (standard: 0,7) |
| `-As` | String | — | — | 3 | — | Enheten för utdatastorleken |

## Examples

### Example 1

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters 10 -SizeInMeters 1 -As "centimeters"
```

Beräknar den skenbara storleken av ett 1 meter objekt på 10 meters avstånd.

### Example 2

```powershell
Get-ApparentSizeAtArmLength 10 1
```

Beräknar den skenbara storleken med hjälp av positionsparametrar.
