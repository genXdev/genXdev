# Get-ChildProcesses

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Hämtar alla processer som är avkomlingar till den aktuella PowerShell-processen.

## Description

* Undersöker alla körprocesser och identifierar de som har den aktuella
  PowerShell-processen som en förfader i sin överordnade processkedja.
* Detta inkluderar både direkta underprocesser och deras avkomlingar
  (barnbarn, etc.).

## Syntax

```powershell
Get-ChildProcesses [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ChildProcesses
```

Hämta alla underordnade processer till den aktuella PowerShell-sessionen.

### Example 2

```powershell
Get-ChildProcesses -Verbose
```

Hämta underordnade processer med utförlig utdata som visar detaljerad processidentifiering.
