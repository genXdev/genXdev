# Get-CmdletMetaData

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Hämtar metadata för en angiven GenXdev-cmdlet, med möjlighet att översätta hjälptext till ett annat språk.

## Description

Extraherar och returnerar omfattande metadata om en GenXdev-cmdlet inklusive
dess synopsis, beskrivning, parametrar, exempel och annan hjälpinformation.
När ett målspråk anges via parametern -Language kan hjälptexten
översättas med AI-baserade översättningstjänster. Anpassade översättnings-
instruktioner kan ges för att finslipa översättningsresultatet.

## Syntax

```powershell
Get-CmdletMetaData -Name <String> [[-Language] <String>] [[-TranslationInstructions] <String>] [[-Model] <String>] [[-ApiEndpoint] <String>] [[-ApiKey] <String>] [[-NoSupportForJsonSchema] <String>] [-SkipTranslation] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Name` | String | ✅ | — | 0 | — | Cmdletens namn för att hämta metadata |
| `-Language` | String | — | — | 1 | — | sv-SE |
| `-TranslationInstructions` | String | — | — | 2 | — | Anpassade instruktioner för AI-översättningsmodellen. Åsidosätter standardinstruktionerna för cmdlet-metadata-medveten översättning. |
| `-Model` | String | — | — | 3 | — | AI-översättningens modellidentifierare eller mönster |
| `-ApiEndpoint` | String | — | — | 4 | — | API-slutpunktens URL för AI-översättningar |
| `-ApiKey` | String | — | — | 5 | — | API-nyckeln för autentiserade AI-översättningar |
| `-NoSupportForJsonSchema` | String | — | — | 6 | — | Anger att LLM inte har stöd för JSON-scheman |
| `-SkipTranslation` | SwitchParameter | — | — | Named | — | Hoppa över LLM-baserad översättning; behåll hjälptexten på källspråket även när -Language anges. |

## Examples

### Get-CmdletMetaData -Name "Find-Item"

```powershell
Get-CmdletMetaData -Name "Find-Item"
```

Hämtar metadata för cmdleten Find-Item på standardspråket.

### Get-CmdletMetaData -Name "Find-Item" -Language "nl-NL"

```powershell
Get-CmdletMetaData -Name "Find-Item" -Language "nl-NL"
```

Hämtar metadata för Find-Item-cmdleten med holländska översättningar.
