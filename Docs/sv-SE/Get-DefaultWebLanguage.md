# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Hämtar standardnyckeln för språk på webben baserat på systemets aktuella språkinställningar.

## Description

* Hämtar aktuella system språk- och kulturinställningar och mappar dem
  till motsvarande webb språkordlista nyckel som används av översättningstjänster.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Returnerar "Engelska" för ett engelskt system, "Nederländska" för ett nederländskt system, etc.
