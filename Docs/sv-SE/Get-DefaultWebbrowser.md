# Get-DefaultWebbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Returnerar den konfigurerade standardwebbläsaren för den aktuella användaren.

## Description

* Hämtar information om systemets standardwebbläsare genom att fråga Windows-registret.
* Returnerar en hashtabell som innehåller webbläsarens namn, beskrivning, ikonsökväg och körbar sökväg.
* Funktionen kontrollerar både användarpreferenser och systemomfattande webbläsarregistreringar för att avgöra standardwebbläsaren.

## Syntax

```powershell
Get-DefaultWebbrowser [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebbrowser | Format-List
```

Få detaljerad information om standardwebbläsaren.

### Example 2

```powershell
$browser = Get-DefaultWebbrowser
& $browser.Path https://www.github.com/
```

Starta standardwebbläsaren med en specifik URL.
