# Get-EscapeVelocityByMassAndRadius

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Beräknar flykthastighet.

## Description

Använder v = sqrt(2 G M / r).

## Syntax

```powershell
Get-EscapeVelocityByMassAndRadius -MassInKilograms <Double> -RadiusInMeters <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Kroppens massa i kg |
| `-RadiusInMeters` | Double | ✅ | — | 1 | `0` | Radie i meter |
| `-As` | String | — | — | 2 | — | Utdataenhet för hastighet |

## Examples

### Example 1

```powershell
Get-EscapeVelocityByMassAndRadius -MassInKilograms 5.972e24 -RadiusInMeters 6371000 -As "km/h"
```

Beräknar flykthastigheten för en kropp med jordens massa och radie.

### Example 2

```powershell
Get-EscapeVelocityByMassAndRadius 1e26 10000000
```

Beräknar flykthastighet med standardenheter m/s.
