# Get-FreeFallDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Beräknar fallsträckan under fritt fall för en given tidsperiod.

## Description

Beräknar fallsträckan under fritt fall med hjälp av en numerisk metod som tar hänsyn till luftmotstånd och ändhastighet.

## Syntax

```powershell
Get-FreeFallDistance -DurationInSeconds <Double> [[-TerminalVelocityInMetersPerSecond] <Double>] [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | Falltiden i sekunder |
| `-TerminalVelocityInMetersPerSecond` | Double | — | — | 1 | `0` | Terminalhastigheten i meter per sekund (standard: 53) |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Accelerationen på grund av gravitation i m/s² (standard: 9,81) |
| `-As` | String | — | — | 3 | — | Enheten för utdatans avstånd |

## Examples

### Example 1

```powershell
Get-FreeFallDistance -DurationInSeconds 10 -TerminalVelocityInMetersPerSecond 53 -As "feet"
```

Beräknar sträckan fallit på 10 sekunder i fot.

### Example 2

```powershell
Get-FreeFallDistance 5
```

Beräknar avståndet på 5 sekunder i meter.
