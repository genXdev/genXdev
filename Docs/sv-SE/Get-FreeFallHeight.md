# Get-FreeFallHeight

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Beräknar fallhöjden under fritt fall för en given tidsperiod.

## Description

Beräknar fallsträckan under fritt fall med hjälp av en numerisk metod som tar hänsyn till luftmotstånd och ändhastighet.

## Syntax

```powershell
Get-FreeFallHeight -DurationInSeconds <Double> [[-TerminalVelocityInMs] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | Falltiden i sekunder |
| `-TerminalVelocityInMs` | Double | — | — | 1 | `0` | Terminalhastigheten i meter per sekund (standard: 53 m/s för människa) |
| `-As` | String | — | — | 2 | — | Enheten för utmatningshöjden |

## Examples

### Example 1

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -TerminalVelocityInMs 53
```

Beräknar den fallna höjden på 10 sekunder med standard mänsklig terminalhastighet.

### Example 2

```powershell
Get-FreeFallHeight 5
```

Beräknar fallhöjden på 5 sekunder med hjälp av positionsparameter och standard sluthastighet.

### Example 3

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -As "feet"
```

Beräknar fallhöjden på 10 sekunder och returnerar resultatet i fot.
