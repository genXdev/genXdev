# Get-FreeFallTime

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Beräknar tiden det tar för ett föremål att falla en given höjd under fritt fall.

## Description

Beräknar tidslängden som krävs för ett objekt att falla en angiven höjd med hjälp av en numerisk metod som tar hänsyn till luftmotstånd och terminalhastighet. Beräkningen använder små tidssteg för att noggrant modellera fysiken för fallande objekt med realistiska terminalhastighetsbegränsningar.

## Syntax

```powershell
Get-FreeFallTime -HeightInMeters <Double> [[-TerminalVelocityInMs] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-HeightInMeters` | Double | ✅ | — | 0 | `0` | Höjden att falla i meter |
| `-TerminalVelocityInMs` | Double | — | — | 1 | `0` | Terminalhastigheten i meter per sekund (standard: 53 m/s för människa) |
| `-As` | String | — | — | 2 | — | Enheten för utmatningstiden |

## Examples

### Example 1

```powershell
Get-FreeFallTime -HeightInMeters 100 -TerminalVelocityInMs 53
```

Beräknar tiden som krävs för att falla 100 meter med standard mänsklig terminalhastighet.

### Example 2

```powershell
Get-FreeFallTime 50
```

Beräknar tiden som krävs för att falla 50 meter med hjälp av positionsparameter och standardterminalhastighet.

### Example 3

```powershell
Get-FreeFallTime -HeightInMeters 100 -As "minutes"
```

Beräknar tiden det tar att falla 100 meter och returnerar resultatet i minuter.
