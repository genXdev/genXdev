# Get-Ideas

> **Module:** GenXdev.Coding | **Type:** Function | **Aliases:** `ideas`

## Synopsis

> Visar idéer från en README.md-fil.

## Description

Visar alla idéer från avsnittet "## Ideas" i en README.md-fil. Kan använda antingen README-filen på aktuell plats, PowerShell-profilkatalogen eller OneDrive-katalogen.

## Syntax

```powershell
Get-Ideas [[-First] <Int32>] [-Ascending] [-SortByDate] [-UseHomeREADME] [-UseOneDriveREADME] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-UseHomeREADME` | SwitchParameter | — | — | Named | — | Använd README i PowerShell-profilkatalogen |
| `-UseOneDriveREADME` | SwitchParameter | — | — | Named | — | Använd README i OneDrive-katalogen |
| `-SortByDate` | SwitchParameter | — | — | Named | — | Sortera rader efter datum (yyyyMMdd-prefix) istället för prioritet |
| `-Ascending` | SwitchParameter | — | — | Named | — | Vänd sorteringsordningen (stigande istället för fallande) |
| `-First` | Int32 | — | — | 0 | `0` | Begränsa utdata till de första N raderna |

## Examples

### Ideas -UseHomeREADME

```powershell
Ideas -UseHomeREADME
```

### Ideas -UseOneDriveREADME

```powershell
Ideas -UseOneDriveREADME
```
