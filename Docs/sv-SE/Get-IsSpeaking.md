# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> Returnerar sant om text-till-tal-motorn talar.

## Description

Kontrollerar tillståndet för både standard- och anpassade talsyntesmotorer för att avgöra om någon av dem för närvarande talar. Returnerar true om tal pågår, annars false.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Returnerar sant om text-till-tal-motorn talar.

### Example 2

```powershell
iss
```

Returnerar sant om talsyntesmotorn talar (med alias).
