# Get-LightTravelTimeByDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Beräknar tiden för ljus att färdas en sträcka.

## Description

Använder t = d / c, där c beror på medium.

## Syntax

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters <Double> [[-As] <String>] [<CommonParameters>]

Get-LightTravelTimeByDistance [[-SpeedOfLightInMetersPerSecond] <Double>] [<CommonParameters>]

Get-LightTravelTimeByDistance -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | Avstånd i meter |
| `-SpeedOfLightInMetersPerSecond` | Double | — | — | 1 | `0` | Ljusets hastighet i m/s (standard: 299792458) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | Mediet genom vilket ljus färdas *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | Tidsenhet för utdata |

## Examples

### Example 1

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters 149597870700 -Medium "water" -As "minutes"
```

Beräknar tiden för ljus att färdas sträckan till solen genom vatten, i minuter.

### Example 2

```powershell
Get-LightTravelTimeByDistance 300000000 -SpeedOfLightInMetersPerSecond 225000000
```

Beräknar ljusets tid att färdas 300000000 meter vid 225000000 m/s.
