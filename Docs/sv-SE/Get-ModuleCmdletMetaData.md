# Get-ModuleCmdletMetaData

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Hämtar metadata för alla cmdlets i en PowerShell-modul.

## Description

Hämtar fullständig cmdlet-metadata (sammanfattning, beskrivning, parametrar, exempel, utdata och alias) för varje cmdlet i den angivna modulen, och lägger till egenskaperna SubModuleName och CmdletType till varje resultat.

Submodultilldelning använder två oberoende vägar:

- Skript-cmdlets (.ps1): källfil matchas mot .psm1-dotkälla katalogmappningar (Functions\<SubModuleName>\FileName.ps1).
- Kompilerade cmdlets (.dll): namnrymd från ImplementationType.Namespace (t.ex. GenXdev.FileSystem).

## Syntax

```powershell
Get-ModuleCmdletMetaData -ModuleName <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-Language <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [-SkipTranslation] [-TranslationInstructions <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String | ✅ | — | 0 | — | Namnet på PowerShell-modulen som cmdlet-metadata ska hämtas för |
| `-Language` | String | — | — | Named | — | @{response=sv-SE} |
| `-Model` | String | — | — | Named | — | AI-översättningens modellidentifierare eller mönster |
| `-ApiEndpoint` | String | — | — | Named | — | API-slutpunktens URL för AI-översättningar |
| `-ApiKey` | String | — | — | Named | — | API-nyckeln för autentiserade AI-översättningar |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | Anger att LLM inte har stöd för JSON-scheman |
| `-TranslationInstructions` | String | — | — | Named | — | @{response=Anpassade AI-översättningsinstruktioner} |
| `-SkipTranslation` | SwitchParameter | — | — | Named | — | Hoppa över LLM-baserad översättning |

## Examples

### Get-ModuleCmdletMetaData -ModuleName 'GenXdev'

```powershell
Get-ModuleCmdletMetaData -ModuleName 'GenXdev'
```

Returnerar metadata för alla cmdlets i GenXdev-modulen.

### Get-ModuleCmdletMetaData -ModuleName 'GenXdev' -Language 'nl-NL'

```powershell
Get-ModuleCmdletMetaData -ModuleName 'GenXdev' -Language 'nl-NL'
```

Returnerar nederländsk-översatt metadata för alla GenXdev-cmdletar.
