# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Returnerar ett fönsterhjälpobjekt för PowerShell-terminalens huvudfönster.

## Description

* Hämtar ett WindowObj-hjälpobjekt som representerar huvudfönstret för
  den aktuella PowerShell-värdprocessen.
* Detta möjliggör manipulering och interaktion med själva terminalfönstret.

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

Hämtar WindowObj för det aktuella PowerShell-terminalfönstret.
