# Get-RefractionAngleByIncidentAngleAndIndices

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Beräknar refraktionsvinkeln med Snells lag.

## Description

Använder θ2 = arcsin( (n1 / n2) sin θ1 ).

## Syntax

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees <Double> -IndexOfRefraction1 <Double> -IndexOfRefraction2 <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-IncidentAngleInDegrees` | Double | ✅ | — | 0 | `0` | Infallsvinkel i grader |
| `-IndexOfRefraction1` | Double | ✅ | — | 1 | `0` | Brytningsindex för första mediet |
| `-IndexOfRefraction2` | Double | ✅ | — | 2 | `0` | Andra mediets brytningsindex |
| `-As` | String | — | — | 3 | — | Utdataenhet för vinkel |

## Examples

### Example 1

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees 30 -IndexOfRefraction1 1 -IndexOfRefraction2 1.33 -As "radians"
```

Beräknar brytningsvinkeln när ljus passerar från luft (n=1) till vatten (n=1,33) vid 30 graders infallsvinkel.

### Example 2

```powershell
Get-RefractionAngleByIncidentAngleAndIndices 45 1 1.5
```

Beräknar refraktionsvinkeln när ljus passerar från luft (n=1) till glas (n=1,5) vid 45 graders infallsvinkel.
