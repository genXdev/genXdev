# Get-SoundTravelDistanceByTime

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Beräknar ljudets avstånd under en given tid.

## Description

Använder formeln sträcka = hastighet * tid, med standardhastighet för ljud i luft.

## Syntax

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds <Double> [[-As] <String>] [<CommonParameters>]

Get-SoundTravelDistanceByTime [[-SpeedOfSoundInMetersPerSecond] <Double>] [<CommonParameters>]

Get-SoundTravelDistanceByTime -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-TimeInSeconds` | Double | ✅ | — | 0 | `0` | Tiden i sekunder |
| `-SpeedOfSoundInMetersPerSecond` | Double | — | — | 1 | `0` | Ljudets hastighet i m/s (standard: 343) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | Mediet genom vilket ljud färdas *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | Enheten för utdatans avstånd |

## Examples

### Example 1

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds 5 -Medium "water" -As "kilometers"
```

Beräknar hur långt ljud färdas i vatten under 5 sekunder och omvandlar resultatet till kilometer.

### Example 2

```powershell
Get-SoundTravelDistanceByTime 10 -SpeedOfSoundInMetersPerSecond 1480
```

Beräknar sträckan med hjälp av en specifik ljudhastighet.
