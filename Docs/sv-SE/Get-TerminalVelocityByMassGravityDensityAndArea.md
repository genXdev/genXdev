# Get-TerminalVelocityByMassGravityDensityAndArea

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Beräknar terminalhastighet.

## Description

Använder v = sqrt(2 m g / (ρ A C)).

## Syntax

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms <Double> [[-GravityInMetersPerSecondSquared] <Double>] -DensityInKilogramsPerCubicMeter <Double> -AreaInSquareMeters <Double> [[-DragCoefficient] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Massa i kg |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 1 | `0` | Gravitation i m/s² (standard: 9,81) |
| `-DensityInKilogramsPerCubicMeter` | Double | ✅ | — | 2 | `0` | Fluidens densitet i kg/m³ |
| `-AreaInSquareMeters` | Double | ✅ | — | 3 | `0` | Tvärsnittsarea i m² |
| `-DragCoefficient` | Double | — | — | 4 | `0` | Motståndskoefficient (standardvärde: 0,5) |
| `-As` | String | — | — | 5 | — | Utdataenhet för hastighet |

## Examples

### Example 1

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms 80 -DensityInKilogramsPerCubicMeter 1.225 -AreaInSquareMeters 0.7 -DragCoefficient 1.0 -As "mph"
```

Beräknar terminalhastigheten för en massa på 80 kg i luft med en luftmotståndskoefficient på 1,0 och matar ut resultatet i mph.

### Example 2

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea 70 1.225 0.8 0.8
```

Använder positionsparametrar för att beräkna terminalhastighet.
