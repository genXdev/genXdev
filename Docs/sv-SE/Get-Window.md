# Get-Window

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Hämtar fönsterinformation för angivna processer eller fönsterhandtag.

## Description

* Hämtar fönsterinformation med processnamn, ID eller fönsterhandtag.
* Returnerar WindowObj-objekt som innehåller detaljer om huvudfönster för matchande processer.
* Stöder jokertecken vid sökning efter processnamn.

## Syntax

```powershell
Get-Window [[-ProcessName] <String>] [-ProcessId <Int32>] [-WindowHandle <Int64>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ProcessName` | String | — | ✅ (ByPropertyName) | 0 | — | Namnet på processen för att hämta fönsterinformation 🌐 *Supports wildcards* |
| `-ProcessId` | Int32 | — | ✅ (ByPropertyName) | Named | `0` | ID för processen att hämta fönsterinformation för |
| `-WindowHandle` | Int64 | — | ✅ (ByPropertyName) | Named | `0` | Fönsterhandtag att hämta information för |

## Examples

### Example 1

```powershell
Get-Window -ProcessName "notepad"
```

Hämtar fönsterinformation för alla processer med namnet "notepad".

### Example 2

```powershell
Get-Window -ProcessId 1234
```

Hämtar fönsterinformation för processen med ID 1234.

### Example 3

```powershell
Get-Window -WindowHandle 45678
```

Hämtar fönsterinformation för fönstret med handtag 45678.
