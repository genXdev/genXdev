# Import-GenXdevBookmarkletMenu

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Importerar GenXdev JavaScript-bokmärken till webbläsarens bokmärkessamlingar.

## Description

Den här funktionen skannar en katalog efter GenXdev-bokmärkesfiler med

## Syntax

```powershell
Import-GenXdevBookmarkletMenu [[-SnippetsPath] <String>] [[-TargetFolder] <String>] [-Chrome] [-Edge] [-Firefox] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-SnippetsPath` | String | — | — | 0 | `"$($MyInvocation.MyCommand.Module.ModuleBase)\Bookmarklets"` | Sökväg till katalogen som innehåller bokmärkeskodsfiler |
| `-TargetFolder` | String | — | — | 1 | `""` | Målbokmärkesmapp i webbläsarens bokmärkesstruktur |
| `-Edge` | SwitchParameter | — | — | Named | — | Importera bokmärkeslets till Microsoft Edge-webbläsaren |
| `-Chrome` | SwitchParameter | — | — | Named | — | Importera bokmärkeslets till Google Chrome-webbläsaren |
| `-Firefox` | SwitchParameter | — | — | Named | — | Importera bokmärkeslets till webbläsaren Mozilla Firefox |

## Examples

### Import-GenXdevBookmarkletMenu -Edge

```powershell
Import-GenXdevBookmarkletMenu -Edge
```

Importerar alla bookmarklet-filer från standardklippsökvägen till Microsoft Edges bokmärkesfältmapp.

### Import-GenXdevBookmarkletMenu -SnippetsPath "C:\MyBookmarklets" -Chrome -WhatIf

```powershell
Import-GenXdevBookmarkletMenu -SnippetsPath "C:\MyBookmarklets" -Chrome -WhatIf
```

Visar vilka bokmärkeslets som skulle importeras från den angivna sökvägen till Google Chrome utan att faktiskt utföra importoperationen.
