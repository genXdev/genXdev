# Invoke-AIPowershellCommand

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `hint`

## Synopsis

> Genererar och kör PowerShell-kommandon med AI-assistans.

## Description

Använder AI-modeller för att generera PowerShell-kommandon baserat på naturliga språkfrågor. Funktionen kan antingen skicka kommandon direkt till PowerShell-fönstret eller kopiera dem till urklipp. Den utnyttjar AI-modeller för att tolka naturligt språk och omvandla det till körbara PowerShell-kommandon med omfattande parameterstöd för olika AI-backends.

## Syntax

```powershell
Invoke-AIPowershellCommand -PowershellCmdline <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-LLMQueryType <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-PowershellCmdline` | String | ✅ | — | 0 | — | Den naturliga språkfrågan för att generera ett kommando för |
| `-LLMQueryType` | String | — | — | Named | `'Coding'` | Typen av LLM-fråga |
| `-Model` | String | — | — | Named | — | Modellidentifieraren eller mönstret som ska användas för AI-operationer |
| `-ApiEndpoint` | String | — | — | Named | — | API-ändpunkts-URL:en för AI-operationer |
| `-ApiKey` | String | — | — | Named | — | API-nyckel för autentiserade AI-operationer |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | Anger att LLM inte har stöd för JSON-scheman |

## Examples

### Invoke-AIPowershellCommand -Query "list all running processes"

```powershell
Invoke-AIPowershellCommand -Query "list all running processes"
```

Genererar ett PowerShell-kommando för att lista körande processer.

### hint "list files modified today"

```powershell
hint "list files modified today"
```

Använder aliaset för att generera ett kommando för att hitta filer som ändrats idag.

### Invoke-AIPowershellCommand -Query "stop service" -Clipboard

```powershell
Invoke-AIPowershellCommand -Query "stop service" -Clipboard
```

Genererar ett kommando för att stoppa en tjänst och kopierar det till urklipp.
