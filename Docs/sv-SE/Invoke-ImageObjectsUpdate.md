# Invoke-ImageObjectsUpdate

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `imageobjectdetection`

## Synopsis

> Uppdaterar metadata för objektdetektering för bildfiler i en angiven katalog.

## Description

Denna funktion bearbetar bilder i en angiven katalog för att detektera objekt med hjälp av artificiell intelligens. Den skapar JSON-metadatafiler som innehåller detekterade objekt, deras positioner, konfidensnivåer och etiketter. Funktionen stöder batchbearbetning med konfigurerbara konfidensgränser och kan valfritt hoppa över befintliga metadatafiler eller försöka igen med tidigare misslyckade detektioner.

## Syntax

```powershell
Invoke-ImageObjectsUpdate [[-ImageDirectories] <String[]>] [-ClearSession] [-Language <String>] [-LLMQueryType <String>] [-Model <String>] [-OnlyNew] [-PreferencesDatabasePath <String>] [-Recurse] [-RetryFailed] [-SessionOnly] [-SkipSession] [-TimeoutSeconds <Int32>] [<CommonParameters>]
```

## Examples

### Invoke-ImageObjectsUpdate -ImageDirectories @("C:\Photos", "D:\Pictures") -Recurse

```powershell
Invoke-ImageObjectsUpdate -ImageDirectories @("C:\Photos", "D:\Pictures") -Recurse
```

Detta exempel bearbetar alla bilder i C:\Photos och D:\Pictures och alla underkataloger med standardinställningar och 0.5 konfidenströskel.

### Invoke-ImageObjectsUpdate @("C:\Photos", "C:\Archive") -RetryFailed -OnlyNew

```powershell
Invoke-ImageObjectsUpdate @("C:\Photos", "C:\Archive") -RetryFailed -OnlyNew
```

Det här exemplet bearbetar endast nya bilder och försöker igen med tidigare misslyckade i flera kataloger med hjälp av positionsparametersyntax.

### Invoke-ImageObjectsUpdate -ImageDirectories "C:\Photos" -UseGPU `     -ConfidenceThreshold 0.7

```powershell
Invoke-ImageObjectsUpdate -ImageDirectories "C:\Photos" -UseGPU `
    -ConfidenceThreshold 0.7
```

Det här exemplet använder GPU-acceleration med ett högre konfidensvärde på 0,7 för mer exakta men färre objektsdetekteringar.
