# New-SQLServerDatabase

> **Module:** GenXdev.Data.SqlServer | **Type:** Function | **Aliases:** `nsqldb`

## Synopsis

> Skapar en ny SQL Server-databas.

## Description

Skapar en ny SQL Server-databas med det angivna namnet på den angivna servern. Kräver lämpliga behörigheter för att skapa databaser på den mål-SQL Server-instansen. Om databasen redan finns kommer åtgärden att hoppas över. Stöder filbaserad databasskapning med explicita .mdf/.ldf-sökvägar.

## Syntax

```powershell
New-SQLServerDatabase -DatabaseName <String> [[-Server] <String>] [-DataFilePath <String>] [-DetachAfterCreation] [-LogFilePath <String>] [<CommonParameters>]

New-SQLServerDatabase -ConnectionString <String> [<CommonParameters>]

New-SQLServerDatabase [-ConsentToThirdPartySoftwareInstallation] [-ForceConsent] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DatabaseName` | String | ✅ | — | 0 | — | Namnet på den SQL Server-databas som ska skapas *(Parameter set: )* |
| `-Server` | String | — | — | 1 | `'.'` | SQL Server-instansens namn *(Parameter set: )* |
| `-ConnectionString` | String | ✅ | — | 0 | — | Anslutningssträngen för att ansluta till SQL Server *(Parameter set: )* |
| `-ForceConsent` | SwitchParameter | — | — | Named | — | Tvinga en samtyckesfråga även om inställningen är inställd för SQL Server-paketinstallation. |
| `-ConsentToThirdPartySoftwareInstallation` | SwitchParameter | — | — | Named | — | Godkänn automatiskt installation av tredjepartsprogramvara och ange permanent flagga för SQL Server-paketet. |
| `-DataFilePath` | String | — | — | Named | — | Den valfria sökvägen till datafilen (.mdf) för att skapa en filbaserad databas. *(Parameter set: )* |
| `-LogFilePath` | String | — | — | Named | — | Valfri loggfilssökväg (.ldf) för filbaserad databasskapning. *(Parameter set: )* |
| `-DetachAfterCreation` | SwitchParameter | — | — | Named | — | Koppla bort databas efter skapande (för filbaserade databaser). *(Parameter set: )* |

## Examples

### New-SQLServerDatabase -DatabaseName "MyNewDatabase" -Server "localhost"

```powershell
New-SQLServerDatabase -DatabaseName "MyNewDatabase" -Server "localhost"
```

### New-SQLServerDatabase "MyNewDatabase"

```powershell
New-SQLServerDatabase "MyNewDatabase"
```

### New-SQLServerDatabase -DatabaseName "MyDB" -ConnectionString "Server=.;..."

```powershell
New-SQLServerDatabase -DatabaseName "MyDB" -ConnectionString "Server=.;..."
```

### New-SQLServerDatabase -DatabaseName "ImageIndex" -DataFilePath "C:\Data\ImageIndex.mdf" -DetachAfterCreation

```powershell
New-SQLServerDatabase -DatabaseName "ImageIndex" -DataFilePath "C:\Data\ImageIndex.mdf" -DetachAfterCreation
```

### New-SQLServerDatabase -DatabaseName "MyDB" -Server "." -ConsentToThirdPartySoftwareInstallation

```powershell
New-SQLServerDatabase -DatabaseName "MyDB" -Server "." -ConsentToThirdPartySoftwareInstallation
```
