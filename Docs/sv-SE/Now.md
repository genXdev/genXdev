# Now

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Returnerar aktuell systemdatum och tid som ett DateTime-objekt.

## Description

Ger ett enkelt sätt att hämta aktuellt systemdatum och tid utan några parametrar. Returnerar ett standard .NET DateTime-objekt som kan användas för datum- och tidsberäkningar, formatering och jämförelser.

## Syntax

```powershell
Now [<CommonParameters>]
```

## Examples

### Now Returns the current system date and time as a DateTime object

```powershell
Now
Returns the current system date and time as a DateTime object
```

### $timestamp = Now Stores the current date and time in a variable for later use

```powershell
$timestamp = Now
Stores the current date and time in a variable for later use
```

## Outputs

- `DateTime`
