# Save-DesktopScreenShot

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Tar en skärmdump på den angivna bildskärmen och sparar den till den angivna sökvägen.

## Description

* Tar en skärmbild av skrivbordet från angiven bildskärm och sparar
  den som en bildfil.
* Stödjer flera bildskärmar och identifierar automatiskt rätt skärmgränser
  och skalningsfaktorer för hög-DPI-skärmar.

## Syntax

```powershell
Save-DesktopScreenShot -Monitor <Int32> -OutputPath <String> [-Quality <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | ✅ | ✅ (ByPropertyName) | 0 | `0` | Index på övervakare att fånga (0=primär, 1+=specifik övervakare, -1=alla övervakare) |
| `-OutputPath` | String | ✅ | ✅ (ByPropertyName) | 1 | — | Fullständig sökväg där skärmbilden kommer att sparas |
| `-Quality` | Int32 | — | — | Named | `0` | JPEG-kvalitetsnivå (1-100), gäller endast för JPEG-format |

## Examples

### Example 1

```powershell
Save-DesktopScreenShot -Monitor 0 -OutputPath "C:\Screenshots\primary.png"
```

Tar en skärmdump av den primära skärmen och sparar den som PNG.

### Example 2

```powershell
Save-DesktopScreenShot -Monitor 1 -OutputPath "C:\Screenshots\monitor2.jpg" -Quality 95
```

Tar en skärmdump av den andra bildskärmen med hög kvalitet JPEG.

### Example 3

```powershell
Save-DesktopScreenShot -Monitor -1 -OutputPath "C:\Screenshots\all_monitors.png"
```

Fångar alla skärmar kombinerade till en enda skärmbild.
