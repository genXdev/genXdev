# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Tal om det aktuella datumet med text-till-tal-syntes.

## Description

Konverterar det aktuella datumet till ett naturligt språkformat och tillkännager det med systemets text-till-tal-motor. Datumet uttalas i formatet: 'Det är [veckodag], [månad] [dag] [år]'

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
