# Search-GenXdevCmdlet

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `searchcmdlet`

## Synopsis

> Söker efter en GenXdev-cmdlet och öppnar den eventuellt i en IDE för redigering.

## Description

Denna funktion söker efter GenXdev-cmdlets med hjälp av ett mönster och öppnar eventuellt den funna cmdletens källfil i Visual Studio Code eller Visual Studio för redigering. Den hämtar cmdlet-information och ger kortkommandon för att snabbt navigera till funktionsdefinitionen.

## Syntax

```powershell
Search-GenXdevCmdlet [[-CmdletName] <String>] [[-ModuleName] <String[]>] [-Code] [-EditCmdlet] [-FromScripts] [-NoLocal] [-OnlyPublished] [-VisualStudio] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CmdletName` | String | — | — | 0 | — | Sök mönster för att filtrera cmdlets 🌐 *Supports wildcards* |
| `-ModuleName` | String[] | — | ✅ (ByValue, ByPropertyName) | 1 | — | Modulnamn för GenXdev att söka 🌐 *Supports wildcards* |
| `-NoLocal` | SwitchParameter | — | — | Named | — | Hoppa över sökning i lokala modulsökvägar |
| `-OnlyPublished` | SwitchParameter | — | — | Named | — | Begränsa sökningen till endast publicerade modulsökvägar |
| `-FromScripts` | SwitchParameter | — | — | Named | — | Sök i skriptfiler istället för modulfiler |
| `-Code` | SwitchParameter | — | — | Named | — | Öppna den hittade cmdleten i Visual Studio Code |
| `-VisualStudio` | SwitchParameter | — | — | Named | — | Öppna den hittade cmdleten i Visual Studio |
| `-EditCmdlet` | SwitchParameter | — | — | Named | — | Öppnar även filen i redigeringsprogrammet efter att ha hittat den. |

## Examples

### Search-GenXdevCmdlet -CmdletName "Get-*" -Code Opens first matching cmdlet starting with "Get-" in Visual Studio Code.

```powershell
Search-GenXdevCmdlet -CmdletName "Get-*" -Code
Opens first matching cmdlet starting with "Get-" in Visual Studio Code.
```

### searchcmdlet "Set-GitConfig" -EditCmdlet Searches for Set-GitConfig cmdlet and opens it for editing.

```powershell
searchcmdlet "Set-GitConfig" -EditCmdlet
Searches for Set-GitConfig cmdlet and opens it for editing.
```
