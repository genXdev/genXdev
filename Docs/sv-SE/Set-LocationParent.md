# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> Ändrar den aktuella platsen till den överordnade katalogen och listar dess innehåll.

## Description

* Navigerar upp en katalognivå i filsystemets hierarki.
* Visar innehållet i den nya aktuella katalogen.
* Ger ett praktiskt '..'-alias för snabb katalognavigering.

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

Ändrar till den överordnade katalogen och listar dess innehåll.

### Example 2

```powershell
..
```

Använder aliaset för att byta till överordnad katalog.
