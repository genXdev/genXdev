# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> Navigerar upp två katalognivåer i filsystemshierarkin.

## Description

* Ändrar den aktuella arbetskatalogen till morföräldrakatalogen (två nivåer upp).
* Visar innehållet i den resulterande katalogen.

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Ändrar till den överordnade katalogen och visar dess innehåll.

### Example 2

```powershell
...
```

Använder aliaset '...' för samma åtgärd.
