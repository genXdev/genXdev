# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> Navigerar upp fem katalognivåer i filsystemshierarkin.

## Description

* Ändrar den aktuella arbetskatalogen genom att flytta upp fem föräldrakatalognivåer.
* Visar innehållet i den resulterande katalogen.
* Ger en praktisk genvägsalias '......' för snabb navigering.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Navigerar upp fem katalognivåer från den aktuella platsen.

### Example 2

```powershell
......
```

Använder aliaset för att navigera upp fem katalognivåer.
