# Set-MonitorPowerOff

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `poweroff`

## Synopsis

> Stänger av strömmen till alla anslutna bildskärmar.

## Description

Använder Windows API-anrop för att skicka en avstängningssignal till alla anslutna bildskärmar. Detta motsvarar att trycka på den fysiska strömknappen på dina bildskärmar. Bildskärmarna går in i energisparläge men kan väckas genom att flytta musen eller trycka på en tangent.

## Syntax

```powershell
Set-MonitorPowerOff [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOff
```

Stänger av alla anslutna bildskärmar.

### Example 2

```powershell
poweroff
```

Stänger av alla anslutna bildskärmar med hjälp av aliaset.
