# Start-VlcMediaPlayerNextInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcnext`

## Synopsis

> Flyttar VLC Media Player till nästa objekt i den aktuella spellistan.

## Description

Denna funktion skickar tangenttryckningen 'n' till VLC Media Player för att hoppa till nästa spår eller mediaobjekt i den för närvarande laddade spellistan. Den fokuserar VLC-fönstret, skickar nästa kommando och återställer fokus till PowerShell-konsolen. Funktionen inkluderar stöd för ShouldProcess för bekräftelseprompter när det behövs.

## Syntax

```powershell
Start-VlcMediaPlayerNextInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerNextInPlaylist

```powershell
Start-VlcMediaPlayerNextInPlaylist
```

### vlcnext

```powershell
vlcnext
```
