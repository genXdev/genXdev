# Start-VlcMediaPlayerPreviousInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcprev`, `vlcback`

## Synopsis

> Går till föregående objekt i VLC Media Player-spellistan.

## Description

Denna funktion skickar tangentkommandot 'p' till VLC Media Player för att navigera till föregående objekt i den aktuella spellistan. Funktionen stöder WhatIf-operationer och återställer fokus efter att ha skickat kommandot.

## Syntax

```powershell
Start-VlcMediaPlayerPreviousInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerPreviousInPlaylist

```powershell
Start-VlcMediaPlayerPreviousInPlaylist
```

### vlcprev

```powershell
vlcprev
```

### vlcback

```powershell
vlcback
```
