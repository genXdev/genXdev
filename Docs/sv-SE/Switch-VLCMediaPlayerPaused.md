# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> Växlar paus/uppspelningsläge för VLC Media Player.

## Description

Denna funktion skickar en mellanslagstangent till VLC Media Player för att växla mellan pausad och spelande status. Den återställer automatiskt fokus till det ursprungliga fönstret efter att ha skickat tangentkommandot.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

Växlar paus/uppspelningsstatus för VLC Media Player.

### vlcpause

```powershell
vlcpause
```

Använder aliaset för att växla mellan paus/uppspelningsläge.

### vlcplay

```powershell
vlcplay
```

Använder det alternativa aliaset för att växla mellan paus/uppspelningsläge.
