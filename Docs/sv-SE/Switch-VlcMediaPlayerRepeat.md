# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> Växlar repeteringsläget i VLC Media Player.

## Description

Denna funktion skickar tangentkommandot 'r' till VLC Media Player för att växla mellan olika uppspelningslägen (ingen upprepning, upprepa aktuell, upprepa alla). Funktionen öppnar VLC om det inte redan körs och återställer fokus till föregående fönster efter att kommandot har skickats.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
