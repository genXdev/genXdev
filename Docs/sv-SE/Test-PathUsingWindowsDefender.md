# Test-PathUsingWindowsDefender

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `virusscan`, `HasNoVirus`

## Synopsis

> Söker igenom filer eller kataloger efter skadlig kod med Windows Defender.

## Description

* Utför en riktad genomsökning av angivna filer eller kataloger med hjälp av Windows Defenders kommandoradsgränssnitt (MpCmdRun.exe).
* Kan genomsöka i detekteringsläge endast eller med automatisk hotreparation aktiverad.
* Returnerar true om inga hot upptäcks, false om hot hittas eller genomsökningen misslyckas.

## Syntax

```powershell
Test-PathUsingWindowsDefender -FilePath <String> [-EnableRemediation] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Sökvägen till filen eller katalogen som ska genomsökas |
| `-EnableRemediation` | SwitchParameter | — | — | Named | `False` | Instruerar Windows Defender att vidta åtgärder mot hot |

## Examples

### Example 1

```powershell
Test-PathUsingWindowsDefender -FilePath "C:\Downloads\file.exe" -Verbose
```

Genomsöker den angivna filen och visar utförlig utdata.

### Example 2

```powershell
virusscan "C:\Downloads\file.exe" -EnableRemediation
```

Skannar filen med hjälp av aliaset och aktiverar åtgärd.

### Example 3

```powershell
"C:\Downloads\file.exe" | HasNoVirus
```

Rör filvägen till cmdleten med hjälp av ett alias.
