# WriteFileOutput

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Matar ut filinformation med hyperlänkade visningsnamn för förbättrad användarupplevelse.

## Description

Denna funktion bearbetar filinmatningsobjekt och matar ut dem i ett användarvänligt
format med hyperlänkade filnamn när de visas i konsolen. Den hanterar
olika inmatningstyper inklusive strängar, FileInfo-objekt och andra objekt
typer. Funktionen upptäcker automatiskt om utdata omdirigeras eller
fångas upp i en pipeline och justerar sitt beteende därefter.

## Syntax

```powershell
WriteFileOutput -CallerInvocation <Object> -Input <Object> [-FullPaths] [-Prefix <String>] [-RelativeBasePath <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CallerInvocation` | Object | ✅ | ✅ (ByValue) | 0 | — | Anropsinformationen från anropsfunktionen |
| `-Input` | Object | ✅ | ✅ (ByValue) | 1 | — | Indataobjektet att bearbeta, vilket kan vara en sökväg eller ett objekt |
| `-Prefix` | String | — | — | Named | — | En valfri strängprefix som ska läggas till före utdatavisningen för ytterligare sammanhang |
| `-RelativeBasePath` | String | — | — | Named | — | Grundläggande sökväg för att generera relativa filvägar i utdata |
| `-FullPaths` | SwitchParameter | — | — | Named | — | Tvingar utdata att använda fullständiga absoluta sökvägar istället för relativa sökvägar |

## Examples

### WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"

```powershell
WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"
```

### Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_

```powershell
Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_
```
