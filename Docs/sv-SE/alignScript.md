# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Returnerar en sträng (med ändrad indragning) av en angiven skriptblockssträng

## Description

Ändrar indragningen av en scriptblock-sträng samtidigt som den ursprungliga kodblocksindragningen respekteras

## Syntax

```powershell
alignScript [<CommonParameters>]
```
