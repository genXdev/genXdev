# resetdefaultmonitor

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Återställer standardkonfiguration för sekundär bildskärm.

## Description

Det här skriptet återställer standardkonfigurationen för den sekundära skärmen i systemet,
och ställer in den sekundära skärmen till det ursprungliga standardvärdet.
Detta är användbart för användare som vill återgå till sin tidigare flerskärmskonfiguration efter att ha använt sida-vid-sida-inställningar.
Se även: funktionen 'sida-vid-sida' för att växla till sida-vid-sida-läge för nya fönster.

## Syntax

```powershell
resetdefaultmonitor [<CommonParameters>]
```

## Examples

### secondscreen

```powershell
secondscreen
```

Återställer standardkonfigurationen för den sekundära monitorn på systemet.
