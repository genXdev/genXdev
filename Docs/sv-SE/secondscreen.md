# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Ställer in standardkonfigurationen för den andra skärmen.

## Description

Ställer in standardbeteendet för GenXdev-fönsteröppningar så att de visas på den sekundära bildskärmen. Detta är användbart för användare med en enda bildskärm eller för dem som föredrar att placera fönster sida vid sida. Se även cmdlet 'sidebyside' och 'restoredefaultmonitor'.

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```
