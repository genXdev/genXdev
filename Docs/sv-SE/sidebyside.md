# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Anger standardkonfigurationen för sida-vid-sida.

## Description

Ställer in standardbeteendet för GenXdev-fönsteröppningar så att de visas sida vid sida med PowerShell. Detta är användbart för användare med en enda bildskärm eller de som föredrar sida vid sida-fönsterlayouter. Se även cmdlet 'secondscreen' och 'restoredefaultmonitor'.

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```
