# ConvertTo-HashTable

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> แปลง PSCustomObject เป็น HashTable แบบเรียกซ้ำ

## Description

* ฟังก์ชันนี้แปลง PSCustomObject และ PSCustomObject ที่ซ้อนกันทั้งหมด
  เป็น HashTables
* จัดการกับอาร์เรย์และประเภทคอลเลกชันอื่นๆ โดยการประมวลผลแต่ละองค์ประกอบ
  แบบวนซ้ำ

## Syntax

```powershell
ConvertTo-HashTable -InputObject <Object> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputObject` | Object | ✅ | ✅ (ByValue) | 0 | — | PSCustomObject ที่จะแปลงเป็น HashTable |

## Examples

### Example 1

```powershell
$object = [PSCustomObject]@{
    Name = "John"
    Age = 30
    Details = [PSCustomObject]@{
        City = "New York"
    }
}
$hashTable = ConvertTo-HashTable -InputObject $object
```

แปลง PSCustomObject เป็น HashTable พร้อมคุณสมบัติแบบซ้อน
