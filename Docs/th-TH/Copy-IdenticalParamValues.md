# Copy-IdenticalParamValues

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คัดลอกค่าพารามิเตอร์จากพารามิเตอร์ที่ถูกผูกไว้ไปยัง hashtable ใหม่ตามพารามิเตอร์ที่เป็นไปได้ของฟังก์ชันอื่น

## Description

* cmdlet นี้สร้าง hashtable ใหม่ที่มีเฉพาะค่าพารามิเตอร์
  ที่ตรงกับพารามิเตอร์ที่กำหนดในฟังก์ชันเป้าหมายที่ระบุ
* hashtable ที่ได้สามารถใช้เรียกฟังก์ชันโดยใช้
  PowerShell splatting
* พารามิเตอร์แบบสวิตช์จะรวมอยู่ในผลลัพธ์ก็ต่อเมื่อถูกระบุอย่างชัดเจน
  และตั้งค่าเป็น $true ในพารามิเตอร์ที่ถูกผูกไว้
* พารามิเตอร์แบบสวิตช์ที่ไม่มีอยู่จะถูกแยกออกจากผลลัพธ์เพื่อรักษา
  ความหมายของพารามิเตอร์ที่เหมาะสม

## Syntax

```powershell
Copy-IdenticalParamValues -BoundParameters <Object[]> -FunctionName <String> [[-DefaultValues] <Object>] [<CommonParameters>]
```

| `-FunctionName` | String | ✅ | — | 1 | — | ชื่อฟังก์ชันเป้าหมายเพื่อกรองพารามิเตอร์ |
| `-DefaultValues` | Object | — | — | 2 | — | ค่าเริ่มต้นสำหรับพารามิเตอร์ |

## Examples

### Example 1

```powershell
function Test-Function {
    [CmdletBinding()]
    param(
        [Parameter(Mandatory = $true)]
        [string] $Path,
        [Parameter(Mandatory = $false)]
        [switch] $Recurse
    )

    $params = GenXdev\Copy-IdenticalParamValues `
        -BoundParameters $PSBoundParameters `
        -FunctionName 'Get-ChildItem'

    Get-ChildItem @params
}
```

สาธิตการคัดลอกพารามิเตอร์ที่ถูกผูกไว้เพื่อใช้กับ splatting
