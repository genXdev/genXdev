# CurrentUserHasElevatedRights

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> ตรวจสอบว่าผู้ใช้ปัจจุบันมีสิทธิ์ระดับสูงหรือไม่

## Description

กำหนดว่าผู้ใช้ Windows ปัจจุบันมีสิทธิ์ผู้ดูแลระบบหรือผู้ดำเนินการสำรองข้อมูลหรือไม่ โดยการตรวจสอบบทบาทหลักด้านความปลอดภัย ส่งคืน true หากผู้ใช้มีสิทธิ์ระดับสูง มิฉะนั้นส่งคืน false มีการจัดการข้อผิดพลาดที่แข็งแกร่งสำหรับปัญหาที่เกี่ยวข้องกับความปลอดภัยและการเข้าถึง

## Syntax

```powershell
CurrentUserHasElevatedRights [<CommonParameters>]
```

## Examples

### $hasRights = CurrentUserHasElevatedRights ##############################################################################

```powershell
$hasRights = CurrentUserHasElevatedRights
##############################################################################
```

## Outputs

- `Boolean
Returns true if user has admin or backup operator rights, false otherwise.`
- `Boolean`
