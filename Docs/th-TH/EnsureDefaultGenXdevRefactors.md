# EnsureDefaultGenXdevRefactors

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> รับประกันว่าคำจำกัดความการปรับโครงสร้าง GenXdev เริ่มต้นพร้อมใช้งาน

## Description

ฟังก์ชันนี้สร้างและบำรุงรักษาคำจำกัดความการปรับโครงสร้างเริ่มต้นสำหรับโมดูล PowerShell ของ GenXdev โดยตั้งค่าการปรับโครงสร้างสำหรับเอกสารและการจัดรูปแบบ การแปลงเป็น C# และพร้อมท์ขอความยินยอมในการติดตั้ง

## Syntax

```powershell
EnsureDefaultGenXdevRefactors [-Force] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Force` | SwitchParameter | — | — | Named | — | บังคับให้สร้างนิยามการปรับโครงสร้างที่มีอยู่ใหม่ |

## Examples

### EnsureDefaultGenXdevRefactors

```powershell
EnsureDefaultGenXdevRefactors
```

### EnsureDefaultGenXdevRefactors -Force

```powershell
EnsureDefaultGenXdevRefactors -Force
```
