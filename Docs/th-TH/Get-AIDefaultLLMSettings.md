# Get-AIDefaultLLMSettings

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> รับการกำหนดค่าเริ่มต้น LLM ทั้งหมดที่มีสำหรับการดำเนินการ AI ใน
GenXdev.AI

## Description

ดึงข้อมูลชุดการตั้งค่าเริ่มต้นของโมเดลภาษาขนาดใหญ่ (LLM) ทั้งหมดที่กำหนดค่าไว้สำหรับการดำเนินการ AI ผลลัพธ์สามารถกรองตามประเภทคำถาม ตัวระบุโมเดล จุดสิ้นสุด API หรือคีย์ API รองรับการดึงข้อมูลทั้งแบบตามเซสชันและตามการตั้งค่าถาวร โดยมีตัวเลือกในการล้างสถานะเซสชันหรือข้ามไปทั้งหมดเพื่ออ่านจากค่าที่เก็บไว้โดยตรง

## Syntax

```powershell
Get-AIDefaultLLMSettings [[-LLMQueryType] <String>] [-ApiEndpoint <String>] [-ApiKey <String>] [-ClearSession] [-Model <String>] [-NoSupportForJsonSchema <String>] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

### Get-AIDefaultLLMSettings -LLMQueryType "Coding"

```powershell
Get-AIDefaultLLMSettings -LLMQueryType "Coding"
```
