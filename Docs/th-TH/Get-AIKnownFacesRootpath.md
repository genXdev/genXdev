# Get-AIKnownFacesRootpath

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> รับไดเรกทอรีที่กำหนดค่าสำหรับไฟล์ภาพใบหน้าที่ใช้ในการดำเนินการของ GenXdev.AI

## Description

ฟังก์ชันนี้ดึงข้อมูลไดเรกทอรีใบหน้าสากลที่ใช้โดยโมดูล GenXdev.AI สำหรับการรู้จำใบหน้าและการดำเนินการ AI ต่างๆ โดยจะตรวจสอบตัวแปร Global ก่อน (เว้นแต่จะระบุ SkipSession) จากนั้นจะกลับไปใช้ค่ากำหนดถาวร และสุดท้ายใช้ค่าเริ่มต้นของระบบ

## Syntax

```powershell
Get-AIKnownFacesRootpath [[-FacesDirectory] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FacesDirectory` | String | — | — | 0 | — | เส้นทางไดเรกทอรีสำหรับไฟล์ภาพใบหน้า |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | ใช้การตั้งค่าสำรองที่เก็บไว้ในเซสชันสำหรับการกำหนดค่า AI เช่น ภาษา ชุดรูปภาพ ฯลฯ |
| `-ClearSession` | SwitchParameter | — | — | Named | — | ล้างการตั้งค่าเซสชัน (ตัวแปรส่วนกลาง) ก่อนที่จะเรียกค้น |
| `-PreferencesDatabasePath` | String | — | — | Named | — | เส้นทางฐานข้อมูลสำหรับไฟล์ข้อมูลการตั้งค่า |
| `-SkipSession` | SwitchParameter | — | — | Named | — | อย่าใช้การตั้งค่าสำรองที่เก็บในเซสชันสำหรับค่ากำหนดของ AI เช่น ภาษา, ชุดรูปภาพ ฯลฯ |

## Examples

### Get-AIKnownFacesRootpath

```powershell
Get-AIKnownFacesRootpath
```

รับไดเรกทอรีใบหน้าที่กำหนดค่าปัจจุบันจากตัวแปรสากลหรือค่ากำหนด

### Get-AIKnownFacesRootpath -SkipSession

```powershell
Get-AIKnownFacesRootpath -SkipSession
```

รับเฉพาะไดเรกทอรีใบหน้าที่กำหนดค่าไว้จากการตั้งค่าถาวร โดยไม่สนใจการตั้งค่าเซสชันใด ๆ

### Get-AIKnownFacesRootpath -ClearSession

```powershell
Get-AIKnownFacesRootpath -ClearSession
```

ล้างการตั้งค่าไดเรกทอรีใบหน้าเซสชันและดึงไดเรกทอรีจากการตั้งค่าถาวร

### Get-AIKnownFacesRootpath "C:\MyFaces"

```powershell
Get-AIKnownFacesRootpath "C:\MyFaces"
```

ส่งคืนไดเรกทอรีที่ระบุหลังจากขยายพาธ
