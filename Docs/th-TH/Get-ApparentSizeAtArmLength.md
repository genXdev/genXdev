# Get-ApparentSizeAtArmLength

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คำนวณขนาดที่ปรากฏของวัตถุที่ระยะแขนเหยียดออก

## Description

คำนวณขนาดที่ปรากฏโดยใช้การประมาณมุมเล็ก

## Syntax

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters <Double> -SizeInMeters <Double> [[-ArmLengthInMeters] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | ระยะทางถึงวัตถุในหน่วยเมตร |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | ขนาดจริงของวัตถุในหน่วยเมตร |
| `-ArmLengthInMeters` | Double | — | — | 2 | `0` | ความยาวแขนเป็นเมตร (ค่าเริ่มต้น: 0.7) |
| `-As` | String | — | — | 3 | — | หน่วยสำหรับขนาดเอาต์พุต |

## Examples

### Example 1

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters 10 -SizeInMeters 1 -As "centimeters"
```

คำนวณขนาดที่ปรากฏของวัตถุขนาด 1 เมตร ที่ระยะ 10 เมตร

### Example 2

```powershell
Get-ApparentSizeAtArmLength 10 1
```

คำนวณขนาดที่ปรากฏโดยใช้พารามิเตอร์ตำแหน่ง
