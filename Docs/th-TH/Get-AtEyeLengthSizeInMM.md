# Get-AtEyeLengthSizeInMM

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คำนวณขนาดปรากฏในหน่วยมิลลิเมตรของวัตถุที่ระยะแขนยื่น

## Description

คำนวณขนาดที่วัตถุจะปรากฏเมื่อมองจากระยะแขนของผู้ใหญ่ที่ยื่นออกไป (ประมาณ 0.7 เมตร) โดยกำหนดขนาดจริงและระยะห่างของวัตถุ การคำนวณใช้การประมาณมุมเล็กสำหรับขนาดเชิงมุม

## Syntax

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters <Double> -SizeInMeters <Double> [-EyeToArmLengthInMeters <Double>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | ระยะทางไปยังวัตถุเป็นเมตร. |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | ขนาดจริงของวัตถุเป็นเมตร |
| `-EyeToArmLengthInMeters` | Double | — | — | Named | `0` | ระยะห่างของความยาวแขนเป็นเมตร ค่าเริ่มต้นคือ 0.7 |

## Examples

### Example 1

```powershell
Get-AtEyeLengthSizeInMM -DistanceInMeters 10 -SizeInMeters 1
```

คำนวณขนาดที่ปรากฏเมื่อมองจากระยะแขนสำหรับวัตถุที่มีขนาด 1 เมตรที่ระยะ 10 เมตร
