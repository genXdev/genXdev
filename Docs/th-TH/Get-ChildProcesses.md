# Get-ChildProcesses

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> ดึงข้อมูลกระบวนการทั้งหมดที่เป็นกระบวนการสืบทอดของกระบวนการ PowerShell ปัจจุบัน

## Description

* ตรวจสอบกระบวนการที่กำลังทำงานอยู่ทั้งหมด และระบุกระบวนการที่มี
  กระบวนการ PowerShell ปัจจุบันเป็นบรรพบุรุษในสายกระบวนการหลัก
* ซึ่งรวมถึงทั้งกระบวนการลูกโดยตรงและลูกหลานของพวกมัน
  (หลาน ฯลฯ)

## Syntax

```powershell
Get-ChildProcesses [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ChildProcesses
```

รับข้อมูลกระบวนการย่อยทั้งหมดของเซสชัน PowerShell ปัจจุบัน

### Example 2

```powershell
Get-ChildProcesses -Verbose
```

รับกระบวนการย่อยพร้อมผลลัพธ์โดยละเอียดที่แสดงข้อมูลการตรวจสอบกระบวนการแบบละเอียด
