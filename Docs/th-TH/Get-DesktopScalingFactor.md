# Get-DesktopScalingFactor

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> ดึงค่าสเกลการแสดงผลของ Windows (การตั้งค่า DPI) สำหรับจอภาพที่ระบุ

## Description

* รับค่าปัจจัยการปรับขนาดหน้าจอ Windows ที่กำหนดค่าสำหรับจอภาพในระบบ
* ปัจจัยการปรับขนาดแสดงเป็นเปอร์เซ็นต์ โดยที่ 100 แทนการปรับขนาดมาตรฐาน (96 DPI) ค่าทั่วไปคือ 100, 125, 150 และ 200
* หากไม่ได้ระบุจอภาพ จะคืนค่าปัจจัยการปรับขนาดสำหรับจอภาพหลัก

## Syntax

```powershell
Get-DesktopScalingFactor [[-Monitor] <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | — | — | 0 | `0` | ดัชนีจอภาพที่ต้องการตรวจสอบ (0 = จอภาพหลัก) |

## Examples

### Example 1

```powershell
Get-DesktopScalingFactor -Monitor 0
```

ส่งคืนเปอร์เซ็นต์ปัจจัยการปรับขนาดสำหรับจอภาพหลัก

### Example 2

```powershell
Get-DesktopScalingFactor 1
```

ส่งคืนเปอร์เซ็นต์ปัจจัยการปรับขนาดสำหรับจอภาพที่สองโดยใช้พารามิเตอร์ตำแหน่ง

## Parameter Details

### `-Monitor <Int32>`

> ดัชนีจอภาพที่ต้องการตรวจสอบ (0 = จอภาพหลัก)

| Property | Value |
|:---|:---|
| **Required?** | No |
| **Position?** | 0 |
| **Default value** | `0` |
| **Accept pipeline input?** | False |
| **Aliases** | `m`, `mon` |
| **Accept wildcard characters?** | No |
