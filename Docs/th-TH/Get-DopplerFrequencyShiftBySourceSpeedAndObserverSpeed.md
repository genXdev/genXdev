# Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คำนวณความถี่ที่ถูกเลื่อนเนื่องจากดอปเปลอร์

## Description

ใช้ f' = f * (v + vo) / (v - vs) ความเร็วเป็นบวกเมื่อเคลื่อนที่เข้าหากัน

## Syntax

```powershell
Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed -OriginalFrequencyInHertz <Double> -SourceSpeedInMetersPerSecond <Double> -ObserverSpeedInMetersPerSecond <Double> [[-As] <String>] [<CommonParameters>]

Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed [[-SpeedOfSoundInMetersPerSecond] <Double>] [<CommonParameters>]

Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-OriginalFrequencyInHertz` | Double | ✅ | — | 0 | `0` | ความถี่เดิมในหน่วยเฮิรตซ์ |
| `-SourceSpeedInMetersPerSecond` | Double | ✅ | — | 1 | `0` | ความเร็วต้นทางในหน่วย m/s (ค่าบวกเมื่อเข้าหาผู้สังเกต) |
| `-ObserverSpeedInMetersPerSecond` | Double | ✅ | — | 2 | `0` | ความเร็วของผู้สังเกตในหน่วย m/s (บวกเมื่อเข้าหาแหล่งกำเนิด) |
| `-SpeedOfSoundInMetersPerSecond` | Double | — | — | 3 | `0` | ความเร็วของเสียงในหน่วย m/s (ค่าเริ่มต้น: 343) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 3 | — | ตัวกลาง *(Parameter set: )* |
| `-As` | String | — | — | 4 | — | หน่วยเอาต์พุตสำหรับความถี่ |

## Examples

### Example 1

```powershell
Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed -OriginalFrequencyInHertz 440 -SourceSpeedInMetersPerSecond 10 -ObserverSpeedInMetersPerSecond 5 -Medium "water" -As "kilohertz"
```

คำนวณความถี่ที่เปลี่ยนเนื่องจากดอปเปลอร์สำหรับโทนเสียง 440 Hz เมื่อแหล่งกำเนิดเคลื่อนที่เข้าหาผู้สังเกตด้วยความเร็ว 10 m/s ผู้สังเกตเคลื่อนที่เข้าหาแหล่งกำเนิดด้วยความเร็ว 5 m/s ในตัวกลางที่เป็นน้ำ ผลลัพธ์เป็นกิโลเฮิรตซ์

### Example 2

```powershell
Get-DopplerFrequencyShiftBySourceSpeedAndObserverSpeed 440 10 0 -SpeedOfSoundInMetersPerSecond 1480
```

คำนวณความถี่ที่เลื่อนเนื่องจากดอปเปลอร์สำหรับคลื่นเสียง 440 Hz โดยแหล่งกำเนิดเคลื่อนที่ด้วยความเร็ว 10 m/s ผู้สังเกตอยู่นิ่ง ความเร็วเสียง 1480 m/s
