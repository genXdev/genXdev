# Get-DragForceByVelocityDensityAreaAndCoefficient

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คำนวณแรงต้าน

## Description

ใช้ F = 1/2 C ρ A v²

## Syntax

```powershell
Get-DragForceByVelocityDensityAreaAndCoefficient -VelocityInMetersPerSecond <Double> -DensityInKilogramsPerCubicMeter <Double> -AreaInSquareMeters <Double> -Coefficient <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-VelocityInMetersPerSecond` | Double | ✅ | — | 0 | `0` | ความเร็วเป็นเมตรต่อวินาที |
| `-DensityInKilogramsPerCubicMeter` | Double | ✅ | — | 1 | `0` | ความหนาแน่นของของไหลในหน่วย kg/m³ |
| `-AreaInSquareMeters` | Double | ✅ | — | 2 | `0` | พื้นที่หน้าตัดในหน่วยตารางเมตร |
| `-Coefficient` | Double | ✅ | — | 3 | `0` | ค่าสัมประสิทธิ์แรงต้าน |
| `-As` | String | — | — | 4 | — | หน่วยผลลัพธ์สำหรับแรง |

## Examples

### Example 1

```powershell
Get-DragForceByVelocityDensityAreaAndCoefficient -VelocityInMetersPerSecond 10 -DensityInKilogramsPerCubicMeter 1.225 -AreaInSquareMeters 1 -Coefficient 0.5 -As "poundforce"
```

คำนวณแรงต้านอากาศโดยใช้ความเร็ว 10 เมตร/วินาที ความหนาแน่นอากาศ 1.225 กิโลกรัม/ลูกบาศก์เมตร พื้นที่ 1 ตารางเมตร และค่าสัมประสิทธิ์ 0.5 ผลลัพธ์เป็นหน่วยปอนด์แรง

### Example 2

```powershell
Get-DragForceByVelocityDensityAreaAndCoefficient 20 1.225 2 0.3
```

คำนวณแรงต้านโดยใช้พารามิเตอร์ตำแหน่ง: ความเร็ว 20 m/s, ความหนาแน่น 1.225 kg/m³, พื้นที่ 2 m², สัมประสิทธิ์ 0.3
