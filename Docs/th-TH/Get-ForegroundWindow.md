# Get-ForegroundWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> รับค่า handle ของหน้าต่างที่กำลังทำงานอยู่ด้านหน้าในขณะนั้น

## Description

* cmdlet นี้เรียกค้นหาตัวระบุหน้าต่าง (HWND) ของหน้าต่างที่
  กำลังอยู่ในเบื้องหน้าและมีการโฟกัสแป้นพิมพ์
* มันใช้ฟังก์ชัน Windows API ผ่าน P/Invoke เพื่อทำงานร่วมกับ
  ไลบรารี User32.dll

## Syntax

```powershell
Get-ForegroundWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ForegroundWindow
```

ดึงและส่งกลับค่า IntPtr handle ของหน้าต่างที่กำลังทำงานอยู่

### Example 2

```powershell
$windowHandle = Get-ForegroundWindow
Write-Host "Active window handle: $windowHandle"
```

จัดเก็บแฮนเดิลของหน้าต่างโฟกัสในตัวแปรและแสดงผล
