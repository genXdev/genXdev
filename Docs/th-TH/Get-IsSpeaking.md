# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> ส่งคืนค่าเป็นจริงหากเอ็นจินข้อความเป็นเสียงกำลังพูดอยู่

## Description

ตรวจสอบสถานะของทั้งซินธิไซเซอร์เสียงพูดเริ่มต้นและที่กำหนดเองเพื่อระบุว่ากำลังพูดอยู่หรือไม่ ส่งคืนค่า true หากกำลังพูดอยู่ และ false หากไม่

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

ส่งคืนค่าเป็นจริงหากเอ็นจินข้อความเป็นเสียงกำลังพูดอยู่

### Example 2

```powershell
iss
```

คืนค่า true หากเอ็นจินข้อความเป็นคำพูดกำลังพูด (โดยใช้นามแฝง)
