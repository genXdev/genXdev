# Get-KineticEnergyByMassAndVelocity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คำนวณพลังงานจลน์

## Description

ใช้ KE = 1/2 mv²

## Syntax

```powershell
Get-KineticEnergyByMassAndVelocity -MassInKilograms <Double> -VelocityInMetersPerSecond <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | มวลเป็นกิโลกรัม |
| `-VelocityInMetersPerSecond` | Double | ✅ | — | 1 | `0` | ความเร็วเป็นเมตรต่อวินาที |
| `-As` | String | — | — | 2 | — | หน่วยส่งออกสำหรับพลังงาน |

## Examples

### Example 1

```powershell
Get-KineticEnergyByMassAndVelocity -MassInKilograms 10 -VelocityInMetersPerSecond 5 -As "calories"
```

คำนวณพลังงานจลน์สำหรับมวล 10 กิโลกรัมเคลื่อนที่ด้วยความเร็ว 5 เมตร/วินาที และแสดงผลเป็นแคลอรี่

### Example 2

```powershell
Get-KineticEnergyByMassAndVelocity 5 10
```

คำนวณพลังงานจลน์สำหรับมวล 5 กก. ที่เคลื่อนที่ด้วยความเร็ว 10 ม./วินาที โดยใช้พารามิเตอร์ตามตำแหน่ง
