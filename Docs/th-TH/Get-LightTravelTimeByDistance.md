# Get-LightTravelTimeByDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คำนวณเวลาที่แสงใช้เดินทางตามระยะทาง

## Description

ใช้ t = d / c โดย c ขึ้นอยู่กับตัวกลาง

## Syntax

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters <Double> [[-As] <String>] [<CommonParameters>]

Get-LightTravelTimeByDistance [[-SpeedOfLightInMetersPerSecond] <Double>] [<CommonParameters>]

Get-LightTravelTimeByDistance -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | ระยะทางในหน่วยเมตร |
| `-SpeedOfLightInMetersPerSecond` | Double | — | — | 1 | `0` | ความเร็วแสงในหน่วยเมตรต่อวินาที (ค่าเริ่มต้น: 299792458) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | ตัวกลางที่แสงเดินทางผ่าน *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | หน่วยเอาต์พุตสำหรับเวลา |

## Examples

### Example 1

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters 149597870700 -Medium "water" -As "minutes"
```

คำนวณเวลาเดินทางของแสงถึงดวงอาทิตย์ผ่านน้ำ โดยมีหน่วยเป็นนาที

### Example 2

```powershell
Get-LightTravelTimeByDistance 300000000 -SpeedOfLightInMetersPerSecond 225000000
```

คำนวณเวลาที่แสงใช้เดินทาง 300000000 เมตร ด้วยความเร็ว 225000000 เมตร/วินาที
