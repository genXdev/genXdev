# Get-MagnificationByObjectDistanceAndImageDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คำนวณกำลังขยายสำหรับเลนส์

## Description

ใช้ m = - (ระยะภาพ / ระยะวัตถุ)

## Syntax

```powershell
Get-MagnificationByObjectDistanceAndImageDistance -ObjectDistanceInMeters <Double> -ImageDistanceInMeters <Double> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ObjectDistanceInMeters` | Double | ✅ | — | 0 | `0` | ระยะห่างของวัตถุในหน่วยเมตร |
| `-ImageDistanceInMeters` | Double | ✅ | — | 1 | `0` | ระยะห่างของภาพในหน่วยเมตร |

## Examples

### Example 1

```powershell
Get-MagnificationByObjectDistanceAndImageDistance -ObjectDistanceInMeters 0.5 -ImageDistanceInMeters 1
```

คำนวณกำลังขยายด้วยระยะวัตถุ 0.5 ม. และระยะภาพ 1 ม.

### Example 2

```powershell
Get-MagnificationByObjectDistanceAndImageDistance 0.3 0.6
```

คำนวณกำลังขยายโดยใช้พารามิเตอร์ตำแหน่ง
