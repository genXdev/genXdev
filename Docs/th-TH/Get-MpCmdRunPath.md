# Get-MpCmdRunPath

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> รับเส้นทางไปยังไฟล์ปฏิบัติการ MpCmdRun.exe ของ Windows Defender

## Description

* ค้นหาและส่งคืนพาธแบบเต็มไปยังยูทิลิตี้บรรทัดคำสั่งของ Windows Defender (MpCmdRun.exe)
* cmdlet จะตรวจสอบตำแหน่งการติดตั้งมาตรฐานใน Program Files และมีการจัดการข้อผิดพลาดอย่างเหมาะสมหากไม่พบไฟล์

## Syntax

```powershell
Get-MpCmdRunPath [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$defenderPath = Get-MpCmdRunPath
```

ดึงเส้นทางแบบเต็มไปยังเครื่องมือบรรทัดคำสั่งของ Windows Defender
