# Get-PlaywrightSessionReference

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> รับการอ้างอิงถึงเซสชันเบราว์เซอร์ Playwright ปัจจุบัน

## Description

ส่งคืน hashtable ที่ประกอบด้วย Playwright Page, BrowserContext และข้อมูล session สำหรับหน้าเบราว์เซอร์ปัจจุบัน การอ้างอิงนี้สามารถใช้กับ Invoke-WebbrowserEvaluation -ByReference เพื่อกำหนดเป้าหมายหน้าเฉพาะ

สามารถระบุประเภทเบราว์เซอร์ได้โดยใช้สวิตช์ -Edge, -Chrome, -Chromium, -Firefox หรือ -Webkit หากไม่ระบุสวิตช์ จะเลือกใช้ Chromium เป็นค่าเริ่มต้น

## Syntax

```powershell
Get-PlaywrightSessionReference [-Chrome] [-Chromium] [-Edge] [-Firefox] [-Webkit] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Edge` | SwitchParameter | — | — | Named | — | ต้องการให้ใช้เซสชันเบราว์เซอร์ Microsoft Edge |
| `-Chrome` | SwitchParameter | — | — | Named | — | ต้องการใช้เซสชันเบราว์เซอร์ Google Chrome |
| `-Chromium` | SwitchParameter | — | — | Named | — | ควรใช้ Microsoft Edge หรือ Google Chrome ขึ้นอยู่กับว่าเบราว์เซอร์เริ่มต้นคืออะไร |
| `-Firefox` | SwitchParameter | — | — | Named | — | ต้องการใช้งานเซสชันเบราว์เซอร์ Firefox |
| `-Webkit` | SwitchParameter | — | — | Named | — | ต้องการใช้เซสชันเบราว์เซอร์ WebKit |

## Examples

### Get a reference to the current browser session $sessionRef = Get-PlaywrightSessionReference

```powershell
Get a reference to the current browser session
$sessionRef = Get-PlaywrightSessionReference
```

### Store the reference and use it later to execute JavaScript $ref = Get-PlaywrightSessionReference Invoke-WebbrowserEvaluation "document.title" -ByReference $ref

```powershell
Store the reference and use it later to execute JavaScript
$ref = Get-PlaywrightSessionReference
Invoke-WebbrowserEvaluation "document.title" -ByReference $ref
```
