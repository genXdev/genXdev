# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> ส่งคืนออบเจ็กต์ตัวช่วยหน้าต่างสำหรับหน้าต่างหลักของเทอร์มินัล PowerShell

## Description

* ดึงอ็อบเจ็กต์ตัวช่วย WindowObj ที่แสดงถึงหน้าต่างหลักของ
  กระบวนการโฮสต์ PowerShell ปัจจุบัน
* ซึ่งช่วยให้สามารถจัดการและโต้ตอบกับหน้าต่างเทอร์มินัลได้เอง

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

ดึงข้อมูล WindowObj สำหรับหน้าต่าง PowerShell เทอร์มินัลปัจจุบัน
