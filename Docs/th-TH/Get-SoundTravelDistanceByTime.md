# Get-SoundTravelDistanceByTime

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คำนวณระยะทางที่เสียงเดินทางในเวลาที่กำหนด

## Description

ใช้สูตร ระยะทาง = ความเร็ว * เวลา โดยมีความเร็วเริ่มต้นของเสียงในอากาศ

## Syntax

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds <Double> [[-As] <String>] [<CommonParameters>]

Get-SoundTravelDistanceByTime [[-SpeedOfSoundInMetersPerSecond] <Double>] [<CommonParameters>]

Get-SoundTravelDistanceByTime -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-TimeInSeconds` | Double | ✅ | — | 0 | `0` | เวลาในหน่วยวินาที |
| `-SpeedOfSoundInMetersPerSecond` | Double | — | — | 1 | `0` | ความเร็วของเสียงในหน่วย m/s (ค่าเริ่มต้น: 343) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | สื่อกลางที่เสียงเดินทางผ่าน *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | หน่วยสำหรับระยะทางที่ส่งออก |

## Examples

### Example 1

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds 5 -Medium "water" -As "kilometers"
```

คำนวณระยะทางที่เสียงเดินทางในน้ำภายในเวลา 5 วินาที และแปลงผลลัพธ์เป็นกิโลเมตร

### Example 2

```powershell
Get-SoundTravelDistanceByTime 10 -SpeedOfSoundInMetersPerSecond 1480
```

คำนวณระยะทางโดยใช้ความเร็วเสียงที่กำหนด
