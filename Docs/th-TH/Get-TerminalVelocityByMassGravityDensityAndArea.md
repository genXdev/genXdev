# Get-TerminalVelocityByMassGravityDensityAndArea

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คำนวณความเร็วปลาย

## Description

ใช้ v = sqrt(2 m g / (ρ A C)).

## Syntax

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms <Double> [[-GravityInMetersPerSecondSquared] <Double>] -DensityInKilogramsPerCubicMeter <Double> -AreaInSquareMeters <Double> [[-DragCoefficient] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | มวลเป็นกิโลกรัม |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 1 | `0` | แรงโน้มถ่วงในหน่วย m/s² (ค่าเริ่มต้น: 9.81) |
| `-DensityInKilogramsPerCubicMeter` | Double | ✅ | — | 2 | `0` | ความหนาแน่นของของไหลในหน่วย kg/m³ |
| `-AreaInSquareMeters` | Double | ✅ | — | 3 | `0` | พื้นที่หน้าตัดในหน่วยตารางเมตร |
| `-DragCoefficient` | Double | — | — | 4 | `0` | ค่าสัมประสิทธิ์แรงต้าน (ค่าเริ่มต้น: 0.5) |
| `-As` | String | — | — | 5 | — | หน่วยผลลัพธ์สำหรับความเร็ว |

## Examples

### Example 1

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms 80 -DensityInKilogramsPerCubicMeter 1.225 -AreaInSquareMeters 0.7 -DragCoefficient 1.0 -As "mph"
```

คำนวณความเร็วปลายสำหรับมวล 80 กก. ในอากาศที่มีค่าสัมประสิทธิ์แรงต้านเท่ากับ 1.0 โดยแสดงผลลัพธ์เป็นไมล์ต่อชั่วโมง

### Example 2

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea 70 1.225 0.8 0.8
```

ใช้พารามิเตอร์แบบตำแหน่งเพื่อคำนวณความเร็วปลายทาง
