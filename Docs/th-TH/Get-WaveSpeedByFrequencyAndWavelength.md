# Get-WaveSpeedByFrequencyAndWavelength

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> คำนวณความเร็วคลื่น

## Description

ใช้ v = f λ.

## Syntax

```powershell
Get-WaveSpeedByFrequencyAndWavelength -FrequencyInHertz <Double> -WavelengthInMeters <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FrequencyInHertz` | Double | ✅ | — | 0 | `0` | ความถี่ในหน่วย Hz |
| `-WavelengthInMeters` | Double | ✅ | — | 1 | `0` | ความยาวคลื่นในหน่วยเมตร |
| `-As` | String | — | — | 2 | — | หน่วยผลลัพธ์สำหรับความเร็ว |

## Examples

### Example 1

```powershell
Get-WaveSpeedByFrequencyAndWavelength -FrequencyInHertz 440 -WavelengthInMeters 0.78 -As "km/h"
```

คำนวณความเร็วคลื่นโดยใช้ v = f * λ

### Example 2

```powershell
Get-WaveSpeedByFrequencyAndWavelength 1000 0.34
```

คำนวณความเร็วคลื่นด้วยพารามิเตอร์ตำแหน่ง
