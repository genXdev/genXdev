# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> ส่งคืนพจนานุกรมแบบย้อนกลับสำหรับภาษาทั้งหมดที่ Google Search รองรับ

## Description

* ส่งกลับพจนานุกรมที่แมปชื่อภาษากับรหัสภาษาที่สอดคล้องกัน
  สำหรับทุกภาษาที่รองรับโดย Google Search
* มีประโยชน์สำหรับการค้นหารหัสภาษาในข้อความค้นหาทางเว็บและงานแปลภาษา

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

ดึงข้อมูลการแมปชื่อภาษากับรหัสของภาษา
