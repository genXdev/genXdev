# Initialize-SearchPaths

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> เริ่มต้นและกำหนดค่าเส้นทางการค้นหาระบบสำหรับการจัดการแพ็กเกจ

## Description

* สร้างรายการเส้นทางค้นหาที่ครอบคลุมโดยรวมตำแหน่งระบบเริ่มต้น
  เส้นทาง Chocolatey เส้นทางเครื่องมือพัฒนา และเส้นทางแพ็คเกจที่กำหนดเอง
* อัปเดตตัวแปรสภาพแวดล้อม PATH ของระบบด้วยเส้นทางที่รวมเหล่านี้

## Syntax

```powershell
Initialize-SearchPaths [[-WorkspaceFolder] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-WorkspaceFolder` | String | — | — | 0 | — | เส้นทางโฟลเดอร์พื้นที่ทำงานที่จะใช้สำหรับโมดูลโหนดและเส้นทาง PowerShell |

## Examples

### Example 1

```powershell
Initialize-SearchPaths
```

เริ่มต้นพาธการค้นหาโดยใช้โฟลเดอร์พื้นที่ทำงานเริ่มต้น

### Example 2

```powershell
Initialize-SearchPaths -WorkspaceFolder "C:\workspace"
```

เริ่มต้นเส้นทางการค้นหาโดยใช้โฟลเดอร์พื้นที่ทำงานที่ระบุ
