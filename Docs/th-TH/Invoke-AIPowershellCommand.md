# Invoke-AIPowershellCommand

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** `hint`

## Synopsis

> สร้างและดำเนินการคำสั่ง PowerShell โดยใช้ความช่วยเหลือจาก AI

## Description

ใช้โมเดล AI เพื่อสร้างคำสั่ง PowerShell ตามคำถามภาษาธรรมชาติ ฟังก์ชันนี้สามารถส่งคำสั่งไปยังหน้าต่าง PowerShell โดยตรงหรือคัดลอกไปยังคลิปบอร์ด ใช้งานโมเดล AI เพื่อตีความภาษาธรรมชาติและแปลงเป็นคำสั่ง PowerShell ที่สามารถทำงานได้ พร้อมรองรับพารามิเตอร์ที่ครอบคลุมสำหรับแบ็กเอนด์ AI ต่างๆ

## Syntax

```powershell
Invoke-AIPowershellCommand -PowershellCmdline <String> [-ApiEndpoint <String>] [-ApiKey <String>] [-LLMQueryType <String>] [-Model <String>] [-NoSupportForJsonSchema <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-PowershellCmdline` | String | ✅ | — | 0 | — | ข้อความค้นหาภาษาธรรมชาติเพื่อสร้างคำสั่งสำหรับ |
| `-LLMQueryType` | String | — | — | Named | `'Coding'` | ประเภทของการสอบถาม LLM |
| `-Model` | String | — | — | Named | — | รหัสหรือรูปแบบของโมเดลที่ใช้สำหรับการดำเนินการ AI |
| `-ApiEndpoint` | String | — | — | Named | — | URL ปลายทางของ API สำหรับการดำเนินการ AI |
| `-ApiKey` | String | — | — | Named | — | คีย์ API สำหรับการดำเนินงาน AI ที่ผ่านการรับรองความถูกต้อง |
| `-NoSupportForJsonSchema` | String | — | — | Named | — | บ่งชี้ว่า LLM ไม่รองรับ JSON schemas |

## Examples

### Invoke-AIPowershellCommand -Query "list all running processes"

```powershell
Invoke-AIPowershellCommand -Query "list all running processes"
```

สร้างคำสั่ง PowerShell เพื่อแสดงรายการกระบวนการที่กำลังทำงานอยู่

### hint "list files modified today"

```powershell
hint "list files modified today"
```

ใช้นามแฝงเพื่อสร้างคำสั่งสำหรับค้นหาไฟล์ที่ถูกแก้ไขในวันนี้

### Invoke-AIPowershellCommand -Query "stop service" -Clipboard

```powershell
Invoke-AIPowershellCommand -Query "stop service" -Clipboard
```

สร้างคำสั่งเพื่อหยุดบริการและคัดลอกไปยังคลิปบอร์ด
