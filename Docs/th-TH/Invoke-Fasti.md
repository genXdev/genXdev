# Invoke-Fasti

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** `fasti`

## Synopsis

> แยกไฟล์บีบอัดในไดเรกทอรีปัจจุบันไปยังโฟลเดอร์ของตัวเอง และลบไฟล์เหล่านั้นหลังจากนั้น

## Description

แยกไฟล์บีบอัดรูปแบบทั่วไป (zip, 7z, tar ฯลฯ) ที่พบในไดเรกทอรีปัจจุบันโดยอัตโนมัติ ไปยังโฟลเดอร์แต่ละโฟลเดอร์ที่ตั้งชื่อตามไฟล์บีบอัดแต่ละไฟล์ หลังจากแยกสำเร็จ ไฟล์บีบอัดต้นฉบับจะถูกลบ ต้องติดตั้ง 7-Zip บนระบบ

## Syntax

```powershell
Invoke-Fasti [[-Password] <String>] [-ExtractOutputToo] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Password` | String | — | — | 0 | — | ป้อนรหัสผ่านสำหรับไฟล์เก็บถาวรที่เข้ารหัส |
| `-ExtractOutputToo` | SwitchParameter | — | — | Named | — | แยกไฟล์บีบอัดที่พบในโฟลเดอร์ที่แยกออกมาแบบเรียกซ้ำ |

## Examples

### PS C:\Downloads> Invoke-Fasti

```powershell
PS C:\Downloads> Invoke-Fasti
```

### PS C:\Downloads> fasti

```powershell
PS C:\Downloads> fasti
```
