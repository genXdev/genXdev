# Open-YabAIBattle

> **Module:** GenXdev.Queries.Websites | **Type:** Function | **Aliases:** `yabbattle`

## Synopsis

> เปิดเกม YabAI Battle ในเว็บเบราว์เซอร์

## Description

ฟังก์ชันนี้จะเปิดเกม YabAI Battle ในเว็บเบราว์เซอร์พร้อมตัวเลือกการปรับแต่งที่หลากหลายสำหรับการวางตำแหน่งหน้าต่าง การเลือกเบราว์เซอร์ และโหมดการแสดงผล โดยเป็น wrapper รอบฟังก์ชัน GenXdev\Open-Webbrowser ที่มีการสร้าง URL เฉพาะของเกมและการจัดการพารามิเตอร์

ฟังก์ชันรองรับสองโหมดเกม:
- โหมด Battle: เริ่มการต่อสู้ AI กับ AI ใหม่
- โหมด Spectate: ดูการต่อสู้ AI ที่มีอยู่

ตัวเลือกการวางตำแหน่งเบราว์เซอร์รวมถึงโหมดซ้าย ขวา บน ล่าง กึ่งกลาง และเต็มหน้าจอพร้อมรองรับหลายจอภาพ ฟังก์ชันจัดการการตั้งค่าโหมดแอปพลิเคชันและการจัดการส่วนขยายเบราว์เซอร์โดยอัตโนมัติเพื่อประสบการณ์การเล่นเกมที่เหมาะสมที่สุด

## Syntax

```powershell
Open-YabAIBattle [[-Language] <String>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-Monitor <Int32>] [-NewWindow] [-NoApplicationMode] [-NoBorders] [-NoBrowserExtensions] [-NoFullScreen] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-ShowWindow] [-SideBySide] [-SkipSession] [-SpectateOnly] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-YabAIBattle -Edge -Private -NoFullScreen

```powershell
Open-YabAIBattle -Edge -Private -NoFullScreen
```

เปิดเกม YabAI Battle ใน Microsoft Edge โดยใช้โหมดการเรียกดูส่วนตัวโดยไม่ต้องเต็มหน้าจอ

### yabbattle -e -incognito -nfs

```powershell
yabbattle -e -incognito -nfs
```

เปิดเกมโดยใช้นามแฝงสำหรับ Edge, โหมดไม่ระบุตัวตน และไม่เต็มหน้าจอ

### Open-YabAIBattle -SpectateOnly -Chrome

```powershell
Open-YabAIBattle -SpectateOnly -Chrome
```

เปิดเกมในโหมดดูการแข่งขันโดยใช้ Google Chrome เพื่อชมการต่อสู้ของ AI

### Open-YabAIBattle -Language "Spanish" -Monitor 1 -Left

```powershell
Open-YabAIBattle -Language "Spanish" -Monitor 1 -Left
```

เปิดเกมด้วยอินเทอร์เฟซภาษาสเปนที่อยู่ทางด้านซ้ายของจอภาพที่ 1
