# Publish-ReleaseNotesToManifest

> **Module:** GenXdev.Coding | **Type:** Function | **Aliases:** `pubrelnotes`

## Synopsis

> เผยแพร่บันทึกการเปลี่ยนแปลงที่ยังไม่เสร็จสมบูรณ์จาก README.md ไปยังแถบรายการโมดูล

## Description

อ่านส่วน "## Release notes" จากไฟล์ README.md รวบรวมบันทึกการเผยแพร่ที่ยังไม่เสร็จสมบูรณ์ทั้งหมด (บรรทัดที่ขึ้นต้นด้วย - ☐) เขียนลงในฟิลด์ ReleaseNotes ของ GenXdev.psd1 และทำเครื่องหมายว่าสมบูรณ์ (☐ → ☒) ใน README อัปเดตทั้งสองไฟล์แบบอะตอมมิกในการดำเนินการเดียว

## Syntax

```powershell
Publish-ReleaseNotesToManifest [[-ManifestPath] <String>] [-UseHomeREADME] [-UseOneDriveREADME] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-UseHomeREADME` | SwitchParameter | — | — | Named | — | ใช้ README ในไดเรกทอรีโปรไฟล์ของ PowerShell |
| `-UseOneDriveREADME` | SwitchParameter | — | — | Named | — | ใช้ README ในไดเรกทอรี OneDrive |
| `-ManifestPath` | String | — | — | 0 | — | เส้นทางไปยังไฟล์โมดูล manifest .psd1 |

## Examples

### Publish-ReleaseNotesToManifest

```powershell
Publish-ReleaseNotesToManifest
```

### Publish-ReleaseNotesToManifest -ManifestPath ".\MyModule.psd1"

```powershell
Publish-ReleaseNotesToManifest -ManifestPath ".\MyModule.psd1"
```
