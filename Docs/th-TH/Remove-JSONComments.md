# Remove-JSONComments

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> ลบความคิดเห็นออกจากเนื้อหา JSON

## Description

* ประมวลผลเนื้อหา JSON และลบความคิดเห็นทั้งแบบบรรทัดเดียวและหลายบรรทัด
  พร้อมคงโครงสร้าง JSON ไว้
* มีประโยชน์สำหรับการทำความสะอาดไฟล์ JSON ที่มีความคิดเห็นเอกสารประกอบ
  ก่อนทำการแยกวิเคราะห์
* รองรับทั้งความคิดเห็นแบบบรรทัดเดียว (//) และความคิดเห็นแบบหลายบรรทัด (/* */)

## Syntax

```powershell
Remove-JSONComments -Json <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Json` | String | ✅ | — | 0 | — | ด้านล่างนี้คือคำอธิบายของพารามิเตอร์ใน cmdlet ของ PowerShell

-ActionType

ระบุประเภทการดำเนินการของกิจกรรม กิจกรรมประเภทนี้จะเป็นตัวกำหนดตรรกะเริ่มต้น ค่าเริ่มต้นคือ Notify |

## Examples

### Example 1

```powershell
$jsonContent = @'
{
    // This is a comment
    "name": "test", /* inline comment */
    "value": 123
}
'@ -split "`n"
Remove-JSONComments -Json $jsonContent
```

ลบความคิดเห็นออกจากเนื้อหา JSON ที่เก็บไว้ในตัวแปร

### Example 2

```powershell
$jsonContent | Remove-JSONComments
```

ประมวลผลเนื้อหา JSON จากไปป์ไลน์
