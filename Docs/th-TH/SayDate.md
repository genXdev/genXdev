# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> พูดวันที่ปัจจุบันโดยใช้การสังเคราะห์ข้อความเป็นเสียง

## Description

แปลงวันที่ปัจจุบันเป็นรูปแบบภาษาธรรมชาติและประกาศโดยใช้เอนจินข้อความเป็นคำพูดของระบบ โดยวันที่ถูกพูดในรูปแบบ: "วันนี้เป็นวัน [ชื่อวัน] ที่ [เดือน] [วันที่] [ปี]"

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
