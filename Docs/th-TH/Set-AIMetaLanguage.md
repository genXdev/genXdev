# Set-AIMetaLanguage

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> ตั้งค่าภาษาเริ่มต้นและไดเรกทอรีรูปภาพ (ตามตัวเลือก) สำหรับการดำเนินการเมทาดาทาของรูปภาพของ GenXdev.AI

## Description

ฟังก์ชันนี้กำหนดค่าภาษาเริ่มต้นทั่วโลกสำหรับการดำเนินการ metadata ของรูปภาพในโมดูล GenXdev.AI โดยสามารถตั้งค่าไดเรกทอรีรูปภาพทั่วโลกได้ตามต้องการ การตั้งค่าทั้งสองจะถูกบันทึกไว้ในพื้นที่จัดเก็บค่ากำหนดของโมดูลเพื่อใช้ในเซสชันต่างๆ

## Syntax

```powershell
Set-AIMetaLanguage [[-Language] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Language` | String | — | — | 0 | — | ภาษาเริ่มต้นสำหรับการดำเนินการเมทาดาทาของรูปภาพ |
| `-PreferencesDatabasePath` | String | — | — | Named | — | เส้นทางฐานข้อมูลสำหรับไฟล์ข้อมูลการตั้งค่า |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | ใช้การตั้งค่าสำรองที่เก็บไว้ในเซสชันสำหรับการกำหนดค่า AI เช่น ภาษา ชุดรูปภาพ ฯลฯ |
| `-ClearSession` | SwitchParameter | — | — | Named | — | ล้างการตั้งค่าทางเลือกที่จัดเก็บในเซสชันสำหรับการตั้งค่า AI เช่น ภาษา คอลเลกชันรูปภาพ ฯลฯ |
| `-SkipSession` | SwitchParameter | — | — | Named | — | อย่าใช้การตั้งค่าสำรองที่เก็บในเซสชันสำหรับค่ากำหนดของ AI เช่น ภาษา, ชุดรูปภาพ ฯลฯ |

## Examples

### Set-AIMetaLanguage -Language "Spanish" -ImageDirectories @("C:\Images", "D:\Photos")

```powershell
Set-AIMetaLanguage -Language "Spanish" -ImageDirectories @("C:\Images", "D:\Photos")
```

ตั้งค่าไดเรกทอรีภาษาและภาพอย่างถาวรในค่ากำหนด

### Set-AIMetaLanguage "French"

```powershell
Set-AIMetaLanguage "French"
```

ตั้งค่าภาษาให้คงอยู่ในค่ากำหนดอย่างถาวร

### Set-AIMetaLanguage -Language "German" -SessionOnly

```powershell
Set-AIMetaLanguage -Language "German" -SessionOnly
```

ตั้งค่าภาษาสำหรับเซสชันปัจจุบันเท่านั้น (ตัวแปร Global)

### Set-AIMetaLanguage -ClearSession

```powershell
Set-AIMetaLanguage -ClearSession
```

ล้างการตั้งค่าภาษาของเซสชัน (ตัวแปรส่วนกลาง) โดยไม่ส่งผลต่อค่ากำหนดถาวร
