# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> เปลี่ยนตำแหน่งปัจจุบันไปยังไดเรกทอรีหลักและแสดงรายการเนื้อหา

## Description

* นำทางขึ้นไปหนึ่งระดับไดเรกทอรีในลำดับชั้นของระบบไฟล์
* แสดงเนื้อหาของไดเรกทอรีปัจจุบันใหม่
* มีนามแฝง '..' ที่สะดวกสำหรับการนำทางไดเรกทอรีอย่างรวดเร็ว

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

เปลี่ยนไปยังไดเรกทอรีหลักและแสดงรายการเนื้อหา

### Example 2

```powershell
..
```

Uses alias เพื่อเปลี่ยนไปยังไดเรกทอรีหลัก
