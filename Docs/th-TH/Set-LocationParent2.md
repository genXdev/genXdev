# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> นำทางขึ้นไปสองระดับไดเรกทอรีในลำดับชั้นของระบบไฟล์

## Description

* เปลี่ยนไดเรกทอรีการทำงานปัจจุบันไปยังไดเรกทอรีปู่ย่า (ขึ้นไปสองระดับ)
* แสดงเนื้อหาของไดเรกทอรีที่ได้

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

เปลี่ยนไปยังไดเรกทอรีหลักของพาเรนต์และแสดงเนื้อหาในนั้น

### Example 2

```powershell
...
```

ใช้นามแฝง '...' สำหรับการดำเนินการเดียวกัน
