# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....`

## Synopsis

> เลื่อนระดับขึ้นไปสามไดเรกทอรีในลำดับชั้นของระบบไฟล์

## Description

* เปลี่ยนไดเรกทอรีการทำงานปัจจุบันโดยเลื่อนขึ้นไปยังไดเรกทอรีหลักสามระดับในลำดับชั้นของระบบไฟล์
* แสดงเนื้อหาของไดเรกทอรีผลลัพธ์

## Syntax

```powershell
Set-LocationParent3 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

นำทางขึ้นไปสามระดับไดเรกทอรีจากตำแหน่งปัจจุบัน

### Example 2

```powershell
....
```
