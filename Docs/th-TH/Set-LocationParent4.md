# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> นำทางขึ้นไปยังสี่ระดับไดเรกทอรีในลำดับชั้นของระบบไฟล์

## Description

* เลื่อนขึ้นไปยังไดเรกทอรีระดับที่สี่จากตำแหน่งปัจจุบันในระบบไฟล์
* หลังจากนำทางแล้ว จะแสดงเนื้อหาของไดเรกทอรีที่เป็นผลลัพธ์

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

นำทางขึ้นไปยังไดเรกทอรีที่สูงขึ้นสี่ระดับจากตำแหน่งปัจจุบัน

### Example 2

```powershell
.....
```
