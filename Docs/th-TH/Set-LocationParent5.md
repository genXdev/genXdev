# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> นำทางขึ้นไปห้าระดับไดเรกทอรีในลำดับชั้นระบบไฟล์

## Description

* เปลี่ยนไดเรกทอรีการทำงานปัจจุบันโดยเลื่อนขึ้นไปยังระดับไดเรกทอรีหลักห้าระดับ
* แสดงเนื้อหาของไดเรกทอรีผลลัพธ์
* มีนามแฝงที่สะดวก '......' สำหรับการนำทางอย่างรวดเร็ว

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

นำทางขึ้นไป 5 ระดับไดเร็กทอรีจากตำแหน่งปัจจุบัน

```powershell

```
