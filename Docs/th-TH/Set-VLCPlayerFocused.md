# Set-VLCPlayerFocused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `showvlc`, `vlcf`, `fvlc`

## Synopsis

> กำหนดโฟกัสไปที่หน้าต่างโปรแกรมเล่นสื่อ VLC

## Description

ค้นหาอินสแตนซ์ที่กำลังทำงานของ VLC media player และนำหน้าต่างของมันมาอยู่เบื้องหน้า ทำให้เป็นหน้าต่างที่ใช้งานอยู่ หาก VLC ไม่ได้ทำงานอยู่ ให้ดำเนินการต่อไปอย่างเงียบ ๆ โดยไม่มีข้อผิดพลาด ใช้การเรียก API ของ Windows เพื่อจัดการโฟกัสของหน้าต่าง

## Syntax

```powershell
Set-VLCPlayerFocused [<CommonParameters>]
```

## Examples

### Set-VLCPlayerFocused Brings the VLC player window to front and gives it focus

```powershell
Set-VLCPlayerFocused
Brings the VLC player window to front and gives it focus
```

### vlcf Same operation using the short alias

```powershell
vlcf
Same operation using the short alias
```
