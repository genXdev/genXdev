# Set-WindowsWallpaper

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> กำหนดวอลเปเปอร์แบบสุ่มจากไดเรกทอรีที่ระบุ

## Description

* เลือกไฟล์ภาพแบบสุ่มจากไดเรกทอรีที่ระบุและตั้งเป็นภาพพื้นหลังเดสก์ท็อปของ Windows
* รองรับรูปแบบภาพ JPG/JPEG และกำหนดค่าภาพพื้นหลังเป็น "พอดีกับหน้าจอ" ตามค่าเริ่มต้น

## Syntax

```powershell
Set-WindowsWallpaper [[-InputObject] <Object>] [-AllDrives] [-NoRecurse] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputObject` | Object | — | ✅ (ByValue, ByPropertyName) | 0 | — | เส้นทางไปยังไดเรกทอรีที่มีภาพพื้นหลัง |
| `-AllDrives` | SwitchParameter | — | — | Named | `False` | ค้นหาในไดรฟ์ที่พร้อมใช้งานทั้งหมด |
| `-NoRecurse` | SwitchParameter | — | — | Named | `False` | อย่ามองหาในโฟลเดอร์ย่อย |

## Examples

### Example 1

```powershell
Set-WindowsWallpaper -InputObject "C:\Wallpapers\*.jpg"
```

ตั้งค่าวอลเปเปอร์แบบสุ่มจากไดเรกทอรี C:\Wallpapers

### Example 2

```powershell
nextbg
```

ตั้งวอลเปเปอร์แบบสุ่มจากไดเรกทอรีเริ่มต้นโดยใช้นามแฝง
