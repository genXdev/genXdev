# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> สลับสถานะหยุดชั่วคราว/เล่นของ VLC Media Player

## Description

ฟังก์ชันนี้ส่งปุ่ม space ไปยัง VLC Media Player เพื่อสลับระหว่างสถานะหยุดชั่วคราวและกำลังเล่น โดยจะคืนโฟกัสกลับไปยังหน้าต่างเดิมโดยอัตโนมัติหลังจากส่งคำสั่งคีย์

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

สลับสถานะหยุดชั่วคราว/เล่นของ VLC Media Player

### vlcpause

```powershell
vlcpause
```

ใช้นามแฝงเพื่อสลับสถานะหยุดชั่วคราว/เล่น

### vlcplay

```powershell
vlcplay
```

ใช้ชื่อแทนเพื่อสลับสถานะหยุดชั่วคราว/เล่น
