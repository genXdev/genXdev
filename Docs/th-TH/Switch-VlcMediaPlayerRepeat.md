# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> สลับโหมดเล่นซ้ำใน VLC Media Player

## Description

ฟังก์ชันนี้ส่งคำสั่งคีย์ 'r' ไปยัง VLC Media Player เพื่อสลับระหว่างโหมดเล่นซ้ำต่างๆ (ไม่เล่นซ้ำ, เล่นซ้ำรายการปัจจุบัน, เล่นซ้ำทั้งหมด) ฟังก์ชันจะเปิด VLC หากยังไม่ได้เปิดอยู่ และคืนโฟกัสไปยังหน้าต่างก่อนหน้าหลังจากส่งคำสั่ง

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
