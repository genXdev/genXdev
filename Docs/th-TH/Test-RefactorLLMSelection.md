# Test-RefactorLLMSelection

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> ประเมินไฟล์ต้นฉบับเพื่อตรวจสอบความเหมาะสมในการปรับโครงสร้างโดยใช้การวิเคราะห์ด้วย LLM

## Description

ใช้การวิเคราะห์ด้วยโมเดลภาษาขนาดใหญ่ (LLM) เพื่อตรวจสอบว่าไฟล์ซอร์สโค้ดควรถูกเลือกสำหรับการปรับโครงสร้างตามเกณฑ์ที่กำหนดหรือไม่ ฟังก์ชันจะประมวลผลเนื้อหาไฟล์ผ่านการสอบถาม LLM และส่งคืนผลลัพธ์แบบบูลีน

## Syntax

```powershell
Test-RefactorLLMSelection -RefactorDefinition <GenXdev.Helpers.RefactorDefinition> -Path <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-RefactorDefinition` | GenXdev.Helpers.RefactorDefinition | ✅ | — | 0 | — | นิยามการปรับโครงสร้างที่ประกอบด้วยการตั้งค่า LLM |
| `-Path` | String | ✅ | — | 1 | — | เส้นทางไปยังไฟล์ต้นฉบับที่ต้องการประเมิน |

```powershell
Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"
```
