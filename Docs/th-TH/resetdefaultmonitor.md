# resetdefaultmonitor

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> เรียกคืนการกำหนดค่าจอภาพสำรองเริ่มต้น

## Description

สคริปต์นี้จะคืนค่าการกำหนดค่าจอภาพรองเริ่มต้นของระบบให้กลับเป็นค่าเริ่มต้นเดิม ซึ่งมีประโยชน์สำหรับผู้ใช้ที่ต้องการกลับไปใช้การตั้งค่าหลายจอภาพก่อนหน้านี้หลังจากใช้การกำหนดค่าแบบเคียงข้างกัน ดูเพิ่มเติม: ฟังก์ชัน 'sidebyside' เพื่อสลับไปใช้โหมดเคียงข้างกันสำหรับหน้าต่างใหม่

## Syntax

```powershell
resetdefaultmonitor [<CommonParameters>]
```

## Examples

### secondscreen

```powershell
secondscreen
```

คืนค่าการกำหนดค่าจอภาพรองเริ่มต้นสำหรับระบบ
