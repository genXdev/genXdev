# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> ตั้งค่าการกำหนดค่าจอภาพที่สองเริ่มต้น

## Description

กำหนดพฤติกรรมเริ่มต้นสำหรับการเปิดหน้าต่าง GenXdev ให้อยู่บนจอภาพที่สอง
ซึ่งมีประโยชน์สำหรับผู้ใช้ที่มีจอภาพเดียวหรือผู้ที่ชอบการจัดวางหน้าต่างแบบเคียงข้างกัน
ดูเพิ่มเติมที่ cmdlet 'sidebyside' และ 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

ตั้งค่าเริ่มต้นสำหรับการเปิดหน้าต่าง GenXdev ให้อยู่ที่จอภาพรอง
