# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> กำหนดค่าการจัดวางแบบเคียงข้างเริ่มต้น

## Description

กำหนดพฤติกรรมเริ่มต้นสำหรับการเปิดหน้าต่าง GenXdev ให้วางเคียงข้างกับ PowerShell
มีประโยชน์สำหรับผู้ใช้ที่มีจอภาพเดียวหรือผู้ที่ชอบการจัดวางหน้าต่างแบบเคียงข้าง
ดูเพิ่มเติมที่ cmdlet 'secondscreen' และ 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

ตั้งค่าเริ่มต้นของหน้าต่าง GenXdev ให้เปิดเคียงข้างกับ PowerShell
