# Add-LineToREADME

> **Module:** GenXdev.Coding | **Type:** Function | **Aliases:** —

## Synopsis

> Belirtilen bir bölüme README.md işaretleme dosyasına bir satır ekler.

## Description

Bir README.md dosyasını bularak belirtilen bir bölüme yeni bir satır ekler.
Bölüm yoksa oluşturabilir. Satırları kod blokları olarak biçimlendirmeyi
ve değiştirilen bölümü göstermeyi destekler.

Önce geçerli dizinde arar, ardından README konumunu bulmak için üst dizinlerde
ilerler. Bulunamazsa, PowerShell profil dizinindeki README'yi kullanır.

## Syntax

```powershell
Add-LineToREADME -Section <String> [-Ascending] [-Code] [-Done] [-First <Int32>] [-Line <String>] [-Priority <Int32>] [-Show] [-SortByDate] [-UseHomeREADME] [-UseOneDriveREADME] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Line` | String | — | — | Named | `''` | README'ye eklenecek satır |
| `-Section` | String | ✅ | — | 0 | — | Satırın ekleneceği bölüm |
| `-Code` | SwitchParameter | — | — | Named | — | Değişiklik yaptıktan sonra Visual Studio Code'da aç |
| `-Show` | SwitchParameter | — | — | Named | — | Değiştirilen bölümü göster |
| `-Done` | SwitchParameter | — | — | Named | — | Öğeyi tamamlanmış olarak işaretle |
| `-UseHomeREADME` | SwitchParameter | — | — | Named | — | PowerShell profil dizinindeki README dosyasını kullan |
| `-UseOneDriveREADME` | SwitchParameter | — | — | Named | — | OneDrive dizininde README kullan |
| `-Priority` | Int32 | — | — | Named | `1` | Sıralama önceliği (yüksek = önce gösterilir, varsayılan 1) |
| `-SortByDate` | SwitchParameter | — | — | Named | — | Satırları öncelik yerine tarihe (yyyyAAgg öneki) göre sırala |
| `-Ascending` | SwitchParameter | — | — | Named | — | Sıralama düzenini tersine çevir (azalan yerine artan) |
| `-First` | Int32 | — | — | Named | `0` | Çıktıyı ilk N satırla sınırla |

## Examples

### Add-LineToREADME -Line "New feature" -Section "## Features"

```powershell
Add-LineToREADME -Line "New feature" -Section "## Features"
```

### Add-LineToREADME "High prio item" "## Issues" -Priority 5

```powershell
Add-LineToREADME "High prio item" "## Issues" -Priority 5
```
