# Assert-ModuleDefinition

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> AI yardımıyla PowerShell kaynak kod dosyalarını yeniden düzenlemeye yardımcı olur.

## Description

Bu fonksiyon, PowerShell kodunun yapay zeka kullanarak yeniden düzenlenmesi sürecini otomatikleştirir.
Prompt şablonlarını yönetir, etkin IDE'yi (VS Code veya Visual Studio) algılar
ve klavye otomasyonu aracılığıyla yeniden düzenleme iş akışını düzenler.
Fonksiyon, hem modül bildirim dosyasını (.psd1) hem de modül betik dosyasını (.psm1)
işleyebilir.

## Syntax

```powershell
Assert-ModuleDefinition -ModuleName <String> [-EditPrompt] [-Prompt <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Modülün adı |
| `-Prompt` | String | — | — | Named | — | Eklenti bu bilgisayarda bulunamadı. Lütfen geçerli bir eklenti adı girin ve tekrar deneyin. |
| `-EditPrompt` | SwitchParameter | — | — | Named | — | .PARAMETER AdSoyad
Filtrelenecek kullanıcının adını ve soyadını belirtir. |

## Examples

### Assert-ModuleDefinition -ModuleName "MyModule" -EditPrompt

```powershell
Assert-ModuleDefinition -ModuleName "MyModule" -EditPrompt
```

### "MyModule" | Assert-ModuleDefinition

```powershell
"MyModule" | Assert-ModuleDefinition
```
