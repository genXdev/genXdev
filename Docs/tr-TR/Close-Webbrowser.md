# Close-Webbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `wbc`

## Synopsis

> Bir veya daha fazla web tarayıcı örneğini seçmeli olarak kapatır.

## Description

Web tarayıcı örneklerini kapatma üzerinde ayrıntılı kontrol sağlar. Belirli tarayıcıları (Edge, Chrome, Firefox) hedefleyebilir veya tüm tarayıcıları kapatabilir. Hem ana pencereleri hem de arka plan işlemlerini kapatmayı destekler.

## Syntax

```powershell
Close-Webbrowser [-Chrome] [-Chromium] [-Edge] [-Firefox] [<CommonParameters>]

Close-Webbrowser [-All] [-Headless] [-IncludeBackgroundProcesses] [-PlayWright] [-Webkit] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Edge` | SwitchParameter | — | — | Named | — | Microsoft Edge tarayıcı örneklerini kapatır *(Parameter set: )* |
| `-Chrome` | SwitchParameter | — | — | Named | — | Google Chrome tarayıcı örneklerini kapatır *(Parameter set: )* |
| `-Chromium` | SwitchParameter | — | — | Named | — | Varsayılan Chromium tabanlı tarayıcıyı kapatır *(Parameter set: )* |
| `-Firefox` | SwitchParameter | — | — | Named | — | Firefox tarayıcı örneklerini kapatır *(Parameter set: )* |
| `-PlayWright` | SwitchParameter | — | — | Named | — | İşletim Sistemi Tarayıcısı Yerine Playwright Tarafından Yönetilen Tarayıcıyı Kullanın |
| `-Webkit` | SwitchParameter | — | — | Named | — | Playwright tarafından yönetilen WebKit tarayıcısını açar. -PlayWright anlamına gelir. |
| `-Headless` | SwitchParameter | — | — | Named | — | Tarayıcıyı görünür bir pencere olmadan çalıştırın |
| `-All` | SwitchParameter | — | — | Named | — | Tüm kayıtlı modern tarayıcılarda açar |
| `-IncludeBackgroundProcesses` | SwitchParameter | — | — | Named | — | Arka plan görevleri dahil tüm örnekleri kapatır |

## Examples

### Close-Webbrowser -Chrome -Firefox -IncludeBackgroundProcesses Closes all Chrome and Firefox instances including background processes

```powershell
Close-Webbrowser -Chrome -Firefox -IncludeBackgroundProcesses
Closes all Chrome and Firefox instances including background processes
```

### wbc -a -bg Closes all browser instances including background processes using aliases

```powershell
wbc -a -bg
Closes all browser instances including background processes using aliases
```
