# Convert-PhysicsUnit

> **Module:** GenXdev.Helpers.Physics | **Type:** Function | **Aliases:** —

## Synopsis

> Aynı kategori içindeki bir fiziksel birimi başka bir birime dönüştürür.

## Description

Bu işlev, uzunluk, zaman, hız gibi kategorilerdeki uyumlu birimler arasında fiziksel büyüklükleri dönüştürür. Kategoriyi birimlerden çıkarır ve uyumsuz dönüşümler için hata fırlatır.

## Syntax

```powershell
Convert-PhysicsUnit -Value <Double> -FromUnit <String> -ToUnit <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Value` | Double | ✅ | — | 0 | — | Dönüştürülecek sayısal değer |
| `-FromUnit` | String | ✅ | — | 1 | — | Giriş değerinin birimi |
| `-ToUnit` | String | ✅ | — | 2 | — | İstenen çıktı birimi |

## Examples

### Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"

```powershell
Convert-PhysicsUnit -Value 100 -FromUnit "meters" -ToUnit "feet"
```

100 metreyi feet'e dönüştürür.

### Convert-PhysicsUnit 10 "seconds" "minutes"

```powershell
Convert-PhysicsUnit 10 "seconds" "minutes"
```

Konumsal parametreler kullanarak 10 saniyeyi dakikaya dönüştürür.
