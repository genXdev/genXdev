# EnsureCopilotKeyboardShortCut

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Visual Studio Code'da GitHub Copilot Sohbet klavye kısayollarını yapılandırır.

## Description

Bu işlev, GitHub Copilot Chat'in dosya ekleme özelliğinin Visual Studio Code'da uygun bir klavye kısayoluna (Ctrl+Shift+Alt+F12) sahip olmasını sağlar.

Mevcut Copilot ekleme kısayollarını kaldırır ve bunları geçerli komutla (github.copilot.chat.attachFile) değiştirir.

Ayrıca, maksimize edilmiş paneli açıp kapatmak için Alt+` (ters tırnak) kısayolunu ekler.

## Syntax

```powershell
EnsureCopilotKeyboardShortCut [<CommonParameters>]
```

## Examples

### EnsureCopilotKeyboardShortCut

```powershell
EnsureCopilotKeyboardShortCut
```
