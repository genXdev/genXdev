# EnsureWireGuard

> **Module:** GenXdev.Windows.WireGuard | **Type:** Function | **Aliases:** —

## Synopsis

> WireGuard VPN hizmetinin Docker konteyneri aracılığıyla kurulu ve çalışır durumda olmasını sağlar.

## Description

Bu işlev, Docker Desktop kullanarak WireGuard VPN hizmetini kurar ve yönetir. Docker Desktop'ın çalıştığından otomatik olarak emin olur, en son WireGuard Docker imajını çeker, kalıcı depolama birimleri oluşturur ve sağlık izleme ile yeniden başlatma yetenekleri dahil olmak üzere kapsayıcı yaşam döngüsünü yönetir.

WireGuard, son teknoloji kriptografi kullanan basit, hızlı ve modern bir VPN'dir. Geleneksel VPN çözümlerine (OpenVPN gibi) kıyasla üstün performans ve basitlik sunar, minimum yapılandırma yükü ve mükemmel platformlar arası destek sağlar.

## Syntax

```powershell
EnsureWireGuard [[-ContainerName] <String>] [[-VolumeName] <String>] [[-ServicePort] <Int32>] [[-HealthCheckTimeout] <Int32>] [[-HealthCheckInterval] <Int32>] [[-ImageName] <String>] [[-PUID] <String>] [[-PGID] <String>] [[-TimeZone] <String>] [-Bottom] [-Centered] [-ClearSession] [-FocusWindow] [-Force] [-Fullscreen] [-Height <Int32>] [-Left] [-NoBorders] [-RestoreFocus] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-ShowWindow] [-SideBySide] [-SkipSession] [-Width <Int32>] [<CommonParameters>]
```
