# Get-AIKnownFacesRootpath

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> GenXdev.AI işlemlerinde kullanılan yüz görüntü dosyaları için yapılandırılmış dizini alır.

## Description

Bu işlev, GenXdev.AI modülü tarafından çeşitli yüz tanıma ve AI işlemleri için kullanılan genel yüz dizinini alır. Önce Global değişkenleri kontrol eder (SkipSession belirtilmediyse), ardından kalıcı tercihlere döner ve son olarak sistem varsayılanlarını kullanır.

## Syntax

```powershell
Get-AIKnownFacesRootpath [[-FacesDirectory] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FacesDirectory` | String | — | — | 0 | — | Yüz görüntü dosyalarının dizin yolu |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | Yapay zeka tercihleri (Dil, Görsel Koleksiyonları vb.) için oturumda depolanan alternatif ayarları kullanın |
| `-ClearSession` | SwitchParameter | — | — | Named | — | Almadan önce oturum ayarını (Genel değişken) temizle |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Tercih veri dosyaları için veritabanı yolu |
| `-SkipSession` | SwitchParameter | — | — | Named | — | AI tercihleri (Dil, Görüntü koleksiyonları vb.) için oturumda depolanan alternatif ayarları kullanmayın |

## Examples

### Get-AIKnownFacesRootpath

```powershell
Get-AIKnownFacesRootpath
```

Genel değişkenlerden veya tercihlerden yapılandırılmış yüzler dizinini alır.

### Get-AIKnownFacesRootpath -SkipSession

```powershell
Get-AIKnownFacesRootpath -SkipSession
```

Yalnızca kalıcı tercihlerden yapılandırılmış yüzler dizinini alır, herhangi bir oturum ayarını yok sayar.

### Get-AIKnownFacesRootpath -ClearSession

```powershell
Get-AIKnownFacesRootpath -ClearSession
```

Oturum yüzleri dizini ayarını temizler ve ardından kalıcı tercihlerden dizini alır.

### Get-AIKnownFacesRootpath "C:\MyFaces"

```powershell
Get-AIKnownFacesRootpath "C:\MyFaces"
```

Yolu genişlettikten sonra belirtilen dizini döndürür.
