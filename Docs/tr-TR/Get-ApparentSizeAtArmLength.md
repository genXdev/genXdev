# Get-ApparentSizeAtArmLength

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Bir nesnenin kol mesafesindeki görünen boyutunu hesaplar.

## Description

Küçük açı yaklaşımını kullanarak görünür boyutu hesaplar.

## Syntax

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters <Double> -SizeInMeters <Double> [[-ArmLengthInMeters] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | Nesneye olan mesafe metre cinsinden |
| `-SizeInMeters` | Double | ✅ | — | 1 | `0` | Nesnenin metre cinsinden gerçek boyutu |
| `-ArmLengthInMeters` | Double | — | — | 2 | `0` | Kol uzunluğu metre cinsinden (varsayılan: 0.7) |
| `-As` | String | — | — | 3 | — | Çıktı boyutu için birim |

## Examples

### Example 1

```powershell
Get-ApparentSizeAtArmLength -DistanceInMeters 10 -SizeInMeters 1 -As "centimeters"
```

10 metre mesafedeki 1 metrelik bir nesnenin görünen boyutunu hesaplar.

### Example 2

```powershell
Get-ApparentSizeAtArmLength 10 1
```

Konumsal parametreleri kullanarak görünen boyutu hesaplar.
