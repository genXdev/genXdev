# Get-BCP47LanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Tüm yüklü BCP 47 dil etiketlerini ve bunların görünen adlarını içeren bir sözlük döndürür.

## Description

```text
* BCP 47 kültür kodlarını (ör. en-US, nl-NL, de-DE) kendilerine ve İngilizce görünen adlarını kodlara eşleyen bir sözlük oluşturur.
* Sistemde yüklü olan tüm belirli kültürleri numaralandırmak için .NET CultureInfo API'sini kullanır ve bu, PowerShell yardım dili klasör adları ve çeviri hedef dilleri için yetkili bilgi kaynağı haline getirir.
* Sözlük anahtarları şunları içerir:
  - Tam BCP 47 etiketleri (ör. "en-US", "nl-NL", "zh-Hans-CN")
  - İngilizce görünen adlar (ör. "English (United States)", "Dutch (Netherlands)")
  - En yaygın bölgelerine eşlenen basit dil adları (ör. "English" -> "en-US", "Dutch" -> "nl-NL")
```
## Syntax

```powershell
Get-BCP47LanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-BCP47LanguageDictionary
```

Tam BCP 47 dil sözlüğünü alın.

### Example 2

```powershell
$dict = Get-BCP47LanguageDictionary
$dict["nl-NL"]  # returns "nl-NL"
$dict["Dutch"]  # returns "nl-NL"
```

Yardım oluşturma için bir dil kodunu doğrulayın ve çözümleyin.
