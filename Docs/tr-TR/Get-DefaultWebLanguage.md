# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Sistemin geçerli dil ayarlarına göre varsayılan web dil anahtarını alır.

## Description

* Geçerli sistem dilini ve kültür ayarlarını alır ve bunları çeviri hizmetleri tarafından kullanılan ilgili web dili sözlük anahtarına eşler.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Bir İngilizce sistemi için "İngilizce", bir Felemenkçe sistemi için "Felemenkçe" vb. döndürür.
