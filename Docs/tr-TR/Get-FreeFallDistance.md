# Get-FreeFallDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Serbest düşüş sırasında belirli bir süre için düşülen mesafeyi hesaplar.

## Description

Hava direncini ve terminal hızı hesaba katan sayısal bir yöntem kullanarak serbest düşüş sırasında düşülen mesafeyi hesaplar.

## Syntax

```powershell
Get-FreeFallDistance -DurationInSeconds <Double> [[-TerminalVelocityInMetersPerSecond] <Double>] [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | Saniye cinsinden düşüş süresi |
| `-TerminalVelocityInMetersPerSecond` | Double | — | — | 1 | `0` | Saniyede metre cinsinden terminal hızı (varsayılan: 53) |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Yerçekimi ivmesi m/s² cinsinden (varsayılan: 9.81) |
| `-As` | String | — | — | 3 | — | Çıktı mesafesi için birim |

## Examples

### Example 1

```powershell
Get-FreeFallDistance -DurationInSeconds 10 -TerminalVelocityInMetersPerSecond 53 -As "feet"
```

10 saniyede düşülen mesafeyi fit cinsinden hesaplar.

### Example 2

```powershell
Get-FreeFallDistance 5
```

5 saniyedeki mesafeyi metre cinsinden hesaplar.
