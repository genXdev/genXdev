# Get-FreeFallHeight

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Belirli bir süre boyunca serbest düşüşte düşülen yüksekliği hesaplar.

## Description

Hava direncini ve terminal hızı hesaba katan sayısal bir yöntem kullanarak serbest düşüş sırasında düşülen mesafeyi hesaplar.

## Syntax

```powershell
Get-FreeFallHeight -DurationInSeconds <Double> [[-TerminalVelocityInMs] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | Saniye cinsinden düşüş süresi |
| `-TerminalVelocityInMs` | Double | — | — | 1 | `0` | Saniyede metre cinsinden terminal hızı (varsayılan: insan için 53 m/s) |
| `-As` | String | — | — | 2 | — | Çıktı yüksekliği için birim |

## Examples

### Example 1

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -TerminalVelocityInMs 53
```

Varsayılan insan terminal hızıyla 10 saniyede düşülen yüksekliği hesaplar.

### Example 2

```powershell
Get-FreeFallHeight 5
```

Konumsal parametre ve varsayılan terminal hızı kullanılarak 5 saniyede düşülen yüksekliği hesaplar.

### Example 3

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -As "feet"
```

10 saniyede düşülen yüksekliği hesaplar ve sonucu fit cinsinden döndürür.
