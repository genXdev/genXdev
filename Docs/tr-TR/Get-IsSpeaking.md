# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> Metin-konuşma motoru konuşuyorsa true değerini döndürür.

## Description

Varsayılan ve özelleştirilmiş konuşma sentezleyicilerinin durumunu kontrol ederek herhangi birinin şu anda konuşup konuşmadığını belirler. Konuşma devam ediyorsa true, değilse false döndürür.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Metin-konuşma motoru konuşuyorsa true değerini döndürür.

### Example 2

```powershell
iss
```

Metin-konuşma motorunun konuşup konuşmadığını döndürür (takma ad kullanarak).
