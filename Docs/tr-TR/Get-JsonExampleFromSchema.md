# Get-JsonExampleFromSchema

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Bir JSON Şema tanımından insan tarafından okunabilir bir örnek JSON dizisi oluşturur.

## Description

Herhangi bir JSON Şemasını (OpenAI json_schema formatı veya ham şema) LLM sistem talimatlarına enjekte etmek için uygun, kompakt bir örnek JSON dizesine dönüştürür. İç içe nesneleri, dizileri, skaler türleri, enumları, birleşimleri, oneOf/anyOf/allOf bileşimlerini ve date-time ile email gibi biçim ipuçlarını işler.

İşlev, şemayı yinelemeli olarak dolaşarak bir PowerShell nesne ağacı oluşturur, ardından ConvertTo-Json aracılığıyla bunu kompakt bir JSON dizesine serileştirir.

## Syntax

```powershell
Get-JsonExampleFromSchema [[-Schema] <Object>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Schema` | Object | — | — | 0 | — | Bir JSON Şeması tanımı (JSON dizesi veya PSCustomObject). OpenAI json_schema sarmalayıcı biçimini ve ham JSON Şeması nesnelerini destekler. |

## Examples

### $schema = '{ "type": "object", "properties": { "name": { "type": "string" } } }' Get-JsonExampleFromSchema -Schema $schema

```powershell
$schema = '{ "type": "object", "properties": { "name": { "type": "string" } } }'
Get-JsonExampleFromSchema -Schema $schema
```

@{name=...}

### $openAiSchema = '{ "type": "json_schema", "json_schema": { "name": "response", "strict": true, "schema": { "type": "object", "properties": { "items": { "type": "array", "items": { "type": "object", "properties": { "id": { "type": "integer" }, "label": { "type": "string" } } } } } } } }' Get-JsonExampleFromSchema -Schema $openAiSchema

```powershell
$openAiSchema = '{ "type": "json_schema", "json_schema": { "name": "response",
"strict": true, "schema": { "type": "object", "properties": {
"items": { "type": "array", "items": { "type": "object", "properties": {
"id": { "type": "integer" }, "label": { "type": "string" } } } } } } } }'
Get-JsonExampleFromSchema -Schema $openAiSchema
```

Oluşturur: {"items":[{"id":0,"label":"..."}]}
