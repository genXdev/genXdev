# Get-LightTravelTimeByDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Işığın bir mesafeyi kat etmesi için geçen süreyi hesaplar.

## Description

Ortama bağlı olarak c ile t = d / c kullanılır.

## Syntax

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters <Double> [[-As] <String>] [<CommonParameters>]

Get-LightTravelTimeByDistance [[-SpeedOfLightInMetersPerSecond] <Double>] [<CommonParameters>]

Get-LightTravelTimeByDistance -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DistanceInMeters` | Double | ✅ | — | 0 | `0` | Metre cinsinden mesafe |
| `-SpeedOfLightInMetersPerSecond` | Double | — | — | 1 | `0` | Işık hızı (m/s cinsinden, varsayılan: 299792458) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | Işığın içinden geçtiği ortam *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | Zaman için çıktı birimi |

## Examples

### Example 1

```powershell
Get-LightTravelTimeByDistance -DistanceInMeters 149597870700 -Medium "water" -As "minutes"
```

Su içinde güneşe olan mesafeyi ışığın kat etmesi için geçen süreyi, dakika cinsinden hesaplar.

### Example 2

```powershell
Get-LightTravelTimeByDistance 300000000 -SpeedOfLightInMetersPerSecond 225000000
```

Işığın 300000000 metreyi 225000000 m/s hızla kat etmesi için gereken süreyi hesaplar.
