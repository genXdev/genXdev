# Get-PotentialEnergyByMassHeightAndGravity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Yerçekimi potansiyel enerjisini hesaplar.

## Description

PE = m g h kullanır.

## Syntax

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms <Double> -HeightInMeters <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | kg cinsinden kütle |
| `-HeightInMeters` | Double | ✅ | — | 1 | `0` | Metre cinsinden yükseklik |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | m/s² cinsinden yerçekimi (varsayılan: 9,81) |
| `-As` | String | — | — | 3 | — | Enerji için çıkış birimi |

## Examples

### Example 1

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms 10 -HeightInMeters 5 -As "calories"
```

10 kg kütleli bir cismin 5 metre yükseklikteki yerçekimi potansiyel enerjisini hesaplar ve sonucu kalori cinsinden verir.

### Example 2

```powershell
Get-PotentialEnergyByMassHeightAndGravity 5 10
```

Konumsal parametreleri kullanarak potansiyel enerji hesaplamayı gösterir.
