# Get-PowershellMainWindowProcess

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> PowerShell terminalini barındıran pencere için işlem nesnesini döndürür.

## Description

* Geçerli PowerShell işleminden başlayarak işlem ağacında yukarı doğru ilerleyerek terminali barındıran ana pencereyi bulur.
* Doğrudan üst işlemin bir ana pencere tanıtıcısı yoksa, ana pencereleri olan benzer işlemleri arar.
* PowerShell oturumunu içeren gerçek terminal penceresi işlemini (Windows Terminal, ConHost vb.) tanımlamak için kullanışlıdır.

## Syntax

```powershell
Get-PowershellMainWindowProcess [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$hostProcess = Get-PowershellMainWindowProcess
Write-Host "PowerShell is hosted in: $($hostProcess.ProcessName)"
```

Mevcut PowerShell oturumunu barındıran işlemi döndürür ve adını görüntüler.
