# Get-ResonantFrequencyByLengthAndSpeed

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Kapalı bir boru için rezonans frekansını hesaplar.

## Description

Temel frekans için f = v / (4 L) kullanır.

## Syntax

```powershell
Get-ResonantFrequencyByLengthAndSpeed -LengthInMeters <Double> [[-As] <String>] [<CommonParameters>]

Get-ResonantFrequencyByLengthAndSpeed -SpeedInMetersPerSecond <Double> [<CommonParameters>]

Get-ResonantFrequencyByLengthAndSpeed -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-LengthInMeters` | Double | ✅ | — | 0 | `0` | Metre cinsinden uzunluk |
| `-SpeedInMetersPerSecond` | Double | ✅ | — | 1 | `0` | Dalga hızı m/s cinsinden *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | Ortam *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | Frekans için çıkış birimi |

## Examples

### Example 1

```powershell
Get-ResonantFrequencyByLengthAndSpeed -LengthInMeters 0.5 -Medium "air" -As "kilohertz"
```

Havada 0,5 metrelik bir boru için rezonans frekansını hesaplar, kilohertz cinsinden çıktı verir.

### Example 2

```powershell
Get-ResonantFrequencyByLengthAndSpeed 1 -SpeedInMetersPerSecond 343
```

343 m/s dalga hızına sahip 1 metrelik bir boru için rezonans frekansını hesaplar.
