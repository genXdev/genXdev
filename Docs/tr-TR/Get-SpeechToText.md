# Get-SpeechToText

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> OpenAI'nin Whisper konuşma tanıma modelini kullanarak ses dosyalarını metne dönüştürür.

## Description

Whisper.NET kütüphanesini kullanarak ses dosyalarını işler ve konuşmayı metne dönüştürür. Bu kütüphane, OpenAI'in Whisper otomatik konuşma tanıma (ASR) sistemini uygular. Birden çok dili, çeviri yeteneklerini ve çeşitli transkripsiyon kalitesi ayarlarını destekler.

## Examples

### Example 1

```powershell
Get-SpeechToText -Input "C:\audio\recording.wav"
```

Varsayılan ayarları kullanarak bir ses dosyasını metne dönüştürür.

### Example 2

```powershell
Get-ChildItem "C:\audio\*.wav" | Get-SpeechToText
```

Bir dizindeki tüm WAV dosyalarını yazıya döker.

### Example 3

```powershell
Get-SpeechToText -Input "audio.mp3" -LanguageIn "es" -WithTranslate
```

İspanyolca sesi yazıya döküp İngilizceye çevirir.

### Example 4

```powershell
Get-SpeechToText -Input "recording.wav" -Passthru -WithTokenTimestamps
```

Hassas zamanlama bilgilerine sahip SegmentData nesnelerini döndürür.
