# Get-TerminalVelocityByMassGravityDensityAndArea

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Terminal hızını hesaplar.

## Description

v = sqrt(2 m g / (ρ A C)) kullanılır.

## Syntax

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms <Double> [[-GravityInMetersPerSecondSquared] <Double>] -DensityInKilogramsPerCubicMeter <Double> -AreaInSquareMeters <Double> [[-DragCoefficient] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | kg cinsinden kütle |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 1 | `0` | m/s² cinsinden yerçekimi (varsayılan: 9,81) |
| `-DensityInKilogramsPerCubicMeter` | Double | ✅ | — | 2 | `0` | kg/m³ cinsinden sıvı yoğunluğu |
| `-AreaInSquareMeters` | Double | ✅ | — | 3 | `0` | m² cinsinden kesit alanı |
| `-DragCoefficient` | Double | — | — | 4 | `0` | Sürükleme katsayısı (varsayılan: 0.5) |
| `-As` | String | — | — | 5 | — | Hız için çıktı birimi |

## Examples

### Example 1

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms 80 -DensityInKilogramsPerCubicMeter 1.225 -AreaInSquareMeters 0.7 -DragCoefficient 1.0 -As "mph"
```

80 kg kütleli bir cismin hava sürtünme katsayısı 1.0 iken terminal hızını mph cinsinden hesaplar.

### Example 2

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea 70 1.225 0.8 0.8
```

Terminal hızı hesaplamak için konumsal parametreler kullanır.
