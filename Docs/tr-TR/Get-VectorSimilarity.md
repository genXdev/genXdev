# Get-VectorSimilarity

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> İki vektör arasındaki kosinüs benzerliğini hesaplar.

## Description

İki sayısal vektörü (dizi) girdi olarak alır ve kosinüs benzerliklerini hesaplar. Sonuç, vektörlerin ne kadar yakından ilişkili olduğunu gösterir; 0 tamamen farklı, 1 ise aynı anlamına gelir.

## Syntax

```powershell
Get-VectorSimilarity -Vector1 <Double[]> -Vector2 <Double[]> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Vector1` | Double[] | ✅ | — | 0 | — | İlk vektör sayı dizisi |
| `-Vector2` | Double[] | ✅ | — | 1 | — | İkinci vektör sayı dizisi |

## Examples

### Example 1

```powershell
$v1 = @(0.12, -0.45, 0.89)
$v2 = @(0.15, -0.40, 0.92)
Get-VectorSimilarity -Vector1 $v1 -Vector2 $v2
```

Yaklaşık olarak 0,998 değerini döndürür, yüksek benzerlik gösterir.
