# Import-GenXdevModules

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** `reloadgenxdev`

## Synopsis

> Tüm GenXdev PowerShell modüllerini global kapsama aktarır.

## Description

GenXdev modülleri için üst dizini tarar ve her birini global kapsama aktarır. Çalışma dizinini korumak için konum yığını yönetimini kullanır ve başarılı ile başarısız aktarımlar için görsel geri bildirim sağlar. Aktarım işlemi sırasında işlev sayısındaki değişiklikleri izler.

## Syntax

```powershell
Import-GenXdevModules [-DebugFailedModuleDefinitions] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DebugFailedModuleDefinitions` | SwitchParameter | — | — | Named | — | Başarısız modül tanımlamaları için hata ayıklama çıktısını etkinleştir |

## Examples

### Import-GenXdevModules -DebugFailedModuleDefinitions Imports modules with debug output for failures

```powershell
Import-GenXdevModules -DebugFailedModuleDefinitions
Imports modules with debug output for failures
```

### reloadgenxdev Imports all modules using the alias

```powershell
reloadgenxdev
Imports all modules using the alias
```
