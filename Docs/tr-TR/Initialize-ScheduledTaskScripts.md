# Initialize-ScheduledTaskScripts

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Belirtilen aralıklarla PowerShell betiklerini çalıştıran zamanlanmış görevler oluşturur.

## Description

PowerShell betiklerini çeşitli aralıklarla çalıştıran zamanlanmış görevler oluşturur ve yapılandırır:
- Sistem başlangıcı
- Kullanıcı oturum açması
- Belirli günlerin her saati (ör. Pazartesi 13:00)
- Belirli saatlerde günlük (ör. her gün 15:00)
Her görev, geçerli kullanıcı bağlamında yükseltilmiş ayrıcalıklarla çalışır.

## Syntax

```powershell
Initialize-ScheduledTaskScripts [[-FilePath] <String>] [[-Prefix] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | — | — | 0 | `''` | Görev betiklerinin oluşturulacağı dizin yolu |
| `-Prefix` | String | — | — | 1 | `'PS'` | Zamanlanmış görev adları için önek |

## Examples

### Initialize-ScheduledTaskScripts -FilePath "C:\Tasks" -Prefix "MyTasks"

```powershell
Initialize-ScheduledTaskScripts -FilePath "C:\Tasks" -Prefix "MyTasks"
```

### Initialize-ScheduledTaskScripts

```powershell
Initialize-ScheduledTaskScripts
```
