# Invoke-OnEachGenXdevModule

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** `foreach-genxdev-module-do`

## Synopsis

> Çalışma alanındaki her GenXdev modülünde bir betik bloğu yürütür.

## Description

Bu işlev, çalışma alanındaki GenXdev modülleri arasında döngü yapar ve her bir modüle karşı sağlanan bir betik bloğunu yürütür. Modülleri ad desenine göre filtreleyebilir, yerel modülleri hariç tutabilir, yalnızca yayınlanmış modülleri dahil edebilir veya modüller yerine betikleri işleyebilir. İşlev, betik bloğunu yürütmeden önce otomatik olarak doğru modül dizinine gider.

## Syntax

```powershell
Invoke-OnEachGenXdevModule -Script <ScriptBlock> [[-ModuleName] <String[]>] [-FromScripts] [-IncludeScripts] [-NoLocal] [-OnlyPublished] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Script` | ScriptBlock | ✅ | — | 0 | — | Her bir GenXdev modülü için yürütülecek betik bloğu |
| `-ModuleName` | String[] | — | — | 1 | `@('GenXdev*')` | Modül adlarına uygulanacak filtre 🌐 *Supports wildcards* |
| `-NoLocal` | SwitchParameter | — | — | Named | — | Yerel geliştirme modüllerini işleme dahil etmez |
| `-OnlyPublished` | SwitchParameter | — | — | Named | — | Yalnızca LICENSE ve README.md dosyalarına sahip yayınlanmış modülleri içerir |
| `-FromScripts` | SwitchParameter | — | — | Named | — | Modül dizinleri yerine betik dizinini işle |
| `-IncludeScripts` | SwitchParameter | — | — | Named | — | Normal modüllerin yanı sıra betikler dizinini de içerir |

## Examples

### Invoke-OnEachGenXdevModule -Script { Write-Host $args[0].Name }

```powershell
Invoke-OnEachGenXdevModule -Script { Write-Host $args[0].Name }
```

Tüm GenXdev modül adlarını listeler.

### foreach-genxdev-module-do {     param($ModuleObj, $isScriptsFolder, $isSubModule, $subModuleName)     Get-ChildItem } -ModuleName "GenXdev.AI"

```powershell
foreach-genxdev-module-do {
    param($ModuleObj, $isScriptsFolder, $isSubModule, $subModuleName)
    Get-ChildItem
} -ModuleName "GenXdev.AI"
```

GenXdev.AI modül dizininin içeriğini listelemek için diğer adı kullanır.
