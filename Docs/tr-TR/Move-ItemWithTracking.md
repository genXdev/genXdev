# Move-ItemWithTracking

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Dosyaları ve dizinleri, dosya sistemi bağlantılarını ve başvurularını koruyarak taşır.

## Description

* Bağlantı takibi etkinleştirilmiş dosya ve dizinleri taşımak için Windows MoveFileEx API'sini kullanır ve dosya sistemi referanslarının korunmasını sağlar.
* Kaynak yol Git kontrolü altındaysa, yeniden adlandırmayı Git'te izlemek için git mv kullanmayı dener.
* Git mevcut değilse veya git mv başarısız olursa MoveFileEx'e geri döner.

## Syntax

```powershell
Move-ItemWithTracking -Path <String> -Destination <String> [-Force] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Taşınacak dosya/dizin kaynak yolu |
| `-Destination` | String | ✅ | — | 1 | — | Taşınacak hedef yol |
| `-Force` | SwitchParameter | — | — | Named | `False` | Hedef varsa üzerine yaz |

## Examples

### Example 1

```powershell
Move-ItemWithTracking -Path "C:\temp\oldfile.txt" -Destination "D:\newfile.txt"
```

Dosya sistem bağlantılarını ve Git takibini koruyarak bir dosyayı taşır.

### Example 2

```powershell
"C:\temp\olddir" | Move-ItemWithTracking -Destination "D:\newdir" -Force
```

Bir dizini taşır, varsa hedefin üzerine yazar.
