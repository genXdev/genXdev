# Now

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Geçerli sistem tarihini ve saatini DateTime nesnesi olarak döndürür.

## Description

Herhangi bir parametre olmadan geçerli sistem tarihini ve saatini almak için basit bir yol sağlar. Tarih-saat hesaplamaları, biçimlendirme ve karşılaştırmalar için kullanılabilen standart bir .NET DateTime nesnesi döndürür.

## Syntax

```powershell
Now [<CommonParameters>]
```

## Examples

### Now Returns the current system date and time as a DateTime object

```powershell
Now
Returns the current system date and time as a DateTime object
```

### $timestamp = Now Stores the current date and time in a variable for later use

```powershell
$timestamp = Now
Stores the current date and time in a variable for later use
```

## Outputs

- `DateTime`
