# Open-BuiltWithSiteInfo

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Bir web tarayıcısında BuiltWith web sitesi sorgularını açar.

## Description

BuiltWith sorgusunu, yapılandırılabilir monitör ayarları ve tarayıcı davranışı seçenekleriyle bir web tarayıcısında açar. İşlev, BuiltWith.com'un, web siteleri tarafından kullanılan web teknolojileri, çerçeveler ve hizmetler hakkında ayrıntılı bilgi sağlayan teknoloji profilleme hizmetini sorgulayarak web sitesi teknoloji yığınlarını analiz eder.

BuiltWith.com, herhangi bir web sitesi tarafından kullanılan web teknolojilerini, çerçeveleri, içerik yönetim sistemlerini, analitik araçlarını, barındırma sağlayıcılarını ve diğer yazılım bileşenlerini tanımlayan kapsamlı bir teknoloji arama hizmetidir. Bu işlev, BuiltWith web sitesine manuel olarak gitmeden teknoloji yığınlarını hızlı bir şekilde analiz etmek için bir PowerShell arayüzü sağlar.

## Syntax

```powershell
Open-BuiltWithSiteInfo -Queries <String[]> [[-Language] <String>] [[-Monitor] <Int32>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-BuiltWithSiteInfo -Queries "microsoft.com" -Monitor 0

```powershell
Open-BuiltWithSiteInfo -Queries "microsoft.com" -Monitor 0
```

Microsoft.com için BuiltWith teknoloji analizini varsayılan monitörde açar.

### "microsoft.com" | Open-BuiltWithSiteInfo -m -1

```powershell
"microsoft.com" | Open-BuiltWithSiteInfo -m -1
```

İzleme konumu atılarak işlem hattı girişi kullanarak Microsoft.com'u analiz eder.
