# Open-IMDBQuery

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** `imdb`

## Synopsis

> Bir IMDB arama sorgusunu web tarayıcısında açar.

## Description

Internet Movie Database (IMDb) sorgusunu kapsamlı yapılandırma seçenekleriyle bir web tarayıcısında açar. Bu işlev, PowerShell üzerinden doğrudan filmler, TV şovları, aktörler ve diğer eğlence içeriklerini aramaya olanak tanıyarak IMDb arama işlevselliği için kullanışlı bir sarmalayıcı sağlar.

İşlev, arama terimlerini kabul eder ve uygun IMDb arama URL'lerini oluşturur, ardından bunları belirtilen web tarayıcısında özelleştirilebilir pencere konumlandırma, tarayıcı seçimi ve görüntüleme seçenekleriyle açar. Edge, Chrome ve Firefox dahil tüm büyük tarayıcıları destekler; gizli gezinti, tam ekran modu ve pencere yönetimi seçenekleri sunar.

Ana özellikler arasında çoklu monitör desteği, dil yerelleştirmesi, klavye otomasyonu ve kapsamlı tarayıcı yapılandırma seçenekleri bulunur. İşlev, aynı anda birden fazla arama sorgusunu işleyebilir ve hem etkileşimli kullanım hem de otomatik iş akışları için seçenekler sağlar.

## Syntax

```powershell
Open-IMDBQuery -Queries <String[]> [[-Language] <String>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-Monitor <Int32>] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-IMDBQuery -Queries "The Matrix" -Monitor 0

```powershell
Open-IMDBQuery -Queries "The Matrix" -Monitor 0
```

Varsayılan monitörde "The Matrix" için bir IMDB araması açar.

### imdb "The Matrix" -m 0

```powershell
imdb "The Matrix" -m 0
```

"The Matrix" için takma ad ve kısayol parametrelerini kullanarak bir IMDB araması açar.

### "Inception", "Interstellar" | Open-IMDBQuery -Language "French" -Chrome

```powershell
"Inception", "Interstellar" | Open-IMDBQuery -Language "French" -Chrome
```

Chrome'da Fransızca dil sonuçlarıyla IMDB'de birden çok film arar.

### Open-IMDBQuery -Queries "Tom Hanks" -FullScreen -RestoreFocus

```powershell
Open-IMDBQuery -Queries "Tom Hanks" -FullScreen -RestoreFocus
```

Tom Hanks'i tam ekran modunda IMDB'de arar, ardından odağı PowerShell'e geri verir.
