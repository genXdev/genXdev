# Open-SourceFileInIde

> **Module:** GenXdev.Coding | **Type:** Function | **Aliases:** `editcode`

## Synopsis

> Tercih edilen IDE'de (Visual Studio Code veya Visual Studio) bir kaynak dosyası açar.

## Description

Bu işlev, belirtilen bir kaynak dosyayı Visual Studio Code veya Visual Studio'da açar. Dosyayı açtıktan sonra doğrudan belirli bir satır numarasına gidebilir ve isteğe bağlı olarak IDE'ye klavye girişleri gönderebilir. İşlev, geçerli ana bilgisayar işlemine, çalışan uygulamalara veya kullanıcı seçimine göre hangi IDE'nin kullanılacağını belirlemeye çalışır.

## Syntax

```powershell
Open-SourceFileInIde -Path <String> [[-LineNo] <Int32>] [[-KeysToSend] <String[]>] [-Code] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-VisualStudio] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Açılacak kaynak dosyanın yolu |
| `-LineNo` | Int32 | — | — | 1 | `0` | Gidilecek satır numarası |
| `-KeysToSend` | String[] | — | — | 2 | `@()` | Uygulamaya gönderilecek klavye girişleri dizisi |
| `-Code` | SwitchParameter | — | — | Named | — | Dosyayı açmak için IDE |
| `-VisualStudio` | SwitchParameter | — | — | Named | — | Visual Studio'da Aç |
| `-SendKeyEscape` | SwitchParameter | — | — | Named | — | Kaçış kontrol karakterlerini ve değiştiricileri |
| `-SendKeyHoldKeyboardFocus` | SwitchParameter | — | — | Named | — | Klavye odağını hedef pencerede tut |
| `-SendKeyUseShiftEnter` | SwitchParameter | — | — | Named | — | Enter yerine Shift+Enter kullanın |
| `-SendKeyDelayMilliSeconds` | Int32 | — | — | Named | — | Farklı giriş dizeleri arasındaki milisaniye cinsinden gecikme |

## Examples

### Open-SourceFileInIde -Path "C:\Projects\MyScript.ps1" -LineNo 25 -Code

```powershell
Open-SourceFileInIde -Path "C:\Projects\MyScript.ps1" -LineNo 25 -Code
```

### editcode "C:\Projects\MyScript.ps1" 25

```powershell
editcode "C:\Projects\MyScript.ps1" 25
```
