# Open-Webbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** `wb`

## Synopsis

> URL'leri isteğe bağlı konumlandırma ve stil ile bir veya daha fazla tarayıcı penceresinde açar.

## Description

Bu işlev, pencere konumlandırma, tarayıcı seçimi ve davranış özelleştirme için kapsamlı seçenekler sunan, tarayıcı başlatma etrafında gelişmiş bir sarmalayıcı sağlar. Edge, Chrome ve Firefox dahil olmak üzere birden çok tarayıcıyı, özel gezinti, uygulama modu ve hassas pencere yönetimi gibi özelliklerle destekler.

Temel özellikler:
- Akıllı tarayıcı algılama ve seçimi
- Pencere konumlandırma (sol, sağ, üst, alt, ortalanmış, tam ekran)
- Otomatik veya manuel monitör seçimi ile çoklu monitör desteği
- Özel/gezici tarama modu desteği
- Dikkat dağıtmayan gezinme için uygulama modu
- Uzantı ve açılır pencere engelleme seçenekleri
- Odak yönetimi ve pencere manipülasyonu
- Birden çok tarayıcıda toplu URL açma
- Tarayıcı pencerelerine tuş vuruşu otomasyonu

İşlev, sistem yeteneklerini otomatik olarak algılayabilir ve davranışı buna göre ayarlayabilir. Sistemde yüklü olmayan tarayıcılar için işlemler hatasız olarak sessizce atlanır.


## Examples

### wb -PlayWright https://github.com

```powershell
wb -PlayWright https://github.com
```

Playwright tarafından yönetilen bir Chromium tarayıcısında GitHub'ı açar.

### Open-Webbrowser -Url "https://github.com"

```powershell
Open-Webbrowser -Url "https://github.com"
```

Varsayılan tarayıcıda GitHub'ı açar.

### Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left

```powershell
Open-Webbrowser -Url "https://stackoverflow.com" -Monitor 1 -Left
```

Stack Overflow'u monitör 1'in sol yarısında açar.

### wb "https://google.com" -m 0 -fs

```powershell
wb "https://google.com" -m 0 -fs
```

Birincil monitörde Google'ı tam ekran modunda takma adlar kullanarak açar.

### Open-Webbrowser -Chrome -Private -NewWindow

```powershell
Open-Webbrowser -Chrome -Private -NewWindow
```

Gizli modda yeni bir Chrome penceresi açar.

### "https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All

```powershell
"https://github.com", "https://stackoverflow.com" | Open-Webbrowser -All
```

Yüklü tüm tarayıcılarda birden fazla URL'yi işlem hattı üzerinden açar.

### Open-Webbrowser -Monitor 0 -Right

```powershell
Open-Webbrowser -Monitor 0 -Right
```

Zaten açık olan bir tarayıcı penceresini birincil monitörün sağ tarafına yeniden konumlandırır.

### Open-Webbrowser -ApplicationMode -Url "https://app.example.com"

```powershell
Open-Webbrowser -ApplicationMode -Url "https://app.example.com"
```

Bir web uygulamasını tarayıcı kontrolleri olmadan uygulama modunda açar.
