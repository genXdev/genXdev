# Rename-InProject

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** `rip`

## Synopsis

> Bir proje dizini boyunca metin değiştirme işlemi gerçekleştirir.

## Description

Bir projedeki dosya ve dizinlerde metin değişiklikleri yapmak için özyinelemeli olarak arama yapar. Hem dosya/dizin adlarını hem de dosya içeriklerini işler. Bozulmayı önlemek için ortak ikili dosyaları ve depo klasörlerini (.git, .svn) atlar. Dosya işlemleri için BOM olmadan UTF-8 kodlaması kullanır. Hem büyük/küçük harfe duyarlı hem de duyarsız değiştirme modlarını destekler.

## Syntax

```powershell
Rename-InProject [[-Source] <String>] -FindText <String> -ReplacementText <String> [-CaseInsensitive] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Source` | String | — | — | 0 | — | Dizin, dosya yolu veya dizin+arama maskesi |
| `-FindText` | String | ✅ | — | 1 | — | Bulunacak metin (büyük/küçük harf duyarlılığı CaseInsensitive parametresi tarafından kontrol edilir) |
| `-ReplacementText` | String | ✅ | — | 2 | — | Eşleşmelerle değiştirilecek metin |
| `-CaseInsensitive` | SwitchParameter | — | — | Named | — | Büyük/küçük harf duyarsız metin değişimi |

## Examples

### Rename-InProject -Source .\src\*.js -FindText "oldName" `     -ReplacementText "newName"

```powershell
Rename-InProject -Source .\src\*.js -FindText "oldName" `
    -ReplacementText "newName"
```

### rip . "MyClass" "MyNewClass" -WhatIf

```powershell
rip . "MyClass" "MyNewClass" -WhatIf
```

### rip . "OLDNAME" "NewName" -CaseInsensitive

```powershell
rip . "OLDNAME" "NewName" -CaseInsensitive
```
