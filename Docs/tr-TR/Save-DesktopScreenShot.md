# Save-DesktopScreenShot

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Belirtilen monitörün ekran görüntüsünü yakalar ve belirtilen yola kaydeder.

## Description

* Belirtilen monitörden masaüstünün ekran görüntüsünü alır ve bir görüntü dosyası olarak kaydeder.
* Birden fazla monitörü destekler ve yüksek DPI ekranlar için doğru ekran sınırlarını ve ölçekleme faktörlerini otomatik olarak algılar.

## Syntax

```powershell
Save-DesktopScreenShot -Monitor <Int32> -OutputPath <String> [-Quality <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | ✅ | ✅ (ByPropertyName) | 0 | `0` | Yakalanacak monitörü belirtin (0=birincil, 1+=belirli monitör, -1=tüm monitörler) |
| `-OutputPath` | String | ✅ | ✅ (ByPropertyName) | 1 | — | Ekran görüntüsünün kaydedileceği tam yol |
| `-Quality` | Int32 | — | — | Named | `0` | JPEG kalite seviyesi (1-100), yalnızca JPEG formatı için geçerlidir |

## Examples

### Example 1

```powershell
Save-DesktopScreenShot -Monitor 0 -OutputPath "C:\Screenshots\primary.png"
```

Birincil monitörün ekran görüntüsünü alır ve PNG olarak kaydeder.

### Example 2

```powershell
Save-DesktopScreenShot -Monitor 1 -OutputPath "C:\Screenshots\monitor2.jpg" -Quality 95
```

İkinci monitörün yüksek kaliteli JPEG ekran görüntüsünü alır.

### Example 3

```powershell
Save-DesktopScreenShot -Monitor -1 -OutputPath "C:\Screenshots\all_monitors.png"
```

Tüm monitörleri tek bir ekran görüntüsünde birleştirir.
