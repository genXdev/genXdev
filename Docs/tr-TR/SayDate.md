# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Geçerli tarihi metin-konuşma sentezi kullanarak söyler.

## Description

Geçerli tarihi doğal dil biçimine dönüştürür ve sistemin metin okuma motoru kullanarak duyurur. Tarih şu biçimde söylenir: "Bugün [haftanın günü], [ay] [gün] [yıl]"

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
