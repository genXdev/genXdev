# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> Geçerli konumu üst dizine değiştirir ve içeriğini listeler.

## Description

* Dosya sistemi hiyerarşisinde bir üst dizin seviyesine gider.
* Yeni geçerli dizinin içeriğini görüntüler.
* Hızlı dizin gezinmesi için kullanışlı bir '..' takma adı sağlar.

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

Ana dizine geçer ve içeriğini listeler.

### Example 2

```powershell
..
```

Üst dizine geçmek için diğer adı kullanır.
