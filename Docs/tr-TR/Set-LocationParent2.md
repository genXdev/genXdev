# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> Dosya sistemi hiyerarşisinde iki dizin seviyesi yukarı çıkar.

## Description

* Geçerli çalışma dizinini üst üstteki dizine (iki seviye yukarı) değiştirir.
* Ortaya çıkan dizinin içeriğini görüntüler.

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Üst dizine geçer ve içeriğini görüntüler.

### Example 2

```powershell
...
```

Aynı işlem için '...' diğer adını kullanır.
