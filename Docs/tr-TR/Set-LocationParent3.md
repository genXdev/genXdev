# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....`

## Synopsis

> Dosya sistemi hiyerarşisinde üç dizin seviyesi yukarı gider.

## Description

* Dosya sistemi hiyerarşisinde üç üst dizin yukarı giderek geçerli çalışma dizinini değiştirir.
* Ortaya çıkan dizinin içeriğini görüntüler.

## Syntax

```powershell
Set-LocationParent3 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

Geçerli konumdan üç dizin yukarı gider.

### Example 2

```powershell
....
```

Üç dizin düzeyine çıkmak için takma adı kullanır.
