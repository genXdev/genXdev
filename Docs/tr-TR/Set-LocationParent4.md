# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> Dosya sistemi hiyerarşisinde dört dizin düzeyi yukarı gider.

## Description

* Geçerli dosya sistemindeki konumdan dört dizin yukarı taşınır.
* Gezinme işleminden sonra, sonuç dizininin içeriğini görüntüler.

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Geçerli konumdan dört dizin seviyesi yukarı gider.

### Example 2

```powershell
.....
```

Dört dizin seviyesi yukarı gitmek için takma adı kullanır.
