# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> Dosya sistemi hiyerarşisinde beş dizin seviyesi yukarı gider.

## Description

* Geçerli çalışma dizinini beş üst dizin yukarı taşıyarak değiştirir.
* Sonuç dizininin içeriğini görüntüler.
* Hızlı gezinme için '......' kullanışlı bir kısayol takma adı sağlar.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Geçerli konumdan beş dizin düzeyi yukarı gider.

### Example 2

```powershell
......
```

Beş dizin seviyesi yukarı gitmek için takma adı kullanır.
