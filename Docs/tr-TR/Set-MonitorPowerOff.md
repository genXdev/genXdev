# Set-MonitorPowerOff

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `poweroff`

## Synopsis

> Bağlı tüm monitörlerin gücünü kapatır.

## Description

Bağlı tüm monitörlere güç kapatma sinyali göndermek için Windows API çağrılarını kullanır. Bu, monitörlerinizdeki fiziksel güç düğmesine basmaya eşdeğerdir. Monitörler güç tasarrufu moduna geçer ancak fareyi hareket ettirerek veya bir tuşa basarak uyandırılabilir.

## Syntax

```powershell
Set-MonitorPowerOff [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOff
```

Bağlı tüm monitörleri kapatır.

### Example 2

```powershell
poweroff
```

Takma adı kullanarak tüm bağlı monitörleri kapatır.
