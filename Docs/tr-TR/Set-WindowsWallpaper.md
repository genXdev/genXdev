# Set-WindowsWallpaper

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Belirtilen bir dizinden rastgele bir duvar kağıdı ayarlar.

## Description

* Belirtilen dizinden rastgele bir resim dosyası seçer ve onu Windows masaüstü duvar kağıdı olarak ayarlar.
* JPG/JPEG resim formatlarını destekler ve varsayılan olarak duvar kağıdını ekrana "sığdır" şeklinde yapılandırır.

## Syntax

```powershell
Set-WindowsWallpaper [[-InputObject] <Object>] [-AllDrives] [-NoRecurse] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputObject` | Object | — | ✅ (ByValue, ByPropertyName) | 0 | — | Duvar kağıdı görüntülerini içeren dizinin yolu |
| `-AllDrives` | SwitchParameter | — | — | Named | `False` | Tüm mevcut sürücüler arasında arama yap |
| `-NoRecurse` | SwitchParameter | — | — | Named | `False` | Alt dizinlere yineleme yapma |

## Examples

### Example 1

```powershell
Set-WindowsWallpaper -InputObject "C:\Wallpapers\*.jpg"
```

C:\Wallpapers dizininden rastgele bir duvar kağıdı ayarlar.

### Example 2

```powershell
nextbg
```

Varsayılan dizinden takma ad kullanarak rastgele bir duvar kağıdı ayarlar.
