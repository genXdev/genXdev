# Start-VlcMediaPlayerNextInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcnext`

## Synopsis

> VLC Media Player'ı geçerli oynatma listesindeki bir sonraki öğeye ilerletir.

## Description

Bu işlev, şu anda yüklü olan oynatma listesinde bir sonraki parçaya veya medya öğesine geçmek için VLC Medya Oynatıcıya 'n' tuş vuruşunu gönderir. VLC penceresine odaklanır, sonraki komutunu gönderir ve PowerShell konsoluna odağı geri verir. İşlev, gerektiğinde onay istemleri için ShouldProcess desteği içerir.

## Syntax

```powershell
Start-VlcMediaPlayerNextInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerNextInPlaylist

```powershell
Start-VlcMediaPlayerNextInPlaylist
```

### vlcnext

```powershell
vlcnext
```
