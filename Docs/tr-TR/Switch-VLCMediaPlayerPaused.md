# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> VLC Medya Oynatıcısının duraklat/oynat durumunu değiştirir.

## Description

Bu işlev, VLC Medya Oynatıcı'ya duraklatma ve oynatma durumları arasında geçiş yapmak için bir boşluk tuşu gönderir. Tuş komutunu gönderdikten sonra odağı otomatik olarak orijinal pencereye geri yükler.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

VLC Media Player'ın duraklat/oynat durumunu değiştirir.

### vlcpause

```powershell
vlcpause
```

Duraklat/devam et durumunu değiştirmek için takma adı kullanır.

### vlcplay

```powershell
vlcplay
```

Duraklat/oynat durumunu değiştirmek için alternatif takma adı kullanır.
