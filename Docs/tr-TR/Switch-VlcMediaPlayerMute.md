# Switch-VlcMediaPlayerMute

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcmute`, `vlcunmute`

## Synopsis

> VLC Medya Oynatıcısının sessiz durumunu değiştirir.

## Description

Bu işlev, sesin kapatılması ve açılması arasında geçiş yapmak için VLC Medya Oynatıcı'ya 'm' tuşunu gönderir. İşlev, VLC'yi odaklar, sessiz/sesi aç komutunu gönderir ve ardından odağı daha önce etkin olan pencereye geri verir.

## Syntax

```powershell
Switch-VlcMediaPlayerMute [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerMute

```powershell
Switch-VlcMediaPlayerMute
```

Tam işlev adını kullanarak VLC Medya Oynatıcı'nın sessiz durumunu değiştirir.

### vlcmute

```powershell
vlcmute
```

Kısa takma adı kullanarak VLC Medya Oynatıcısının sessiz durumunu değiştirir.
