# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> VLC Medya Oynatıcı'da tekrar modunu açar/kapatır.

## Description

Bu işlev, farklı tekrar modları (tekrar yok, geçerli öğeyi tekrarla, tümünü tekrarla) arasında geçiş yapmak için VLC Medya Oynatıcı'ya 'r' tuş komutunu gönderir. İşlev, VLC zaten çalışmıyorsa onu açar ve komutu gönderdikten sonra odağı önceki pencereye geri verir.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
