# Test-RefactorLLMSelection

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Yeniden düzenleme uygunluğu için kaynak dosyaları LLM analizi kullanarak değerlendirir.

## Description

Bir kaynak kodu dosyasının belirtilen kriterlere göre yeniden düzenleme için seçilip seçilmeyeceğini belirlemek amacıyla Dil Öğrenme Modeli (LLM) analizini kullanır. İşlev, dosya içeriğini bir LLM sorgusu aracılığıyla işler ve bir boolean yanıt döndürür.

## Syntax

```powershell
Test-RefactorLLMSelection -RefactorDefinition <GenXdev.Helpers.RefactorDefinition> -Path <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-RefactorDefinition` | GenXdev.Helpers.RefactorDefinition | ✅ | — | 0 | — | LLM ayarlarını içeren refactor tanımı |
| `-Path` | String | ✅ | — | 1 | — | Değerlendirilecek kaynak dosyanın yolu |

## Examples

### Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"

```powershell
Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"
```

### $def | Test-RefactorLLMSelection -Path source.ps1

```powershell
$def | Test-RefactorLLMSelection -Path source.ps1
```
