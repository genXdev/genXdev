# WriteFileOutput

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Kullanıcı deneyimini geliştirmek için köprü bağlantılı görüntüleme adlarıyla dosya bilgilerini çıkarır.

## Description

Bu işlev, dosya girdi nesnelerini işler ve konsolda görüntülendiğinde, köprü bağlantılı dosya adlarıyla kullanıcı dostu bir biçimde çıktılar. Farklı girdi türlerini (dizeler, FileInfo nesneleri ve diğer nesne türleri) işler. İşlev, çıktının yönlendirilip yönlendirilmediğini veya bir ardışık düzende yakalanıp yakalanmadığını otomatik olarak algılar ve davranışını buna göre ayarlar.

## Syntax

```powershell
WriteFileOutput -CallerInvocation <Object> -Input <Object> [-FullPaths] [-Prefix <String>] [-RelativeBasePath <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CallerInvocation` | Object | ✅ | ✅ (ByValue) | 0 | — | Çağıran işlevden gelen çağırma bilgisi |
| `-Input` | Object | ✅ | ✅ (ByValue) | 1 | — | İşlenecek girdi nesnesi, bir dosya yolu veya nesne olabilir |
| `-Prefix` | String | — | — | Named | — | Çıktı görüntüsüne ek bağlam için eklenebilecek isteğe bağlı bir dize öneki |
| `-RelativeBasePath` | String | — | — | Named | — | Çıktıda göreli dosya yolları oluşturmak için temel yol |
| `-FullPaths` | SwitchParameter | — | — | Named | — | Çıktıyı göreli yollar yerine tam mutlak yollar kullanmaya zorlar |

## Examples

### WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"

```powershell
WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"
```

### Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_

```powershell
Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_
```
