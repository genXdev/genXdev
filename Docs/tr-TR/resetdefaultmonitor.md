# resetdefaultmonitor

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Varsayılan ikincil monitör yapılandırmasını geri yükler.

## Description

Bu betik, sistem için varsayılan ikincil monitör yapılandırmasını geri yükler ve ikincil monitörü orijinal varsayılan değerine ayarlar. Bu, yan yana yapılandırmaları kullandıktan sonra önceki çoklu monitör kurulumlarına dönmek isteyen kullanıcılar için kullanışlıdır.
Ayrıca bakınız: yeni pencereler için yan yana moda geçmek üzere 'sidebyside' işlevi.

## Syntax

```powershell
resetdefaultmonitor [<CommonParameters>]
```

## Examples

### secondscreen

```powershell
secondscreen
```

Sistem için varsayılan ikincil monitör yapılandırmasını geri yükler.
