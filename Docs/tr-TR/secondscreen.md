# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Varsayılan ikinci monitör yapılandırmasını ayarlar.

## Description

GenXdev pencere açılışlarının varsayılan davranışını ikincil monitörde olacak şekilde ayarlar.
Tek monitörü olan veya yan yana pencere düzenlerini tercih eden kullanıcılar için faydalıdır.
Ayrıca 'sidebyside' ve 'restoredefaultmonitor' cmdlet'lerine bakın

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

GenXdev pencere açılışları için varsayılanları ikincil monitör olarak ayarlar
