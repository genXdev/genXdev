# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Varsayılan yan yana yapılandırmayı ayarlar.

## Description

GenXdev pencere açılışlarının varsayılan davranışını PowerShell ile yan yana olacak şekilde ayarlar.
Bu, tek monitörü olan veya yan yana pencere düzenlerini tercih eden kullanıcılar için kullanışlıdır.
Ayrıca 'secondscreen' ve 'restoredefaultmonitor' cmdlet'lerine bakın.

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

GenXdev pencere açılışları için varsayılanları PowerShell ile yan yana olacak şekilde ayarlar
