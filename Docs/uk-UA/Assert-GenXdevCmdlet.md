# Assert-GenXdevCmdlet

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `improvecmdlet`

## Synopsis

> Покращує документацію та реалізацію командлетів GenXdev за допомогою штучного інтелекту.

## Description

Ця функція покращує командлети GenXdev шляхом аналізу та вдосконалення їх коду за допомогою AI-промптів. Вона може інтегрувати командлети в модулі, оновлювати документацію та перевіряти правильність реалізації. Функція підтримує власні шаблони промптів і може відкривати файли у Visual Studio Code або Visual Studio.

## Syntax

```powershell
Assert-GenXdevCmdlet [[-CmdletName] <String>] [[-PromptKey] <String>] [[-Prompt] <String>] [-Code] [-FromScripts] [-Integrate] [-ModuleName <String[]>] [-NoLocal] [-OnlyPublished] [-VisualStudio] [<CommonParameters>]

Assert-GenXdevCmdlet [-EditPrompt] [<CommonParameters>]
```
