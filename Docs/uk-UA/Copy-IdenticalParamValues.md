# Copy-IdenticalParamValues

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Копіює значення параметрів зі зв'язаних параметрів у нову хеш-таблицю на основі можливих параметрів іншої функції.

## Description

* Цей командлет створює нову хеш-таблицю, що містить лише ті значення параметрів,
  які збігаються з параметрами, визначеними у вказаній цільовій функції.
* Отримана хеш-таблиця може бути використана для виклику функції за допомогою
  сплатінгу PowerShell.
* Параметри-перемикачі включаються в результат лише у тому випадку, якщо вони були
  явно передані та встановлені у $true у зв'язаних параметрах.
* Відсутні параметри-перемикачі виключаються з результату для збереження
  правильної семантики параметрів.

## Syntax

```powershell
Copy-IdenticalParamValues -BoundParameters <Object[]> -FunctionName <String> [[-DefaultValues] <Object>] [<CommonParameters>]
```

| `-FunctionName` | String | ✅ | — | 1 | — | Назва цільової функції для фільтрації параметрів |
| `-DefaultValues` | Object | — | — | 2 | — | Значення за замовчуванням для параметрів |

## Examples

### Example 1

```powershell
function Test-Function {
    [CmdletBinding()]
    param(
        [Parameter(Mandatory = $true)]
        [string] $Path,
        [Parameter(Mandatory = $false)]
        [switch] $Recurse
    )

    $params = GenXdev\Copy-IdenticalParamValues `
        -BoundParameters $PSBoundParameters `
        -FunctionName 'Get-ChildItem'

    Get-ChildItem @params
}
```

Демонструє копіювання зв'язаних параметрів для використання з splatting.
