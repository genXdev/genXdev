# Copy-PDFsFromGoogleQuery

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Завантажує PDF-файли, знайдені через результати пошуку Google.

## Description

Виконує пошук Google у раніше вибраній вкладці веб-браузера та завантажує всі знайдені PDF-файли в поточний каталог. Підтримує кілька запитів і фільтрацію за мовою.

## Syntax

```powershell
Copy-PDFsFromGoogleQuery -Queries <String[]> [[-Max] <Int32>] [[-Language] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Queries` | String[] | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Пошукові запити для запиту до Google щодо PDF-файлів |
| `-Max` | Int32 | — | — | 1 | `200` | Максимальна кількість результатів для отримання (за замовчуванням: 200) |
| `-Language` | String | — | — | 2 | — | Необов'язковий мовний фільтр для результатів пошуку |

## Examples

### Open-Webbrowser Select-WebbrowserTab $null = New-Item -ItemType Directory -Name pdfs Set-Location pdfs Copy-PDFsFromGoogleQuery "scientific paper co2" -Max 50 -Language "English"

```powershell
Open-Webbrowser
Select-WebbrowserTab
$null = New-Item -ItemType Directory -Name pdfs
Set-Location pdfs
Copy-PDFsFromGoogleQuery "scientific paper co2" -Max 50 -Language "English"
```
