# CurrentUserHasElevatedRights

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Перевіряє, чи поточний користувач має підвищені права.

## Description

Визначає, чи поточний користувач Windows має права адміністратора або оператора резервного копіювання, перевіряючи їх ролі суб'єкта безпеки. Повертає true, якщо користувач має підвищені права, інакше false. Реалізує надійну обробку помилок для проблем, пов'язаних з безпекою та доступом.

## Syntax

```powershell
CurrentUserHasElevatedRights [<CommonParameters>]
```

## Examples

### $hasRights = CurrentUserHasElevatedRights ##############################################################################

```powershell
$hasRights = CurrentUserHasElevatedRights
##############################################################################
```

## Outputs

- `Boolean
Returns true if user has admin or backup operator rights, false otherwise.`
- `Boolean`
