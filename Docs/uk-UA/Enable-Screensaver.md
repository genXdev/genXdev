# Enable-Screensaver

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Запускає налаштовану заставку Windows.

## Description

Активує системну заставку Windows шляхом запуску виконуваного файлу за замовчуванням (scrnsave.scr) з ключем /s для негайного запуску.

## Syntax

```powershell
Enable-Screensaver [<CommonParameters>]
```

## Examples

### Enable-Screensaver

```powershell
Enable-Screensaver
```
