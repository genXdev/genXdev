# EnsureDefaultGenXdevRefactors

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** —

## Synopsis

> Забезпечує наявність стандартних визначень рефакторингу GenXdev.

## Description

Ця функція створює та підтримує стандартні визначення рефакторингу для
модулів PowerShell GenXdev. Вона налаштовує рефактори для документації та форматування,
перетворення C# та підказок згоди на встановлення.

## Syntax

```powershell
EnsureDefaultGenXdevRefactors [-Force] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Force` | SwitchParameter | — | — | Named | — | Примусове відтворення існуючих визначень рефакторингу |

## Examples

### EnsureDefaultGenXdevRefactors

```powershell
EnsureDefaultGenXdevRefactors
```

### EnsureDefaultGenXdevRefactors -Force

```powershell
EnsureDefaultGenXdevRefactors -Force
```
