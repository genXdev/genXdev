# Get-ActiveUser

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** `gusers`

## Synopsis

> Отримує список унікальних імен користувачів з активних системних процесів.

## Description

Запитує всі запущені процеси на системі, отримує пов'язане ім'я користувача для кожного процесу та повертає дедуплікований список користувачів, які мають активні процеси. Це корисно для системного адміністрування та моніторингу безпеки.

## Syntax

```powershell
Get-ActiveUser [<CommonParameters>]
```

## Examples

### Get-ActiveUser Returns a list of all unique usernames with active processes.

```powershell
Get-ActiveUser
Returns a list of all unique usernames with active processes.
```

### gusers Uses the alias to get the same results. ##############################################################################

```powershell
gusers
Uses the alias to get the same results.
##############################################################################
```
