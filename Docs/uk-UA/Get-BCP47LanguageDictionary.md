# Get-BCP47LanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Повертає словник усіх встановлених мовних тегів BCP 47 та їхніх відображуваних імен.

## Description

```text
* Створює словник, який зіставляє коди культур BCP 47 (наприклад, en-US, nl-NL, de-DE) з ними самими та їхніми англійськими відображуваними назвами з кодами.
* Використовує API .NET CultureInfo для переліку всіх конкретних культур, встановлених у системі, що робить його авторитетним джерелом істини для назв папок довідки PowerShell та мов перекладу.
* Ключі словника включають:
  - Повні теги BCP 47 (наприклад, "en-US", "nl-NL", "zh-Hans-CN")
  - Англійські відображувані назви (наприклад, "English (United States)", "Dutch (Netherlands)")
  - Прості назви мов, зіставлені з їхнім найпоширенішим регіоном
    (наприклад, "English" -> "en-US", "Dutch" -> "nl-NL")
```
## Syntax

```powershell
Get-BCP47LanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-BCP47LanguageDictionary
```

Отримайте повний словник мов BCP 47.

### Example 2

```powershell
$dict = Get-BCP47LanguageDictionary
$dict["nl-NL"]  # returns "nl-NL"
$dict["Dutch"]  # returns "nl-NL"
```

Перевірте та вирішіть код мови для генерації довідки.
