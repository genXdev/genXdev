# Get-ChildProcesses

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Отримує всі процеси, які є нащадками поточного процесу PowerShell.

## Description

* Перевіряє всі працюючі процеси та визначає ті, що мають поточний
  процес PowerShell як предка у своєму ланцюжку батьківських процесів.
* Це включає як безпосередні дочірні процеси, так і їх нащадків
  (внуки тощо).

## Syntax

```powershell
Get-ChildProcesses [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ChildProcesses
```

Отримати всі дочірні процеси поточного сеансу PowerShell.

### Example 2

```powershell
Get-ChildProcesses -Verbose
```

Отримати дочірні процеси з докладним виведенням, що показує детальну інформацію про виявлення процесів.
