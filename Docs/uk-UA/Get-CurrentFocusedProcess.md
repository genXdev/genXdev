# Get-CurrentFocusedProcess

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Отримує об'єкт процесу вікна, яке наразі має фокус введення з клавіатури.

## Description

* Цей командлет використовує виклики Windows API через P/Invoke для ідентифікації та повернення об'єкта процесу, пов'язаного з поточним активним вікном.
* Він використовує функції бібліотеки User32.dll GetForegroundWindow та GetWindowThreadProcessId, щоб визначити, яке вікно має фокус, та отримати ідентифікатор пов'язаного з ним процесу.

## Syntax

```powershell
Get-CurrentFocusedProcess [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$focusedProcess = Get-CurrentFocusedProcess
Write-Host "Active window process: $($focusedProcess.ProcessName)"
```

Отримує об'єкт процесу для вікна, яке наразі має фокус клавіатури.
