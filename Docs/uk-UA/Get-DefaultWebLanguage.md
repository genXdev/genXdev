# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Отримує ключ мови веб-інтерфейсу за замовчуванням на основі поточних мовних налаштувань системи.

## Description

* Отримує поточні налаштування системної мови та культури та зіставляє їх
  з відповідним ключем словника веб-мови, який використовується службами
  перекладу.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Повертає "Англійська" для системи з англійською мовою, "Голландська" для системи з голландською мовою тощо.
