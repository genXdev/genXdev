# Get-DefaultWebbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Повертає налаштований веб-браузер за замовчуванням для поточного користувача.

## Description

* Отримує інформацію про стандартний веб-браузер системи шляхом запиту до реєстру Windows.
* Повертає хеш-таблицю, що містить назву браузера, опис, шлях до значка та шлях до виконуваного файлу.
* Функція перевіряє як налаштування користувача, так і загальносистемні реєстрації браузерів, щоб визначити браузер за замовчуванням.

## Syntax

```powershell
Get-DefaultWebbrowser [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebbrowser | Format-List
```

Отримайте детальну інформацію про браузер за замовчуванням.

### Example 2

```powershell
$browser = Get-DefaultWebbrowser
& $browser.Path https://www.github.com/
```

Запустити браузер за замовчуванням із зазначеним URL.
