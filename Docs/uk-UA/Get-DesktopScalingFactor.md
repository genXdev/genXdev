# Get-DesktopScalingFactor

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Отримує коефіцієнт масштабування дисплея Windows (налаштування DPI) для вказаного монітора.

## Description

* Отримує поточний коефіцієнт масштабування дисплея Windows, налаштований для монітора
  у системі.
* Коефіцієнт масштабування виражається у відсотках, де 100 позначає
  стандартне масштабування (96 DPI). Поширені значення: 100, 125, 150 та 200.
* Якщо монітор не вказано, повертає коефіцієнт масштабування для основного
  дисплея.

## Syntax

```powershell
Get-DesktopScalingFactor [[-Monitor] <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | — | — | 0 | `0` | Індекс монітора для перевірки (0 = основний монітор) |

## Examples

### Example 1

```powershell
Get-DesktopScalingFactor -Monitor 0
```

Повертає відсоток коефіцієнта масштабування для основного монітора.

### Example 2

```powershell
Get-DesktopScalingFactor 1
```

Повертає відсоток коефіцієнта масштабування для другого монітора, використовуючи позиційний параметр.
