# Get-DragForceByVelocityDensityAreaAndCoefficient

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Обчислює силу опору.

## Description

Використовує F = 1/2 C ρ A v².

## Syntax

```powershell
Get-DragForceByVelocityDensityAreaAndCoefficient -VelocityInMetersPerSecond <Double> -DensityInKilogramsPerCubicMeter <Double> -AreaInSquareMeters <Double> -Coefficient <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-VelocityInMetersPerSecond` | Double | ✅ | — | 0 | `0` | Швидкість у м/с |
| `-DensityInKilogramsPerCubicMeter` | Double | ✅ | — | 1 | `0` | Густина рідини в кг/м³ |
| `-AreaInSquareMeters` | Double | ✅ | — | 2 | `0` | Площа поперечного перерізу в м² |
| `-Coefficient` | Double | ✅ | — | 3 | `0` | Коефіцієнт лобового опору |
| `-As` | String | — | — | 4 | — | Одиниця виміру сили |

## Examples

### Example 1

```powershell
Get-DragForceByVelocityDensityAreaAndCoefficient -VelocityInMetersPerSecond 10 -DensityInKilogramsPerCubicMeter 1.225 -AreaInSquareMeters 1 -Coefficient 0.5 -As "poundforce"
```

Обчислює силу опору, використовуючи швидкість 10 м/с, густину повітря 1,225 кг/м³, площу 1 м² та коефіцієнт 0,5, з виведенням результату в фунт-силах.

### Example 2

```powershell
Get-DragForceByVelocityDensityAreaAndCoefficient 20 1.225 2 0.3
```

Обчислює силу опору за допомогою позиційних параметрів: швидкість 20 м/с, густина 1.225 кг/м³, площа 2 м², коефіцієнт 0.3.
