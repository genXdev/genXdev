# Get-EscapeVelocityByMassAndRadius

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Обчислює другу космічну швидкість.

## Description

Використовує v = sqrt(2 G M / r).

## Syntax

```powershell
Get-EscapeVelocityByMassAndRadius -MassInKilograms <Double> -RadiusInMeters <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Маса тіла в кг |
| `-RadiusInMeters` | Double | ✅ | — | 1 | `0` | Радіус у метрах |
| `-As` | String | — | — | 2 | — | Одиниця виведення для швидкості |

## Examples

### Example 1

```powershell
Get-EscapeVelocityByMassAndRadius -MassInKilograms 5.972e24 -RadiusInMeters 6371000 -As "km/h"
```

Обчислює швидкість втечі для тіла з масою та радіусом Землі.

### Example 2

```powershell
Get-EscapeVelocityByMassAndRadius 1e26 10000000
```

Обчислює швидкість втечі, використовуючи одиниці м/с за замовчуванням.
