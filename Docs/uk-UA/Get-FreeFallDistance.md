# Get-FreeFallDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Обчислює відстань падіння під час вільного падіння для заданої тривалості часу.

## Description

Обчислює відстань падіння під час вільного падіння за допомогою чисельного методу, який враховує опір повітря та кінцеву швидкість.

## Syntax

```powershell
Get-FreeFallDistance -DurationInSeconds <Double> [[-TerminalVelocityInMetersPerSecond] <Double>] [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | Тривалість падіння в секундах |
| `-TerminalVelocityInMetersPerSecond` | Double | — | — | 1 | `0` | Кінцева швидкість у метрах за секунду (за замовчуванням: 53) |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Прискорення вільного падіння в м/с² (за замовчуванням: 9.81) |
| `-As` | String | — | — | 3 | — | Одиниця виміру для вихідної відстані |

## Examples

### Example 1

```powershell
Get-FreeFallDistance -DurationInSeconds 10 -TerminalVelocityInMetersPerSecond 53 -As "feet"
```

Обчислює відстань падіння за 10 секунд у футах.

### Example 2

```powershell
Get-FreeFallDistance 5
```

Обчислює відстань за 5 секунд у метрах.
