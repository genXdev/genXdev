# Get-FreeFallHeight

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Обчислює висоту падіння під час вільного падіння за заданий проміжок часу.

## Description

Обчислює відстань падіння під час вільного падіння за допомогою чисельного методу, який враховує опір повітря та кінцеву швидкість.

## Syntax

```powershell
Get-FreeFallHeight -DurationInSeconds <Double> [[-TerminalVelocityInMs] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | Тривалість падіння в секундах |
| `-TerminalVelocityInMs` | Double | — | — | 1 | `0` | Кінцева швидкість у метрах за секунду (за замовчуванням: 53 м/с для людини) |
| `-As` | String | — | — | 2 | — | Одиниця виміру для висоти виводу |

## Examples

### Example 1

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -TerminalVelocityInMs 53
```

Обчислює висоту падіння за 10 секунд зі стандартною кінцевою швидкістю людини.

### Example 2

```powershell
Get-FreeFallHeight 5
```

Обчислює висоту падіння за 5 секунд за допомогою позиційного параметра та кінцевої швидкості за замовчуванням.

### Example 3

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -As "feet"
```

Обчислює висоту падіння за 10 секунд і повертає результат у футах.
