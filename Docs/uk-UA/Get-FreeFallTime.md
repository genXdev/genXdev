# Get-FreeFallTime

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Обчислює час, необхідний для падіння об'єкта з заданої висоти під час вільного падіння.

## Description

Обчислює тривалість часу, необхідного для падіння об'єкта із заданої висоти, використовуючи чисельний метод, який враховує опір повітря та кінцеву швидкість. Розрахунок використовує малі кроки часу для точного моделювання фізики падіння об'єктів з реалістичними обмеженнями кінцевої швидкості.

## Syntax

```powershell
Get-FreeFallTime -HeightInMeters <Double> [[-TerminalVelocityInMs] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-HeightInMeters` | Double | ✅ | — | 0 | `0` | Висота падіння в метрах |
| `-TerminalVelocityInMs` | Double | — | — | 1 | `0` | Кінцева швидкість у метрах за секунду (за замовчуванням: 53 м/с для людини) |
| `-As` | String | — | — | 2 | — | Одиниця виміру часу виведення |

## Examples

### Example 1

```powershell
Get-FreeFallTime -HeightInMeters 100 -TerminalVelocityInMs 53
```

Обчислює час, необхідний для падіння на 100 метрів зі стандартною кінцевою швидкістю людини.

### Example 2

```powershell
Get-FreeFallTime 50
```

Обчислює час, необхідний для падіння на 50 метрів, використовуючи позиційний параметр і кінцеву швидкість за замовчуванням.

### Example 3

```powershell
Get-FreeFallTime -HeightInMeters 100 -As "minutes"
```

Обчислює час, необхідний для падіння з висоти 100 метрів, і повертає результат у хвилинах.
