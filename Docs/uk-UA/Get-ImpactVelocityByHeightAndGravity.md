# Get-ImpactVelocityByHeightAndGravity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Обчислює швидкість удару за висотою.

## Description

Використовує v = sqrt(2 g h), ігноруючи опір повітря.

## Syntax

```powershell
Get-ImpactVelocityByHeightAndGravity -HeightInMeters <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-HeightInMeters` | Double | ✅ | — | 0 | `0` | Висота в метрах |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 1 | `0` | Гравітація в м/с². Стандартне значення 9,81. |
| `-As` | String | — | — | 2 | — | Одиниця виміру швидкості. За замовчуванням 'м/с'. |

## Examples

### Example 1

```powershell
Get-ImpactVelocityByHeightAndGravity -HeightInMeters 100 -As "km/h"
```

Обчислює швидкість удару при падінні з висоти 100 метрів і виводить у км/год.

### Example 2

```powershell
Get-ImpactVelocityByHeightAndGravity 50
```

Обчислює швидкість удару при падінні з висоти 50 метрів, використовуючи стандартні одиниці вимірювання.
