# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> Повертає true, якщо механізм перетворення тексту в мову говорить.

## Description

Перевіряє стан як стандартного, так і налаштованого синтезатора мовлення, щоб визначити, чи говорить зараз якийсь із них. Повертає true, якщо мовлення триває, і false в іншому випадку.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Повертає true, якщо механізм перетворення тексту в мову говорить.

### Example 2

```powershell
iss
```

Повертає true, якщо механізм синтезу мовлення говорить (використовуючи псевдонім).
