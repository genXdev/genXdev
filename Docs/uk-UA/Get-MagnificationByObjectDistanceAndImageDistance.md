# Get-MagnificationByObjectDistanceAndImageDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Обчислює збільшення для лінзи.

## Description

Використовується m = - (відстань зображення / відстань об'єкта).

## Syntax

```powershell
Get-MagnificationByObjectDistanceAndImageDistance -ObjectDistanceInMeters <Double> -ImageDistanceInMeters <Double> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ObjectDistanceInMeters` | Double | ✅ | — | 0 | `0` | Відстань об'єкта в метрах |
| `-ImageDistanceInMeters` | Double | ✅ | — | 1 | `0` | Відстань зображення в метрах |

## Examples

### Example 1

```powershell
Get-MagnificationByObjectDistanceAndImageDistance -ObjectDistanceInMeters 0.5 -ImageDistanceInMeters 1
```

Обчислює збільшення з відстанню до об'єкта 0.5 м та відстанню до зображення 1 м.

### Example 2

```powershell
Get-MagnificationByObjectDistanceAndImageDistance 0.3 0.6
```

Обчислює збільшення за допомогою позиційних параметрів.
