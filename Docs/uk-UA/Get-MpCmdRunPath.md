# Get-MpCmdRunPath

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Отримує шлях до виконуваного файлу MpCmdRun.exe Захисника Windows.

## Description

* Знаходить і повертає повний шлях до командного рядка Захисника Windows
  (MpCmdRun.exe).
* Командлет перевіряє стандартне розташування встановлення в Program Files
  і надає відповідну обробку помилок, якщо файл не знайдено.

## Syntax

```powershell
Get-MpCmdRunPath [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$defenderPath = Get-MpCmdRunPath
```

Отримує повний шлях до інструменту командного рядка Windows Defender.
