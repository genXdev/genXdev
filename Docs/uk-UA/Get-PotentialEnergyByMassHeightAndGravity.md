# Get-PotentialEnergyByMassHeightAndGravity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Обчислює потенціальну енергію гравітації.

## Description

Використовує PE = m g h.

## Syntax

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms <Double> -HeightInMeters <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Маса в кг |
| `-HeightInMeters` | Double | ✅ | — | 1 | `0` | Висота в метрах |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Гравітація в м/с² (за замовчуванням: 9,81) |
| `-As` | String | — | — | 3 | — | Одиниця виміру енергії |

## Examples

### Example 1

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms 10 -HeightInMeters 5 -As "calories"
```

Обчисліть потенціальну енергію гравітації для тіла масою 10 кг на висоті 5 метрів, виводячи результат у калоріях.

### Example 2

```powershell
Get-PotentialEnergyByMassHeightAndGravity 5 10
```

Демонструє використання позиційних параметрів для обчислення потенціальної енергії.
