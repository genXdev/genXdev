# Get-RefractionAngleByIncidentAngleAndIndices

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Обчислює кут заломлення за законом Снелла.

## Description

Використовує θ2 = arcsin( (n1 / n2) sin θ1 ).

## Syntax

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees <Double> -IndexOfRefraction1 <Double> -IndexOfRefraction2 <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-IncidentAngleInDegrees` | Double | ✅ | — | 0 | `0` | Кут падіння в градусах |
| `-IndexOfRefraction1` | Double | ✅ | — | 1 | `0` | Показник заломлення першого середовища |
| `-IndexOfRefraction2` | Double | ✅ | — | 2 | `0` | Показник заломлення другого середовища |
| `-As` | String | — | — | 3 | — | Одиниця виводу для кута |

## Examples

### Example 1

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees 30 -IndexOfRefraction1 1 -IndexOfRefraction2 1.33 -As "radians"
```

Обчислює кут заломлення, коли світло переходить з повітря (n=1) у воду (n=1,33) під кутом падіння 30 градусів.

### Example 2

```powershell
Get-RefractionAngleByIncidentAngleAndIndices 45 1 1.5
```

Обчислює кут заломлення, коли світло переходить із повітря (n=1) у скло (n=1.5) під кутом падіння 45 градусів.
