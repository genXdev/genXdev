# Get-SoundTravelDistanceByTime

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Обчислює відстань, яку проходить звук за заданий час.

## Description

Використовує формулу відстань = швидкість * час, зі стандартною швидкістю звуку в повітрі.

## Syntax

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds <Double> [[-As] <String>] [<CommonParameters>]

Get-SoundTravelDistanceByTime [[-SpeedOfSoundInMetersPerSecond] <Double>] [<CommonParameters>]

Get-SoundTravelDistanceByTime -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-TimeInSeconds` | Double | ✅ | — | 0 | `0` | Час у секундах |
| `-SpeedOfSoundInMetersPerSecond` | Double | — | — | 1 | `0` | Швидкість звуку в м/с (за замовчуванням: 343) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | Середовище, через яке поширюється звук *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | Одиниця виміру для вихідної відстані |

## Examples

### Example 1

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds 5 -Medium "water" -As "kilometers"
```

Обчислює відстань, яку звук проходить у воді за 5 секунд, і перетворює результат у кілометри.

### Example 2

```powershell
Get-SoundTravelDistanceByTime 10 -SpeedOfSoundInMetersPerSecond 1480
```

Обчислює відстань, використовуючи конкретну швидкість звуку.
