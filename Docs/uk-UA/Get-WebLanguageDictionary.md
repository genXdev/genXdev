# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Повертає обернений словник для всіх мов, підтримуваних Пошуком Google.

## Description

* Повертає словник зіставлення назв мов із відповідними кодами мов для всіх мов, які підтримуються Google Пошуком.
* Корисно для пошуку кодів мов у веб-запитах і завданнях перекладу.

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

Отримує словник, який зіставляє назви мов з їхніми кодами.
