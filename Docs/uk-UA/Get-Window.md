# Get-Window

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Отримує інформацію про вікна для вказаних процесів або дескрипторів вікон.

## Description

* Отримує інформацію про вікна за іменем процесу, ідентифікатором або дескриптором вікна.
* Повертає об'єкти WindowObj, що містять деталі про головні вікна відповідних процесів.
* Підтримує символи підстановки під час пошуку за іменем процесу.

## Syntax

```powershell
Get-Window [[-ProcessName] <String>] [-ProcessId <Int32>] [-WindowHandle <Int64>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ProcessName` | String | — | ✅ (ByPropertyName) | 0 | — | Назва процесу для отримання інформації про вікно 🌐 *Supports wildcards* |
| `-ProcessId` | Int32 | — | ✅ (ByPropertyName) | Named | `0` | Ідентифікатор процесу для отримання інформації про вікно |
| `-WindowHandle` | Int64 | — | ✅ (ByPropertyName) | Named | `0` | Дескриптор вікна, для якого потрібно отримати інформацію |

## Examples

### Example 1

```powershell
Get-Window -ProcessName "notepad"
```

Отримує інформацію про вікна для всіх процесів з ім'ям "notepad".

### Example 2

```powershell
Get-Window -ProcessId 1234
```

Отримує інформацію про вікна для процесу з ідентифікатором 1234.

### Example 3

```powershell
Get-Window -WindowHandle 45678
```

Отримує інформацію про вікно для вікна з дескриптором 45678.
