# Initialize-SearchPaths

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Ініціалізує та налаштовує системні шляхи пошуку для керування пакетами.

## Description

* Створює комплексний список шляхів пошуку, поєднуючи стандартні системні розташування, шляхи Chocolatey, шляхи інструментів розробки та власні шляхи пакетів.
* Оновлює системну змінну середовища PATH цими об'єднаними шляхами.

## Syntax

```powershell
Initialize-SearchPaths [[-WorkspaceFolder] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-WorkspaceFolder` | String | — | — | 0 | — | Шлях до папки робочої області, який використовувати для модулів node та шляхів PowerShell. |

## Examples

### Example 1

```powershell
Initialize-SearchPaths
```

Ініціалізує шляхи пошуку, використовуючи теку робочого простору за замовчуванням.

### Example 2

```powershell
Initialize-SearchPaths -WorkspaceFolder "C:\workspace"
```

Ініціалізує шляхи пошуку за допомогою певної папки робочої області.
