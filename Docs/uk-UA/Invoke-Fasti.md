# Invoke-Fasti

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** `fasti`

## Synopsis

> Витягує архівні файли в поточному каталозі до власних папок і видаляє їх після цього.

## Description

Автоматично витягує поширені архівні формати (zip, 7z, tar тощо), знайдені в поточному каталозі, в окремі папки, названі за іменем кожного архіву. Після успішного витягування оригінальні архівні файли видаляються. Для роботи потрібно, щоб на системі було встановлено 7-Zip.

## Syntax

```powershell
Invoke-Fasti [[-Password] <String>] [-ExtractOutputToo] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Password` | String | — | — | 0 | — | Введіть пароль для зашифрованого(их) архіву(ів) |
| `-ExtractOutputToo` | SwitchParameter | — | — | Named | — | Рекурсивно витягувати архіви, знайдені в розпакованих теках |

## Examples

### PS C:\Downloads> Invoke-Fasti

```powershell
PS C:\Downloads> Invoke-Fasti
```

### PS C:\Downloads> fasti

```powershell
PS C:\Downloads> fasti
```
