# Invoke-OnEachGenXdevModule

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** `foreach-genxdev-module-do`

## Synopsis

> Виконує блок сценарію для кожного модуля GenXdev у робочому просторі.

## Description

Ця функція проходить через модулі GenXdev у робочій області та виконує наданий блок сценарію для кожного модуля. Вона може фільтрувати модулі за шаблоном назви, виключати локальні модулі, включати лише опубліковані модулі або обробляти сценарії замість модулів. Функція автоматично переходить до правильної теки модуля перед виконанням блоку сценарію.

## Syntax

```powershell
Invoke-OnEachGenXdevModule -Script <ScriptBlock> [[-ModuleName] <String[]>] [-FromScripts] [-IncludeScripts] [-NoLocal] [-OnlyPublished] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Script` | ScriptBlock | ✅ | — | 0 | — | Блок сценарію для виконання кожного модуля GenXdev |
| `-ModuleName` | String[] | — | — | 1 | `@('GenXdev*')` | Фільтр для застосування до назв модулів 🌐 *Supports wildcards* |
| `-NoLocal` | SwitchParameter | — | — | Named | — | Виключає локальні модулі розробки з обробки |
| `-OnlyPublished` | SwitchParameter | — | — | Named | — | Включає лише опубліковані модулі, які мають файли LICENSE та README.md |
| `-FromScripts` | SwitchParameter | — | — | Named | — | Обробляти каталог сценаріїв замість каталогів модулів |
| `-IncludeScripts` | SwitchParameter | — | — | Named | — | Включає каталог сценаріїв на додаток до звичайних модулів |

## Examples

### Invoke-OnEachGenXdevModule -Script { Write-Host $args[0].Name }

```powershell
Invoke-OnEachGenXdevModule -Script { Write-Host $args[0].Name }
```

Виводить список усіх назв модулів GenXdev.

### foreach-genxdev-module-do {     param($ModuleObj, $isScriptsFolder, $isSubModule, $subModuleName)     Get-ChildItem } -ModuleName "GenXdev.AI"

```powershell
foreach-genxdev-module-do {
    param($ModuleObj, $isScriptsFolder, $isSubModule, $subModuleName)
    Get-ChildItem
} -ModuleName "GenXdev.AI"
```

Використовує псевдонім для виведення вмісту каталогу модуля GenXdev.AI.
