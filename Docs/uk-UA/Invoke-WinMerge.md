# Invoke-WinMerge

> **Module:** GenXdev.AI | **Type:** Function | **Aliases:** —

## Synopsis

> Запускає WinMerge для порівняння двох файлів поруч.

## Description

Запускає застосунок WinMerge для порівняння вихідних і цільових файлів у режимі перегляду різниці поруч. Функція перевіряє наявність обох вхідних файлів і переконується, що WinMerge належним чином встановлено перед запуском. Забезпечує додаткову функцію очікування для призупинення виконання до закриття WinMerge.

## Syntax

```powershell
Invoke-WinMerge -SourcecodeFilePath <String> -TargetcodeFilePath <String> [-Wait] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-SourcecodeFilePath` | String | ✅ | — | 0 | — | Шлях до вихідного файлу для порівняння |
| `-TargetcodeFilePath` | String | ✅ | — | 1 | — | Шлях до цільового файлу для порівняння |
| `-Wait` | SwitchParameter | — | — | Named | — | Зачекайте, поки WinMerge закриється, перш ніж продовжувати |

## Examples

### Invoke-WinMerge -SourcecodeFilePath "C:\source\file1.txt" `                 -TargetcodeFilePath "C:\target\file2.txt" `                 -Wait

```powershell
Invoke-WinMerge -SourcecodeFilePath "C:\source\file1.txt" `
                -TargetcodeFilePath "C:\target\file2.txt" `
                -Wait
```

### merge "C:\source\file1.txt" "C:\target\file2.txt"

```powershell
merge "C:\source\file1.txt" "C:\target\file2.txt"
```
