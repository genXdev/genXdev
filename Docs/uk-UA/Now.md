# Now

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Повертає поточну системну дату та час як об'єкт DateTime.

## Description

Надає простий спосіб отримати поточну системну дату та час без жодних параметрів. Повертає стандартний об'єкт DateTime .NET, який можна використовувати для обчислень дати та часу, форматування та порівнянь.

## Syntax

```powershell
Now [<CommonParameters>]
```

## Examples

### Now Returns the current system date and time as a DateTime object

```powershell
Now
Returns the current system date and time as a DateTime object
```

### $timestamp = Now Stores the current date and time in a variable for later use

```powershell
$timestamp = Now
Stores the current date and time in a variable for later use
```

## Outputs

- `DateTime`
