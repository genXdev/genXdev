# Open-BuiltWithSiteInfo

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Відкриває запити вебсайту BuiltWith у веббраузері.

## Description

Відкриває запит BuiltWith у веб-браузері з налаштовуваними параметрами монітора та параметрами поведінки браузера. Функція аналізує технологічні стеки веб-сайтів, запитуючи службу профілювання технологій BuiltWith.com, яка надає детальну інформацію про веб-технології, фреймворки та сервіси, що використовуються веб-сайтами.

BuiltWith.com — це комплексна служба пошуку технологій, яка ідентифікує веб-технології, фреймворки, системи керування вмістом, інструменти аналітики, хостинг-провайдерів та інші програмні компоненти, що використовуються будь-яким веб-сайтом. Ця функція забезпечує інтерфейс PowerShell для швидкого аналізу технологічних стеків без необхідності вручну переходити на веб-сайт BuiltWith.

## Syntax

```powershell
Open-BuiltWithSiteInfo -Queries <String[]> [[-Language] <String>] [[-Monitor] <Int32>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-BuiltWithSiteInfo -Queries "microsoft.com" -Monitor 0

```powershell
Open-BuiltWithSiteInfo -Queries "microsoft.com" -Monitor 0
```

Відкриває аналіз технологій BuiltWith для Microsoft.com на моніторі за замовчуванням.

### "microsoft.com" | Open-BuiltWithSiteInfo -m -1

```powershell
"microsoft.com" | Open-BuiltWithSiteInfo -m -1
```

Аналізує microsoft.com за допомогою введення через конвеєр, при цьому позиціонування монітора ігнорується.
