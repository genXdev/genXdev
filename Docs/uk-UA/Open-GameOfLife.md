# Open-GameOfLife

> **Module:** GenXdev.Queries.Websites | **Type:** Function | **Aliases:** `gameoflife`, `conway`

## Synopsis

> Відкриває симуляцію гри в життя Конвея у веб-браузері.

## Description

Відкриває інтерактивну симуляцію гри «Життя» Конвея у вікні веб-браузера з розширеними можливостями позиціонування та налаштування. Гра «Життя» Конвея — це клітинний автомат, розроблений математиком Джоном Конвеєм у 1970 році, що складається з сітки клітин, які можуть перебувати в одному з двох станів: живі або мертві.

Ця функція забезпечує комплексне керування браузером, включаючи позиціонування вікна, вибір браузера, приватний перегляд та можливості автоматизованої взаємодії. Симуляція запускається за адресою https://conway.genxdev.net/ та підтримує різні режими взаємодії.

## Examples

### Open-GameOfLife -Monitor 1 -FullScreen

```powershell
Open-GameOfLife -Monitor 1 -FullScreen
```

Відкриває гру «Життя» Конвея в повноекранному режимі на моніторі 1.

### Open-GameOfLife -Language "French" -Chrome -Private

```powershell
Open-GameOfLife -Language "French" -Chrome -Private
```

Відкриває Game of Life французькою мовою в Chrome у режимі інкогніто.

### gameoflife -m 0 -app

```powershell
gameoflife -m 0 -app
```

Відкриває гру «Життя» на основному моніторі в режимі додатка за допомогою псевдоніма.

### conway -Edge -Left -Width 800 -Height 600

```powershell
conway -Edge -Left -Width 800 -Height 600
```

Відкриває гру 'Життя' Конвея в Microsoft Edge, розташовану зліва, із заданими розмірами.
