# Open-StackOverflowQuery

> **Module:** GenXdev.Queries.Webbrowser | **Type:** Function | **Aliases:** `qso`

## Synopsis

> Відкриває пошукові запити Stack Overflow у веб-браузері.

## Description

Відкриває один або кілька пошукових запитів Stack Overflow у веб-браузері з
розширеними параметрами конфігурації браузера та дисплея. Ця функція є
покращеною оболонкою функції пошуку Stack Overflow з широкими
можливостями налаштування позиції вікна, вибору браузера та поведінки.

Ключові особливості:
- Підтримка кількох пошукових запитів з введенням через конвеєр
- Розумне виявлення та вибір браузера (Edge, Chrome, Firefox, усі браузери)
- Розширене позиціонування вікна (ліворуч, праворуч, зверху, знизу, по центру, на весь екран)
- Підтримка кількох моніторів з автоматичним або ручним вибором монітора
- Підтримка приватного/інкогніто режиму перегляду
- Режим додатка для перегляду без відволікань
- Підтримка локалізації мови для міжнародних результатів пошуку
- Параметри блокування розширень і спливаючих вікон
- Керування фокусом та маніпуляції з вікнами
- Автоматизація натискання клавіш у вікнах браузера
- Опції повернення URL для робочих процесів автоматизації

Функція автоматично визначає можливості системи та відповідно
налаштовує поведінку. Для браузерів, не встановлених у системі,
операції безшумно пропускаються без помилок.

```powershell
Open-StackOverflowQuery -Queries <String[]> [[-Language] <String>] [-AcceptLang <String>] [-All] [-ApplicationMode] [-Bottom] [-Centered] [-Chrome] [-Chromium] [-ClearSession] [-DisablePopupBlocker] [-Edge] [-Firefox] [-FocusWindow] [-Force] [-FullScreen] [-Headless] [-Height <Int32>] [-KeysToSend <String[]>] [-Left] [-Maximize] [-Monitor <Int32>] [-NewWindow] [-NoBorders] [-NoBrowserExtensions] [-PassThru] [-PlayWright] [-Private] [-RestoreFocus] [-ReturnOnlyURL] [-ReturnURL] [-Right] [-SendKeyDelayMilliSeconds <Int32>] [-SendKeyEscape] [-SendKeyHoldKeyboardFocus] [-SendKeyUseShiftEnter] [-SessionOnly] [-SetForeground] [-SetRestored] [-SideBySide] [-SkipSession] [-Top] [-Webkit] [-Width <Int32>] [-X <Int32>] [-Y <Int32>] [<CommonParameters>]
```

## Examples

### Open-StackOverflowQuery -Queries "powershell array" -Monitor 0

```powershell
Open-StackOverflowQuery -Queries "powershell array" -Monitor 0
```

Відкриває пошук Stack Overflow для "powershell array" на головному моніторі.

### qso "powershell array" -mon 0

```powershell
qso "powershell array" -mon 0
```

Відкриває пошук на Stack Overflow з використанням псевдоніма та позиціонуванням монітора.

### "powershell", "array manipulation" | Open-StackOverflowQuery -Language "English" -Chrome

```powershell
"powershell", "array manipulation" | Open-StackOverflowQuery -Language "English" -Chrome
```

Відкриває кілька пошуків у Stack Overflow у Chrome з англійською мовою переваги.

### Open-StackOverflowQuery -Queries "c# linq" -ReturnURL

```powershell
Open-StackOverflowQuery -Queries "c# linq" -ReturnURL
```

Повертає URL пошуку у Stack Overflow, не відкриваючи браузер.
