# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Вимовляє поточну дату за допомогою синтезу тексту в мовлення.

## Description

Перетворює поточну дату у формат природної мови та озвучує її за допомогою системного механізму перетворення тексту в мовлення. Дата вимовляється у форматі: «Сьогодні [день тижня], [місяць] [день] [рік]»

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
