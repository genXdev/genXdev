# Set-ForegroundWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Переводить указане вікно на передній план і робить його активним вікном.

## Description

Робить вікно передплановим за допомогою кількох методів Win32 API для максимальної надійності. Спочатку намагається використати API SwitchToThisWindow, потім, за необхідності, повертається до SetForegroundWindow.

## Syntax

```powershell
Set-ForegroundWindow -WindowHandle <IntPtr> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-WindowHandle` | IntPtr | ✅ | — | 0 | `0` | Дескриптор вікна, який встановити як активне вікно |

## Examples

### Example 1

```powershell
$hwnd = (Get-Process notepad).MainWindowHandle
Set-ForegroundWindow -WindowHandle $hwnd
```

Виводить вікно Блокнота на передній план, використовуючи дескриптор вікна.

### Example 2

```powershell
$hwnd = (Get-Process notepad).MainWindowHandle
Set-ForegroundWindow $hwnd
```

Використовує позиційний параметр для компактного синтаксису.
