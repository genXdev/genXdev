# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> Змінює поточне розташування на батьківський каталог та виводить список його вмісту.

## Description

* Переходить на один рівень вгору в ієрархії файлової системи.
* Відображає вміст нового поточного каталогу.
* Надає зручний псевдонім '..' для швидкої навігації по каталогах.

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

Змінює батьківський каталог і виводить список його вмісту.

### Example 2

```powershell
..
```

Використовує псевдонім для переходу до батьківського каталогу.
