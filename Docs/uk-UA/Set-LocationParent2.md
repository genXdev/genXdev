# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> Переміщується на два рівні вгору в ієрархії файлової системи.

## Description

* Змінює поточний робочий каталог на каталог батька батька (на два рівні вгору).
* Відображає вміст отриманого каталогу.

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Переходить до батьківського каталогу та відображає його вміст.

### Example 2

```powershell
...
```

Використовує псевдонім '...' для тієї самої операції.
