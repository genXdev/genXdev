# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....`

## Synopsis

> Переміщується на чотири рівні каталогів вгору в ієрархії файлової системи.

## Description

* Переміщується на чотири рівні каталогів вгору від поточного місця у файловій системі.
* Після навігації відображає вміст отриманого каталогу.

## Syntax

```powershell
Set-LocationParent4 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

Переходить на чотири рівні каталогів вгору від поточного розташування.

### Example 2

```powershell
.....
```

Використовує псевдонім для переходу на чотири рівні каталогів угору.
