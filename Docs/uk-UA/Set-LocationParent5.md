# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......`

## Synopsis

> Переміщується на п'ять рівнів вгору в ієрархії файлової системи.

## Description

* Змінює поточний робочий каталог, переходячи на п'ять рівнів батьківських каталогів вище.
* Відображає вміст отриманого каталогу.
* Надає зручний скорочений псевдонім '......' для швидкої навігації.

## Syntax

```powershell
Set-LocationParent5 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

Переміщується на п'ять рівнів директорій вгору від поточного розташування.

### Example 2

```powershell
......
```

Використовує псевдонім для переходу на п'ять рівнів каталогів вгору.
