# Set-WindowsWallpaper

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Встановлює випадкове тло з вказаної папки.

## Description

* Випадково обирає файл зображення з вказаної теки та встановлює його як шпалери робочого столу Windows.
* Підтримує формати зображень JPG/JPEG та за замовчуванням налаштовує шпалери на "підгонку" під екран.

## Syntax

```powershell
Set-WindowsWallpaper [[-InputObject] <Object>] [-AllDrives] [-NoRecurse] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputObject` | Object | — | ✅ (ByValue, ByPropertyName) | 0 | — | Шлях до каталогу, що містить зображення шпалер |
| `-AllDrives` | SwitchParameter | — | — | Named | `False` | Пошук на всіх доступних дисках |
| `-NoRecurse` | SwitchParameter | — | — | Named | `False` | Не входити до підкаталогів |

## Examples

### Example 1

```powershell
Set-WindowsWallpaper -InputObject "C:\Wallpapers\*.jpg"
```

Встановлює випадковий шпалери з каталогу C:\Wallpapers.

### Example 2

```powershell
nextbg
```

Встановлює випадковий шпалери з каталогу за замовчуванням за допомогою псевдоніма.
