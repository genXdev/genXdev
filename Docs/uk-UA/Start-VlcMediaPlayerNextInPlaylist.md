# Start-VlcMediaPlayerNextInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcnext`

## Synopsis

> Переводить VLC Media Player до наступного елемента в поточному списку відтворення.

## Description

Ця функція надсилає натискання клавіші 'n' до VLC Media Player, щоб перейти до наступного треку або медіа-елемента в поточному завантаженому списку відтворення. Вона фокусує вікно VLC, надсилає команду переходу та відновлює фокус на консолі PowerShell. Функція включає підтримку ShouldProcess для запитів підтвердження, коли це необхідно.

## Syntax

```powershell
Start-VlcMediaPlayerNextInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerNextInPlaylist

```powershell
Start-VlcMediaPlayerNextInPlaylist
```

### vlcnext

```powershell
vlcnext
```
