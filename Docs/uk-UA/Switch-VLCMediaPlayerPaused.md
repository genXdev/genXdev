# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> Перемикає стан паузи/відтворення медіаплеєра VLC.

## Description

Ця функція надсилає клавішу пробілу до VLC Media Player, щоб перемикати між станами паузи та відтворення. Після надсилання команди клавіші вона автоматично повертає фокус на початкове вікно.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

Перемикає стан паузи/відтворення медіаплеєра VLC.

### vlcpause

```powershell
vlcpause
```

Використовує псевдонім для перемикання стану пауза/відтворення.

### vlcplay

```powershell
vlcplay
```

Використовує альтернативний псевдонім для перемикання стану паузи/відтворення.
