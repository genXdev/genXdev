# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> Перемикає режим повтору в медіаплеєрі VLC.

## Description

Ця функція надсилає команду клавіші 'r' до VLC Media Player для перемикання між різними режимами повтору (без повтору, повтор поточного, повтор усіх). Функція відкриває VLC, якщо він ще не запущений, і відновлює фокус на попередньому вікні після надсилання команди.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
