# VSCode

> **Module:** GenXdev.Coding | **Type:** Function | **Aliases:** —

## Synopsis

> Відкриває один або декілька файлів у Visual Studio Code.

## Description

Ця функція приймає шляхи до файлів і відкриває їх у Visual Studio Code. Вона розширює шляхи та перевіряє існування файлів перед спробою відкрити їх. Функція підтримує як прямі шляхи до файлів, так і вхідні дані через конвеєр, що робить її ідеальною для швидкого відкриття кількох файлів з пошуку в терміналі.

## Syntax

```powershell
VSCode -FilePath <String[]> [-Copilot] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String[] | ✅ | ✅ (ByValue) | 0 | — | Шлях до файлу, який потрібно відкрити у VSCode |
| `-Copilot` | SwitchParameter | — | — | Named | — | Додати вихідний файл до сеансу редагування Copilot |

## Examples

### VSCode -FilePath "C:\path\to\file.txt" -Copilot

```powershell
VSCode -FilePath "C:\path\to\file.txt" -Copilot
```

### Get-ChildItem *.js -Recurse | VSCode

```powershell
Get-ChildItem *.js -Recurse | VSCode
```
