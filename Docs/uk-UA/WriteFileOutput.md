# WriteFileOutput

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Виводить інформацію про файли з гіперпосиланням на відображувані імена для покращеного досвіду користувача.

## Description

Ця функція обробляє вхідні файлові об'єкти та виводить їх у зручному для користувача форматі з гіперпосиланнями на назви файлів при відображенні в консолі. Вона підтримує різні типи вхідних даних, включаючи рядки, об'єкти FileInfo та інші типи об'єктів. Функція автоматично визначає, чи вивід перенаправляється або захоплюється в каналі, і відповідно змінює свою поведінку.

## Syntax

```powershell
WriteFileOutput -CallerInvocation <Object> -Input <Object> [-FullPaths] [-Prefix <String>] [-RelativeBasePath <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-CallerInvocation` | Object | ✅ | ✅ (ByValue) | 0 | — | Інформація про виклик з викликаючої функції |
| `-Input` | Object | ✅ | ✅ (ByValue) | 1 | — | Зовнішній об'єкт для обробки, який може бути шляхом до файлу або об'єктом |
| `-Prefix` | String | — | — | Named | — | Необов'язковий рядковий префікс для додавання до виводу для додаткового контексту |
| `-RelativeBasePath` | String | — | — | Named | — | Базовий шлях для створення відносних шляхів файлів у вихідних даних |
| `-FullPaths` | SwitchParameter | — | — | Named | — | Змушує вихідні дані використовувати повні абсолютні шляхи замість відносних |

## Examples

### WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"

```powershell
WriteFileOutput -CallerInvocation $MyInvocation -Input "C:\temp\file.txt"
```

### Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_

```powershell
Get-ChildItem | WriteFileOutput -CallerInvocation $MyInvocation -Input $_
```
