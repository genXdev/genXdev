# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Повертає рядок (із зміненою відступністю) наданого рядка скриптового блоку

## Description

Змінює відступ рядка скриптблока, зберігаючи початкові відступи кодового блоку

## Syntax

```powershell
alignScript [<CommonParameters>]
```
