# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Встановлює конфігурацію другого монітора за замовчуванням.

## Description

Встановлює поведінку за замовчуванням для відкриття вікон GenXdev на вторинному моніторі. Це корисно для користувачів з одним монітором або для тих, хто віддає перевагу розташуванню вікон поруч. Дивіться також командлети 'sidebyside' та 'restoredefaultmonitor'.

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Встановлює типові значення для відкриття вікон GenXdev на вторинному моніторі
