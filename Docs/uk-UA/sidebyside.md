# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Встановлює стандартну конфігурацію поруч.

## Description

Встановлює поведінку за замовчуванням для відкриття вікон GenXdev поруч із PowerShell.
Це корисно для користувачів з одним монітором або тих, хто віддає перевагу розташуванню вікон поруч.
Див. також командлети 'secondscreen' та 'restoredefaultmonitor'.

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

Встановлює типові параметри відкриття вікон GenXdev поряд із PowerShell
