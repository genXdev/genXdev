# ConvertTo-HashTable

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Chuyển đổi PSCustomObject thành HashTable một cách đệ quy.

## Description

* Hàm này chuyển đổi một PSCustomObject và tất cả các thuộc tính PSCustomObject lồng nhau của nó thành HashTables.
* Nó xử lý các mảng và các loại tập hợp khác bằng cách xử lý từng phần tử một cách đệ quy.

## Syntax

```powershell
ConvertTo-HashTable -InputObject <Object> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputObject` | Object | ✅ | ✅ (ByValue) | 0 | — | PSCustomObject cần chuyển đổi thành HashTable |

## Examples

### Example 1

```powershell
$object = [PSCustomObject]@{
    Name = "John"
    Age = 30
    Details = [PSCustomObject]@{
        City = "New York"
    }
}
$hashTable = ConvertTo-HashTable -InputObject $object
```

Chuyển đổi một PSCustomObject thành HashTable với các thuộc tính lồng nhau.
