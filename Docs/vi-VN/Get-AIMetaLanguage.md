# Get-AIMetaLanguage

> **Module:** GenXdev.AI.Queries | **Type:** Cmdlet | **Aliases:** `getimgmetalang`

## Synopsis

> Lấy ngôn ngữ mặc định đã cấu hình cho các thao tác siêu dữ liệu hình ảnh.

## Description

* Cmdlet này truy xuất ngôn ngữ mặc định được sử dụng bởi mô-đun GenXdev.AI
  cho các thao tác siêu dữ liệu hình ảnh.
* Nó kiểm tra các biến Global trước (trừ khi chỉ định SkipSession), sau đó
  dự phòng vào các tùy thích bền vững, và cuối cùng sử dụng các mặc định hệ thống.

## Syntax

```powershell
Get-AIMetaLanguage [[-Language] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Language` | String | — | — | 0 | — | Ngôn ngữ mặc định cho các thao tác siêu dữ liệu hình ảnh |
| `-PreferencesDatabasePath` | String | — | — | Named | — | Đường dẫn cơ sở dữ liệu cho các tệp dữ liệu ưu tiên |
| `-SessionOnly` | SwitchParameter | — | — | Named | `False` | Sử dụng cài đặt thay thế được lưu trong phiên làm việc cho các tùy chọn AI như Ngôn ngữ, Bộ sưu tập hình ảnh, v.v. |
| `-ClearSession` | SwitchParameter | — | — | Named | `False` | Xóa cài đặt phiên (biến toàn cục) trước khi truy xuất |
| `-SkipSession` | SwitchParameter | — | — | Named | `False` | Không sử dụng các thiết lập thay thế được lưu trong phiên làm việc cho các tùy chọn AI như Ngôn ngữ, Bộ sưu tập hình ảnh, v.v. |

## Examples

### Example 1

```powershell
Get-AIMetaLanguage
```

Lấy ngôn ngữ hiện đang được cấu hình từ các biến toàn cục hoặc tùy chọn.

### Example 2

```powershell
Get-AIMetaLanguage -SkipSession
```

Lấy ngôn ngữ đã cấu hình từ các tùy chọn ưu tiên liên tục, bỏ qua phiên làm việc.

### Example 3

```powershell
Get-AIMetaLanguage -ClearSession
```

Xóa cài đặt phiên và lấy ngôn ngữ từ tùy chọn cố định.
