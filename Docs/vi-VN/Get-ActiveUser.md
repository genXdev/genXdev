# Get-ActiveUser

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** `gusers`

## Synopsis

> Truy xuất danh sách các tên người dùng duy nhất từ các quy trình hệ thống đang hoạt động.

## Description

Truy vấn tất cả các tiến trình đang chạy trên hệ thống, trích xuất tên người dùng liên quan đến mỗi tiến trình, và trả về danh sách người dùng đã được loại bỏ trùng lặp có các tiến trình đang hoạt động. Điều này hữu ích cho quản trị hệ thống và giám sát bảo mật.

## Syntax

```powershell
Get-ActiveUser [<CommonParameters>]
```

## Examples

### Get-ActiveUser Returns a list of all unique usernames with active processes.

```powershell
Get-ActiveUser
Returns a list of all unique usernames with active processes.
```

### gusers Uses the alias to get the same results. ##############################################################################

```powershell
gusers
Uses the alias to get the same results.
##############################################################################
```
