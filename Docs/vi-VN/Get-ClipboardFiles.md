# Get-ClipboardFiles

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `getclipfiles`, `gcbf`

## Synopsis

> Lấy các tệp từ bảng tạm Windows đã được thiết lập cho các thao tác tệp như sao chép/dán.

## Description

* Hàm này truy xuất các đường dẫn tệp từ clipboard Windows đã được thiết lập trước đó cho các thao tác tệp.
* Nó tự động xử lý cả chế độ luồng STA và MTA, đảm bảo tương thích trên các ngữ cảnh thực thi PowerShell khác nhau.
* Hàm xác thực sự tồn tại của tệp và chỉ trả về các tệp/thư mục tồn tại dưới dạng các đối tượng tương tự như đầu ra của Get-ChildItem hoặc Get-Item.

## Syntax

```powershell
Get-ClipboardFiles [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-ClipboardFiles
```

Lấy tất cả tệp hiện có trong bảng tạm dưới dạng các đối tượng hệ thống tệp.

### Example 2

```powershell
$clipboardFiles = Get-ClipboardFiles
$clipboardFiles | ForEach-Object { Write-Host $_.FullName }
```

Lấy các tệp trong clipboard và hiển thị đường dẫn đầy đủ của chúng.

### Example 3

```powershell
Get-ClipboardFiles | Where-Object { $_.Extension -eq '.txt' }
```

Chỉ lấy các tệp văn bản từ bảng tạm.
