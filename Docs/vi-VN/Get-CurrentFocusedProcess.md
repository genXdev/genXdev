# Get-CurrentFocusedProcess

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Truy xuất đối tượng tiến trình của cửa sổ hiện đang có tiêu điểm bàn phím.

## Description

* Lệnh ghép này sử dụng các lệnh gọi API Windows thông qua P/Invoke để xác định và trả về đối tượng Process liên kết với cửa sổ đang được chọn hiện tại.
* Nó tận dụng các hàm GetForegroundWindow và GetWindowThreadProcessId từ User32.dll để xác định cửa sổ nào đang có tiêu điểm và lấy ID tiến trình liên kết của nó.

## Syntax

```powershell
Get-CurrentFocusedProcess [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$focusedProcess = Get-CurrentFocusedProcess
Write-Host "Active window process: $($focusedProcess.ProcessName)"
```

Truy xuất đối tượng tiến trình cho cửa sổ hiện đang có tiêu điểm bàn phím.
