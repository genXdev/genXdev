# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Lấy khóa ngôn ngữ web mặc định dựa trên cài đặt ngôn ngữ hiện tại của hệ thống.

## Description

Lấy cài đặt ngôn ngữ và văn hóa hệ thống hiện tại và ánh xạ chúng đến khóa từ điển ngôn ngữ web tương ứng được sử dụng bởi các dịch vụ dịch thuật.

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

Trả về "Tiếng Anh" cho hệ thống tiếng Anh, "Tiếng Hà Lan" cho hệ thống tiếng Hà Lan, v.v.
