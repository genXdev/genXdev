# Get-DefaultWebbrowser

> **Module:** GenXdev.Webbrowser | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Trả về trình duyệt web mặc định đã được cấu hình cho người dùng hiện tại.

## Description

* Truy xuất thông tin về trình duyệt web mặc định của hệ thống bằng cách truy vấn Registry Windows.
* Trả về một hashtable chứa tên, mô tả, đường dẫn biểu tượng và đường dẫn tệp thực thi của trình duyệt.
* Hàm kiểm tra cả tùy chọn người dùng và đăng ký trình duyệt toàn hệ thống để xác định trình duyệt mặc định.

## Syntax

```powershell
Get-DefaultWebbrowser [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebbrowser | Format-List
```

Lấy thông tin chi tiết về trình duyệt mặc định.

### Example 2

```powershell
$browser = Get-DefaultWebbrowser
& $browser.Path https://www.github.com/
```

Khởi chạy trình duyệt mặc định với một URL cụ thể.
