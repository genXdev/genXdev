# Get-DesktopScalingFactor

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Truy xuất hệ số tỷ lệ hiển thị Windows (cài đặt DPI) cho một màn hình được chỉ định.

## Description

* Lấy hệ số tỷ lệ hiển thị Windows hiện tại được cấu hình cho một màn hình trong hệ thống.
* Hệ số tỷ lệ được biểu thị dưới dạng phần trăm, trong đó 100 đại diện cho tỷ lệ tiêu chuẩn (96 DPI). Các giá trị phổ biến là 100, 125, 150 và 200.
* Nếu không chỉ định màn hình, trả về hệ số tỷ lệ cho màn hình chính.

## Syntax

```powershell
Get-DesktopScalingFactor [[-Monitor] <Int32>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Monitor` | Int32 | — | — | 0 | `0` | Chỉ mục màn hình cần kiểm tra (0 = màn hình chính) |

## Examples

### Example 1

```powershell
Get-DesktopScalingFactor -Monitor 0
```

Trả về tỷ lệ phần trăm hệ số tỷ lệ cho màn hình chính.

### Example 2

```powershell
Get-DesktopScalingFactor 1
```

Trả về tỷ lệ phần trăm tỷ lệ thu phóng cho màn hình thứ hai bằng cách sử dụng tham số vị trí.
