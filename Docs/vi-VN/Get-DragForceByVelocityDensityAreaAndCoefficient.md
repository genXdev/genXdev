# Get-DragForceByVelocityDensityAreaAndCoefficient

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Tính toán lực cản.

## Description

Sử dụng F = 1/2 C ρ A v².

## Syntax

```powershell
Get-DragForceByVelocityDensityAreaAndCoefficient -VelocityInMetersPerSecond <Double> -DensityInKilogramsPerCubicMeter <Double> -AreaInSquareMeters <Double> -Coefficient <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-VelocityInMetersPerSecond` | Double | ✅ | — | 0 | `0` | Vận tốc tính bằng m/s |
| `-DensityInKilogramsPerCubicMeter` | Double | ✅ | — | 1 | `0` | Mật độ chất lỏng tính bằng kg/m³ |
| `-AreaInSquareMeters` | Double | ✅ | — | 2 | `0` | Diện tích mặt cắt ngang tính bằng m² |
| `-Coefficient` | Double | ✅ | — | 3 | `0` | Hệ số cản |
| `-As` | String | — | — | 4 | — | Đơn vị đầu ra cho lực |

## Examples

### Example 1

```powershell
Get-DragForceByVelocityDensityAreaAndCoefficient -VelocityInMetersPerSecond 10 -DensityInKilogramsPerCubicMeter 1.225 -AreaInSquareMeters 1 -Coefficient 0.5 -As "poundforce"
```

Tính toán lực cản sử dụng vận tốc 10 m/s, mật độ không khí 1.225 kg/m³, diện tích 1 m², và hệ số 0.5, xuất ra đơn vị pound-lực.

### Example 2

```powershell
Get-DragForceByVelocityDensityAreaAndCoefficient 20 1.225 2 0.3
```

Tính lực cản sử dụng tham số vị trí: vận tốc 20 m/s, mật độ 1.225 kg/m³, diện tích 2 m², hệ số 0.3.
