# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss`

## Synopsis

> Trả về true nếu công cụ chuyển văn bản thành giọng nói đang nói.

## Description

Kiểm tra trạng thái của cả bộ tổng hợp giọng nói mặc định và tùy chỉnh để xác định xem có bộ nào hiện đang nói hay không. Trả về true nếu giọng nói đang được thực hiện, false nếu không.

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

Trả về true nếu công cụ chuyển văn bản thành giọng nói đang nói.

### Example 2

```powershell
iss
```

Trả về true nếu công cụ chuyển văn bản thành giọng nói đang nói (sử dụng bí danh).
