# Get-LLMJsonOutput

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Trích xuất JSON hợp lệ từ văn bản phản hồi của LLM bằng cách sử dụng các phương pháp heuristic tốt nhất.

## Description

Nhận văn bản phản hồi thô từ LLM có thể chứa dấu hiệu markdown, bình luận hoặc nội dung không phải JSON khác và cố gắng trích xuất JSON sạch, hợp lệ từ đó. Sử dụng chiến lược nhiều bước: đầu tiên loại bỏ dấu hiệu markdown, sau đó thử phân tích toàn bộ chuỗi dưới dạng JSON, và nếu thất bại, sử dụng trích xuất theo hướng schema, tiếp theo là các phương pháp dự phòng heuristic. Hàm xử lý: - Dấu hiệu khung mã markdown (...) - Đối tượng JSON ({...}) và mảng ([...]) - Chuỗi, số, boolean và giá trị ngày-giờ - Trích xuất theo hướng schema sử dụng gợi ý kiểu json_schema

## Syntax

```powershell
Get-LLMJsonOutput -Text <String> [[-ResponseFormat] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Text` | String | ✅ | — | 0 | — | @{response=Dùng để chỉ các khoảng chỉ mục.} |
| `-ResponseFormat` | String | — | — | 1 | — | @{response=Đối số này chỉ định máy chủ proxy. Nếu không được chỉ định, không có proxy nào được sử dụng.} |

## Examples

### Get-LLMJsonOutput -Văn bản '

Get-LLMJsonOutput -Văn bản '

```json
{"result": "hello"}
```

Trích xuất: {"result": "hello"}

### $schema = '{"type":"json_schema","json_schema":{"name":"r","schema":{"type":"object"}}}' Get-LLMJsonOutput -Text 'The answer is {"key": "value"}' -ResponseFormat $schema

```powershell
$schema = '{"type":"json_schema","json_schema":{"name":"r","schema":{"type":"object"}}}'
Get-LLMJsonOutput -Text 'The answer is {"key": "value"}' -ResponseFormat $schema
```

Trích xuất: {"key": "value"}
