# Get-MagnificationByObjectDistanceAndImageDistance

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Tính toán độ phóng đại cho một thấu kính.

## Description

Sử dụng m = - (khoảng cách ảnh / khoảng cách vật).

## Syntax

```powershell
Get-MagnificationByObjectDistanceAndImageDistance -ObjectDistanceInMeters <Double> -ImageDistanceInMeters <Double> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ObjectDistanceInMeters` | Double | ✅ | — | 0 | `0` | Khoảng cách đối tượng tính bằng mét |
| `-ImageDistanceInMeters` | Double | ✅ | — | 1 | `0` | Khoảng cách ảnh (tính bằng mét) |

## Examples

### Example 1

```powershell
Get-MagnificationByObjectDistanceAndImageDistance -ObjectDistanceInMeters 0.5 -ImageDistanceInMeters 1
```

Tính độ phóng đại với khoảng cách vật là 0.5m và khoảng cách ảnh là 1m.

### Example 2

```powershell
Get-MagnificationByObjectDistanceAndImageDistance 0.3 0.6
```

Tính toán độ phóng đại bằng cách sử dụng các tham số vị trí.
