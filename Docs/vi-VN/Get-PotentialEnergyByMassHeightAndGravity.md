# Get-PotentialEnergyByMassHeightAndGravity

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Tính toán thế năng hấp dẫn.

## Description

Sử dụng PE = m g h.

## Syntax

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms <Double> -HeightInMeters <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Khối lượng (kg) |
| `-HeightInMeters` | Double | ✅ | — | 1 | `0` | Chiều cao tính bằng mét |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Trọng lực tính bằng m/s² (mặc định: 9,81) |
| `-As` | String | — | — | 3 | — | Đơn vị đo năng lượng |

## Examples

### Example 1

```powershell
Get-PotentialEnergyByMassHeightAndGravity -MassInKilograms 10 -HeightInMeters 5 -As "calories"
```

Tính toán năng lượng thế hấp dẫn cho một khối lượng 10kg ở độ cao 5 mét, xuất kết quả dưới dạng calo.

### Example 2

```powershell
Get-PotentialEnergyByMassHeightAndGravity 5 10
```

Trình bày cách sử dụng các tham số vị trí để tính toán thế năng.
