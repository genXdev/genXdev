# Get-ProjectileRangeByInitialSpeedAndAngle

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Tính toán tầm bay của một đạn.

## Description

Sử dụng công thức R = (v² sin(2θ)) / g cho chuyển động của vật thể lý tưởng.

## Syntax

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle -InitialSpeedInMetersPerSecond <Double> -AngleInDegrees <Double> [[-GravityInMetersPerSecondSquared] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InitialSpeedInMetersPerSecond` | Double | ✅ | — | 0 | `0` | Tốc độ ban đầu (m/s) |
| `-AngleInDegrees` | Double | ✅ | — | 1 | `0` | Góc phóng tính bằng độ |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 2 | `0` | Trọng lực tính bằng m/s² (mặc định: 9,81) |
| `-As` | String | — | — | 3 | — | Đơn vị đầu ra cho phạm vi |

## Examples

### Example 1

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle -InitialSpeedInMetersPerSecond 20 -AngleInDegrees 45 -As "feet"
```

Tính toán tầm xa sử dụng công thức chuyển động của vật phóng.

### Example 2

```powershell
Get-ProjectileRangeByInitialSpeedAndAngle 30 30
```

Tính toán phạm vi với các tham số vị trí.
