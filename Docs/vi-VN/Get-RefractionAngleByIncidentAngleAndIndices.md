# Get-RefractionAngleByIncidentAngleAndIndices

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Tính góc khúc xạ sử dụng định luật Snell.

## Description

Sử dụng θ2 = arcsin( (n1 / n2) sin θ1 ).

## Syntax

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees <Double> -IndexOfRefraction1 <Double> -IndexOfRefraction2 <Double> [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-IncidentAngleInDegrees` | Double | ✅ | — | 0 | `0` | Góc tới tính bằng độ |
| `-IndexOfRefraction1` | Double | ✅ | — | 1 | `0` | Chiết suất của môi trường thứ nhất |
| `-IndexOfRefraction2` | Double | ✅ | — | 2 | `0` | Chiết suất của môi trường thứ hai |
| `-As` | String | — | — | 3 | — | Đơn vị đầu ra cho góc |

## Examples

### Example 1

```powershell
Get-RefractionAngleByIncidentAngleAndIndices -IncidentAngleInDegrees 30 -IndexOfRefraction1 1 -IndexOfRefraction2 1.33 -As "radians"
```

Tính góc khúc xạ khi ánh sáng truyền từ không khí (n=1) vào nước (n=1,33) ở góc tới 30 độ.

### Example 2

```powershell
Get-RefractionAngleByIncidentAngleAndIndices 45 1 1.5
```

Tính góc khúc xạ khi ánh sáng truyền từ không khí (n=1) sang thủy tinh (n=1,5) với góc tới 45 độ.
