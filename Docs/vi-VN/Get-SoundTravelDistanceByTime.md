# Get-SoundTravelDistanceByTime

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Tính toán quãng đường âm thanh truyền đi trong một khoảng thời gian nhất định.

## Description

Sử dụng công thức distance = speed * time, với tốc độ mặc định là tốc độ âm thanh trong không khí.

## Syntax

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds <Double> [[-As] <String>] [<CommonParameters>]

Get-SoundTravelDistanceByTime [[-SpeedOfSoundInMetersPerSecond] <Double>] [<CommonParameters>]

Get-SoundTravelDistanceByTime -Medium <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-TimeInSeconds` | Double | ✅ | — | 0 | `0` | Thời gian tính bằng giây |
| `-SpeedOfSoundInMetersPerSecond` | Double | — | — | 1 | `0` | Tốc độ âm thanh trong m/s (mặc định: 343) *(Parameter set: )* |
| `-Medium` | String | ✅ | — | 1 | — | Môi trường mà âm thanh truyền qua *(Parameter set: )* |
| `-As` | String | — | — | 2 | — | Đơn vị cho khoảng cách đầu ra |

## Examples

### Example 1

```powershell
Get-SoundTravelDistanceByTime -TimeInSeconds 5 -Medium "water" -As "kilometers"
```

Tính toán quãng đường âm thanh truyền trong nước trong 5 giây và chuyển đổi kết quả sang kilômét.

### Example 2

```powershell
Get-SoundTravelDistanceByTime 10 -SpeedOfSoundInMetersPerSecond 1480
```

Tính toán khoảng cách bằng cách sử dụng tốc độ âm thanh cụ thể.
