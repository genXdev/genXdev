# Get-TerminalVelocityByMassGravityDensityAndArea

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Tính vận tốc cuối.

## Description

Sử dụng v = sqrt(2 m g / (ρ A C)).

## Syntax

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms <Double> [[-GravityInMetersPerSecondSquared] <Double>] -DensityInKilogramsPerCubicMeter <Double> -AreaInSquareMeters <Double> [[-DragCoefficient] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-MassInKilograms` | Double | ✅ | — | 0 | `0` | Khối lượng (kg) |
| `-GravityInMetersPerSecondSquared` | Double | — | — | 1 | `0` | Trọng lực tính bằng m/s² (mặc định: 9,81) |
| `-DensityInKilogramsPerCubicMeter` | Double | ✅ | — | 2 | `0` | Mật độ chất lỏng tính bằng kg/m³ |
| `-AreaInSquareMeters` | Double | ✅ | — | 3 | `0` | Diện tích mặt cắt ngang tính bằng m² |
| `-DragCoefficient` | Double | — | — | 4 | `0` | Hệ số cản (mặc định: 0.5) |
| `-As` | String | — | — | 5 | — | Đơn vị đầu ra cho vận tốc |

## Examples

### Example 1

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea -MassInKilograms 80 -DensityInKilogramsPerCubicMeter 1.225 -AreaInSquareMeters 0.7 -DragCoefficient 1.0 -As "mph"
```

Tính vận tốc cuối cùng cho một khối lượng 80 kg trong không khí với hệ số cản 1.0, xuất kết quả theo đơn vị dặm/giờ.

### Example 2

```powershell
Get-TerminalVelocityByMassGravityDensityAndArea 70 1.225 0.8 0.8
```

Sử dụng tham số vị trí để tính vận tốc cuối cùng.
