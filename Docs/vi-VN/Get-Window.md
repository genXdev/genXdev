# Get-Window

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Lấy thông tin cửa sổ cho các tiến trình hoặc xử lý cửa sổ đã chỉ định.

## Description

* Truy xuất thông tin cửa sổ bằng tên tiến trình, ID hoặc handle cửa sổ.
* Trả về các đối tượng WindowObj chứa chi tiết về các cửa sổ chính của các tiến trình phù hợp.
* Hỗ trợ ký tự đại diện khi tìm kiếm theo tên tiến trình.

## Syntax

```powershell
Get-Window [[-ProcessName] <String>] [-ProcessId <Int32>] [-WindowHandle <Int64>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ProcessName` | String | — | ✅ (ByPropertyName) | 0 | — | Tên của tiến trình để lấy thông tin cửa sổ 🌐 *Supports wildcards* |
| `-ProcessId` | Int32 | — | ✅ (ByPropertyName) | Named | `0` | ID của tiến trình để lấy thông tin cửa sổ |
| `-WindowHandle` | Int64 | — | ✅ (ByPropertyName) | Named | `0` | Xử lý cửa sổ để lấy thông tin |

## Examples

### Example 1

```powershell
Get-Window -ProcessName "notepad"
```

Truy xuất thông tin cửa sổ cho tất cả các tiến trình có tên "notepad".

### Example 2

```powershell
Get-Window -ProcessId 1234
```

Truy xuất thông tin cửa sổ cho tiến trình có ID 1234.

### Example 3

```powershell
Get-Window -WindowHandle 45678
```

Truy xuất thông tin cửa sổ cho cửa sổ có mã xử lý 45678.
