# Import-BrowserBookmarks

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Nhập dấu trang từ một tệp hoặc bộ sưu tập vào trình duyệt web.

## Description

Nhập dấu trang vào Microsoft Edge hoặc Google Chrome từ tệp CSV hoặc tập hợp các đối tượng dấu trang. Các dấu trang được thêm vào thanh dấu trang của trình duyệt hoặc các thư mục được chỉ định. Nhập từ Firefox hiện không được hỗ trợ.

## Syntax

```powershell
Import-BrowserBookmarks [[-InputFile] <String>] [<CommonParameters>]

Import-BrowserBookmarks [[-Bookmarks] <Array>] [<CommonParameters>]

Import-BrowserBookmarks [-Chrome] [-Edge] [-Firefox] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputFile` | String | — | — | 0 | — | Đường dẫn đến tệp CSV chứa dấu trang cần nhập *(Parameter set: )* |
| `-Bookmarks` | Array | — | — | 0 | — | Bộ sưu tập bookmark để nhập *(Parameter set: )* |
| `-Chrome` | SwitchParameter | — | — | Named | — | Nhập vào Google Chrome |
| `-Edge` | SwitchParameter | — | — | Named | — | Nhập vào Microsoft Edge |
| `-Firefox` | SwitchParameter | — | — | Named | — | Nhập vào Firefox (không được hỗ trợ) |

## Examples

### Import-BrowserBookmarks -InputFile "C:\MyBookmarks.csv" -Edge Imports bookmarks from the CSV file into Microsoft Edge.

```powershell
Import-BrowserBookmarks -InputFile "C:\MyBookmarks.csv" -Edge
Imports bookmarks from the CSV file into Microsoft Edge.
```

### $bookmarks = @(     @{         Name = "Microsoft";         URL = "https://microsoft.com";         Folder = "Tech"     } ) Import-BrowserBookmarks -Bookmarks $bookmarks -Chrome Imports a collection of bookmarks into Google Chrome.

```powershell
$bookmarks = @(
    @{
        Name = "Microsoft";
        URL = "https://microsoft.com";
        Folder = "Tech"
    }
)
Import-BrowserBookmarks -Bookmarks $bookmarks -Chrome
Imports a collection of bookmarks into Google Chrome.
```
