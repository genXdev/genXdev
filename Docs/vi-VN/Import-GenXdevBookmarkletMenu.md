# Import-GenXdevBookmarkletMenu

> **Module:** GenXdev.Webbrowser | **Type:** Function | **Aliases:** —

## Synopsis

> Nhập các bookmarklet JavaScript GenXdev vào bộ sưu tập dấu trang trình duyệt.

## Description

Hàm này quét một thư mục để tìm các tệp dấu trang GenXdev có phần mở rộng .

## Syntax

```powershell
Import-GenXdevBookmarkletMenu [[-SnippetsPath] <String>] [[-TargetFolder] <String>] [-Chrome] [-Edge] [-Firefox] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-SnippetsPath` | String | — | — | 0 | `"$($MyInvocation.MyCommand.Module.ModuleBase)\Bookmarklets"` | Đường dẫn đến thư mục chứa các tệp đoạn mã bookmarklet |
| `-TargetFolder` | String | — | — | 1 | `""` | Thư mục dấu trang mục tiêu trong cấu trúc dấu trang trình duyệt |
| `-Edge` | SwitchParameter | — | — | Named | — | Nhập bookmarklets vào trình duyệt Microsoft Edge |
| `-Chrome` | SwitchParameter | — | — | Named | — | Nhập bookmarklet vào trình duyệt Google Chrome |
| `-Firefox` | SwitchParameter | — | — | Named | — | Nhập bookmarklets vào trình duyệt Mozilla Firefox |

## Examples

### Import-GenXdevBookmarkletMenu -Edge

```powershell
Import-GenXdevBookmarkletMenu -Edge
```

Nhập tất cả các tệp bookmarklet từ thư mục đoạn mã mặc định vào thư mục thanh dấu trang của Microsoft Edge.

### Import-GenXdevBookmarkletMenu -SnippetsPath "C:\MyBookmarklets" -Chrome -WhatIf

```powershell
Import-GenXdevBookmarkletMenu -SnippetsPath "C:\MyBookmarklets" -Chrome -WhatIf
```

Hiển thị các bookmarklet nào sẽ được nhập từ đường dẫn đã chỉ định vào Google Chrome mà không thực sự thực hiện thao tác nhập.
