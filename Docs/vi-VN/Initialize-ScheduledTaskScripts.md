# Initialize-ScheduledTaskScripts

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> Tạo các tác vụ đã lên lịch chạy các tập lệnh PowerShell vào các khoảng thời gian xác định.

## Description

Tạo và cấu hình các tác vụ đã lên lịch thực thi tập lệnh PowerShell vào nhiều khoảng thời gian khác nhau bao gồm:
- Khởi động hệ thống
- Đăng nhập người dùng
- Mỗi giờ vào những ngày cụ thể (ví dụ: Thứ Hai lúc 13:00)
- Hàng ngày vào những giờ cụ thể (ví dụ: mỗi ngày lúc 15:00)
Mỗi tác vụ chạy với đặc quyền nâng cao trong ngữ cảnh của người dùng hiện tại.

## Syntax

```powershell
Initialize-ScheduledTaskScripts [[-FilePath] <String>] [[-Prefix] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-FilePath` | String | — | — | 0 | `''` | Đường dẫn thư mục nơi các tập lệnh tác vụ sẽ được tạo |
| `-Prefix` | String | — | — | 1 | `'PS'` | Tiền tố cho tên tác vụ đã lên lịch |

## Examples

### Initialize-ScheduledTaskScripts -FilePath "C:\Tasks" -Prefix "MyTasks"

```powershell
Initialize-ScheduledTaskScripts -FilePath "C:\Tasks" -Prefix "MyTasks"
```

### Initialize-ScheduledTaskScripts

```powershell
Initialize-ScheduledTaskScripts
```
