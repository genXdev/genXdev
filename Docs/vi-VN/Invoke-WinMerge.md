# Invoke-WinMerge

> **Module:** GenXdev.AI | **Type:** Function | **Aliases:** —

## Synopsis

> Khởi chạy WinMerge để so sánh hai tệp cạnh nhau.

## Description

Khởi chạy ứng dụng WinMerge để so sánh tệp nguồn và tệp đích trong chế độ xem khác biệt song song. Hàm xác thực sự tồn tại của cả hai tệp đầu vào và đảm bảo WinMerge được cài đặt đúng cách trước khi khởi chạy. Cung cấp chức năng chờ tùy chọn để tạm dừng thực thi cho đến khi WinMerge đóng.

## Syntax

```powershell
Invoke-WinMerge -SourcecodeFilePath <String> -TargetcodeFilePath <String> [-Wait] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-SourcecodeFilePath` | String | ✅ | — | 0 | — | Đường dẫn đến tệp nguồn để so sánh |
| `-TargetcodeFilePath` | String | ✅ | — | 1 | — | Đường dẫn đến tệp đích để so sánh |
| `-Wait` | SwitchParameter | — | — | Named | — | Chờ WinMerge đóng trước khi tiếp tục |

## Examples

### Invoke-WinMerge -SourcecodeFilePath "C:\source\file1.txt" `                 -TargetcodeFilePath "C:\target\file2.txt" `                 -Wait

```powershell
Invoke-WinMerge -SourcecodeFilePath "C:\source\file1.txt" `
                -TargetcodeFilePath "C:\target\file2.txt" `
                -Wait
```

### merge "C:\source\file1.txt" "C:\target\file2.txt"

```powershell
merge "C:\source\file1.txt" "C:\target\file2.txt"
```
