# Now

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Trả về ngày và giờ hiện tại của hệ thống dưới dạng đối tượng DateTime.

## Description

Cung cấp một cách đơn giản để lấy ngày và giờ hiện tại của hệ thống mà không cần tham số nào. Trả về một đối tượng DateTime .NET tiêu chuẩn có thể được sử dụng để tính toán, định dạng và so sánh ngày giờ.

## Syntax

```powershell
Now [<CommonParameters>]
```

## Examples

### Now Returns the current system date and time as a DateTime object

```powershell
Now
Returns the current system date and time as a DateTime object
```

### $timestamp = Now Stores the current date and time in a variable for later use

```powershell
$timestamp = Now
Stores the current date and time in a variable for later use
```

## Outputs

- `DateTime`
