# Open-PlayWrightBrowser

> **Module:** GenXdev.Webbrowser.Playwright | **Type:** Function | **Aliases:** `spb`

## Synopsis

> Khởi động trình duyệt do Playwright quản lý với hồ sơ người dùng liên tục.

## Description

<details>
<summary><b>Expand description</b></summary>

Khởi chạy một phiên trình duyệt được hỗ trợ bởi Playwright. Tham số BrowserType quyết định công cụ trình duyệt và chế độ khởi chạy nào được sử dụng:

- ChromeNormal / EdgeNormal: Khởi chạy Chrome hoặc Edge đã cài trên hệ điều hành của bạn thông qua Channel API của Playwright, tránh thanh thông báo "được kiểm soát bởi tự động hóa" và các vấn đề chống bot.
- ChromiumNormal: Tự động phát hiện Chrome hoặc Edge (cái nào đã cài và được đặt làm mặc định hệ thống) và sử dụng Channel API. Dự phòng xuống Playwright Chromium tích hợp nếu không có cái nào được cài.
- ChromiumPlaywright: Luôn sử dụng tệp nhị phân Chromium tích hợp của Playwright (không bao giờ dùng trình duyệt cài trên hệ điều hành).
- FirefoxPlaywright: Sử dụng tệp nhị phân Firefox tích hợp của Playwright. Channel API không khả dụng cho Firefox trong ràng buộc Playwright .NET.
- WebKitPlaywright: Sử dụng tệp nhị phân WebKit tích hợp của Playwright.

Các biện pháp chống phát hiện được áp dụng cho trình duyệt dựa trên Chromium:
- Vô hiệu hóa cờ tự động hóa blink (loại bỏ thanh thông báo)
- Đặt viewport thành null và khởi động tối đa hóa để thay đổi kích thước cửa sổ gốc
- Sử dụng lược đồ màu 'không ưu tiên' để tránh nền trắng
- Đặt các đối số trình duyệt tiêu chuẩn (không-chạy-lần-đầu, không-kiểm-tra-trình-duyệt-mặc-định)

Hồ sơ liên tục cho mỗi BrowserType được lưu trữ dưới GenXdev AppData. Trình duyệt đã khởi chạy và bối cảnh/trang của nó được lưu trữ trong $Global:GenXdevPlaywright để sử dụng bởi các lệnh ghép khác.

</details>


## Examples

### Open-PlayWrightBrowser Launches your OS-installed Chrome or Edge (auto-detected) with a persistent profile and anti-detection measures.

```powershell
Open-PlayWrightBrowser
Launches your OS-installed Chrome or Edge (auto-detected) with a
persistent profile and anti-detection measures.
```

### Open-PlayWrightBrowser -BrowserType EdgeNormal Launches Microsoft Edge via Playwright's Channel API.

```powershell
Open-PlayWrightBrowser -BrowserType EdgeNormal
Launches Microsoft Edge via Playwright's Channel API.
```

### Open-PlayWrightBrowser -BrowserType ChromiumPlaywright Uses the bundled Playwright Chromium binary.

```powershell
Open-PlayWrightBrowser -BrowserType ChromiumPlaywright
Uses the bundled Playwright Chromium binary.
```

### Open-PlayWrightBrowser -BrowserType FirefoxPlaywright -Headless Launches the bundled Firefox in headless mode.

```powershell
Open-PlayWrightBrowser -BrowserType FirefoxPlaywright -Headless
Launches the bundled Firefox in headless mode.
```

### Open-PlayWrightBrowser -Width 1280 -Height 720 -Force Restarts the browser with a 1280x720 viewport.

```powershell
Open-PlayWrightBrowser -Width 1280 -Height 720 -Force
Restarts the browser with a 1280x720 viewport.
```
