# ReadJsonWithRetry

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** —

## Synopsis

> Đọc tệp JSON với logic thử lại và tự động dọn dẹp khóa.

## Description

Cố gắng đọc tệp JSON với logic thử lại để xử lý truy cập đồng thời. Thực hiện tự động dọn dẹp các tệp khóa cũ. Trả về bảng băm rỗng nếu tệp không tồn tại.

## Syntax

```powershell
ReadJsonWithRetry -FilePath <String> [[-MaxRetries] <Int32>] [[-RetryDelayMs] <Int32>] [-AsHashtable] [<CommonParameters>]
```
