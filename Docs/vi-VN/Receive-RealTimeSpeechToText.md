# Receive-RealTimeSpeechToText

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Sử dụng mô hình Whisper AI để chuyển đổi đầu vào âm thanh thời gian thực thành văn bản.

## Description

Cmdlet này ghi lại âm thanh từ micro hoặc máy tính để bàn và phiên âm thành văn bản theo thời gian thực bằng mô hình Whisper AI. Nó hỗ trợ nhiều nguồn âm thanh, phát hiện im lặng và nhiều tùy chọn cấu hình để nhận dạng giọng nói.

## Examples

### Example 1

```powershell
Receive-RealTimeSpeechToText
```

Sử dụng cơ bản với microphone.

### Example 2

```powershell
Receive-RealTimeSpeechToText -UseDesktopAudioCapture
```

Sử dụng tính năng thu âm màn hình máy tính.

### Example 3

```powershell
Receive-RealTimeSpeechToText -AudioDevice "Microphone*"
```

Sử dụng thiết bị âm thanh cụ thể.

### Example 4

```powershell
Receive-RealTimeSpeechToText -MaxDurationOfSilence "00:00:05"
```

Với tính năng phát hiện im lặng.
