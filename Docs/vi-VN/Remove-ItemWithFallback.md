# Remove-ItemWithFallback

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** `rmf`

## Synopsis

> Loại bỏ các tệp hoặc thư mục với nhiều cơ chế dự phòng để xóa đáng tin cậy.

## Description

Hàm này cung cấp cách mạnh mẽ để xóa tệp và thư mục bằng cách thử các phương pháp xóa tuần tự:
1. Xóa trực tiếp qua các phương thức System.IO để có hiệu suất tốt nhất
2. Sử dụng lệnh ghép Remove-Item nhận biết nhà cung cấp PowerShell dự phòng
3. Đánh dấu để xóa khi khởi động lại hệ thống nếu các phương pháp khác thất bại
Điều này đảm bảo độ tin cậy tối đa khi xóa các mục trên các nhà cung cấp khác nhau.

## Syntax

```powershell
Remove-ItemWithFallback -Path <String> [-CountRebootDeletionAsSuccess] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Path` | String | ✅ | ✅ (ByValue, ByPropertyName) | 0 | — | Đường dẫn đến mục cần xóa |
| `-CountRebootDeletionAsSuccess` | SwitchParameter | — | — | Named | `$false` | The `-CountRebootDeletionAsSuccess` parameter. |

## Examples

### Remove-ItemWithFallback -Path "C:\temp\myfile.txt" Attempts to remove the file using all available methods.

```powershell
Remove-ItemWithFallback -Path "C:\temp\myfile.txt"
Attempts to remove the file using all available methods.
```

### "C:\temp\mydir" | rif Uses the alias 'rif' to remove a directory through the pipeline.

```powershell
"C:\temp\mydir" | rif
Uses the alias 'rif' to remove a directory through the pipeline.
```
