# Rename-InProject

> **Module:** GenXdev.FileSystem | **Type:** Function | **Aliases:** `rip`

## Synopsis

> Thực hiện thay thế văn bản trong toàn bộ thư mục dự án.

## Description

Đệ quy tìm kiếm qua các tệp và thư mục trong một dự án để thực hiện thay thế văn bản. Xử lý cả tên tệp/thư mục và nội dung tệp. Bỏ qua các tệp nhị phân phổ biến và thư mục kho lưu trữ (.git, .svn) để tránh hỏng. Sử dụng mã hóa UTF-8 không có BOM cho các thao tác tệp. Hỗ trợ cả chế độ thay thế phân biệt chữ hoa chữ thường và không phân biệt chữ hoa chữ thường.

## Syntax

```powershell
Rename-InProject [[-Source] <String>] -FindText <String> -ReplacementText <String> [-CaseInsensitive] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Source` | String | — | — | 0 | — | Thư mục, đường dẫn tệp, hoặc thư mục+mặt nạ tìm kiếm |
| `-FindText` | String | ✅ | — | 1 | — | Văn bản cần tìm (phân biệt chữ hoa/thường được kiểm soát bởi tham số CaseInsensitive) |
| `-ReplacementText` | String | ✅ | — | 2 | — | Văn bản cần thay thế các kết quả khớp với |
| `-CaseInsensitive` | SwitchParameter | — | — | Named | — | Thực hiện thay thế văn bản không phân biệt chữ hoa chữ thường |

## Examples

### Rename-InProject -Source .\src\*.js -FindText "oldName" `     -ReplacementText "newName"

```powershell
Rename-InProject -Source .\src\*.js -FindText "oldName" `
    -ReplacementText "newName"
```

### rip . "MyClass" "MyNewClass" -WhatIf

```powershell
rip . "MyClass" "MyNewClass" -WhatIf
```

### rip . "OLDNAME" "NewName" -CaseInsensitive

```powershell
rip . "OLDNAME" "NewName" -CaseInsensitive
```
