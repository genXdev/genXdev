# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Phát âm ngày hiện tại bằng cách sử dụng tổng hợp chuyển văn bản thành giọng nói.

## Description

Chuyển đổi ngày hiện tại thành định dạng ngôn ngữ tự nhiên và thông báo bằng công cụ chuyển văn bản thành giọng nói của hệ thống. Ngày tháng được đọc theo định dạng: 'Hôm nay là [thứ], ngày [ngày] tháng [tháng] năm [năm]'

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
