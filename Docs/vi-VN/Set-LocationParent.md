# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..`

## Synopsis

> Thay đổi vị trí hiện tại sang thư mục cha và liệt kê nội dung của nó.

## Description

* Di chuyển lên một cấp thư mục trong hệ thống phân cấp tệp.
* Hiển thị nội dung của thư mục hiện tại mới.
* Cung cấp bí danh '..' thuận tiện để điều hướng thư mục nhanh.

## Syntax

```powershell
Set-LocationParent [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

Chuyển đến thư mục cha và liệt kê nội dung của nó.

### Example 2

```powershell
..
```

Sử dụng bí danh để chuyển đến thư mục cha.
