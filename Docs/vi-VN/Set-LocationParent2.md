# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> Điều hướng lên hai cấp thư mục trong hệ thống phân cấp hệ thống tệp.

## Description

* Thay đổi thư mục làm việc hiện tại sang thư mục ông bà (hai cấp lên trên).
* Hiển thị nội dung của thư mục kết quả.

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

Chuyển đến thư mục gốc và hiển thị nội dung của nó.

```powershell
...
```
