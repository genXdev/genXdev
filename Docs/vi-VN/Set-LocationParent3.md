# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....`

## Synopsis

> Di chuyển lên ba cấp thư mục trong hệ thống phân cấp tệp.

## Description

* Thay đổi thư mục làm việc hiện tại bằng cách di chuyển lên ba cấp thư mục cha trong hệ thống phân cấp tệp.
* Hiển thị nội dung của thư mục kết quả.

## Syntax

```powershell
Set-LocationParent3 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

Điều hướng lên ba cấp thư mục từ vị trí hiện tại.

### Example 2

```powershell
....
```

Sử dụng bí danh để điều hướng lên ba cấp thư mục.
