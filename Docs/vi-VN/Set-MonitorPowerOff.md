# Set-MonitorPowerOff

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `poweroff`

## Synopsis

> Tắt nguồn tất cả màn hình đã kết nối.

## Description

Sử dụng các lệnh gọi API Windows để gửi tín hiệu tắt nguồn đến tất cả màn hình được kết nối. Điều này tương đương với việc nhấn nút nguồn vật lý trên màn hình của bạn. Các màn hình sẽ chuyển sang chế độ tiết kiệm năng lượng nhưng có thể được đánh thức bằng cách di chuyển chuột hoặc nhấn phím.

## Syntax

```powershell
Set-MonitorPowerOff [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOff
```

Tắt tất cả các màn hình đang kết nối.

### Example 2

```powershell
poweroff
```

Tắt tất cả các màn hình đã kết nối bằng cách sử dụng bí danh.
