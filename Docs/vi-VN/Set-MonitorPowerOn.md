# Set-MonitorPowerOn

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** `wakemonitor`, `monitoroff`

## Synopsis

> Bật nguồn màn hình.

## Description

Sử dụng Windows API thông qua GenXdev.Helpers.WindowObj để đánh thức màn hình từ trạng thái ngủ/tắt nguồn. Điều này hữu ích cho các tập lệnh tự động hóa cần đảm bảo màn hình được bật nguồn.

## Syntax

```powershell
Set-MonitorPowerOn [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-MonitorPowerOn
```

Bật nguồn màn hình.

### Example 2

```powershell
wake-monitor
```

Bật nguồn màn hình bằng cách sử dụng bí danh.
