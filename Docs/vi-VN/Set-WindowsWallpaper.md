# Set-WindowsWallpaper

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> Đặt hình nền ngẫu nhiên từ một thư mục đã chỉ định.

## Description

* Chọn ngẫu nhiên một tệp hình ảnh từ thư mục được chỉ định và đặt nó làm hình nền máy tính Windows.
* Hỗ trợ các định dạng hình ảnh JPG/JPEG và cấu hình hình nền ở chế độ "vừa khít" màn hình theo mặc định.

## Syntax

```powershell
Set-WindowsWallpaper [[-InputObject] <Object>] [-AllDrives] [-NoRecurse] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-InputObject` | Object | — | ✅ (ByValue, ByPropertyName) | 0 | — | Đường dẫn đến thư mục chứa các hình ảnh hình nền |
| `-AllDrives` | SwitchParameter | — | — | Named | `False` | Tìm kiếm trên tất cả các ổ đĩa có sẵn |
| `-NoRecurse` | SwitchParameter | — | — | Named | `False` | Không đệ quy vào các thư mục con |

## Examples

### Example 1

```powershell
Set-WindowsWallpaper -InputObject "C:\Wallpapers\*.jpg"
```

Đặt hình nền ngẫu nhiên từ thư mục C:\Wallpapers.

### Example 2

```powershell
nextbg
```

Đặt hình nền ngẫu nhiên từ thư mục mặc định bằng cách sử dụng bí danh.
