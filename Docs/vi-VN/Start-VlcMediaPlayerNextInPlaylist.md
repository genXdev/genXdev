# Start-VlcMediaPlayerNextInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcnext`

## Synopsis

> Chuyển VLC Media Player đến mục tiếp theo trong danh sách phát hiện tại.

## Description

Chức năng này gửi tổ hợp phím 'n' tới VLC Media Player để chuyển đến bài hát hoặc mục phương tiện tiếp theo trong danh sách phát đang được tải. Nó tập trung cửa sổ VLC, gửi lệnh chuyển tiếp, và khôi phục lại tiêu điểm cho bảng điều khiển PowerShell. Chức năng bao gồm hỗ trợ ShouldProcess cho các lời nhắc xác nhận khi cần.

## Syntax

```powershell
Start-VlcMediaPlayerNextInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerNextInPlaylist

```powershell
Start-VlcMediaPlayerNextInPlaylist
```

### vlcnext

```powershell
vlcnext
```
