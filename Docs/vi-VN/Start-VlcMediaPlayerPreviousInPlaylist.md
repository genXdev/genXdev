# Start-VlcMediaPlayerPreviousInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcprev`, `vlcback`

## Synopsis

> Chuyển đến mục trước đó trong danh sách phát của VLC Media Player.

## Description

Hàm này gửi lệnh phím 'p' đến VLC Media Player để chuyển đến mục trước đó trong danh sách phát hiện tại. Hàm hỗ trợ các thao tác WhatIf và sẽ khôi phục tiêu điểm sau khi gửi lệnh.

## Syntax

```powershell
Start-VlcMediaPlayerPreviousInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerPreviousInPlaylist

```powershell
Start-VlcMediaPlayerPreviousInPlaylist
```

### vlcprev

```powershell
vlcprev
```

### vlcback

```powershell
vlcback
```
