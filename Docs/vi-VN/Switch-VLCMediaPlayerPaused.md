# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> Chuyển đổi trạng thái tạm dừng/phát của VLC Media Player.

## Description

Chức năng này gửi phím cách tới VLC Media Player để chuyển đổi giữa trạng thái tạm dừng và phát. Nó tự động khôi phục tiêu điểm về cửa sổ ban đầu sau khi gửi lệnh phím.

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

Chuyển đổi trạng thái tạm dừng/phát của VLC Media Player.

### vlcpause

```powershell
vlcpause
```

Sử dụng bí danh để chuyển đổi trạng thái tạm dừng/phát.

### vlcplay

```powershell
vlcplay
```

Sử dụng bí danh thay thế để chuyển đổi trạng thái tạm dừng/phát.
