# Switch-VlcMediaPlayerMute

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcmute`, `vlcunmute`

## Synopsis

> Chuyển đổi trạng thái tắt tiếng của VLC Media Player.

## Description

Hàm này gửi phím 'm' tới VLC Media Player để chuyển đổi giữa trạng thái tắt tiếng và bật tiếng. Hàm tập trung vào VLC, gửi lệnh tắt/bật tiếng, sau đó khôi phục tiêu điểm về cửa sổ đang hoạt động trước đó.

## Syntax

```powershell
Switch-VlcMediaPlayerMute [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerMute

```powershell
Switch-VlcMediaPlayerMute
```

Chuyển đổi trạng thái tắt tiếng của VLC Media Player bằng cách sử dụng tên hàm đầy đủ.

### vlcmute

```powershell
vlcmute
```

Chuyển đổi trạng thái tắt tiếng của VLC Media Player bằng cách sử dụng bí danh ngắn.
