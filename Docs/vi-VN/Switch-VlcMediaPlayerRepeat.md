# Switch-VlcMediaPlayerRepeat

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcrepeat`

## Synopsis

> Chuyển đổi chế độ lặp lại trong VLC Media Player.

## Description

Hàm này gửi lệnh phím 'r' đến VLC Media Player để chuyển đổi giữa các chế độ lặp khác nhau (không lặp, lặp hiện tại, lặp tất cả). Hàm sẽ mở VLC nếu chưa chạy và khôi phục tiêu điểm về cửa sổ trước đó sau khi gửi lệnh.

## Syntax

```powershell
Switch-VlcMediaPlayerRepeat [<CommonParameters>]
```

## Examples

### Switch-VlcMediaPlayerRepeat

```powershell
Switch-VlcMediaPlayerRepeat
```

### vlcrepeat

```powershell
vlcrepeat
```
