# Test-RefactorLLMSelection

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Đánh giá các tệp nguồn để xác định khả năng tái cấu trúc bằng phân tích LLM.

## Description

Sử dụng phân tích Mô hình Ngôn ngữ Học (LLM) để xác định xem một tệp mã nguồn có nên được chọn để tái cấu trúc dựa trên các tiêu chí đã chỉ định hay không. Hàm xử lý nội dung tệp thông qua truy vấn LLM và trả về phản hồi boolean.

## Syntax

```powershell
Test-RefactorLLMSelection -RefactorDefinition <GenXdev.Helpers.RefactorDefinition> -Path <String> [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-RefactorDefinition` | GenXdev.Helpers.RefactorDefinition | ✅ | — | 0 | — | Định nghĩa tái cấu trúc chứa các cài đặt LLM |
| `-Path` | String | ✅ | — | 1 | — | Đường dẫn đến tệp nguồn cần đánh giá |

## Examples

### Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"

```powershell
Test-RefactorLLMSelection -RefactorDefinition $refDef -Path "C:\source.ps1"
```
