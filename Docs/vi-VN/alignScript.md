# alignScript

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> Trả về một chuỗi (có thay đổi thụt lề) của một chuỗi scriptblock được cung cấp

## Description

Thay đổi độ thụt lề của chuỗi scriptblock trong khi vẫn tôn trọng độ thụt lề của khối mã gốc

## Syntax

```powershell
alignScript [<CommonParameters>]
```
