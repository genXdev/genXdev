# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Đặt cấu hình màn hình phụ mặc định.

## Description

Đặt hành vi mặc định cho việc mở cửa sổ GenXdev trên màn hình phụ.
Điều này hữu ích cho người dùng có màn hình đơn hoặc những người thích bố trí cửa sổ cạnh nhau.
Xem thêm lệnh ghép 'sidebyside' và 'restoredefaultmonitor'

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

Đặt mặc định cho việc mở cửa sổ GenXdev trên màn hình phụ
