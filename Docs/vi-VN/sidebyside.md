# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> Đặt cấu hình song song mặc định.

## Description

Đặt hành vi mặc định cho các cửa sổ GenXdev mở cạnh nhau với PowerShell.
Điều này hữu ích cho người dùng có một màn hình hoặc những người thích bố trí cửa sổ cạnh nhau.
Xem thêm lệnh 'secondscreen' và 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

Đặt mặc định cho các cửa sổ mở của GenXdev để hiển thị cạnh nhau với PowerShell
