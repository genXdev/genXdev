# Assert-GenXdevDependencyUsage

> **Module:** GenXdev.Coding.PowerShell.Modules | **Type:** Function | **Aliases:** `checkgenxdevdependencies`

## Synopsis

> 验证跨 GenXdev 模块的依赖使用情况，以确保维护正确的模块层次结构。

## Description

此函数用于分析 GenXdev 模块，确保它们遵循正确的依赖层次结构。它检查模块仅引用其 RequiredModules 清单中列出的依赖项，并通过验证模块未引用依赖链中位于其后的模块来防止循环依赖。

## Syntax

```powershell
Assert-GenXdevDependencyUsage [[-ModuleName] <String[]>] [-FromScripts] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-ModuleName` | String[] | — | ✅ (ByValue, ByPropertyName) | 1 | `@('GenXdev*')` | 要应用于模块名称的筛选器 🌐 *Supports wildcards* |
| `-FromScripts` | SwitchParameter | — | — | Named | — | 在脚本文件而非模块文件中搜索 |

## Examples

### Assert-GenXdevDependencyUsage -ModuleName "GenXdev.Coding"

```powershell
Assert-GenXdevDependencyUsage -ModuleName "GenXdev.Coding"
```

### checkgenxdevdependencies "GenXdev*" -FromScripts

```powershell
checkgenxdevdependencies "GenXdev*" -FromScripts
```
