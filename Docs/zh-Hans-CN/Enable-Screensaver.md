# Enable-Screensaver

> **Module:** GenXdev.Windows | **Type:** Function | **Aliases:** —

## Synopsis

> 启动配置的 Windows 屏幕保护程序。

## Description

通过执行默认屏幕保护程序可执行文件（scrnsave.scr）并附加 /s 开关立即启动，激活 Windows 系统屏幕保护程序。

## Syntax

```powershell
Enable-Screensaver [<CommonParameters>]
```

## Examples

### Enable-Screensaver

```powershell
Enable-Screensaver
```
