# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 根据系统的当前语言设置获取默认的Web语言键。

## Description

检索当前系统语言和文化设置，并将其映射到翻译服务所使用的对应网页语言字典键。

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

返回英语系统为“English”，荷兰语系统为“Dutch”，以此类推。
