# Get-DefaultWebLanguage

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 根据系统当前语言设置获取默认的Web语言键。

## Description

获取当前系统语言和区域设置，并将其映射到翻译服务使用的相应网络语言字典键。

## Syntax

```powershell
Get-DefaultWebLanguage [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-DefaultWebLanguage
```

对于英文系统返回"English"，对于荷兰语系统返回"Dutch"，等等。
