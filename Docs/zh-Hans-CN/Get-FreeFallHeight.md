# Get-FreeFallHeight

> **Module:** GenXdev.Helpers.Physics | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 计算给定时间内自由落体下落的高度。

## Description

使用考虑空气阻力和终端速度的数值方法计算自由落体过程中下落的距离。

## Syntax

```powershell
Get-FreeFallHeight -DurationInSeconds <Double> [[-TerminalVelocityInMs] <Double>] [[-As] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-DurationInSeconds` | Double | ✅ | — | 0 | `0` | 坠落持续时间（秒） |
| `-TerminalVelocityInMs` | Double | — | — | 1 | `0` | 终端速度，单位为米每秒（默认值：人类为53米/秒） |
| `-As` | String | — | — | 2 | — | 输出高度单位 |

## Examples

### Example 1

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -TerminalVelocityInMs 53
```

以默认人体终端速度计算10秒内下落的高度。

### Example 2

```powershell
Get-FreeFallHeight 5
```

使用位置参数和默认终端速度计算5秒内下落的高度。

### Example 3

```powershell
Get-FreeFallHeight -DurationInSeconds 10 -As "feet"
```

计算10秒内下落的距离并以英尺为单位返回结果。
