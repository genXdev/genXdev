# Get-IsSpeaking

> **Module:** GenXdev.Console | **Type:** Cmdlet | **Aliases:** `iss

## Synopsis

> 如果文本转语音引擎正在说话，则返回 true。

## Description

检查默认和自定义语音合成器的状态，判断是否有正在进行的语音播放。如果正在播放语音，返回true；否则返回false。

## Syntax

```powershell
Get-IsSpeaking [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-IsSpeaking
```

如果文本转语音引擎正在说话，则返回 true。

### Example 2

```powershell
iss
```

如果文本转语音引擎正在说话（使用别名），则返回 true。
