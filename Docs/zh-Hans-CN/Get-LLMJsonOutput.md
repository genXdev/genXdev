# Get-LLMJsonOutput

> **Module:** GenXdev.Helpers | **Type:** Function | **Aliases:** —

## Synopsis

> 从LLM响应文本中提取有效的JSON，使用最佳启发式方法。

## Description

获取原始LLM响应文本，其中可能包含Markdown代码块、注释或其他非JSON内容，并尝试从中提取干净有效的JSON。采用多步策略：首先去除Markdown代码块标记，然后尝试将整个字符串解析为JSON；若失败，则使用模式引导提取，最后尝试启发式回退方案。

该函数处理：
- Markdown代码块（...）
- JSON对象（{...}）和数组（[...]）
- 字符串、数字、布尔值和日期时间值
- 使用json_schema类型提示进行模式引导提取

## Syntax

```powershell
Get-LLMJsonOutput -Text <String> [[-ResponseFormat] <String>] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Text` | String | ✅ | — | 0 | — | @{response=您是一位乐于助人的助手，专为输出JSON而设计。} |
| `-ResponseFormat` | String | — | — | 1 | — | 可选JSON模式字符串，用于指导提取（OpenAI json_schema格式） |

## Examples

### Example 1

```json
{"result": "hello"}
```

@{response=提取：{"result": "hello"}}

### $schema = '{"type":"json_schema","json_schema":{"name":"r","schema":{"type":"object"}}}' Get-LLMJsonOutput -Text 'The answer is {"key": "value"}' -ResponseFormat $schema

```powershell
$schema = '{"type":"json_schema","json_schema":{"name":"r","schema":{"type":"object"}}}'
Get-LLMJsonOutput -Text 'The answer is {"key": "value"}' -ResponseFormat $schema
```

提取：{"key": "value"}
