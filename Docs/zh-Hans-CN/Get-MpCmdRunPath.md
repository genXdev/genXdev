# Get-MpCmdRunPath

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 获取 Windows Defender MpCmdRun.exe 可执行文件的路径。

## Description

* 定位并返回 Windows Defender 命令行实用程序 (MpCmdRun.exe) 的完整路径。
* 该 cmdlet 会检查 Program Files 中的标准安装位置，并在文件未找到时提供适当的错误处理。

## Syntax

```powershell
Get-MpCmdRunPath [<CommonParameters>]
```

## Examples

### Example 1

```powershell
$defenderPath = Get-MpCmdRunPath
```

检索 Windows Defender 命令行工具的完整路径。
