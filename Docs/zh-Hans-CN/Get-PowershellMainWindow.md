# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 返回一个用于PowerShell终端主窗口的窗口辅助对象。

## Description

* 获取一个 WindowObj 辅助对象，该对象表示当前 PowerShell 主机进程的主窗口。
* 这允许操作和交互终端窗口本身。

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

检索当前 PowerShell 终端窗口的 WindowObj 对象。
