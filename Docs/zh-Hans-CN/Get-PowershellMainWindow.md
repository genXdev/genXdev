# Get-PowershellMainWindow

> **Module:** GenXdev.Windows | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 返回 PowerShell 终端主窗口的窗口帮助对象。

## Description

* 获取一个 WindowObj 帮助器对象，该对象表示当前 PowerShell 主机进程的主窗口。
* 这允许操作和与终端窗口本身进行交互。

## Syntax

```powershell
Get-PowershellMainWindow [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-PowershellMainWindow
```

检索当前 PowerShell 终端窗口的 WindowObj。
