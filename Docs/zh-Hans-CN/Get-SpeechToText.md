# Get-SpeechToText

> **Module:** GenXdev.AI | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 使用OpenAI的Whisper语音识别模型将音频文件转换为文本。

## Description

使用 Whisper.NET 库处理音频文件并将语音转换为文本，该库实现了 OpenAI 的 Whisper 自动语音识别（ASR）系统。支持多种语言、翻译功能以及各种转录质量设置。

## Examples

### Example 1

```powershell
Get-SpeechToText -Input "C:\audio\recording.wav"
```

使用默认设置将音频文件转录为文本。

### Example 2

```powershell
Get-ChildItem "C:\audio\*.wav" | Get-SpeechToText
```

转录目录中的所有WAV文件。

### Example 3

```powershell
Get-SpeechToText -Input "audio.mp3" -LanguageIn "es" -WithTranslate
```

将西班牙语音频转录并翻译成英语。

### Example 4

```powershell
Get-SpeechToText -Input "recording.wav" -Passthru -WithTokenTimestamps
```

返回具有精确计时信息的 SegmentData 对象。
