# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 返回谷歌搜索支持的所有语言的逆向字典。

## Description

返回一个字典，将语言名称映射到谷歌搜索支持的所有语言对应的语言代码。
在网页查询和翻译任务中，用于查找语言代码十分有用。

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

检索将语言名称映射到其代码的字典。
