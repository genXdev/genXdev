# Get-WebLanguageDictionary

> **Module:** GenXdev.Helpers | **Type:** Cmdlet | **Aliases:** —

## Synopsis

> 返回一个反向字典，适用于谷歌搜索支持的所有语言。

## Description

* 返回一个字典，将语言名称映射到 Google 搜索支持的所有语言对应的语言代码。
* 用于在 Web 查询和翻译任务中查找语言代码。

## Syntax

```powershell
Get-WebLanguageDictionary [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Get-WebLanguageDictionary
```

获取将语言名称映射到其代码的字典。
