# SayDate

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> 使用文本转语音合成说出当前日期。

## Description

将当前日期转换为自然语言格式，并使用系统的文本转语音引擎朗读出来。日期以以下格式朗读：'今天是 [星期几]，[月份] [日] [年]'

## Syntax

```powershell
SayDate [<CommonParameters>]
```

## Examples

### SayDate Announces today's date using text-to-speech

```powershell
SayDate
Announces today's date using text-to-speech
```
