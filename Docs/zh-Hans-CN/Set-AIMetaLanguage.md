# Set-AIMetaLanguage

> **Module:** GenXdev.AI.Queries | **Type:** Function | **Aliases:** —

## Synopsis

> 为GenXdev.AI图像元数据操作设置默认语言以及可选的图像目录。

## Description

此函数配置 GenXdev.AI 模块中图像元数据操作的全局默认语言。可选地，它还可以设置全局图像目录。这两个设置都会持久保存在模块的首选项存储中，以便跨会话使用。

## Syntax

```powershell
Set-AIMetaLanguage [[-Language] <String>] [-ClearSession] [-PreferencesDatabasePath <String>] [-SessionOnly] [-SkipSession] [<CommonParameters>]
```

## Parameters

| Name | Type | Required | Pipeline | Position | Default | Description |
|:---|:---|:---:|:---|:---:|:---|:---|
| `-Language` | String | — | — | 0 | — | 图像元数据操作的默认语言 |
| `-PreferencesDatabasePath` | String | — | — | Named | — | 偏好数据文件的数据库路径 |
| `-SessionOnly` | SwitchParameter | — | — | Named | — | 使用会话中存储的替代设置来配置AI偏好，如语言、图像集合等 |
| `-ClearSession` | SwitchParameter | — | — | Named | — | 清除会话中为AI偏好（如语言、图像集等）存储的备用设置 |
| `-SkipSession` | SwitchParameter | — | — | Named | — | 不要使用会话中存储的替代设置来获取AI偏好设置，如语言、图像集合等 |

## Examples

### Set-AIMetaLanguage -Language "Spanish" -ImageDirectories @("C:\Images", "D:\Photos")

```powershell
Set-AIMetaLanguage -Language "Spanish" -ImageDirectories @("C:\Images", "D:\Photos")
```

在首选项中持久化设置语言和图像目录。

### Set-AIMetaLanguage "French"

```powershell
Set-AIMetaLanguage "French"
```

在首选项中持久性地设置语言。

### Set-AIMetaLanguage -Language "German" -SessionOnly

```powershell
Set-AIMetaLanguage -Language "German" -SessionOnly
```

仅为当前会话设置语言（全局变量）。

### Set-AIMetaLanguage -ClearSession

```powershell
Set-AIMetaLanguage -ClearSession
```

清除会话语言设置（全局变量），不影响持久性偏好。
