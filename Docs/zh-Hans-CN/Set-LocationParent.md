# Set-LocationParent

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `..

## Synopsis

> 切换到父目录并列出其内容。

## Description

* 在文件系统层级中向上导航一级。
* 显示新的当前目录的内容。
* 提供便捷的 '..' 别名，用于快速目录导航。

## Syntax

```powershell
Set-LocationParent [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent
```

切换到父目录并列出其内容。

### Example 2

```powershell
..
```

使用别名切换到父目录。
