# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...

## Synopsis

> 在文件系统层次结构中向上导航两级目录。

## Description

* 将当前工作目录更改为祖父目录（向上两级）。
* 显示结果目录的内容。

## Syntax

```powershell
Set-LocationParent2 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

切换到上级目录并显示其内容。
