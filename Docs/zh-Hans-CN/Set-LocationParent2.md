# Set-LocationParent2

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `...`

## Synopsis

> 在文件系统层次结构中向上导航两级目录。

## Description

* 将当前工作目录更改为祖父目录（向上两级）。
* 显示结果目录的内容。

## Syntax

```powershell
Set-LocationParent2 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent2
```

更改到父级目录并显示其内容。

```powershell
...
```

使用 '...' 别名执行相同操作。
