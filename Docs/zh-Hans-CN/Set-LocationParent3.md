# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....`

## Synopsis

> 在文件系统层次结构中向上导航三个目录级别。

## Description

* 通过文件系统层次结构向上移动三个父目录来更改当前工作目录。
* 显示结果目录的内容。

## Syntax

```powershell
Set-LocationParent3 [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

从当前位置向上导航三个目录层级。

### Example 2

```powershell
....
```

使用别名向上导航三级目录。
