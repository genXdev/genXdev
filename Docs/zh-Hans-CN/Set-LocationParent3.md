# Set-LocationParent3

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `....

## Synopsis

> 在文件系统层次结构中向上导航三级目录。

## Description

* 将当前工作目录在文件系统层次结构中向上移动三级父目录
* 显示结果目录的内容

## Syntax

```powershell
Set-LocationParent3 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent3
```

从当前位置向上导航三个目录层级。

```powershell
....
```
