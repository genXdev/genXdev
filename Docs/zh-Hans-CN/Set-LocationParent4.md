# Set-LocationParent4

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `.....

## Synopsis

> 在文件系统层次结构中向上导航四个目录层级。

## Description

从文件系统中的当前位置向上移动四级目录。
* 导航后，显示结果目录的内容。

## Syntax

```powershell
Set-LocationParent4 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent4
```

从当前位置向上导航四个目录层级。

.....
