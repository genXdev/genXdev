# Set-LocationParent5

> **Module:** GenXdev.FileSystem | **Type:** Cmdlet | **Aliases:** `......

## Synopsis

> 在文件系统层级中向上导航五个目录级别。

## Description

* 将当前工作目录向上移动五个父目录级别。
* 显示结果目录的内容。
* 提供便捷的简写别名 '......' 用于快速导航。

## Syntax

```powershell
Set-LocationParent5 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Examples

### Example 1

```powershell
Set-LocationParent5
```

从当前位置向上导航五个目录层级。

### Example 2

```powershell
......
```

使用别名向上导航五个目录层级。
