# Start-VlcMediaPlayerPreviousInPlaylist

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcprev`, `vlcback`

## Synopsis

> 移动到 VLC 媒体播放器播放列表中的上一个项目。

## Description

此函数向 VLC 媒体播放器发送 'p' 键命令以导航到当前播放列表中的上一项。该函数支持 WhatIf 操作，并在发送命令后恢复焦点。

## Syntax

```powershell
Start-VlcMediaPlayerPreviousInPlaylist [<CommonParameters>]
```

## Examples

### Start-VlcMediaPlayerPreviousInPlaylist

```powershell
Start-VlcMediaPlayerPreviousInPlaylist
```

### vlcprev

```powershell
vlcprev
```

### vlcback

```powershell
vlcback
```
