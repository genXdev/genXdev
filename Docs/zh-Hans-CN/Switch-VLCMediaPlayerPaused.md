# Switch-VLCMediaPlayerPaused

> **Module:** GenXdev.Media | **Type:** Function | **Aliases:** `vlcpause`, `vlcplay`

## Synopsis

> 切换 VLC 媒体播放器的暂停/播放状态。

## Description

此函数向 VLC 媒体播放器发送空格键，以在暂停和播放状态之间切换。发送按键命令后，它会自动将焦点恢复到原始窗口。

## Syntax

```powershell
Switch-VLCMediaPlayerPaused [<CommonParameters>]
```

## Examples

### Switch-VLCMediaPlayerPaused

```powershell
Switch-VLCMediaPlayerPaused
```

切换 VLC 媒体播放器的暂停/播放状态。

### vlcpause

```powershell
vlcpause
```

使用别名切換暫停/播放狀態。

### vlcplay

```powershell
vlcplay
```

使用备用别名切换暂停/播放状态。
