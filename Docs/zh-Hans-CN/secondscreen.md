# secondscreen

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> 设置默认第二显示器配置。

## Description

设置 GenXdev 窗口默认在辅助显示器上打开。
这对于单显示器用户或偏好并排窗口布局的用户非常有用。
另请参阅 cmdlet 'sidebyside' 和 'restoredefaultmonitor'。

## Syntax

```powershell
secondscreen [<CommonParameters>]
```

## Examples

### PS> secondscreen

```powershell
PS> secondscreen
```

设置 GenXdev 窗口打开时的默认位置为辅助显示器
