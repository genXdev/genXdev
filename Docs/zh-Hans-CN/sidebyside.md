# sidebyside

> **Module:** GenXdev.Console | **Type:** Function | **Aliases:** —

## Synopsis

> 设置默认的并行配置。

## Description

设置 GenXdev 窗口打开的默认行为，使其与 PowerShell 并排显示。
这对于使用单个显示器的用户或偏好并排窗口布局的用户非常有用。
另请参阅 cmdlet 'secondscreen' 和 'restoredefaultmonitor'

## Syntax

```powershell
sidebyside [<CommonParameters>]
```

## Examples

### PS> sidebyside

```powershell
PS> sidebyside
```

将 GenXdev 窗口的默认打开方式设置为与 PowerShell 并排显示
